# Playwright Mastery

Playwright Mastery is a hands-on course that takes you from your first browser automation script to production-grade end-to-end test suites running in CI. You will learn the Playwright API inside out — locators, assertions, waiting strategies, network interception, visual testing, authentication, parallel execution, and the AI-powered features that set Playwright apart from every other browser automation tool. By the end, you will build a complete test automation pipeline with self-healing locators, AI-generated test scenarios, and continuous integration integration.

## Who this is for

- **QA engineers** transitioning from manual testing or Selenium who need a modern, fast, and reliable automation framework
- **Frontend developers** who want to catch regressions early with automated browser tests integrated into their development workflow
- **SDETs and test architects** designing scalable test infrastructure for teams of 5 to 50 engineers
- **AI engineers** curious about how to combine LLMs with browser automation for intelligent test generation and self-healing

## Prerequisites

- **Basic JavaScript or TypeScript knowledge** — you should be comfortable with arrow functions, async/await, and basic object syntax
- **Node.js 18+ installed** — you will install Playwright via npm from day one
- **A code editor** — VS Code is recommended for the official Playwright extension
- **Comfort in the terminal** — you will run npm commands, start test runners, and configure CI pipelines

## Learning path

1. **Getting Started** — What Playwright is, why it matters, and how to install and run your first test
2. **TypeScript for Playwright** — TypeScript essentials you need to write type-safe Playwright tests
3. **Core Playwright** — Locators, assertions, waiting, actions, page object model, and API testing
4. **Advanced Playwright** — Authentication, visual testing, parallel execution, reporting, trace viewer, and Docker
5. **CI/CD Integration** — GitHub Actions, worker configuration, and production-grade test pipelines
6. **AI-Powered Playwright** — Playwright CLI Agent, MCP server, self-healing locators, and AI test generation
