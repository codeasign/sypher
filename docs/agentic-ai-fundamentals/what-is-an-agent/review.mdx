# What Is an Agent? — Review

*Reading time: 3 minutes*

## Key Takeaways

- An AI agent is a system where an LLM drives a loop of perception, reasoning, action, and memory
- Three essential properties define an agent: autonomy, tool use, and memory
- A plain LLM call produces text; an agent produces action through tool execution
- The agent loop: the model decides what to do, your code executes it, the result feeds back to the model
- Always set a turn limit on agent loops to prevent runaway costs

## Recap in 60 Seconds

1. An agent is an LLM driving a loop with tools and memory — without the loop or tools it is just an API call
2. The agent decides what to do next; your code provides the tools and the loop structure
3. Start with the simplest architecture that works and add agent capabilities only when needed
