# Environment Variables Explained from Zero

⏱ 30 minutes · Difficulty: Beginner

## What This Is

Environment variables are one of those concepts that sounds mysterious until you see what they actually do. This section walks you through everything from "what is a variable in the terminal?" to "how do I securely store my API keys?"

You will learn by doing: you will read variables, set them, write a `.env` file, and connect your first script to an AI service using an environment variable.

## Prerequisites

- You can open a terminal and run a command
- You have Python installed (any version 3.8+)
- You have completed the "Installing Python and Using the Terminal" section

## What You Will Build

By the end of this section you will have:

- A Python script that reads secrets from your environment
- A `.env` file that keeps those secrets out of your source code
- Working knowledge of `os.environ`, `python-dotenv`, and terminal variable syntax
- A reusable pattern you will use in every AI project from now on
