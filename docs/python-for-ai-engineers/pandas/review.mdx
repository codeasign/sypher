# Pandas — Review

⏱ 3 minutes

## Key Takeaways

- A DataFrame is a labeled 2D table; a Series is a single labeled column. Use `df["col"]` to select a column, `df.loc[]` for label-based indexing, `df.iloc[]` for position-based indexing.
- Filter rows with boolean indexing: `df[df["accuracy"] > 0.9]`. Use `&` for AND, `|` for OR, and wrap each condition in parentheses.
- Handle missing data with `dropna()` (remove rows) and `fillna()` (replace missing values).
- Group and aggregate with `df.groupby("col")["value"].mean()`. Always call an aggregation after `groupby()`.
- Merge DataFrames with `pd.merge()` (like SQL JOIN) and concatenate with `pd.concat()`.

## Recap in 60 Seconds

If you only remember three things:

1. A DataFrame is a dict of Series — each column is a named, typed array with a shared index.
2. Use vectorized operations (`df["col"].mean()`) not loops — they are faster and clearer.
3. Always use `&` for AND, `|` for OR, `.loc` for assignment, and never iterate when you could group.
