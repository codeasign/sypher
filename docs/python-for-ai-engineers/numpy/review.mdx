# NumPy — Review

⏱ 3 minutes

## Key Takeaways

- Create arrays with `np.array()`, `np.zeros()`, `np.ones()`, `np.arange()`, `np.linspace()`, and `np.random`.
- Vectorized operations apply element-wise at C speed — write `arr * 2`, not a loop.
- Use boolean indexing (`arr[arr > 0.5]`) for filtering and `np.where()` for conditional selection.
- Slicing returns a view (modifying it changes the original); boolean indexing returns a copy.
- Broadcasting lets you operate on arrays of different shapes when their trailing dimensions match or one is 1.

## Recap in 60 Seconds

If you only remember three things:

1. A NumPy array is a dense memory block of one type — operations apply to all elements at once at C speed.
2. Write array-level expressions, not element-level loops: `arr ** 2` not `[x**2 for x in arr]`.
3. Check `.shape` before operations, use `.copy()` when you need independence from the original.
