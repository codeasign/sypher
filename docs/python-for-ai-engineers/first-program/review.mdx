# Your First Program — Review

⏱ 3 minutes

## Key Takeaways

- A Python script is a sequence of statements executed top to bottom — one step at a time.
- `input()` returns a string; convert with `int()` or `float()` when you need a number.
- `sys.argv` provides command-line arguments, with `sys.argv[0]` being the script name and `sys.argv[1]` onward being the actual arguments.
- Use `import sys` at the top of the file before accessing `sys.argv`.
- String concatenation with `+` joins strings; `len()` returns the number of characters in a string.

## Recap in 60 Seconds

If you only remember three things:

1. A script reads input (via `input()` or `sys.argv`), processes it, and prints output with `print()` — that is the pattern of almost every Python program.
2. Strings need quotes; `input()` returns a string; convert types explicitly when you need numbers.
3. Save your file before running it — the most common "bug" is an unsaved edit.
