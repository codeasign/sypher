# Data Visualization — Review

⏱ 3 minutes

## Key Takeaways

- Use the object-oriented interface: `fig, ax = plt.subplots()` → `ax.plot()` → `ax.set_xlabel()` → `fig.savefig()`.
- Choose the right chart type: line plots for trends, bar charts for comparisons, histograms for distributions, scatter plots for relationships.
- Customize everything on the axes object — colors, labels, limits, grid, legend.
- Always use `bbox_inches="tight"` when saving to avoid cropped labels.
- Close figures with `plt.close(fig)` in loops to prevent memory leaks.

## Recap in 60 Seconds

If you only remember three things:

1. Create with `fig, ax = plt.subplots()`, draw on `ax`, save with `fig.savefig()` — never use the pyplot global state.
2. Labels, legends, and limits go on the axes object — `ax.set_xlabel()`, `ax.legend()`, `ax.set_ylim()`.
3. Save with `bbox_inches="tight"` and close figures in loops to avoid cropping and memory leaks.
