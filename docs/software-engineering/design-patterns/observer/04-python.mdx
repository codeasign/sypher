## Implementation

Below is a complete Python implementation of the Observer pattern using a stock market domain. A `StockExchange` (Subject) tracks prices for AAPL, GOOGL, and MSFT. Three observers — `PriceDisplay`, `AlertSystem`, and `PortfolioTracker` — register with the exchange and react to price changes. The implementation uses Python's ABC module for the abstract observer interface and dataclasses for clean data representation.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────┐
//   │   Observer (ABC)                  │
//   ├──────────────────────────────────┤
//   │   @abstractmethod                │ ← OOP: Abstraction
//   │   update(ticker, price)          │    ISP: single-method contract
//   └────────────────┬─────────────────┘
//                    ▲ inherits
//   ┌────────────────┴─────────────────┐
//   │                │                 │
//   ┌────────────┐ ┌──────────┐ ┌──────────────┐
//   │PriceDisplay│ │AlertSys  │ │Portfolio     │ ← OOP: Polymorphism
//   ├────────────┤ ├──────────┤ ├──────────────┤    LSP: all subclass
//   │ update()   │ │ update() │ │ update()     │     Observer
//   └────────────┘ └──────────┘ └──────────────┘
//
//   ┌──────────────────────────────────┐
//   │  StockExchange                   │ ← OOP: Encapsulation
//   ├──────────────────────────────────┤    SRP: prices + observers
//   │  attach(observer)                │     OCP: new observers
//   │  detach(observer)                │     via attach()
//   │  set_price(ticker, price)        │    DIP: depends on Observer
//   │  get_price(ticker) -> float      │     abstract class
//   └──────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — The `Observer` ABC defines the abstract `update()` method. StockExchange depends on the `Observer` type, not on concrete observer classes.
2. **Encapsulation** — StockExchange hides its `_observers` list and `_prices` dict with underscore prefix convention. Each observer encapsulates its own state and logic.
3. **Inheritance** — `PriceDisplay`, `AlertSystem`, and `PortfolioTracker` all inherit from `Observer` and implement the abstract `update()` method.
4. **Polymorphism** — StockExchange iterates over `Observer` references and calls `update()` on each. The correct implementation runs per observer type.

#### SOLID Principles Applied

5. **SRP** — StockExchange is responsible for price management and observer notification only. Each observer handles one domain concern.
6. **OCP** — New observer types are created by subclassing `Observer` and implementing `update()`. StockExchange is never modified.
7. **LSP** — Every `Observer` subclass is substitutable for the base `Observer` type. The notification loop treats all observers uniformly.
8. **ISP** — The `Observer` ABC defines a single abstract method. No class is forced to implement unused methods.
9. **DIP** — StockExchange depends on the abstract `Observer` class. Concrete observers are injected via `attach()`.

### Observer (Abstract Base Class)

```python
# Observer Pattern — Python — Stock Market example
# ==================================================
# Participants: StockExchange (Subject), PriceDisplay,
# AlertSystem, PortfolioTracker (Concrete Observers)
# OOP: abstraction, encapsulation, inheritance, polymorphism
# SOLID: SRP, OCP, LSP, ISP, DIP
# ==================================================

from abc import ABC, abstractmethod


# ── Observer Interface (Abstract Base Class) ──
class Observer(ABC):
    """Abstract observer — all observers implement update()"""

    @abstractmethod
    def update(self, ticker: str, price: float) -> None:
        """Called when the subject's state changes."""
        pass
```

### StockExchange (Subject)

```python
# ── Subject ──
class StockExchange:
    """Concrete subject — manages stock prices and notifies observers."""

    def __init__(self) -> None:
        self._observers: list[Observer] = []
        self._prices: dict[str, float] = {}

    def attach(self, observer: Observer) -> None:
        """Register an observer."""
        self._observers.append(observer)

    def detach(self, observer: Observer) -> None:
        """Unregister an observer."""
        self._observers.remove(observer)

    def _notify_observers(self, ticker: str, price: float) -> None:
        """Notify all registered observers of a price change."""
        for obs in self._observers:
            obs.update(ticker, price)

    def set_price(self, ticker: str, price: float) -> None:
        """Set a new price and notify observers."""
        self._prices[ticker] = price
        self._notify_observers(ticker, price)

    def get_price(self, ticker: str) -> float:
        """Get the current price of a ticker."""
        return self._prices.get(ticker, 0.0)
```

### PriceDisplay (Concrete Observer)

```python
# ── Concrete Observer 1: PriceDisplay ──
class PriceDisplay(Observer):
    """Displays price updates to the console."""

    def update(self, ticker: str, price: float) -> None:
        print(f"DISPLAY: {ticker} is now ${price:.2f}")
```

### AlertSystem (Concrete Observer)

```python
# ── Concrete Observer 2: AlertSystem ──
class AlertSystem(Observer):
    """Triggers alerts when price changes exceed a threshold percentage."""

    def __init__(self, threshold_percent: float) -> None:
        self._previous_prices: dict[str, float] = {}
        self._threshold = threshold_percent

    def update(self, ticker: str, price: float) -> None:
        if ticker in self._previous_prices:
            old_price = self._previous_prices[ticker]
            change = abs(price - old_price) / old_price * 100.0
            if change >= self._threshold:
                direction = "up" if price >= old_price else "down"
                print(
                    f"ALERT: {ticker} moved {direction} {change:.1f}% "
                    f"(${old_price:.2f} -> ${price:.2f})"
                )
        self._previous_prices[ticker] = price
```

### PortfolioTracker (Concrete Observer)

```python
# ── Concrete Observer 3: PortfolioTracker ──
class PortfolioTracker(Observer):
    """Tracks portfolio value and recalculates when a held stock changes."""

    def __init__(self, exchange: StockExchange) -> None:
        self._holdings: dict[str, int] = {}
        self._total_value: float = 0.0
        self._exchange = exchange

    def add_holding(self, ticker: str, shares: int) -> None:
        """Add a stock holding to the portfolio."""
        self._holdings[ticker] = shares

    def _recalculate(self) -> None:
        """Recalculate total portfolio value from all holdings."""
        self._total_value = 0.0
        for ticker, shares in self._holdings.items():
            price = self._exchange.get_price(ticker)
            self._total_value += price * shares

    def update(self, ticker: str, price: float) -> None:
        if ticker in self._holdings:
            self._recalculate()
            print(f"PORTFOLIO: Updated value is ${self._total_value:.2f}")

    @property
    def total_value(self) -> float:
        return self._total_value
```

### Main / Driver

```python
# ── Main / Driver ──
if __name__ == "__main__":
    exchange = StockExchange()

    display = PriceDisplay()
    alerts = AlertSystem(5.0)
    portfolio = PortfolioTracker(exchange)

    # Register observers
    exchange.attach(display)
    exchange.attach(alerts)
    exchange.attach(portfolio)

    # Set up portfolio holdings
    portfolio.add_holding("AAPL", 10)
    portfolio.add_holding("GOOGL", 5)
    portfolio.add_holding("MSFT", 8)

    print("=== Stock Market Observer Demo ===")
    print()

    exchange.set_price("AAPL", 150.00)
    exchange.set_price("GOOGL", 2800.00)
    exchange.set_price("MSFT", 300.00)

    print()

    # Detach display before a large price swing
    exchange.detach(display)
    exchange.set_price("AAPL", 162.00)  # 8% change -- alert fires, no display
    exchange.attach(display)

    print()

    exchange.set_price("GOOGL", 2750.00)  # < 5% -- no alert
    exchange.set_price("MSFT", 315.00)    # 5% change -- alert fires
```

### Language-Specific Notes

- **ABC module**: Python's `abc.ABC` and `@abstractmethod` decorator provide the formal interface mechanism. The `Observer` ABC cannot be instantiated directly — attempting to do so raises `TypeError` at runtime.
- **Duck typing fallback**: While this implementation uses explicit ABC inheritance, Python's duck typing means any object with an `update(ticker, price)` method could serve as an observer. The ABC adds documentation and runtime safety.
- **Underscore privacy**: Python uses the `_` prefix convention (not enforced by the interpreter) to indicate private members. This is idiomatic Python — true private members with `__name` mangling are reserved for name conflicts in inheritance hierarchies.
- **Type hints**: The implementation uses Python 3.11+ type hints (`list[Observer]`, `dict[str, float]`, `-> None`) for documentation and IDE support. These are not enforced at runtime.
- **Property decorator**: `@property` on `total_value` provides read-only access to the computed portfolio value, consistent with Python's preference for attribute-style access over getter methods.
