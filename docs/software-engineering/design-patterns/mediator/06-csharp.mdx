C# implements the Mediator pattern through interfaces with first-class language support. C#'s interface syntax provides clean separation between the mediator contract and concrete implementations. In the stock market domain, a C# `NasdaqExchange` mediates between `BuyAgent` and `SellAgent` colleagues, matching orders through a centralized order book without agents communicating directly with each other.

## How the Implementation Works

| Aspect | Implementation |
|---|---|
| **Mediator** | `IStockExchange` interface with `RegisterAgent`, `PlaceBid`, `PlaceAsk`, `Broadcast` |
| **ConcreteMediator** | `NasdaqExchange : IStockExchange` maintains agent registry and order book, matches bids against asks |
| **Colleague** | `ITradingAgent` interface with `Name`, `Symbol`, `OnMessage` |
| **ConcreteColleagues** | `BuyAgent` and `SellAgent` each hold name, symbol, quantity, and price limit |
| **Encapsulation** | Colleagues never reference each other — all communication goes through the mediator |
| **SOLID** | Single Responsibility (exchange matches orders, agents represent intent), Interface Segregation (small focused interfaces), Dependency Inversion (both depend on abstractions) |

### IStockExchange (Mediator Interface)
```csharp
// ── Mediator Interface ──
/// <summary>
/// The mediator interface defining the contract for a stock exchange.
/// All communication between trading agents flows through this interface.
/// </summary>
public interface IStockExchange
{
    /// <summary>
    /// Register a trading agent with the exchange.
    /// </summary>
    /// <param name="agent">The agent to register.</param>
    void RegisterAgent(ITradingAgent agent);

    /// <summary>
    /// Place a buy order (bid) on the exchange.
    /// </summary>
    /// <param name="agent">The agent placing the bid.</param>
    /// <param name="symbol">Stock ticker symbol.</param>
    /// <param name="quantity">Number of shares.</param>
    /// <param name="price">Maximum price per share the agent is willing to pay.</param>
    void PlaceBid(ITradingAgent agent, string symbol, int quantity, decimal price);

    /// <summary>
    /// Place a sell order (ask) on the exchange.
    /// </summary>
    /// <param name="agent">The agent placing the ask.</param>
    /// <param name="symbol">Stock ticker symbol.</param>
    /// <param name="quantity">Number of shares.</param>
    /// <param name="price">Minimum price per share the agent is willing to accept.</param>
    void PlaceAsk(ITradingAgent agent, string symbol, int quantity, decimal price);

    /// <summary>
    /// Broadcast a message to all registered agents.
    /// </summary>
    /// <param name="message">The message to broadcast.</param>
    void Broadcast(string message);
}
```

### ITradingAgent (Colleague Interface)
```csharp
// ── Colleague Interface ──
/// <summary>
/// The colleague interface for any participant in the exchange.
/// Agents communicate through the mediator, never directly with each other.
/// </summary>
public interface ITradingAgent
{
    /// <summary>
    /// Gets the name of this trading agent.
    /// </summary>
    string Name { get; }

    /// <summary>
    /// Gets the stock symbol this agent trades.
    /// </summary>
    string Symbol { get; }

    /// <summary>
    /// Receive a message from the mediator.
    /// </summary>
    /// <param name="message">The message content.</param>
    void OnMessage(string message);
}
```

### NasdaqExchange (Concrete Mediator)
```csharp
// ── Concrete Mediator ──
/// <summary>
/// A concrete stock exchange that mediates between trading agents.
/// Maintains a registry of agents and an order book, matching bids against asks.
/// </summary>
public class NasdaqExchange : IStockExchange
{
    private readonly Dictionary<string, ITradingAgent> _agents = new();
    private readonly List<Order> _orderBook = new();
    private readonly List<Trade> _executedTrades = new();

    /// <summary>
    /// Register a trading agent with the exchange.
    /// </summary>
    public void RegisterAgent(ITradingAgent agent)
    {
        string key = agent.Name;
        if (!_agents.TryAdd(key, agent))
        {
            Console.WriteLine($"  [Exchange] Agent '{key}' is already registered.");
            return;
        }

        Console.WriteLine($"  [Exchange] Registered agent '{agent.Name}' trading {agent.Symbol}.");
        Broadcast($"Welcome, {agent.Name}! You are now trading {agent.Symbol} on Nasdaq.");
    }

    /// <summary>
    /// Place a buy order (bid) and attempt to match against existing asks.
    /// </summary>
    public void PlaceBid(ITradingAgent agent, string symbol, int quantity, decimal price)
    {
        Console.WriteLine($"  [Exchange] Bid received: {agent.Name} wants to buy {quantity} {symbol} @ ${price:F2}");

        var bid = new Order(agent.Name, symbol, quantity, price, "Bid");
        _orderBook.Add(bid);

        // Try to match this bid against existing asks.
        MatchOrders();
    }

    /// <summary>
    /// Place a sell order (ask) and attempt to match against existing bids.
    /// </summary>
    public void PlaceAsk(ITradingAgent agent, string symbol, int quantity, decimal price)
    {
        Console.WriteLine($"  [Exchange] Ask received: {agent.Name} wants to sell {quantity} {symbol} @ ${price:F2}");

        var ask = new Order(agent.Name, symbol, quantity, price, "Ask");
        _orderBook.Add(ask);

        // Try to match this ask against existing bids.
        MatchOrders();
    }

    /// <summary>
    /// Broadcast a message to all registered agents.
    /// </summary>
    public void Broadcast(string message)
    {
        foreach (var agent in _agents.Values)
        {
            agent.OnMessage($"[BROADCAST] {message}");
        }
    }

    /// <summary>
    /// Match compatible bids and asks from the order book.
    /// A bid matches an ask when the same symbol, bid price >= ask price, and quantities align.
    /// </summary>
    private void MatchOrders()
    {
        bool matched;
        do
        {
            matched = false;

            for (int i = _orderBook.Count - 1; i >= 0; i--)
            {
                if (_orderBook[i].Type != "Bid")
                    continue;

                var bid = _orderBook[i];

                for (int j = _orderBook.Count - 1; j >= 0; j--)
                {
                    if (_orderBook[j].Type != "Ask")
                        continue;

                    var ask = _orderBook[j];

                    if (bid.Symbol != ask.Symbol)
                        continue;

                    // A bid matches an ask if the bid price is at least the ask price.
                    if (bid.Price < ask.Price)
                        continue;

                    int tradeQuantity = Math.Min(bid.Quantity, ask.Quantity);
                    decimal executionPrice = ask.Price;

                    var trade = new Trade(bid.AgentName, ask.AgentName, bid.Symbol, tradeQuantity, executionPrice);
                    _executedTrades.Add(trade);

                    Console.WriteLine($"  [Exchange] TRADE EXECUTED: {trade.Buyer} bought {trade.Quantity} {trade.Symbol} " +
                        $"from {trade.Seller} @ ${trade.Price:F2}");

                    // Notify the agents.
                    if (_agents.TryGetValue(bid.AgentName, out var buyer))
                    {
                        buyer.OnMessage($"Trade executed: You bought {tradeQuantity} {bid.Symbol} @ ${executionPrice:F2}");
                    }

                    if (_agents.TryGetValue(ask.AgentName, out var seller))
                    {
                        seller.OnMessage($"Trade executed: You sold {tradeQuantity} {ask.Symbol} @ ${executionPrice:F2}");
                    }

                    // Update remaining quantities.
                    Order updatedBid = bid with { Quantity = bid.Quantity - tradeQuantity };
                    Order updatedAsk = ask with { Quantity = ask.Quantity - tradeQuantity };

                    // Remove or update orders.
                    _orderBook.RemoveAt(i);
                    _orderBook.RemoveAt(j < i ? j : j - 1);

                    if (updatedBid.Quantity > 0)
                    {
                        _orderBook.Add(updatedBid);
                    }

                    if (updatedAsk.Quantity > 0)
                    {
                        _orderBook.Add(updatedAsk);
                    }

                    matched = true;
                    break;
                }

                if (matched)
                    break;
            }
        }
        while (matched);
    }

    /// <summary>
    /// Print a summary of all trades executed during this session.
    /// </summary>
    public void PrintTradeSummary()
    {
        Console.WriteLine();
        Console.WriteLine("  ┌" + new string('─', 54) + "┐");
        Console.WriteLine("  │                  TRADE EXECUTION SUMMARY              │");
        Console.WriteLine("  ├" + new string('─', 54) + "┤");
        Console.WriteLine("  │  Total trades executed: {0,-3}                                 │", _executedTrades.Count);
        Console.WriteLine("  ├" + "──────" + "┬" + "──────────" + "┬" + "──────────" + "┬" + "──────────" + "┬" + "───────────────" + "┤");
        Console.WriteLine("  │  #   │ Buyer    │ Seller   │ Symbol   │ Price         │");
        Console.WriteLine("  ├" + "──────" + "┼" + "──────────" + "┼" + "──────────" + "┼" + "──────────" + "┼" + "───────────────" + "┤");

        for (int k = 0; k < _executedTrades.Count; k++)
        {
            var t = _executedTrades[k];
            Console.WriteLine("  │  {0,-3} │ {1,-8} │ {2,-8} │ {3,-8} │ ${4,-10:F2} │",
                k + 1, t.Buyer, t.Seller, t.Symbol, t.Price);
        }

        Console.WriteLine("  └" + "──────" + "┴" + "──────────" + "┴" + "──────────" + "┴" + "──────────" + "┴" + "───────────────" + "┘");
    }
}
```

### BaseTradingAgent (Abstract Base Class)
```csharp
// ── Abstract Base Colleague ──
/// <summary>
/// Abstract base class for trading agents.
/// Provides common functionality like name, symbol, and message handling.
/// </summary>
public abstract class BaseTradingAgent : ITradingAgent
{
    private readonly string _name;
    private readonly string _symbol;

    /// <summary>
    /// Initializes a new instance of the BaseTradingAgent.
    /// </summary>
    /// <param name="name">The agent's display name.</param>
    /// <param name="symbol">The stock ticker symbol this agent trades.</param>
    protected BaseTradingAgent(string name, string symbol)
    {
        _name = name;
        _symbol = symbol;
    }

    /// <summary>
    /// Gets the name of this trading agent.
    /// </summary>
    public string Name => _name;

    /// <summary>
    /// Gets the stock symbol this agent trades.
    /// </summary>
    public string Symbol => _symbol;

    /// <summary>
    /// Receive a message from the mediator.
    /// </summary>
    /// <param name="message">The message content.</param>
    public abstract void OnMessage(string message);
}
```

### BuyAgent (Concrete Colleague 1)
```csharp
// ── Concrete Colleague: BuyAgent ──
/// <summary>
/// A trading agent that places buy orders (bids) on the exchange.
/// </summary>
public class BuyAgent : BaseTradingAgent
{
    private readonly int _quantity;
    private readonly decimal _priceLimit;

    /// <summary>
    /// Initializes a new instance of the BuyAgent.
    /// </summary>
    /// <param name="name">The agent's display name.</param>
    /// <param name="symbol">The stock ticker symbol to buy.</param>
    /// <param name="quantity">Number of shares to buy.</param>
    /// <param name="priceLimit">Maximum price per share.</param>
    public BuyAgent(string name, string symbol, int quantity, decimal priceLimit)
        : base(name, symbol)
    {
        _quantity = quantity;
        _priceLimit = priceLimit;
    }

    /// <summary>
    /// Submit a buy order to the exchange.
    /// </summary>
    /// <param name="exchange">The exchange to submit the order to.</param>
    public void Send(IStockExchange exchange)
    {
        Console.WriteLine($"  [BuyAgent] {Name} submitting bid: {_quantity} {Symbol} @ ${_priceLimit:F2}");
        exchange.PlaceBid(this, Symbol, _quantity, _priceLimit);
    }

    /// <summary>
    /// Receive a message from the exchange.
    /// </summary>
    /// <param name="message">The message content.</param>
    public override void OnMessage(string message)
    {
        Console.WriteLine($"  [BuyAgent:{Name}] {message}");
    }
}
```

### Main / Driver
```csharp
// ── Driver ──
/// <summary>
/// Demonstrates the Mediator pattern with a stock exchange simulation.
/// Multiple trading agents place bids and asks through the NasdaqExchange mediator.
/// </summary>
public class MediatorDemo
{
    public static void Main()
    {
        Console.WriteLine(new string('=', 65));
        Console.WriteLine("MEDIATOR PATTERN — C# Stock Exchange Demo");
        Console.WriteLine(new string('=', 65));
        Console.WriteLine();

        // Create the mediator (stock exchange).
        var exchange = new NasdaqExchange();

        // Create trading agents (colleagues).
        var buyer1 = new BuyAgent("Alice", "AAPL", 100, 150.50m);
        var seller1 = new SellAgent("Bob", "AAPL", 100, 150.50m);
        var buyer2 = new BuyAgent("Charlie", "GOOGL", 50, 175.35m);
        var seller2 = new SellAgent("Diana", "GOOGL", 50, 175.30m);
        var buyer3 = new BuyAgent("Eve", "MSFT", 75, 340.00m);
        var seller3 = new SellAgent("Frank", "MSFT", 75, 340.10m);

        // Register agents with the exchange.
        Console.WriteLine("--- Registering Agents ---");
        exchange.RegisterAgent(buyer1);
        exchange.RegisterAgent(seller1);
        exchange.RegisterAgent(buyer2);
        exchange.RegisterAgent(seller2);
        exchange.RegisterAgent(buyer3);
        exchange.RegisterAgent(seller3);
        Console.WriteLine();

        // Place orders through the mediator.
        Console.WriteLine("--- Placing Orders ---");
        buyer1.Send(exchange);
        Console.WriteLine();
        seller1.Send(exchange);
        Console.WriteLine();
        buyer2.Send(exchange);
        Console.WriteLine();
        seller2.Send(exchange);
        Console.WriteLine();
        buyer3.Send(exchange);
        Console.WriteLine();
        seller3.Send(exchange);
        Console.WriteLine();

        // Print trade summary.
        exchange.PrintTradeSummary();

        Console.WriteLine();
        Console.WriteLine(new string('=', 65));
        Console.WriteLine("All orders processed through the NasdaqExchange mediator.");
    }
}
```

## Language-Specific Notes

- C# uses `IStockExchange` and `ITradingAgent` (the `I` prefix convention) for the interface names. This is a .NET naming convention that visually distinguishes interfaces from classes. The compiler enforces the interface contract identically regardless of the naming convention.

- C# 12 records provide a concise syntax for data-carrying types. The `record Order(...)` and `record Trade(...)` declarations define immutable data objects with positional parameters, auto-generated `Equals`/`GetHashCode`, and deconstruction support — all in a single line each. The `with` keyword enables non-destructive mutation (e.g., `bid with { Quantity = bid.Quantity - tradeQuantity }`).

- The `Dictionary<string, ITradingAgent>.TryAdd` method (introduced in .NET Core 2.0+) atomically adds a key-value pair and returns `false` if the key already exists. This avoids the common pattern of checking `ContainsKey` before adding, which is two operations that could race in a multi-threaded context.

- C# generic collections (`Dictionary<TKey, TValue>`, `List<T>`) are type-safe and avoid boxing value types. The `decimal` type used for prices provides 28-29 significant digits of precision with no floating-point rounding errors — critical for financial applications where `double` would introduce representation errors.

- The abstract base class `BaseTradingAgent` implements `ITradingAgent` and provides common fields (`_name`, `_symbol`) while leaving the `OnMessage` method abstract for subclasses to implement. This is the Template Method pattern at the class level, reducing duplication across `BuyAgent` and `SellAgent`.

- C# top-level statements (available in .NET 6+) allow the `Main` method to be written as a single file without a wrapping class. This example uses the traditional `public class MediatorDemo { public static void Main() }` structure for consistency with the rest of the codebase, but the same code could be written with top-level statements to reduce boilerplate.
