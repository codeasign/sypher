## Overview

The Memento pattern captures an object's internal state so it can be restored later without breaking encapsulation. In a stock portfolio system, every trade changes the state — cash balance and holdings. By saving snapshots before each trade, we can undo operations by restoring a prior snapshot. The **Originator** (Portfolio) creates snapshots and restores from them. The **Memento** (PortfolioSnapshot) holds a frozen copy of state. The **Caretaker** (PortfolioHistory) manages the snapshot stack and never inspects the snapshots, preserving encapsulation.

## How the Implementation Works

The classes below form a classic Memento structure:

```
┌──────────────────────┐       ┌──────────────────────────┐
│     Portfolio        │       │   PortfolioSnapshot      │
│   (Originator)       │       │     (Memento)            │
│  ──────────────────  │       │  ──────────────────────  │
│  - #holdings: Map    │──────>│  - #holdings: Map        │
│  - #cash: number     │       │  - #cash: number         │
│  + save(): Snapshot  │<──────│  + getHoldings(): Map    │
│  + restore(snapshot) │       │  + getCash(): number     │
│  + buyTicker()       │       └───────────▲──────────────┘
│  + sellTicker()      │                   │
└──────────┬───────────┘                   │ manages
           │ uses                          │
           ▼                               │
┌──────────────────────┐   ┌───────────────┴──────────────┐
│       Holding        │   │     PortfolioHistory          │
│   (Value Object)     │   │       (Caretaker)            │
│  ──────────────────  │   │  ──────────────────────────  │
│  - #symbol           │   │  - #snapshots: Snapshot[]    │
│  - #shares           │   │  + push(snapshot)            │
│  - #totalCostBasis   │   │  + pop(): Snapshot           │
│  + averageCost       │   │  + canUndo(): boolean        │
└──────────────────────┘   └──────────────────────────────┘
```

**OOP / SOLID principles:**

- **Single Responsibility** — Portfolio manages trading logic, PortfolioSnapshot freezes state, PortfolioHistory owns the undo stack, Holding is an immutable data container.
- **Open / Closed** — Snapshot internals are closed to the caretaker but open to the originator via controlled accessors.
- **Encapsulation** — ES2022+ `#` private fields prevent any external component from mutating internal state directly. The caretaker never reads snapshot data.
- **Value Object immutability** — Holding is fully immutable; every arithmetic operation produces a new instance rather than mutating fields.

### Holding (Value Object)

```javascript
class Holding {
  #symbol;
  #shares;
  #totalCostBasis;

  constructor(symbol, shares, totalCostBasis) {
    this.#symbol = symbol;
    this.#shares = shares;
    this.#totalCostBasis = totalCostBasis;
  }

  get symbol() {
    return this.#symbol;
  }

  get shares() {
    return this.#shares;
  }

  get totalCostBasis() {
    return this.#totalCostBasis;
  }

  get averageCost() {
    return this.#shares > 0 ? this.#totalCostBasis / this.#shares : 0;
  }
}
```

### PortfolioSnapshot (Memento)

```javascript
class PortfolioSnapshot {
  #holdings;
  #cash;

  constructor(holdings, cash) {
    this.#holdings = holdings;
    this.#cash = cash;
  }

  getHoldings() {
    return this.#holdings;
  }

  getCash() {
    return this.#cash;
  }
}
```

### Portfolio (Originator)

```javascript
class Portfolio {
  #holdings;
  #cash;

  constructor(cash) {
    this.#holdings = new Map();
    this.#cash = cash;
  }

  buyTicker(symbol, shares, price) {
    const cost = shares * price;
    if (cost > this.#cash) {
      throw new Error(
        `Insufficient cash: need $${cost.toFixed(2)}, have $${this.#cash.toFixed(2)}`
      );
    }
    this.#cash -= cost;
    const existing = this.#holdings.get(symbol);
    if (existing) {
      const newShares = existing.shares + shares;
      const newCost = existing.totalCostBasis + cost;
      this.#holdings.set(symbol, new Holding(symbol, newShares, newCost));
    } else {
      this.#holdings.set(symbol, new Holding(symbol, shares, cost));
    }
  }

  sellTicker(symbol, shares) {
    const holding = this.#holdings.get(symbol);
    if (!holding || holding.shares < shares) {
      throw new Error(
        `Insufficient shares of ${symbol}: have ${holding?.shares ?? 0}, want ${shares}`
      );
    }
    const proceeds = shares * holding.averageCost;
    this.#cash += proceeds;
    const remaining = holding.shares - shares;
    if (remaining === 0) {
      this.#holdings.delete(symbol);
    } else {
      const remainingCost = holding.totalCostBasis * (remaining / holding.shares);
      this.#holdings.set(symbol, new Holding(symbol, remaining, remainingCost));
    }
  }

  save() {
    const holdingsCopy = new Map();
    for (const [symbol, holding] of this.#holdings) {
      holdingsCopy.set(
        symbol,
        new Holding(holding.symbol, holding.shares, holding.totalCostBasis)
      );
    }
    return new PortfolioSnapshot(holdingsCopy, this.#cash);
  }

  restore(snapshot) {
    this.#holdings = new Map();
    for (const [symbol, holding] of snapshot.getHoldings()) {
      this.#holdings.set(
        symbol,
        new Holding(holding.symbol, holding.shares, holding.totalCostBasis)
      );
    }
    this.#cash = snapshot.getCash();
  }

  toString() {
    const lines = [`Cash: $${this.#cash.toFixed(2)}`];
    for (const [symbol, holding] of this.#holdings) {
      lines.push(
        `  ${symbol}: ${holding.shares} shares @ $${holding.averageCost.toFixed(2)} avg (cost basis: $${holding.totalCostBasis.toFixed(2)})`
      );
    }
    return lines.join('\n');
  }
}
```

### PortfolioHistory (Caretaker)

```javascript
class PortfolioHistory {
  #snapshots;

  constructor() {
    this.#snapshots = [];
  }

  push(snapshot) {
    this.#snapshots.push(snapshot);
  }

  pop() {
    return this.#snapshots.pop();
  }

  canUndo() {
    return this.#snapshots.length > 0;
  }
}
```

### Main / Driver

```javascript
// ===== Driver =====

const portfolio = new Portfolio(500000);
const history = new PortfolioHistory();

console.log('=== Initial Portfolio ===');
console.log(portfolio.toString());

history.push(portfolio.save());

console.log('\n=== Buying 100 AAPL at $150 ===');
portfolio.buyTicker('AAPL', 100, 150);
console.log(portfolio.toString());
history.push(portfolio.save());

console.log('\n=== Buying 50 GOOGL at $2800 ===');
portfolio.buyTicker('GOOGL', 50, 2800);
console.log(portfolio.toString());
history.push(portfolio.save());

console.log('\n=== Buying 200 MSFT at $330 ===');
portfolio.buyTicker('MSFT', 200, 330);
console.log(portfolio.toString());
history.push(portfolio.save());

console.log('\n=== Undoing MSFT purchase ===');
portfolio.restore(history.pop());
console.log(portfolio.toString());

console.log('\n=== Undoing GOOGL purchase ===');
portfolio.restore(history.pop());
console.log(portfolio.toString());

console.log('\n=== Undoing AAPL purchase (back to initial) ===');
portfolio.restore(history.pop());
console.log(portfolio.toString());
```

## Language-Specific Notes

- ES2022+ `#` private fields give true hard encapsulation at the language level rather than relying on naming conventions like underscore prefix.
- The Memento's wide interface (`getHoldings`, `getCash`) is accessible only to the Originator; the Caretaker never calls these methods, preserving the encapsulation boundary by convention.
- Deep copying inside `save()` is essential — without it, the snapshot would share references with the live Portfolio, breaking the undo guarantee.
- `Map` preserves insertion order, which makes portfolio display deterministic across saves and restores without additional sorting.
- Holding is a value object: all fields are set once in the constructor and never change. Any modification produces a new instance, preventing accidental mutation through shared references.
- The `restore` method fully replaces internal state rather than merging, ensuring a clean rollback to the exact prior state with no residual data.
