## Overview

The Java 17+ implementation of the Memento pattern uses modern language features — records, sealed classes, text blocks, and pattern matching — to implement a stock portfolio with undo support. The `Portfolio` (originator) manages a collection of holdings and a cash balance. Before each state-changing trade, the caller saves a snapshot via `save()`, which returns an opaque `PortfolioSnapshot` record (the memento). The `PortfolioHistory` (caretaker) stores these snapshots on a LIFO stack, enabling multi-level undo. Because the memento is a nested static record, it has privileged access to the originator's private fields, preserving encapsulation while allowing complete state capture and restoration.

**OOP / SOLID breakdown:**

- **Single Responsibility** — `Portfolio` owns business logic (buy, sell) and snapshot creation; `PortfolioHistory` owns only the undo stack; `PortfolioSnapshot` is a pure data carrier.
- **Open/Closed** — Adding new fields to `Portfolio` (e.g., margin debt) requires updating only `save()` and `restore()`; the caretaker and clients are unaffected.
- **Liskov Substitution** — `PortfolioSnapshot` is a record with no subtyping, so no substitution issues arise.
- **Interface Segregation** — The caretaker interacts only through `push`/`pop`/`canUndo`; it never sees the memento's internals.
- **Dependency Inversion** — `PortfolioHistory` depends on `PortfolioSnapshot` (the memento abstraction), not on `Portfolio` directly.

### Holding (Immutable Record)

```java
public record Holding(String symbol, int shares, double totalCostBasis) {
    public Holding {
        if (shares < 0) {
            throw new IllegalArgumentException("Shares cannot be negative");
        }
        if (totalCostBasis < 0) {
            throw new IllegalArgumentException("Total cost basis cannot be negative");
        }
    }

    public double averageCostPerShare() {
        return totalCostBasis / shares;
    }

    @Override
    public String toString() {
        return String.format("%s: %d shares @ $%.2f avg (total $%.2f)",
            symbol, shares, averageCostPerShare(), totalCostBasis);
    }
}
```

### Portfolio (Originator)

```java
public static class Portfolio {
    private final Map<String, Holding> holdings = new LinkedHashMap<>();
    private double cash;

    /**
     * Memento — nested static record.
     * Because it is a member of Portfolio, it can access private fields
     * of the enclosing class (through save() / restore()).
     * External callers see only the record type; they cannot mutate
     * the captured state.
     */
    public record PortfolioSnapshot(Map<String, Holding> holdings, double cash) {}

    public Portfolio(double initialCash) {
        this.cash = initialCash;
    }

    public void buyTicker(String symbol, int shares, double price) {
        double cost = shares * price;
        if (cost > cash) {
            throw new IllegalStateException(
                "Insufficient cash: need $%.2f but have $%.2f".formatted(cost, cash));
        }
        cash -= cost;
        Holding existing = holdings.get(symbol);
        if (existing != null) {
            int newShares = existing.shares() + shares;
            double newCostBasis = existing.totalCostBasis() + cost;
            holdings.put(symbol, new Holding(symbol, newShares, newCostBasis));
        } else {
            holdings.put(symbol, new Holding(symbol, shares, cost));
        }
    }

    public void sellTicker(String symbol, int shares) {
        Holding existing = holdings.get(symbol);
        if (existing == null) {
            throw new IllegalStateException("No holdings for ticker: " + symbol);
        }
        if (existing.shares() < shares) {
            throw new IllegalStateException(
                "Not enough shares of %s: have %d, want %d".formatted(
                    symbol, existing.shares(), shares));
        }
        double avgCost = existing.averageCostPerShare();
        double proceeds = shares * avgCost;
        cash += proceeds;
        int remaining = existing.shares() - shares;
        if (remaining == 0) {
            holdings.remove(symbol);
        } else {
            double remainingCostBasis = existing.totalCostBasis() - (shares * avgCost);
            holdings.put(symbol, new Holding(symbol, remaining, remainingCostBasis));
        }
    }

    public PortfolioSnapshot save() {
        return new PortfolioSnapshot(new LinkedHashMap<>(holdings), cash);
    }

    public void restore(PortfolioSnapshot snapshot) {
        this.holdings.clear();
        this.holdings.putAll(snapshot.holdings());
        this.cash = snapshot.cash();
    }

    @Override
    public String toString() {
        StringBuilder sb = new StringBuilder();
        sb.append(String.format("Portfolio [Cash: $%,.2f]%n", cash));
        if (holdings.isEmpty()) {
            sb.append("  (no holdings)");
        } else {
            for (Holding h : holdings.values()) {
                sb.append("  ").append(h).append(System.lineSeparator());
            }
        }
        return sb.toString();
    }

    /**
     * Utility method using pattern matching for switch (Java 17+ preview,
     * finalized in Java 21). Demonstrates concise dispatch on the sealed
     * Order hierarchy.
     */
    public static String describeOrder(Order order) {
        return switch (order) {
            case BuyOrder b  -> "BUY  %d %s @ $%.2f".formatted(b.shares(), b.symbol(), b.price());
            case SellOrder s -> "SELL %d %s".formatted(s.shares(), s.symbol());
        };
    }
}
```

### PortfolioHistory (Caretaker)

```java
public static class PortfolioHistory {
    private final Deque<Portfolio.PortfolioSnapshot> history = new ArrayDeque<>();

    public void push(Portfolio.PortfolioSnapshot snapshot) {
        history.push(snapshot);
    }

    public Portfolio.PortfolioSnapshot pop() {
        if (history.isEmpty()) {
            throw new java.util.EmptyStackException();
        }
        return history.pop();
    }

    public boolean canUndo() {
        return !history.isEmpty();
    }

    public int size() {
        return history.size();
    }
}
```

### Main / Driver

```java
public static void main(String[] args) {
    // Use text block for the title banner (Java 17+ feature)
    String banner = """
        ╔══════════════════════════════════════════════════╗
        ║   Memento Pattern — Stock Portfolio Demo (Java) ║
        ╚══════════════════════════════════════════════════╝
        """;
    System.out.println(banner);

    Portfolio portfolio = new Portfolio(1_000_000.00);
    PortfolioHistory history = new PortfolioHistory();

    System.out.println("=== Initial State ===");
    System.out.println(portfolio);

    // --- Build a position: buy AAPL, GOOGL, MSFT ---
    System.out.println("--- BUY 100 AAPL @ $150.00 ---");
    System.out.println("  " + Portfolio.describeOrder(new BuyOrder("AAPL", 100, 150.00)));
    portfolio.buyTicker("AAPL", 100, 150.00);
    System.out.println(portfolio);

    System.out.println("--- BUY 50 GOOGL @ $2,800.00 ---");
    System.out.println("  " + Portfolio.describeOrder(new BuyOrder("GOOGL", 50, 2800.00)));
    portfolio.buyTicker("GOOGL", 50, 2800.00);
    System.out.println(portfolio);

    System.out.println("--- BUY 200 MSFT @ $330.00 ---");
    System.out.println("  " + Portfolio.describeOrder(new BuyOrder("MSFT", 200, 330.00)));
    portfolio.buyTicker("MSFT", 200, 330.00);
    System.out.println(portfolio);

    // Save checkpoint after all buys
    System.out.println("--- Saving checkpoint (after all buys) ---");
    history.push(portfolio.save());
    System.out.println("  History size: " + history.size());

    // --- Execute some sells ---
    System.out.println("--- SELL 30 AAPL ---");
    System.out.println("  " + Portfolio.describeOrder(new SellOrder("AAPL", 30)));
    portfolio.sellTicker("AAPL", 30);
    System.out.println(portfolio);

    System.out.println("--- SELL 10 GOOGL ---");
    System.out.println("  " + Portfolio.describeOrder(new SellOrder("GOOGL", 10)));
    portfolio.sellTicker("GOOGL", 10);
    System.out.println(portfolio);

    // --- Undo back to checkpoint ---
    System.out.println("=== Undoing trades ===");
    System.out.println("  Can undo? " + history.canUndo());
    if (history.canUndo()) {
        Portfolio.PortfolioSnapshot checkpoint = history.pop();
        System.out.println("  Restoring checkpoint from history...");
        portfolio.restore(checkpoint);
        System.out.println("  History size after undo: " + history.size());
        System.out.println();
        System.out.println("=== Portfolio After Undo ===");
        System.out.println(portfolio);
    }
}
```

## Language-Specific Notes

- The memento is implemented as a nested static `record` within the `Portfolio` class. This gives it privileged access to the originator's private fields while exposing a fully opaque type to external callers — the caretaker stores and retrieves mementos without ever inspecting their contents.
- `LinkedHashMap` is used for holdings to preserve insertion order, making portfolio display predictable and matching the order in which trades were executed.
- `ArrayDeque` is used instead of the legacy `Stack` class for the caretaker's history. It provides LIFO operations (`push`/`pop`) with better performance, no synchronization overhead, and a modern API.
- The sealed `Order` interface with `BuyOrder` and `SellOrder` records demonstrates how sealed types enable exhaustive pattern matching — the compiler verifies that all permitted subtypes are handled in the `switch` expression.
- Pattern matching for `switch` (a preview in Java 17, finalized in Java 21) allows the `describeOrder` method to dispatch on the sealed `Order` hierarchy without manual `instanceof` chains or casting.
- Text blocks (delimited by `"""` ) provide clean multi-line string formatting for the title banner and could be extended to generate trade confirmations, portfolio summaries, or serialized snapshot descriptions.
