## Implementation

Below is a complete Java implementation of the Flyweight pattern using a car configurator domain. The `CarSpec` interface defines the flyweight contract with a `display()` method. Three concrete flyweight classes (`EngineSpec`, `WheelSpec`, `PaintSpec`) implement this interface with immutable intrinsic state stored in `final` fields. The `CarSpecFactory` maintains concurrent hash maps for thread-safe caching of flyweight objects. The `CarConfiguration` class stores extrinsic state (VIN, owner, license plate, production date) and holds references to shared flyweight objects.

### How the Implementation Works

#### Class Diagram (Pseudocode)


#### OOP Concepts Applied

1. **Abstraction** — The `CarSpec` interface defines the `display()` contract. All client code depends on this abstraction.

2. **Encapsulation** — Flyweight fields are `private final`. The factory encapsulates the cache maps.

3. **Polymorphism** — `CarConfiguration.display()` calls `display()` on each flyweight reference without knowing the concrete type.

#### SOLID Principles Applied

4. **SRP** — Each flyweight class has a single responsibility. The factory manages caching. The configuration represents a car.

5. **OCP** — New flyweight types can be added by implementing `CarSpec` and adding a new cache map in the factory.

6. **LSP** — All concrete flyweights implement `CarSpec` and can be substituted wherever `CarSpec` is expected.

7. **ISP** — The `CarSpec` interface contains only `display()`. Flyweights are not forced to implement unused methods.

8. **DIP** — `CarConfiguration` depends on `CarSpec` abstractions, not on concrete flyweight classes.

### CarSpec (Flyweight Interface)

```java
// Flyweight Pattern — Java — Car Configurator Example
// ==========================================================
// Participants: CarSpec (Flyweight), EngineSpec/WheelSpec/PaintSpec
//   (Concrete Flyweights), CarSpecFactory (FlyweightFactory),
//   CarConfiguration (Client)
// OOP: abstraction, encapsulation, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// ==========================================================

import java.util.Map;
import java.util.concurrent.ConcurrentHashMap;
import java.util.List;
import java.util.ArrayList;

// ── Flyweight interface ──
// OOP role: Abstraction + ISP
// SOLID: ISP, DIP
interface CarSpec {
    String display();
}

```

### EngineSpec

```java
// ── Concrete Flyweight: EngineSpec ──
// Intrinsic state: model, horsepower, displacement, cylinders, fuel type
// OOP role: Inheritance/Polymorphism
// SOLID: LSP, SRP
final class EngineSpec implements CarSpec {
    private final String model;
    private final int horsepower;
    private final double displacement;
    private final int cylinders;
    private final String fuelType;

    EngineSpec(String model, int horsepower, double displacement,
               int cylinders, String fuelType) {
        this.model = model;
        this.horsepower = horsepower;
        this.displacement = displacement;
        this.cylinders = cylinders;
        this.fuelType = fuelType;
    }

    public String getModel() { return model; }
    public int getHorsepower() { return horsepower; }
    public double getDisplacement() { return displacement; }
    public int getCylinders() { return cylinders; }
    public String getFuelType() { return fuelType; }

    @Override
    public String display() {
        return String.format("%s | %d hp | %.1fL | %d cyl | %s",
            model, horsepower, displacement, cylinders, fuelType);
    }
}

```

### CarSpecFactory

```java
// ── Flyweight Factory ──
// OOP role: Composition
// SOLID: SRP (manages flyweight cache), DIP (returns CarSpec abstractions)
final class CarSpecFactory {
    private final Map<String, EngineSpec> engines = new ConcurrentHashMap<>();
    private final Map<String, WheelSpec> wheels = new ConcurrentHashMap<>();
    private final Map<String, PaintSpec> paints = new ConcurrentHashMap<>();

    EngineSpec getEngine(String model, int horsepower, double displacement,
                         int cylinders, String fuelType) {
        return engines.computeIfAbsent(model, k ->
            new EngineSpec(model, horsepower, displacement, cylinders, fuelType));
    }

    WheelSpec getWheel(String key, double diameter, int width,
                       String material, String brand) {
        return wheels.computeIfAbsent(key, k ->
            new WheelSpec(diameter, width, material, brand));
    }

    PaintSpec getPaint(String code, String name, String finish, String hexColor) {
        return paints.computeIfAbsent(code, k ->
            new PaintSpec(code, name, finish, hexColor));
    }

    String getStats() {
        return String.format(
            "Flyweight Cache Stats:\n  Engines: %d unique\n  Wheels:  %d unique\n  Paints:  %d unique",
            engines.size(), wheels.size(), paints.size());
    }
}

```

### CarConfiguration

```java
// ── Client: CarConfiguration ──
// Extrinsic state: VIN, owner, license plate, production date
// References to shared flyweights: engine, wheels, paint
// OOP role: Encapsulation, Composition
// SOLID: SRP (represents one car), DIP (depends on CarSpec abstractions)
final class CarConfiguration {
    private final String vin;
    private final String owner;
    private final String licensePlate;
    private final String productionDate;
    private final EngineSpec engine;
    private final WheelSpec wheels;
    private final PaintSpec paint;

    CarConfiguration(String vin, String owner, String licensePlate,
                     String productionDate, EngineSpec engine,
                     WheelSpec wheels, PaintSpec paint) {
        this.vin = vin;
        this.owner = owner;
        this.licensePlate = licensePlate;
        this.productionDate = productionDate;
        this.engine = engine;
        this.wheels = wheels;
        this.paint = paint;
    }

    public EngineSpec getEngine() { return engine; }
    public WheelSpec getWheels() { return wheels; }
    public PaintSpec getPaint() { return paint; }

    String display() {
        return String.join("\n",
            "--- Car Configuration ---",
            "VIN:             " + vin,
            "Owner:           " + owner,
            "License Plate:   " + licensePlate,
            "Production Date: " + productionDate,
            "Engine:          " + engine.display(),
            "Wheels:          " + wheels.display(),
            "Paint:           " + paint.display(),
            "---"
        );
    }
}

```

### Main / Driver

```java
// ── Main / Driver ──
public final class FlyweightDemo {
    public static void main(String[] args) {
        CarSpecFactory factory = new CarSpecFactory();

        // Create flyweights (each unique spec is created once)
        EngineSpec engine2L = factory.getEngine("2.0L Turbo", 250, 2.0, 4, "Gasoline");
        EngineSpec engine3L = factory.getEngine("3.0L V6", 340, 3.0, 6, "Gasoline");
        EngineSpec engineElectric = factory.getEngine("Electric Drive", 280, 0.0, 0, "Electric");

        WheelSpec wheel18 = factory.getWheel("18x8 Sport", 18.0, 8, "Alloy", "Sport");
        WheelSpec wheel20 = factory.getWheel("20x9 Performance", 20.0, 9, "Forged", "Performance");

        PaintSpec paintRed = factory.getPaint("R3", "Racing Red", "Gloss", "#FF2020");
        PaintSpec paintBlue = factory.getPaint("B7", "Midnight Blue", "Metallic", "#002060");
        PaintSpec paintWhite = factory.getPaint("W1", "Pearl White", "Pearl", "#F5F5F5");

        // Create car configurations that share flyweights
        List<CarConfiguration> cars = new ArrayList<>();
        cars.add(new CarConfiguration("WBA1234567890ABCD", "Alice Johnson", "ABC-1234",
            "2026-01-15", engine2L, wheel18, paintRed));
        cars.add(new CarConfiguration("WBA2345678901BCDE", "Bob Smith", "XYZ-5678",
            "2026-02-20", engine2L, wheel18, paintBlue));
        cars.add(new CarConfiguration("WBA3456789012CDEF", "Carol Williams", "DEF-9012",
            "2026-03-10", engine3L, wheel20, paintRed));
        cars.add(new CarConfiguration("WBA4567890123DEFG", "Dave Brown", "GHI-3456",
            "2026-04-05", engineElectric, wheel20, paintWhite));
        cars.add(new CarConfiguration("WBA5678901234EFGH", "Eve Davis", "JKL-7890",
            "2026-05-18", engine2L, wheel18, paintRed));
        cars.add(new CarConfiguration("WBA6789012345FGHI", "Frank Miller", "MNO-1234",
            "2026-06-01", engine3L, wheel20, paintBlue));

        // Display all car configurations
        System.out.println("=== Car Configurator — Flyweight Pattern ===\n");

        for (CarConfiguration car : cars) {
            System.out.println(car.display());
            System.out.println();
        }

        // Demonstrate flyweight sharing
        System.out.println("=== Flyweight Sharing Verification ===");
        System.out.println("Car 1 and Car 5 share engine: "
            + (cars.get(0).getEngine() == cars.get(4).getEngine()) + " (expected: true)");
        System.out.println("Car 1 and Car 4 share engine: "
            + (cars.get(0).getEngine() == cars.get(3).getEngine()) + " (expected: false)");
        System.out.println("Car 1 and Car 2 share wheels: "
            + (cars.get(0).getWheels() == cars.get(1).getWheels()) + " (expected: true)");
        System.out.println("Car 1 and Car 3 share wheels: "
            + (cars.get(0).getWheels() == cars.get(2).getWheels()) + " (expected: false)");
        System.out.println("Car 1 and Car 5 share paint: "
            + (cars.get(0).getPaint() == cars.get(4).getPaint()) + " (expected: true)");
        System.out.println("Car 1 and Car 4 share paint: "
            + (cars.get(0).getPaint() == cars.get(3).getPaint()) + " (expected: false)");

        System.out.println();
        System.out.println(factory.getStats());
        System.out.println("\nTotal car configurations that share them: " + cars.size());
    }
}
```

### Language-Specific Notes

- **`ConcurrentHashMap` for thread-safe caching** — The factory uses `ConcurrentHashMap` instead of `HashMap` to ensure thread-safe access to the flyweight cache. The `computeIfAbsent()` method is atomic: it checks whether the key exists and creates the flyweight if absent in a single, thread-safe operation. This eliminates the need for explicit `synchronized` blocks and provides better performance under concurrent access.

- **`final` fields for immutability** — All flyweight fields are declared `private final`. The `final` keyword guarantees that the fields are assigned exactly once in the constructor and cannot be reassigned. Together with the absence of setter methods, this ensures that flyweight objects are immutable. The `final` class declaration prevents subclassing, which could otherwise introduce mutable state.

- **`String.format()` for consistent display formatting** — The `display()` methods use `String.format()` with format specifiers (`%s`, `%d`, `%.1f`, `%.0f`) to produce consistent, localized output. This is more maintainable than string concatenation, especially when the format specification changes.

- **`==` for identity comparison** — Java's `==` operator on objects checks reference equality, not structural equality. The driver uses `==` to verify that two `CarConfiguration` objects reference the same flyweight instance. This is both the intent and the verification: the factory ensures that the same key returns the same object reference, and `==` confirms it.

- **`List<CarConfiguration>` for the car collection** — The driver stores car configurations in a `List` (specifically `ArrayList`) to enable indexed access for the sharing verification section. The `List.get(index)` method provides O(1) access to individual cars for comparison.

- **No `Cloneable` or `Serializable`** — Unlike the Prototype pattern, the Flyweight pattern does not require cloning or serialization of flyweight objects. The flyweights are intentionally simple data holders with a `display()` method. The absence of these interfaces keeps the flyweight classes small and focused on their single responsibility.
