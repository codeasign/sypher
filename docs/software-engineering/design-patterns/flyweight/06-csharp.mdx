## Implementation

Below is a complete C# implementation of the Flyweight pattern using a car configurator domain. The `ICarSpec` interface defines the flyweight contract with a `Display()` method. Three concrete flyweight classes (`EngineSpec`, `WheelSpec`, `PaintSpec`) implement this interface with immutable intrinsic state stored in `init`-only properties. The `CarSpecFactory` uses `ConcurrentDictionary` for thread-safe caching of flyweight objects. The `CarConfiguration` class stores extrinsic state (VIN, owner, license plate, production date) and holds references to shared flyweight objects.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────────┐
//   │   <<interface>>                      │
//   │   ICarSpec                            │  ← OOP: Abstraction
//   ├──────────────────────────────────────┤     ISP: minimal contract
//   │ + Display(): string                  │
//   └──────────────────┬───────────────────┘
//                      ▲ implements
//   ┌──────────────────┼───────────────────┐
//   │                  │                   │
//   ┌────────────────┐ ┌────────────────┐ ┌────────────────┐
//   │   EngineSpec   │ │   WheelSpec    │ │   PaintSpec    │
//   ├────────────────┤ ├────────────────┤ ├────────────────┤
//   │ {get; init;}   │ │ {get; init;}   │ │ {get; init;}   │
//   │ + Display()    │ │ + Display()    │ │ + Display()    │
//   └────────────────┘ └────────────────┘ └────────────────┘
//
//   ┌────────────────────────────────────────────────────────┐
//   │   CarSpecFactory (FlyweightFactory)                    │  ← SRP: cache
//   ├────────────────────────────────────────────────────────┤
//   │ - engines: ConcurrentDictionary<string, EngineSpec>    │
//   │ - wheels:  ConcurrentDictionary<string, WheelSpec>     │
//   │ - paints:  ConcurrentDictionary<string, PaintSpec>     │
//   │ + GetEngine(...): EngineSpec                           │
//   │ + GetWheel(...): WheelSpec                             │
//   │ + GetPaint(...): PaintSpec                             │
//   │ + GetStats(): string                                   │
//   └────────────────────────────────────────────────────────┘
//
//   ┌────────────────────────────────────────────────────────┐
//   │   CarConfiguration (Client)                            │
//   ├────────────────────────────────────────────────────────┤
//   │ - vin: string, - owner: string, - plate: string       │
//   │ - prodDate: string                                     │
//   │ - engine: EngineSpec, - wheels: WheelSpec              │
//   │ - paint: PaintSpec                                     │
//   │ + Display(): string                                    │
//   └────────────────────────────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — The `ICarSpec` interface defines the `Display()` contract. All client code depends on this abstraction.

2. **Encapsulation** — Flyweight properties are `{get; init;}`. The factory encapsulates the cache dictionaries.

3. **Polymorphism** — `CarConfiguration.Display()` calls `Display()` on each flyweight reference without knowing the concrete type.

#### SOLID Principles Applied

4. **SRP** — Each flyweight class has a single responsibility. The factory manages caching. The configuration represents a car.

5. **OCP** — New flyweight types can be added by implementing `ICarSpec` and adding a new cache dictionary in the factory.

6. **LSP** — All concrete flyweights implement `ICarSpec` and can be substituted wherever `ICarSpec` is expected.

7. **ISP** — The `ICarSpec` interface contains only `Display()`. Flyweights are not forced to implement unused methods.

8. **DIP** — `CarConfiguration` depends on `ICarSpec` abstractions, not on concrete flyweight classes.

### CarSpec (Flyweight Interface)

```csharp
// Flyweight Pattern — C# — Car Configurator Example
// ==========================================================
// Participants: ICarSpec (Flyweight), EngineSpec/WheelSpec/PaintSpec
//   (Concrete Flyweights), CarSpecFactory (FlyweightFactory),
//   CarConfiguration (Client)
// OOP: abstraction, encapsulation, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// ==========================================================

using System;
using System.Collections.Concurrent;
using System.Collections.Generic;
using System.Text;

// ── Flyweight interface ──
// OOP role: Abstraction + ISP
// SOLID: ISP, DIP
public interface ICarSpec
{
    string Display();
}

```

### EngineSpec

```csharp
// ── Concrete Flyweight: EngineSpec ──
// Intrinsic state: model, horsepower, displacement, cylinders, fuel type
// OOP role: Inheritance/Polymorphism
// SOLID: LSP, SRP
public sealed class EngineSpec : ICarSpec
{
    public string Model { get; init; }
    public int Horsepower { get; init; }
    public double Displacement { get; init; }
    public int Cylinders { get; init; }
    public string FuelType { get; init; }

    public string Display()
    {
        return $"{Model} | {Horsepower} hp | {Displacement:F1}L | {Cylinders} cyl | {FuelType}";
    }
}

```

### CarSpecFactory

```csharp
// ── Flyweight Factory ──
// OOP role: Composition
// SOLID: SRP (manages flyweight cache), DIP (returns ICarSpec abstractions)
public sealed class CarSpecFactory
{
    private readonly ConcurrentDictionary<string, EngineSpec> _engines = new();
    private readonly ConcurrentDictionary<string, WheelSpec> _wheels = new();
    private readonly ConcurrentDictionary<string, PaintSpec> _paints = new();

    public EngineSpec GetEngine(string model, int horsepower, double displacement,
                                int cylinders, string fuelType)
    {
        return _engines.GetOrAdd(model, _ =>
            new EngineSpec
            {
                Model = model,
                Horsepower = horsepower,
                Displacement = displacement,
                Cylinders = cylinders,
                FuelType = fuelType
            });
    }

    public WheelSpec GetWheel(string key, double diameter, int width,
                              string material, string brand)
    {
        return _wheels.GetOrAdd(key, _ =>
            new WheelSpec
            {
                Diameter = diameter,
                Width = width,
                Material = material,
                Brand = brand
            });
    }

    public PaintSpec GetPaint(string code, string name, string finish, string hexColor)
    {
        return _paints.GetOrAdd(code, _ =>
            new PaintSpec
            {
                Code = code,
                Name = name,
                Finish = finish,
                HexColor = hexColor
            });
    }

    public string GetStats()
    {
        var sb = new StringBuilder();
        sb.AppendLine("Flyweight Cache Stats:");
        sb.AppendLine($"  Engines: {_engines.Count} unique");
        sb.AppendLine($"  Wheels:  {_wheels.Count} unique");
        sb.AppendLine($"  Paints:  {_paints.Count} unique");
        return sb.ToString().TrimEnd();
    }
}

```

### CarConfiguration

```csharp
// ── Client: CarConfiguration ──
// Extrinsic state: VIN, owner, license plate, production date
// References to shared flyweights: engine, wheels, paint
// OOP role: Encapsulation, Composition
// SOLID: SRP (represents one car), DIP (depends on ICarSpec abstractions)
public sealed class CarConfiguration
{
    private readonly string _vin;
    private readonly string _owner;
    private readonly string _licensePlate;
    private readonly string _productionDate;
    private readonly EngineSpec _engine;
    private readonly WheelSpec _wheels;
    private readonly PaintSpec _paint;

    public CarConfiguration(string vin, string owner, string licensePlate,
                            string productionDate, EngineSpec engine,
                            WheelSpec wheels, PaintSpec paint)
    {
        _vin = vin;
        _owner = owner;
        _licensePlate = licensePlate;
        _productionDate = productionDate;
        _engine = engine;
        _wheels = wheels;
        _paint = paint;
    }

    public EngineSpec Engine => _engine;
    public WheelSpec Wheels => _wheels;
    public PaintSpec Paint => _paint;

    public string Display()
    {
        var sb = new StringBuilder();
        sb.AppendLine("--- Car Configuration ---");
        sb.AppendLine($"VIN:             {_vin}");
        sb.AppendLine($"Owner:           {_owner}");
        sb.AppendLine($"License Plate:   {_licensePlate}");
        sb.AppendLine($"Production Date: {_productionDate}");
        sb.AppendLine($"Engine:          {_engine.Display()}");
        sb.AppendLine($"Wheels:          {_wheels.Display()}");
        sb.AppendLine($"Paint:           {_paint.Display()}");
        sb.AppendLine("---");
        return sb.ToString().TrimEnd();
    }
}

```

### Main / Driver

```csharp
// ── Main / Driver ──
public static class FlyweightDemo
{
    public static void Main()
    {
        var factory = new CarSpecFactory();

        // Create flyweights (each unique spec is created once)
        var engine2L = factory.GetEngine("2.0L Turbo", 250, 2.0, 4, "Gasoline");
        var engine3L = factory.GetEngine("3.0L V6", 340, 3.0, 6, "Gasoline");
        var engineElectric = factory.GetEngine("Electric Drive", 280, 0.0, 0, "Electric");

        var wheel18 = factory.GetWheel("18x8 Sport", 18.0, 8, "Alloy", "Sport");
        var wheel20 = factory.GetWheel("20x9 Performance", 20.0, 9, "Forged", "Performance");

        var paintRed = factory.GetPaint("R3", "Racing Red", "Gloss", "#FF2020");
        var paintBlue = factory.GetPaint("B7", "Midnight Blue", "Metallic", "#002060");
        var paintWhite = factory.GetPaint("W1", "Pearl White", "Pearl", "#F5F5F5");

        // Create car configurations that share flyweights
        var cars = new List<CarConfiguration>
        {
            new("WBA1234567890ABCD", "Alice Johnson", "ABC-1234",
                "2026-01-15", engine2L, wheel18, paintRed),
            new("WBA2345678901BCDE", "Bob Smith", "XYZ-5678",
                "2026-02-20", engine2L, wheel18, paintBlue),
            new("WBA3456789012CDEF", "Carol Williams", "DEF-9012",
                "2026-03-10", engine3L, wheel20, paintRed),
            new("WBA4567890123DEFG", "Dave Brown", "GHI-3456",
                "2026-04-05", engineElectric, wheel20, paintWhite),
            new("WBA5678901234EFGH", "Eve Davis", "JKL-7890",
                "2026-05-18", engine2L, wheel18, paintRed),
            new("WBA6789012345FGHI", "Frank Miller", "MNO-1234",
                "2026-06-01", engine3L, wheel20, paintBlue)
        };

        // Display all car configurations
        Console.WriteLine("=== Car Configurator — Flyweight Pattern ===\n");

        foreach (var car in cars)
        {
            Console.WriteLine(car.Display());
            Console.WriteLine();
        }

        // Demonstrate flyweight sharing
        Console.WriteLine("=== Flyweight Sharing Verification ===");
        Console.WriteLine($"Car 1 and Car 5 share engine: " +
            $"{ReferenceEquals(cars[0].Engine, cars[4].Engine)} (expected: True)");
        Console.WriteLine($"Car 1 and Car 4 share engine: " +
            $"{ReferenceEquals(cars[0].Engine, cars[3].Engine)} (expected: False)");
        Console.WriteLine($"Car 1 and Car 2 share wheels: " +
            $"{ReferenceEquals(cars[0].Wheels, cars[1].Wheels)} (expected: True)");
        Console.WriteLine($"Car 1 and Car 3 share wheels: " +
            $"{ReferenceEquals(cars[0].Wheels, cars[2].Wheels)} (expected: False)");
        Console.WriteLine($"Car 1 and Car 5 share paint: " +
            $"{ReferenceEquals(cars[0].Paint, cars[4].Paint)} (expected: True)");
        Console.WriteLine($"Car 1 and Car 4 share paint: " +
            $"{ReferenceEquals(cars[0].Paint, cars[3].Paint)} (expected: False)");

        Console.WriteLine();
        Console.WriteLine(factory.GetStats());
        Console.WriteLine($"\nTotal car configurations that share them: {cars.Count}");
    }
}
```

### Language-Specific Notes

- **`sealed` classes for flyweight safety** — All flyweight classes are declared `sealed` to prevent inheritance. Subclassing a flyweight could introduce mutable state or override the `Display()` method in ways that violate the flyweight contract. The `sealed` keyword provides compile-time enforcement of this constraint.

- **`{get; init;}` properties for immutable intrinsic state** — C# 9.0's `init` accessors enable property assignment only during object initialization (in the constructor or object initializer). After initialization, the properties are effectively read-only. This is the idiomatic C# approach to creating immutable objects without writing boilerplate constructor code.

- **`ConcurrentDictionary.GetOrAdd()` for atomic cache operations** — The factory uses `ConcurrentDictionary<TKey, TValue>` with the `GetOrAdd()` method, which atomically checks whether a key exists and creates the value if it does not. This is thread-safe and more concise than the equivalent check-then-create pattern with explicit locking. The lambda parameter `_` (discard) ignores the key argument since the factory methods already receive all necessary parameters.

- **`StringBuilder` for efficient string concatenation** — The `GetStats()` and `Display()` methods use `StringBuilder` for building multi-line strings. `StringBuilder` is more memory-efficient than repeated string concatenation (`+` or `$""`) for strings with many lines because it avoids creating intermediate string objects.

- **`ReferenceEquals()` for identity comparison** — C# uses `ReferenceEquals()` instead of `==` to check reference equality. The `==` operator can be overloaded to provide value equality (as with `string`), so `ReferenceEquals()` is the unambiguous way to check whether two references point to the same object. The driver uses this to verify flyweight sharing.

- **Object initializer syntax for flyweight construction** — The factory uses C# object initializer syntax (`new EngineSpec { Model = model, ... }`) instead of constructor parameters. This is idiomatic in C# for classes with many properties and improves readability by making each property assignment explicit. The `init` accessors allow this syntax while still enforcing immutability after construction.
