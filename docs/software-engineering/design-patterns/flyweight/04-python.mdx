## Implementation

Below is a complete Python implementation of the Flyweight pattern using a car configurator domain. The `CarSpec` abstract base class defines the flyweight contract with a `display()` method. Three concrete flyweight classes (`EngineSpec`, `WheelSpec`, `PaintSpec`) store immutable intrinsic state and are shared across many `CarConfiguration` instances. The `CarSpecFactory` acts as a cache, returning existing flyweight objects when the same key is requested again. Each `CarConfiguration` stores its own extrinsic state alongside references to shared flyweights.

### How the Implementation Works

#### OOP Concepts Applied

1. **Abstraction** — `CarSpec` ABC defines the `display()` contract. All client code depends on this abstraction.

2. **Encapsulation** — Flyweight classes hide their internal state using `__slots__` and provide no public setters.

3. **Polymorphism** — `CarConfiguration.display()` calls `display()` on each flyweight without knowing the concrete type.

#### SOLID Principles Applied

4. **SRP** — Each flyweight class has one responsibility. The factory manages caching. The configuration represents a car.

5. **OCP** — New flyweight types can be added by subclassing `CarSpec` and adding a cache dict in the factory.

6. **LSP** — All concrete flyweights inherit from `CarSpec` and can be substituted wherever `CarSpec` is expected.

7. **ISP** — The `CarSpec` interface contains only `display()`. Flyweights are not forced to implement unused methods.

8. **DIP** — `CarConfiguration` depends on `CarSpec` abstractions, not on concrete flyweight classes.

### CarSpec (Flyweight Interface)

```python
# Flyweight Pattern — Python — Car Configurator Example
# ==========================================================
# Participants: CarSpec (Flyweight), EngineSpec/WheelSpec/PaintSpec
#   (Concrete Flyweights), CarSpecFactory (FlyweightFactory),
#   CarConfiguration (Client)
# OOP: abstraction, encapsulation, polymorphism
# SOLID: SRP, OCP, LSP, ISP, DIP
# ==========================================================

from abc import ABC, abstractmethod
from typing import Dict, Optional


# ── Flyweight interface (ABC) ──
# OOP role: Abstraction + ISP
# SOLID: ISP, DIP
class CarSpec(ABC):
    """Flyweight interface for all car specifications."""

    @abstractmethod
    def display(self) -> str:
        """Return a formatted display string of the specification."""
        pass


```

### EngineSpec

```python
# ── Concrete Flyweight: EngineSpec ──
# Intrinsic state: model, horsepower, displacement, cylinders, fuel type
# OOP role: Inheritance/Polymorphism
# SOLID: LSP, SRP
class EngineSpec(CarSpec):
    """Flyweight storing shared engine specification data."""

    __slots__ = ('_model', '_horsepower', '_displacement', '_cylinders', '_fuel_type')

    def __init__(
        self,
        model: str,
        horsepower: int,
        displacement: float,
        cylinders: int,
        fuel_type: str,
    ) -> None:
        self._model = model
        self._horsepower = horsepower
        self._displacement = displacement
        self._cylinders = cylinders
        self._fuel_type = fuel_type

    @property
    def model(self) -> str:
        return self._model

    @property
    def horsepower(self) -> int:
        return self._horsepower

    @property
    def displacement(self) -> float:
        return self._displacement

    @property
    def cylinders(self) -> int:
        return self._cylinders

    @property
    def fuel_type(self) -> str:
        return self._fuel_type

    def display(self) -> str:
        return (
            f"{self._model} | {self._horsepower} hp | "
            f"{self._displacement}L | {self._cylinders} cyl | {self._fuel_type}"
        )


```

### CarSpecFactory

```python
# ── Flyweight Factory ──
# OOP role: Composition
# SOLID: SRP (manages flyweight cache), DIP (returns CarSpec abstractions)
class CarSpecFactory:
    """Creates and caches flyweight car specification objects."""

    def __init__(self) -> None:
        self._engines: Dict[str, EngineSpec] = {}
        self._wheels: Dict[str, WheelSpec] = {}
        self._paints: Dict[str, PaintSpec] = {}

    def get_engine(
        self,
        model: str,
        horsepower: int,
        displacement: float,
        cylinders: int,
        fuel_type: str,
    ) -> EngineSpec:
        """Return an EngineSpec flyweight, creating it if not cached."""
        if model not in self._engines:
            self._engines[model] = EngineSpec(
                model, horsepower, displacement, cylinders, fuel_type
            )
        return self._engines[model]

    def get_wheel(
        self, key: str, diameter: float, width: int, material: str, brand: str
    ) -> WheelSpec:
        """Return a WheelSpec flyweight, creating it if not cached."""
        if key not in self._wheels:
            self._wheels[key] = WheelSpec(diameter, width, material, brand)
        return self._wheels[key]

    def get_paint(
        self, code: str, name: str, finish: str, hex_color: str
    ) -> PaintSpec:
        """Return a PaintSpec flyweight, creating it if not cached."""
        if code not in self._paints:
            self._paints[code] = PaintSpec(code, name, finish, hex_color)
        return self._paints[code]

    def get_stats(self) -> Dict[str, int]:
        """Return cache statistics as a dictionary."""
        return {
            'engines': len(self._engines),
            'wheels': len(self._wheels),
            'paints': len(self._paints),
            'total': len(self._engines) + len(self._wheels) + len(self._paints),
        }


```

### CarConfiguration

```python
# ── Client: CarConfiguration ──
# Extrinsic state: VIN, owner, license plate, production date
# References to shared flyweights: engine, wheels, paint
# OOP role: Encapsulation, Composition
# SOLID: SRP (represents one car), DIP (depends on CarSpec abstractions)
class CarConfiguration:
    """A car configuration that references shared flyweight spec objects."""

    def __init__(
        self,
        vin: str,
        owner: str,
        license_plate: str,
        production_date: str,
        engine: EngineSpec,
        wheels: WheelSpec,
        paint: PaintSpec,
    ) -> None:
        self._vin = vin
        self._owner = owner
        self._license_plate = license_plate
        self._production_date = production_date
        self._engine = engine
        self._wheels = wheels
        self._paint = paint

    @property
    def engine(self) -> EngineSpec:
        return self._engine

    @property
    def wheels(self) -> WheelSpec:
        return self._wheels

    @property
    def paint(self) -> PaintSpec:
        return self._paint

    def display(self) -> str:
        lines = [
            '--- Car Configuration ---',
            f'VIN:             {self._vin}',
            f'Owner:           {self._owner}',
            f'License Plate:   {self._license_plate}',
            f'Production Date: {self._production_date}',
            f'Engine:          {self._engine.display()}',
            f'Wheels:          {self._wheels.display()}',
            f'Paint:           {self._paint.display()}',
            '---',
        ]
        return '\n'.join(lines)


```

### Main / Driver

```python
# ── Main / Driver ──
def main() -> None:
    factory = CarSpecFactory()

    # Create flyweights (each unique spec is created once)
    engine_2l = factory.get_engine('2.0L Turbo', 250, 2.0, 4, 'Gasoline')
    engine_3l = factory.get_engine('3.0L V6', 340, 3.0, 6, 'Gasoline')
    engine_electric = factory.get_engine('Electric Drive', 280, 0.0, 0, 'Electric')

    wheel_18 = factory.get_wheel('18x8 Sport', 18.0, 8, 'Alloy', 'Sport')
    wheel_20 = factory.get_wheel('20x9 Performance', 20.0, 9, 'Forged', 'Performance')

    paint_red = factory.get_paint('R3', 'Racing Red', 'Gloss', '#FF2020')
    paint_blue = factory.get_paint('B7', 'Midnight Blue', 'Metallic', '#002060')
    paint_white = factory.get_paint('W1', 'Pearl White', 'Pearl', '#F5F5F5')

    # Create car configurations that share flyweights
    cars = [
        CarConfiguration('WBA1234567890ABCD', 'Alice Johnson', 'ABC-1234',
                         '2026-01-15', engine_2l, wheel_18, paint_red),
        CarConfiguration('WBA2345678901BCDE', 'Bob Smith', 'XYZ-5678',
                         '2026-02-20', engine_2l, wheel_18, paint_blue),
        CarConfiguration('WBA3456789012CDEF', 'Carol Williams', 'DEF-9012',
                         '2026-03-10', engine_3l, wheel_20, paint_red),
        CarConfiguration('WBA4567890123DEFG', 'Dave Brown', 'GHI-3456',
                         '2026-04-05', engine_electric, wheel_20, paint_white),
        CarConfiguration('WBA5678901234EFGH', 'Eve Davis', 'JKL-7890',
                         '2026-05-18', engine_2l, wheel_18, paint_red),
        CarConfiguration('WBA6789012345FGHI', 'Frank Miller', 'MNO-1234',
                         '2026-06-01', engine_3l, wheel_20, paint_blue),
    ]

    # Display all car configurations
    print('=== Car Configurator — Flyweight Pattern ===\n')

    for car in cars:
        print(car.display())
        print()

    # Demonstrate flyweight sharing
    print('=== Flyweight Sharing Verification ===')
    car1_engine = cars[0].engine
    car4_engine = cars[3].engine
    car5_engine = cars[4].engine
    print(f'Car 1 and Car 5 share engine: {car1_engine is car5_engine} (expected: True)')
    print(f'Car 1 and Car 4 share engine: {car1_engine is car4_engine} (expected: False)')

    car1_wheels = cars[0].wheels
    car2_wheels = cars[1].wheels
    car3_wheels = cars[2].wheels
    print(f'Car 1 and Car 2 share wheels: {car1_wheels is car2_wheels} (expected: True)')
    print(f'Car 1 and Car 3 share wheels: {car1_wheels is car3_wheels} (expected: False)')

    car1_paint = cars[0].paint
    car4_paint = cars[3].paint
    car5_paint = cars[4].paint
    print(f'Car 1 and Car 5 share paint: {car1_paint is car5_paint} (expected: True)')
    print(f'Car 1 and Car 4 share paint: {car1_paint is car4_paint} (expected: False)')

    print()
    print('=== Factory Cache Statistics ===')
    stats = factory.get_stats()
    print(f'  Engines: {stats["engines"]} unique')
    print(f'  Wheels:  {stats["wheels"]} unique')
    print(f'  Paints:  {stats["paints"]} unique')
    print(f'  Total:   {stats["total"]} unique flyweight objects')
    print(f'  Shared across: {len(cars)} car configurations')


if __name__ == '__main__':
    main()
```

### Language-Specific Notes

- **ABC for flyweight interface** — Python's `abc.ABC` and `@abstractmethod` decorator enforce that all concrete flyweight classes implement the `display()` method. A subclass that does not implement `display()` raises a `TypeError` at instantiation time. This provides compile-time-like safety in a dynamically-typed language.

- **`__slots__` for memory efficiency** — The flyweight classes use `__slots__` to reduce memory overhead. Without `__slots__`, each flyweight instance has a `__dict__` attribute that consumes additional memory. Since flyweights are meant to be shared across many clients, the memory savings from `__slots__` compound significantly when the factory holds many unique flyweights.

- **`@property` for read-only access** — The flyweight classes expose their intrinsic state through `@property` decorators without setter methods. This provides read-only access to the state while maintaining encapsulation. Any attempt to set a property without a setter raises an `AttributeError`. This enforces the flyweight immutability invariant at runtime.

- **`is` operator for sharing verification** — Python's `is` operator checks object identity (reference equality). The driver uses `is` to verify that two `CarConfiguration` objects reference the same flyweight instance. This is only possible because the factory returns the same object reference for the same key. In a non-flyweight implementation, `is` would return `False` for structurally identical objects.

- **`Dict` type hints for the factory cache** — The factory uses Python's `Dict[str, EngineSpec]` type hints to document the cache structure. While Python does not enforce types at runtime, the type hints improve code readability and enable static type checking with tools like `mypy`.

- **`main()` guard** — The driver code is wrapped in a `main()` function and guarded by `if __name__ == '__main__':` to allow the module to be imported without executing the driver. This is a Python best practice that enables the flyweight classes to be imported and used in other modules.
