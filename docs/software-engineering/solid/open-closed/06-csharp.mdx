C#'s interface system, dependency injection, and record types provide first-class support for the Open/Closed Principle. The language's strong typing and modern features (records, primary constructors, pattern matching) make strategy implementations concise and safe.

## The Domain: Log Output Formatter System

A distributed application produces logs in multiple formats for different consumers: developers prefer plain text, monitoring systems consume JSON, compliance requires XML, and dashboards use Markdown. New output formats are requested as integration requirements grow.

### The Violation

The following example shows a `LogFormatter` class that uses a switch statement on a format type enum. Adding a new format requires editing the switch.

```csharp
// VIOLATION: FormatLog uses a switch on LogFormat enum.
// Every new format requires editing this method, risking regressions
// across all existing log formatters.

using System;
using System.Text.Json;

public enum LogFormat
{
    Json,
    Xml,
    PlainText,
    Markdown
}

public record LogEntry(
    DateTime Timestamp,
    string Level,
    string Source,
    string Message,
    Exception? Exception = null
);

// VIOLATION: Monolithic formatter. Adding "csv" or "yaml" means editing
// this switch and retesting every existing format.
public class LogFormatter
{
    // VIOLATION: switch on enum -- each new format adds a case. A missing
    // break or incorrect logic in one case can break all formats.
    public string FormatLog(LogEntry entry, LogFormat format)
    {
        switch (format)
        {
            case LogFormat.Json:
                // JSON serialization
                var jsonObj = new
                {
                    timestamp = entry.Timestamp.ToString("O"),
                    level = entry.Level,
                    source = entry.Source,
                    message = entry.Message,
                    exception = entry.Exception?.ToString()
                };
                return JsonSerializer.Serialize(jsonObj, new JsonSerializerOptions
                {
                    WriteIndented = true
                });

            case LogFormat.Xml:
                // XML formatting
                var xml = $"<log>\n";
                xml += $"  <timestamp>{entry.Timestamp:O}</timestamp>\n";
                xml += $"  <level>{EscapeXml(entry.Level)}</level>\n";
                xml += $"  <source>{EscapeXml(entry.Source)}</source>\n";
                xml += $"  <message>{EscapeXml(entry.Message)}</message>\n";
                if (entry.Exception != null)
                    xml += $"  <exception>{EscapeXml(entry.Exception.ToString())}</exception>\n";
                xml += $"</log>";
                return xml;

            case LogFormat.PlainText:
                // Simple text format
                var text = $"[{entry.Timestamp:yyyy-MM-dd HH:mm:ss}] ";
                text += $"[{entry.Level}] [{entry.Source}] ";
                text += entry.Message;
                if (entry.Exception != null)
                    text += $"\n  Exception: {entry.Exception.Message}";
                return text;

            case LogFormat.Markdown:
                // Markdown table row
                var md = $"| {entry.Timestamp:yyyy-MM-dd HH:mm:ss} ";
                md += $"| {entry.Level} ";
                md += $"| {entry.Source} ";
                md += $"| {entry.Message} |";
                if (entry.Exception != null)
                    md += $"\n> **Exception:** {EscapeMarkdown(entry.Exception.Message)}";
                return md;

            default:
                throw new ArgumentOutOfRangeException(nameof(format), format, null);
        }
    }

    private static string EscapeXml(string s) =>
        s.Replace("&", "&amp;").Replace("<", "&lt;").Replace(">", "&gt;");

    private static string EscapeMarkdown(string s) =>
        s.Replace("|", "\\|").Replace("*", "\\*");
}

public class Program
{
    public static void Main()
    {
        var formatter = new LogFormatter();
        var entry = new LogEntry(
            Timestamp: DateTime.UtcNow,
            Level: "ERROR",
            Source: "PaymentService",
            Message: "Payment declined: insufficient funds",
            Exception: new InvalidOperationException("Insufficient funds")
        );

        Console.WriteLine("=== JSON ===");
        Console.WriteLine(formatter.FormatLog(entry, LogFormat.Json));

        Console.WriteLine("\n=== PlainText ===");
        Console.WriteLine(formatter.FormatLog(entry, LogFormat.PlainText));
    }
}
```

Expected output:

```bash
=== JSON ===
{
  "timestamp": "2026-07-05T00:00:00.0000000Z",
  "level": "ERROR",
  "source": "PaymentService",
  "message": "Payment declined: insufficient funds",
  "exception": "System.InvalidOperationException: Insufficient funds"
}

=== PlainText ===
[2026-07-05 00:00:00] [ERROR] [PaymentService] Payment declined: insufficient funds
  Exception: Insufficient funds
```

### The Correct Implementation: Interface with Strategy

The solution defines an `ILogFormatter` interface. Each format implements the interface. A factory method selects the formatter. Adding a new format means creating a new class.

```csharp
// CORRECT: ILogFormatter interface defines the contract. Each format
// is a separate implementation. The logging pipeline is closed.

using System;
using System.Collections.Generic;
using System.Text.Json;

public record LogEntry(
    DateTime Timestamp,
    string Level,
    string Source,
    string Message,
    Exception? Exception = null
);

// --- Interface contract ---
public interface ILogFormatter
{
    /// <summary>Unique format identifier.</summary>
    string FormatName { get; }

    /// <summary>MIME content type for this format.</summary>
    string ContentType { get; }

    /// <summary>Format a single log entry as a string.</summary>
    string Format(LogEntry entry);
}

// --- Concrete implementations ---

public class JsonFormatter : ILogFormatter
{
    public string FormatName => "json";
    public string ContentType => "application/json";

    public string Format(LogEntry entry)
    {
        var jsonObj = new
        {
            timestamp = entry.Timestamp.ToString("O"),
            level = entry.Level,
            source = entry.Source,
            message = entry.Message,
            exception = entry.Exception?.ToString()
        };
        return JsonSerializer.Serialize(jsonObj, new JsonSerializerOptions
        {
            WriteIndented = true
        });
    }
}

public class XmlFormatter : ILogFormatter
{
    public string FormatName => "xml";
    public string ContentType => "application/xml";

    public string Format(LogEntry entry)
    {
        string Escape(string s) =>
            s.Replace("&", "&amp;").Replace("<", "&lt;").Replace(">", "&gt;");

        var xml = $"<log>\n";
        xml += $"  <timestamp>{entry.Timestamp:O}</timestamp>\n";
        xml += $"  <level>{Escape(entry.Level)}</level>\n";
        xml += $"  <source>{Escape(entry.Source)}</source>\n";
        xml += $"  <message>{Escape(entry.Message)}</message>\n";
        if (entry.Exception != null)
            xml += $"  <exception>{Escape(entry.Exception.ToString())}</exception>\n";
        xml += $"</log>";
        return xml;
    }
}

public class PlainTextFormatter : ILogFormatter
{
    public string FormatName => "plaintext";
    public string ContentType => "text/plain";

    public string Format(LogEntry entry)
    {
        var text = $"[{entry.Timestamp:yyyy-MM-dd HH:mm:ss}] ";
        text += $"[{entry.Level}] [{entry.Source}] ";
        text += entry.Message;
        if (entry.Exception != null)
            text += $"\n  Exception: {entry.Exception.Message}";
        return text;
    }
}

public class MarkdownFormatter : ILogFormatter
{
    public string FormatName => "markdown";
    public string ContentType => "text/markdown";

    public string Format(LogEntry entry)
    {
        string Escape(string s) =>
            s.Replace("|", "\\|").Replace("*", "\\*");

        var md = $"| {entry.Timestamp:yyyy-MM-dd HH:mm:ss} ";
        md += $"| {entry.Level} ";
        md += $"| {entry.Source} ";
        md += $"| {Escape(entry.Message)} |";
        if (entry.Exception != null)
            md += $"\n> **Exception:** {Escape(entry.Exception.Message)}";
        return md;
    }
}

// --- Formatter registry ---
public class FormatterRegistry
{
    private readonly Dictionary<string, ILogFormatter> _formatters = new();

    public void Register(ILogFormatter formatter)
    {
        _formatters[formatter.FormatName] = formatter;
    }

    public ILogFormatter Get(string formatName)
    {
        if (_formatters.TryGetValue(formatName, out var formatter))
            return formatter;
        throw new KeyNotFoundException($"No formatter registered for: {formatName}");
    }

    public IEnumerable<string> AvailableFormats => _formatters.Keys;
}

// --- Core pipeline -- closed for modification ---
public class LogPipeline
{
    private readonly FormatterRegistry _registry;

    public LogPipeline(FormatterRegistry registry)
    {
        _registry = registry;
    }

    public string Format(LogEntry entry, string formatName)
    {
        var formatter = _registry.Get(formatName);
        return formatter.Format(entry);
    }

    public Dictionary<string, string> FormatAll(LogEntry entry)
    {
        var results = new Dictionary<string, string>();
        foreach (var name in _registry.AvailableFormats)
        {
            results[name] = Format(entry, name);
        }
        return results;
    }
}

// --- Main ---
public class Program
{
    public static void Main()
    {
        // Setup registry (simulating dependency injection registration)
        var registry = new FormatterRegistry();
        registry.Register(new JsonFormatter());
        registry.Register(new XmlFormatter());
        registry.Register(new PlainTextFormatter());
        registry.Register(new MarkdownFormatter());

        var pipeline = new LogPipeline(registry);

        var entry = new LogEntry(
            Timestamp: DateTime.UtcNow,
            Level: "ERROR",
            Source: "PaymentService",
            Message: "Payment declined: insufficient funds",
            Exception: new InvalidOperationException("Insufficient funds")
        );

        // Format in each format
        Console.WriteLine("=== JSON ===");
        Console.WriteLine(pipeline.Format(entry, "json"));

        Console.WriteLine("\n=== PlainText ===");
        Console.WriteLine(pipeline.Format(entry, "plaintext"));

        Console.WriteLine("\n=== All formats ===");
        var all = pipeline.FormatAll(entry);
        foreach (var kvp in all)
        {
            Console.WriteLine($"\n--- {kvp.Key} ({kvp.Value.Length} chars) ---");
            var preview = kvp.Value.Length > 100
                ? kvp.Value[..100] + "..."
                : kvp.Value;
            Console.WriteLine(preview);
        }

        // --- Adding a new format without modifying existing code ---
        //
        // New requirement: CSV format for data pipeline ingestion.
        // We create a new class and register it. No existing files change.

        Console.WriteLine("\n=== CSV (new, no edits to existing code) ===");
        registry.Register(new CsvFormatter());
        Console.WriteLine(pipeline.Format(entry, "csv"));
    }
}

// New format -- separate file, no modifications to existing code
public class CsvFormatter : ILogFormatter
{
    public string FormatName => "csv";
    public string ContentType => "text/csv";

    public string Format(LogEntry entry)
    {
        string EscapeCsv(string s) =>
            s.Contains(',') || s.Contains('"') || s.Contains('\n')
                ? $"\"{s.Replace("\"", "\"\"")}\""
                : s;

        // CSV header + data row
        var csv = "timestamp,level,source,message,exception\n";
        csv += $"{EscapeCsv(entry.Timestamp:O)},";
        csv += $"{EscapeCsv(entry.Level)},";
        csv += $"{EscapeCsv(entry.Source)},";
        csv += $"{EscapeCsv(entry.Message)},";
        csv += $"{EscapeCsv(entry.Exception?.ToString() ?? "")}";
        return csv;
    }
}
```

## Key Takeaways

- **Interface contracts** with `ILogFormatter` define the shape every formatter must satisfy. The `FormatName` and `ContentType` properties provide metadata alongside the core `Format` method.
- **Records** (`LogEntry` as a positional record) provide immutable data transfer objects with built-in equality and `ToString()`.
- **`Dictionary<string, ILogFormatter>`** serves as the registry. In production, this would be registered as a service collection in dependency injection.
- **Dependency injection** in .NET makes OCP natural: register `ILogFormatter` implementations in the DI container, and the `LogPipeline` receives them as `IEnumerable<ILogFormatter>`.
- **Local functions** like `Escape()` inside methods keep escaping logic close to where it is used without polluting the class surface.
- **Adding a formatter** is purely additive in C#. Create a new class implementing `ILogFormatter`, register it in the DI container or the registry. No existing code changes.
- **`FormatAll`** iterates over registered formatters, demonstrating that operations composed from the closed core also remain closed.
