Testing AI systems is fundamentally different from testing traditional software. LLM outputs are non-deterministic — the same input can produce different responses. Traditional assertions ("assert output == expected") don't work. A 2023 study by NVIDIA found that 78% of teams deploying LLMs in production had no systematic evaluation pipeline. This leads to silent regressions, safety failures, and unpredictable cost spikes.

## What You'll Build

You'll build **TestAI**, an AI testing framework with:

- **Test runner** — execute test cases against any LLM provider
- **Semantic assertions** — `assert_similar`, `assert_safe`, `assert_contains`, `assert_factual` using LLM-as-judge
- **Regression testing** — compare outputs before/after prompt or model changes
- **Benchmark suites** — pre-built suites for accuracy, safety, latency, and cost
- **CI integration** — run test suites in GitHub Actions with pass/fail reporting
- **Dashboard** — visualize test results over time, track regressions

## Technology Decisions

| Component | Choice | Why this, not X |
|---|---|---|
| Assertion Engine | LLM-as-judge (Claude Haiku) | Covers all assertion types with one model. Dedicated NLP models need training data |
| Test Format | YAML + Python | YAML for config, Python for custom assertions. JSON-only is too limiting |
| CI Format | JUnit XML | Compatible with every CI platform. Custom format requires CI plugin |
| Results DB | SQLite | Zero-config, queryable. JSON files don't support cross-run queries |

## Data Model

```python
from pydantic import BaseModel
from typing import Optional, Literal
from datetime import datetime

class TestCase(BaseModel):
    name: str
    input: str
    expected: Optional[str] = None
    assertions: list[str]  # "similarity>0.8", "safe=true", "contains=X"
    model: str = "claude-haiku-3-5"

class TestRun(BaseModel):
    id: str
    suite_name: str
    total: int
    passed: int
    failed: int
    duration_ms: int
    total_cost: float
    timestamp: datetime

class AssertionResult(BaseModel):
    test_name: str
    assertion: str
    passed: bool
    score: float
    detail: Optional[str] = None
```
