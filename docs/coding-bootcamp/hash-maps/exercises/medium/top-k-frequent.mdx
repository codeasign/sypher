# Top K Frequent Elements

Given an integer array `nums` and an integer `k`, return the `k` most frequent elements. You may return the answer in **any order**.

## Examples

**Example 1:**
```
Input: nums = [1,1,1,2,2,3], k = 2
Output: [1,2]
```

**Example 2:**
```
Input: nums = [1], k = 1
Output: [1]
```

## Constraints

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
- `k` is in the range `[1, the number of unique elements in the array]`.
- It is **guaranteed** that the answer is **unique**.

## Hints

<details>
<summary>Hint 1</summary>

First, count the frequency of each element using a hash map. This gives you a mapping from value to its count.
</details>

<details>
<summary>Hint 2</summary>

To extract the top k, you can use a min-heap of size k. Iterate over the frequency map entries and keep the k largest.
</details>

<details>
<summary>Hint 3</summary>

Or use bucket sort: since the maximum frequency is at most n, create an array of buckets where the index is the frequency, and collect the top k from the highest buckets.
</details>
