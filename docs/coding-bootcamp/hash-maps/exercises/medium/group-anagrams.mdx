# Group Anagrams

Given an array of strings `strs`, group the **anagrams** together. You can return the answer in **any order**.

An **anagram** is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

## Examples

**Example 1:**
```
Input: strs = ["eat","tea","tan","ate","nat","bat"]
Output: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

**Example 2:**
```
Input: strs = [""]
Output: [[""]]
```

**Example 3:**
```
Input: strs = ["a"]
Output: [["a"]]
```

## Constraints

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` consists of lowercase English letters.

## Hints

<details>
<summary>Hint 1</summary>

Two strings are anagrams if they have the same sorted character sequence. Try sorting each string and using the sorted version as a hash map key.
</details>

<details>
<summary>Hint 2</summary>

Alternatively, use a character frequency count (26-element array for lowercase letters) as the key. This avoids the O(k log k) sorting cost.
</details>

<details>
<summary>Hint 3</summary>

Iterate through the input array once. For each string, compute its key, and append the original string to the corresponding group in the hash map.
</details>
