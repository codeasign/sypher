# Ransom Note

**Difficulty:** Easy

Given two strings `ransomNote` and `magazine`, return `true` if `ransomNote` can be constructed by using the letters from `magazine`, and `false` otherwise.

Each letter in `magazine` can only be used **once** in `ransomNote`.

## Example 1

**Input:** `ransomNote = "a"`, `magazine = "b"`  
**Output:** `false`  
**Explanation:** The letter `'a'` does not appear in `magazine`.

## Example 2

**Input:** `ransomNote = "aa"`, `magazine = "ab"`  
**Output:** `false`  
**Explanation:** `magazine` has only one `'a'`, but `ransomNote` needs two.

## Example 3

**Input:** `ransomNote = "aa"`, `magazine = "aab"`  
**Output:** `true`  
**Explanation:** `magazine` has two `'a'`s, so `ransomNote` can be constructed.

## Input Specification

The first line contains the string `ransomNote`.  
The second line contains the string `magazine`.

## Output Specification

Print `true` if `ransomNote` can be constructed from `magazine`'s letters, otherwise print `false`.

## Constraints

- `1 <= ransomNote.length, magazine.length <= 10^5`
- `ransomNote` and `magazine` consist of lowercase English letters only.

## Hints

<details>
<summary>Hint 1</summary>

Count the frequency of each character in the magazine first. A hash map (or an array of size 26) is a natural fit for character frequency counting.

</details>

<details>
<summary>Hint 2</summary>

Once you have the magazine's character counts, iterate through each character in the ransom note and decrement the corresponding count. If any count goes below zero, you cannot construct the ransom note.

</details>

<details>
<summary>Hint 3</summary>

Because the problem is restricted to lowercase English letters, you can use a fixed-size array of length 26 instead of a hash map — index by `char - 'a'`. This gives O(1) space and O(n + m) time where n and m are the lengths of the two strings.

</details>

</ProblemEditor>
