# Number of Provinces

**Difficulty:** Easy

There are `n` cities. Some of them are connected, while some are not. If city
`a` is connected directly to city `b`, and city `b` is connected directly to
city `c`, then city `a` is connected indirectly to city `c`.

A **province** is a group of directly or indirectly connected cities with no
other cities outside the group.

You are given an `n x n` matrix `isConnected` where `isConnected[i][j] = 1`
if the i-th city and the j-th city are directly connected, and
`isConnected[i][j] = 0` otherwise.

Return the total number of provinces.

## Example 1

**Input:**
```
isConnected = [[1,1,0],
               [1,1,0],
               [0,0,1]]
```
**Output:** `2`

**Explanation:**
Cities 0 and 1 are directly connected, forming one province. City 2 is
isolated, forming its own province.

## Example 2

**Input:**
```
isConnected = [[1,0,0],
               [0,1,0],
               [0,0,1]]
```
**Output:** `3`

**Explanation:**
No cities are connected to each other, so each city is its own province.

## Input Specification

The first line contains an integer `n` (the number of cities).
The next `n` lines each contain `n` space-separated integers (0 or 1)
representing the matrix `isConnected`.

## Output Specification

Print a single integer: the number of provinces.

## Constraints

- `1 <= n <= 200`
- `isConnected[i][j]` is 0 or 1
- `isConnected[i][i] == 1` (self-connection)
- `isConnected[i][j] == isConnected[j][i]` (symmetric matrix)

## Hints

<details>
<summary>Hint 1</summary>

Each city is a node in a graph. The matrix is the adjacency matrix. Count
connected components using Union-Find.
</details>

<details>
<summary>Hint 2</summary>

Initialize the DSU with `n` elements. For every pair `(i, j)` where
`isConnected[i][j] == 1`, call `union(i, j)`.
</details>

<details>
<summary>Hint 3</summary>

After processing all connections, count the number of distinct roots. Each
root represents one province.
</details>

</ProblemEditor>
