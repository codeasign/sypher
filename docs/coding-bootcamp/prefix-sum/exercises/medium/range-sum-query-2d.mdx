# Range Sum Query 2D

**Difficulty:** Medium

Given a 2D matrix of integers, handle multiple queries that each ask for the **sum of all elements** inside a rectangle defined by its upper-left corner `(r1, c1)` and lower-right corner `(r2, c2)` (inclusive).

You should precompute a 2D prefix sum table so that each query is answered in **O(1)** after **O(m * n)** preprocessing.

## Example 1

**Input:**
```
matrix = [
  [3, 0, 1, 4, 2],
  [5, 6, 3, 2, 1],
  [1, 2, 0, 1, 5],
  [4, 1, 0, 1, 7],
  [1, 0, 3, 0, 5]
]
```

- `sumRegion(2, 1, 4, 3)` → **8**
- `sumRegion(1, 1, 2, 2)` → **11**
- `sumRegion(1, 2, 2, 4)` → **12**

## Example 2

**Input:**
```
matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]
```

- `sumRegion(0, 0, 2, 2)` → **45** (entire matrix)
- `sumRegion(0, 0, 1, 1)` → **12** (top-left 2×2 block: 1+2+4+5)
- `sumRegion(1, 1, 2, 2)` → **28** (bottom-right 2×2 block: 5+6+8+9)

## Input Specification

- Line 1: two integers `rows cols` — dimensions of the matrix.
- Next `rows` lines: `cols` space-separated integers representing each row.
- Next line: integer `q` — number of queries.
- Next `q` lines: four integers `r1 c1 r2 c2` (0-indexed, inclusive bounds).

## Output Specification

Print `q` lines. Each line contains one integer: the rectangle sum for that query.

## Constraints

- `1 <= rows, cols <= 200`
- `-10^4 <= matrix[i][j] <= 10^4`
- `1 <= q <= 10^4`
- `0 <= r1 <= r2 < rows`
- `0 <= c1 <= c2 < cols`

## Hints

<details>
<summary>Hint 1</summary>

Build a 2D prefix sum table `prefix` of size `(rows+1) x (cols+1)` where `prefix[i][j]` stores the sum of all elements in the rectangle from `(0,0)` to `(i-1, j-1)`. The extra row and column of zeros simplifies boundary handling.

</details>

<details>
<summary>Hint 2</summary>

Fill the prefix table using:
`prefix[i][j] = matrix[i-1][j-1] + prefix[i-1][j] + prefix[i][j-1] - prefix[i-1][j-1]`

This is the 2D inclusion-exclusion recurrence — you add the cell value, the prefix above, and the prefix to the left, then subtract the overlap that was counted twice.

</details>

<details>
<summary>Hint 3</summary>

Answer each query `(r1, c1, r2, c2)` in O(1) using:
`prefix[r2+1][c2+1] - prefix[r1][c2+1] - prefix[r2+1][c1] + prefix[r1][c1]`

Draw a Venn diagram with four rectangles rooted at the origin to convince yourself of the inclusion-exclusion logic.

</details>
