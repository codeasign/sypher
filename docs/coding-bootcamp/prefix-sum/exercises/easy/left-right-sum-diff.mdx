# Left and Right Sum Differences

**Difficulty:** Easy

Given a 0-indexed integer array `nums`, find a 0-indexed integer array `answer` where `answer[i]` is the **absolute difference** between the sum of all elements to the **left** of index `i` and the sum of all elements to the **right** of index `i`.

More formally:
- `answer[i] = |leftSum[i] - rightSum[i]|`
- `leftSum[i]` = sum of `nums[0..i-1]` (empty sum = 0)
- `rightSum[i]` = sum of `nums[i+1..n-1]` (empty sum = 0)

## Example 1

**Input:** `nums = [10, 4, 8, 3]`  
**Output:** `[15, 1, 11, 22]`  
**Explanation:**
- index 0: leftSum = 0, rightSum = 4+8+3 = 15, |0-15| = 15
- index 1: leftSum = 10, rightSum = 8+3 = 11, |10-11| = 1
- index 2: leftSum = 10+4 = 14, rightSum = 3, |14-3| = 11
- index 3: leftSum = 10+4+8 = 22, rightSum = 0, |22-0| = 22

## Example 2

**Input:** `nums = [1, 2, 3, 4, 5]`  
**Output:** `[14, 11, 6, 1, 10]`  
**Explanation:**
- index 0: leftSum=0, rightSum=2+3+4+5=14, |0-14| = 14
- index 1: leftSum=1, rightSum=3+4+5=12, |1-12| = 11
- index 2: leftSum=1+2=3, rightSum=4+5=9, |3-9| = 6
- index 3: leftSum=1+2+3=6, rightSum=5, |6-5| = 1
- index 4: leftSum=1+2+3+4=10, rightSum=0, |10-0| = 10

## Example 3

**Input:** `nums = [5]`  
**Output:** `[0]`  
**Explanation:** A single element has no left or right elements, so both sums are 0 and the difference is 0.

## Input Specification

A single line containing space-separated integers representing the array `nums`.

## Output Specification

Print the answer array as space-separated integers on a single line.

## Constraints

- `1 <= nums.length <= 10^5`
- `1 <= nums[i] <= 10^4` (values may also be 0 or negative in some test cases)

## Hints

<details>
<summary>Hint 1</summary>

Compute the total sum of the array first. Then iterate once from left to right, tracking how much is on the left. The right sum at each index can be derived from total sum minus left sum minus current element.

</details>

<details>
<summary>Hint 2</summary>

As you move from index `i` to `i+1`, `leftSum` grows by `nums[i]`. You do not need a separate pass or a prefix sum array — just maintain a running `leftSum` variable.

</details>

<details>
<summary>Hint 3</summary>

The formula for each index is: `answer[i] = abs(leftSum - (totalSum - leftSum - nums[i]))`. After computing `answer[i]`, update `leftSum += nums[i]` before moving to the next index.

</details>

</ProblemEditor>
