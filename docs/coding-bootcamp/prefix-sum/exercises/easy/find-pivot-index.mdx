# Find Pivot Index

**Difficulty:** Easy

Given an array of integers `nums`, find the **pivot index** where the sum of all elements to the left of the index equals the sum of all elements to the right of the index.

If no such index exists, return `-1`. If there are multiple pivot indices, return the **left-most** one.

## Example 1

**Input:** `nums = [1, 7, 3, 6, 5, 6]`

**Output:** `3`

**Explanation:**
- Left sum = `nums[0] + nums[1] + nums[2]` = 1 + 7 + 3 = 11
- Right sum = `nums[4] + nums[5]` = 5 + 6 = 11
- Both are equal at index 3.

## Example 2

**Input:** `nums = [1, 2, 3]`

**Output:** `-1`

**Explanation:** No index satisfies the condition.

## Input Specification

A single line containing space-separated integers representing the array `nums`.

## Output Specification

Print a single integer — the pivot index (0-based), or `-1` if no pivot exists.

## Constraints

- `1 <= nums.length <= 10^4`
- `-1000 <= nums[i] <= 1000`

## Hints

<details>
<summary>Hint 1</summary>

Compute the total sum of the array first. As you iterate, maintain a running left sum.

</details>

<details>
<summary>Hint 2</summary>

At index `i`, the right sum equals `total_sum - left_sum - nums[i]`. Check if `left_sum == right_sum`.

</details>

<details>
<summary>Hint 3</summary>

Return the first index where the condition is satisfied — that guarantees the left-most pivot.

</details>

</ProblemEditor>
