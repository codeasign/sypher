# Range Sum Query 2D — Solution

## Problem Overview

Given a 2D matrix of integers and a list of queries (each query is a rectangle defined by `(r1, c1, r2, c2)`), return the sum of elements within each rectangle.

## Approach: 2D Prefix Sum

**Idea:** Build a 2D prefix sum array where `prefix[i+1][j+1]` stores the sum of all elements in the rectangle from `(0,0)` to `(i,j)`. Then each query can be answered in O(1) time using the inclusion-exclusion principle.

**Algorithm:**
1. Let `rows = len(matrix)`, `cols = len(matrix[0])`
2. Build a 2D prefix array of size `(rows+1) x (cols+1)` initialized to 0
3. For each cell `(i, j)`:
   - `prefix[i+1][j+1] = matrix[i][j] + prefix[i][j+1] + prefix[i+1][j] - prefix[i][j]`
4. For each query `(r1, c1, r2, c2)`:
   - Sum = `prefix[r2+1][c2+1] - prefix[r1][c2+1] - prefix[r2+1][c1] + prefix[r1][c1]`

**Why This Works:** The 2D prefix sum encodes the sum of the submatrix from the origin to each cell. Using inclusion-exclusion, we can extract any rectangular region in O(1).

**Complexity Analysis:**
- **Time:** O(rows × cols + q) — building the prefix matrix O(rows × cols) and answering q queries in O(1) each
- **Space:** O(rows × cols) — the prefix sum matrix

def range_sum_query_2d(matrix: list, queries: list) -> list:
    if not matrix or not matrix[0]:
        return []

    rows, cols = len(matrix), len(matrix[0])
    # Build 2D prefix sum
    prefix = [[0] * (cols + 1) for _ in range(rows + 1)]

    for i in range(rows):
        row_sum = 0
        for j in range(cols):
            row_sum += matrix[i][j]
            prefix[i + 1][j + 1] = prefix[i][j + 1] + row_sum

    # Answer each query
    results = []
    for r1, c1, r2, c2 in queries:
        total = (prefix[r2 + 1][c2 + 1]
                 - prefix[r1][c2 + 1]
                 - prefix[r2 + 1][c1]
                 + prefix[r1][c1])
        results.append(total)

    return results

        let r1 = q[0] as usize;
        let c1 = q[1] as usize;
        let r2 = q[2] as usize;
        let c2 = q[3] as usize;

      - prefix[r2 + 1][c1]
      + prefix[r1][c1]

func rangeSumQuery2D(matrix [][]int, queries [][]int) []int {
    if len(matrix) == 0 || len(matrix[0]) == 0 {
        return []int{}
    }

    rows, cols := len(matrix), len(matrix[0])
    prefix := make([][]int, rows+1)
    for i := range prefix {
        prefix[i] = make([]int, cols+1)
    }

    for i := 0; i < rows; i++ {
        rowSum := 0
        for j := 0; j < cols; j++ {
            rowSum += matrix[i][j]
            prefix[i+1][j+1] = prefix[i][j+1] + rowSum
        }
    }

    results := make([]int, len(queries))
    for qi, q := range queries {
        r1, c1, r2, c2 := q[0], q[1], q[2], q[3]
        results[qi] = prefix[r2+1][c2+1]
                    - prefix[r1][c2+1]
                    - prefix[r2+1][c1]
                    + prefix[r1][c1]
    }
    return results
}

## Key Takeaways

- **2D prefix sums** reduce rectangle sum queries from O(rows × cols) to O(1) each
- The **inclusion-exclusion formula** `sum = prefix[r2+1][c2+1] - prefix[r1][c2+1] - prefix[r2+1][c1] + prefix[r1][c1]` is the key insight
- Building the prefix array can be done in-place or with extra memory depending on constraints
