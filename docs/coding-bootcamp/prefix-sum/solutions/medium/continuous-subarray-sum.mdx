# Continuous Subarray Sum — Solution

## Problem Overview

Given an integer array `nums` and an integer `k`, return `true` if there exists a subarray of length at least two whose sum is a multiple of `k`.

## Brute Force Approach

**Idea:** Check every subarray of length at least 2, compute its sum, and test divisibility.

**Algorithm:**
1. For each start index `i` from `0` to `n-2`:
   - Initialize `sum = 0`
   - For each end index `j` from `i` to `n-1`:
     - Add `nums[j]` to `sum`
     - If `j - i >= 1` and (k == 0 ? sum == 0 : sum % k == 0), return `true`
2. Return `false`

**Complexity Analysis:**
- **Time:** O(n^2) — two nested loops
- **Space:** O(1) — no extra space

**Why this could be better:** O(n^2) is too slow for n up to 10^5. We can use the modulo property of prefix sums.

## Optimal Approach: Prefix Sum Modulo with Hash Map

**Idea:** If `prefix[j] % k == prefix[i] % k` (for `j - i >= 2`), then the subarray `(i+1..j)` has a sum divisible by `k`. We use a hash map to store the earliest index where each remainder appears.

**Algorithm:**
1. Initialize a hash map `remainderIndex = {0: -1}` (remainder 0 at index -1)
2. Initialize `currentSum = 0`
3. For each index `i` from `0` to `n-1`:
   - `currentSum += nums[i]`
   - Compute `remainder = currentSum % k` (handle negative and k=0)
   - If `k != 0`:
     - If `remainder` exists in map and `i - remainderIndex[remainder] >= 2`, return `true`
     - Otherwise, store `remainderIndex[remainder] = i` (only if not already stored)
   - Else (k == 0): If two consecutive zeros exist, return `true`
4. Return `false`

**Why This Works:** The key insight is that when two prefix sums have the same remainder modulo `k`, the difference between them (the subarray sum) is divisible by `k`. We store only the first occurrence of each remainder to maximize the subarray length.

**Complexity Analysis:**
- **Time:** O(n) — single pass
- **Space:** O(min(n, k)) — hash map size bounded by distinct remainders

import java.util.HashMap;
import java.util.Map;

class Solution {
    public boolean checkSubarraySum(int[] nums, int k) {
        Map<Integer, Integer> remainderIndex = new HashMap<>();
        remainderIndex.put(0, -1);  // remainder 0 before any element

        int currentSum = 0;
        for (int i = 0; i < nums.length; i++) {
            currentSum += nums[i];

            if (k != 0) {
                int remainder = currentSum % k;
                // Handle negative remainders
                if (remainder < 0) remainder += k;

                if (remainderIndex.containsKey(remainder)) {
                    if (i - remainderIndex.get(remainder) >= 2) return true;
                } else {
                    remainderIndex.put(remainder, i);
                }
            } else {
                // k == 0: look for two consecutive zeros
                if (i > 0 && nums[i] == 0 && nums[i - 1] == 0) return true;
            }
        }
        return false;
    }
}

use std::collections::HashMap;

fn check_subarray_sum(nums: &Vec<i32>, k: i32) -> bool {
    let mut remainder_index = HashMap::new();
    remainder_index.insert(0, -1);  // remainder 0 before any element

    let mut current_sum = 0;
    for (i, &num) in nums.iter().enumerate() {
        current_sum += num;

        if k != 0 {
            let mut remainder = current_sum % k;
            if remainder < 0 { remainder += k; }

            match remainder_index.get(&remainder) {
                Some(&prev_idx) => {
                    if (i as i32) - prev_idx >= 2 { return true; }
                }
                None => {
                    remainder_index.insert(remainder, i as i32);
                }
            }
        } else {
            // k == 0: look for two consecutive zeros
            if i > 0 && nums[i] == 0 && nums[i - 1] == 0 { return true; }
        }
    }
    false
}

    for (int i = 0; i < MAX * 2; i++) remainderIndex[i] = -2;  // -2 = unset

## Key Takeaways

- **Remainder equality** is the key insight: same remainder means difference is divisible by k
- Store only the **first occurrence** of each remainder to maximize subarray length
- Handle edge cases: `k == 0`, negative numbers, and the minimum length requirement
