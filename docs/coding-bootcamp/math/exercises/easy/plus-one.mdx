# Plus One

You are given a **large integer** represented as an array of digits, where each `digits[i]` is the i-th digit of the integer. The digits are ordered from most significant to least significant (left-to-right), and no leading zeros are present.

Increment the large integer by one and return the resulting array of digits.

## Examples

**Example 1:**
```
Input: digits = [1, 2, 3]
Output: [1, 2, 4]
Explanation: The array represents the integer 123. Incrementing by one gives 124.
```

**Example 2:**
```
Input: digits = [4, 3, 2, 1]
Output: [4, 3, 2, 2]
Explanation: The array represents the integer 4321. Incrementing by one gives 4322.
```

**Example 3:**
```
Input: digits = [9]
Output: [1, 0]
Explanation: The array represents the integer 9. Incrementing by one gives 10, represented as [1, 0].
```

## Constraints

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` does not contain any leading zeros

## Hints

<details>
<summary>Hint 1</summary>

Start from the least significant digit (rightmost). Add 1, and if the result is 10, set it to 0 and carry the 1 to the next digit.

</details>

<details>
<summary>Hint 2</summary>

If all digits become 0 after processing (e.g., [9, 9, 9] becomes [0, 0, 0]), you need to insert a 1 at the front.

</details>

<details>
<summary>Hint 3</summary>

You can modify the input array in-place to save space, but be careful with the carry propagation.

</details>

</ProblemEditor>
