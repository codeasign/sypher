## GCD and LCM

The **Greatest Common Divisor (GCD)** of two numbers is the largest number that divides both without remainder. The **Least Common Multiple (LCM)** is the smallest number that is divisible by both.

### Euclid's Algorithm

The most efficient way to compute GCD is Euclid's algorithm:

```
GCD(a, b) = GCD(b, a mod b)
```

This relies on the fact that any common divisor of `a` and `b` also divides `a mod b`. The algorithm terminates when `b = 0`, at which point `a` is the GCD.

```python
def gcd(a: int, b: int) -> int:
    while b:
        a, b = b, a % b
    return a

def lcm(a: int, b: int) -> int:
    return a // gcd(a, b) * b  # prevent overflow
```

**Complexity:** O(log min(a, b)) вҖ” the number of iterations is proportional to the number of digits.

## Prime Numbers

A **prime number** is a natural number greater than 1 that is only divisible by 1 and itself.

### Prime Checking

To check if a number `n` is prime, we only need to test divisors up to `вҲҡn`. If `n` has no divisor up to `вҲҡn`, it is prime.

```python
def is_prime(n: int) -> bool:
    if n < 2:
        return False
    if n < 4:
        return True
    if n % 2 == 0 or n % 3 == 0:
        return False
    i = 5
    while i * i <= n:
        if n % i == 0 or n % (i + 2) == 0:
            return False
        i += 6
    return True
```

**Complexity:** O(вҲҡn) вҖ” optimized by checking only numbers of the form `6k Вұ 1`.

### Sieve of Eratosthenes

When finding all primes up to a limit `n`, the Sieve of Eratosthenes is far more efficient than checking each number individually.

```python
def sieve(n: int) -> list[bool]:
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False
    for i in range(2, int(n ** 0.5) + 1):
        if is_prime[i]:
            for j in range(i * i, n + 1, i):
                is_prime[j] = False
    return is_prime
```

**Complexity:** O(n log log n) вҖ” near-linear for practical ranges.

## Modular Arithmetic

Modular arithmetic deals with integers modulo a fixed number `m`. Two numbers are congruent modulo `m` if they have the same remainder when divided by `m`.

### Key Properties

```
(a + b) mod m = ((a mod m) + (b mod m)) mod m
(a * b) mod m = ((a mod m) * (b mod m)) mod m
(a - b) mod m = ((a mod m) - (b mod m) + m) mod m
```

### Modular Exponentiation

Computing `x^n mod m` efficiently uses exponentiation by squaring, reducing O(n) to O(log n).

```python
def mod_pow(x: int, n: int, m: int) -> int:
    result = 1
    x = x % m
    while n > 0:
        if n & 1:
            result = (result * x) % m
        x = (x * x) % m
        n >>= 1
    return result
```

## Common Applications

### Finding All Primes Up to N

Use the Sieve of Eratosthenes when you need to answer many prime queries within a fixed range. The precomputation cost of O(n log log n) is amortized over all queries.

### Modular Arithmetic for Large Numbers

When results exceed 32-bit or 64-bit integer limits, modular arithmetic keeps values manageable. This is essential in:
- Combinatorial problems with large factorials
- Cryptography algorithms
- Hash function computations

### Fast Exponentiation

Exponentiation by squaring computes powers in O(log n) time. It is the foundation of:
- Modular exponentiation in RSA
- Matrix exponentiation for linear recurrences
- Computing large Fibonacci numbers

### Number Properties

- **Even/Odd:** `n & 1` checks the least significant bit
- **Divisibility:** `n % d == 0` checks if `d` divides `n`
- **Digit manipulation:** Repeated modulo and division by 10 extracts digits

## Worked Example: Find the GCD of 48 and 18

Using Euclid's algorithm:

```
GCD(48, 18)
Step 1: 48 Г· 18 = 2 remainder 12  вҶ’  GCD(18, 12)
Step 2: 18 Г· 12 = 1 remainder 6   вҶ’  GCD(12, 6)
Step 3: 12 Г· 6  = 2 remainder 0   вҶ’  GCD(6, 0)
Result: 6
```

The algorithm takes only 3 steps for numbers that might require 48 individual checks using a naive approach.
