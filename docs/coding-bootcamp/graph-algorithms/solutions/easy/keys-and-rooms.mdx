# Keys and Rooms — Solution

## Problem Overview

You start at room 0 and each room contains keys to other rooms. Determine if you can visit **every** room. This is a reachability problem on a directed graph.

## Approach: DFS (Recursive)

**Idea:** Model rooms as nodes and keys as directed edges. Run DFS from room 0 and track visited rooms. If the visited set contains all `n` rooms, return `true`.

**Algorithm:**
1. Build a `visited` boolean array of size `n`
2. Define a recursive DFS function: mark the current room as visited, then recursively visit each key (neighbor) if not yet visited
3. Start DFS from room 0
4. Return `true` if all rooms are visited, `false` otherwise

**Complexity Analysis:**
- **Time:** O(V + E) — we visit each room once and process each key once
- **Space:** O(V) — recursion stack and visited array

## Alternative: BFS (Iterative)

**Algorithm:**
1. Use a queue initialized with room 0
2. Mark room 0 as visited
3. While the queue is not empty: pop a room, for each key in that room, if the corresponding room is not visited, mark it visited and enqueue it
4. Return `true` if visited count == n

Both approaches have the same complexity, but BFS avoids recursion depth issues on large graphs.

import java.util.*;

class Solution {
    public boolean canVisitAllRooms(List<List<Integer>> rooms) {
        int n = rooms.size();
        boolean[] visited = new boolean[n];  // Track visited rooms
        dfs(rooms, 0, visited);              // Start DFS from room 0
        for (boolean v : visited) {          // Check all rooms visited
            if (!v) return false;
        }
        return true;
    }

    private void dfs(List<List<Integer>> rooms, int room, boolean[] visited) {
        visited[room] = true;                // Mark current room as visited
        for (int key : rooms.get(room)) {    // Try each key in this room
            if (!visited[key]) {             // If the target room is unvisited
                dfs(rooms, key, visited);    // Recursively explore it
            }
        }
    }
}

bool canVisitAllRooms(int** rooms, int roomsSize, int* roomsColSize) {
    bool* visited = calloc(roomsSize, sizeof(bool));  // Allocate and zero-initialize
    dfs(rooms, roomsColSize, 0, visited);              // Start DFS from room 0
    for (int i = 0; i < roomsSize; i++) {
        if (!visited[i]) { free(visited); return false; }
    }
    free(visited);
    return true;

## Key Takeaways

- Rooms and keys form a **directed graph** — DFS is the natural solution
- A simple boolean `visited` array is all you need for tracking
- BFS works identically well and avoids deep recursion on large graphs
- The problem reduces to: is the graph rooted at node 0 **connected** to all other nodes?
