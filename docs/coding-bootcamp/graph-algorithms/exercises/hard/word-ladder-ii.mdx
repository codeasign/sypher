# Word Ladder II

**Difficulty:** Hard

A **transformation sequence** from word `beginWord` to word `endWord` using a dictionary `wordList` is a sequence of words `beginWord = s0 -> s1 -> s2 -> ... -> sk = endWord` such that:

- Every adjacent pair of words differs by exactly one letter.
- Every word `si` for `1 <= i <= k` is in `wordList`.

Given two words, `beginWord` and `endWord`, and a dictionary `wordList`, return **all** the shortest transformation sequences from `beginWord` to `endWord`. If no such sequence exists, output `"No sequence"`.

## Example 1

**Input:** `beginWord = "hit"`, `endWord = "cog"`, `wordList = ["hot","dot","dog","lot","log","cog"]`  
**Output:**  
```
hit hot dot dog cog
hit hot lot log cog
```
**Explanation:** Both sequences are length 5 and are the shortest possible.

## Example 2

**Input:** `beginWord = "hit"`, `endWord = "cog"`, `wordList = ["hot","dot","dog","lot","log"]`  
**Output:** `No sequence`  
**Explanation:** `"cog"` is not in wordList, so no transformation sequence exists.

## Input Specification

The first line contains `beginWord`.  
The second line contains `endWord`.  
The third line contains a comma-separated list of words in the dictionary.

## Output Specification

Print each shortest transformation sequence on its own line, with words separated by spaces.  
If no sequence exists, print `No sequence`.

## Constraints

- `1 <= beginWord.length <= 10`
- `endWord.length == beginWord.length`
- `1 <= wordList.length <= 100`
- `wordList[i].length == beginWord.length`
- `beginWord != endWord`
- All words consist of lowercase English letters

## Hints

<details>
<summary>Hint 1</summary>

Use BFS from beginWord to endWord to find the shortest distance and build a graph of valid transformations.
</details>

<details>
<summary>Hint 2</summary>

After BFS, use DFS to reconstruct all paths that follow the shortest distance.
</details>

<details>
<summary>Hint 3</summary>

A word can be transformed to another if they differ by exactly one character. Consider using a wildcard pattern or checking each position.
</details>
