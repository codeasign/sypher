# Clone Graph

**Difficulty:** Medium

Given a reference of a node in a connected undirected graph, return a deep copy (clone) of the graph.

Each node in the graph contains a value (`int`) and a list (`List[Node]`) of its neighbors.

For this exercise, the graph is represented as an adjacency list. Your task is to return a deep copy — a new adjacency list where every node object is newly created (not a reference to the original).

## Example 1

**Input:** `adj = [[2,4],[1,3],[2,4],[1,3]]`  
**Output:** `[[2,4],[1,3],[2,4],[1,3]]`  
**Explanation:** A 4-node cycle graph where node 1 is connected to 2 and 4, node 2 to 1 and 3, etc.

## Example 2

**Input:** `adj = [[]]`  
**Output:** `[[]]`  
**Explanation:** A single isolated node with no neighbors.

## Input Specification

The first line contains an integer `n` representing the number of nodes.  
Each of the next `n` lines contains space-separated integers representing the neighbors of each node (1-indexed values). An empty line means no neighbors.

## Output Specification

Print `n` lines, each containing space-separated integers representing the neighbors of each node in the cloned graph (same format as input).

## Constraints

- `1 <= n <= 100`
- `0 <= adj[i].length <= n - 1`
- `1 <= adj[i][j] <= n`
- The graph is undirected and connected

## Hints

<details>
<summary>Hint 1</summary>

Use a hash map to store cloned nodes: the key is the original node index, the value is the cloned node index (or node object).
</details>

<details>
<summary>Hint 2</summary>

Traverse the graph with DFS or BFS. When you encounter a neighbor that hasn't been cloned yet, clone it first, then add the edge.
</details>

</ProblemEditor>
