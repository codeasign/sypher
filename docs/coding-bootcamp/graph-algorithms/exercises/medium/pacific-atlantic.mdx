# Pacific Atlantic Water Flow

**Difficulty:** Medium

There is an `m x n` rectangular island that borders both the **Pacific Ocean** and **Atlantic Ocean**. The Pacific Ocean touches the island's left and top edges, and the Atlantic Ocean touches the island's right and bottom edges.

The island's elevation map is given as an integer matrix `heights` where `heights[r][c]` represents the height above sea level of the cell at `(r, c)`.

Rain water can flow to neighboring cells (up, down, left, right) only if the neighboring cell's height is **less than or equal to** the current cell's height. Water can flow into the ocean from any cell adjacent to that ocean.

Return a list of grid coordinates `[r, c]` representing cells from which water can flow to **both** the Pacific and Atlantic oceans. The result may be returned in any order.

## Example 1

**Input:**
```
heights = [
  [1, 2, 2, 3, 5],
  [3, 2, 3, 4, 4],
  [2, 4, 5, 3, 1],
  [6, 7, 1, 4, 5],
  [5, 1, 1, 2, 4]
]
```

**Output:** `[[0,4],[1,3],[1,4],[2,2],[3,0],[3,1],[4,0]]`

**Explanation:** The shaded cells can flow water to both the Pacific (top/left) and Atlantic (bottom/right) oceans.

## Example 2

**Input:** `heights = [[1]]`

**Output:** `[[0,0]]`

**Explanation:** A single cell touches both oceans.

## Input Specification

The first line contains two integers `m` and `n` (the number of rows and columns).  
The next `m` lines each contain `n` space-separated integers representing the height values.

## Output Specification

Print each result cell on its own line as `r c` (row and column, 0-indexed), sorted in ascending order by row then by column.

## Constraints

- `m == heights.length`
- `n == heights[r].length`
- `1 <= m, n <= 200`
- `0 <= heights[r][c] <= 10^5`

## Hints

<details>
<summary>Hint 1</summary>

Instead of checking each cell and seeing if it can reach both oceans (which is expensive), try working **backwards**: start BFS/DFS from the ocean borders and find which cells are reachable going **uphill** (to equal or higher cells).

</details>

<details>
<summary>Hint 2</summary>

Run two separate BFS/DFS passes — one starting from all Pacific border cells (top row + left column) and one from all Atlantic border cells (bottom row + right column). In each pass, move to a neighboring cell only if its height is **greater than or equal to** the current cell.

</details>

<details>
<summary>Hint 3</summary>

After both BFS/DFS passes, the answer is the **intersection** of the two sets of reachable cells — cells that both the Pacific BFS and Atlantic BFS can reach. Use two boolean matrices or two sets to track visited cells for each ocean.

</details>
