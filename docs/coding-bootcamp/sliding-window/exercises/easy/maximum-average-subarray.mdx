# Maximum Average Subarray I

**Difficulty:** Easy

You are given an integer array `nums` consisting of `n` elements, and an integer `k`.

Find a contiguous subarray whose length is equal to `k` that has the maximum average value and return this value.

## Example 1

**Input:** `nums = [1,12,-5,-6,50,3]`, `k = 4`  
**Output:** `12.75000`  
**Explanation:** Maximum average is (12 - 5 - 6 + 50) / 4 = 51 / 4 = 12.75

## Example 2

**Input:** `nums = [5]`, `k = 1`  
**Output:** `5.00000`

## Input Specification

The first line contains two integers: `n` (length of array) and `k` (window size). The second line contains `n` space-separated integers.

## Output Specification

Print the maximum average value rounded to 5 decimal places.

## Constraints

- `1 <= k <= n <= 10⁵`
- `-10⁴ <= nums[i] <= 10⁴`

## Hints

<details>
<summary>Hint 1</summary>

Use a sliding window of size k. Compute the sum of the first window, then slide: subtract the element leaving the window, add the new element entering.
</details>

<details>
<summary>Hint 2</summary>

Track the maximum sum seen. The maximum average is the maximum sum divided by k.
</details>
