# Find Minimum in Rotated Sorted Array — Solution

## Approach: Binary Search

**Idea:** In a rotated sorted array, the minimum is the pivot point. Use binary search comparing `mid` with `right` to determine which half contains the minimum.

**Algorithm:**
1. `left = 0, right = n-1`.
2. While `left < right`:
   - `mid = (left + right) // 2`.
   - If `nums[mid] > nums[right]`: min is in right half → `left = mid + 1`.
   - Else: min is in left half (including mid) → `right = mid`.
3. Return `nums[left]`.

**Complexity:** O(log n) time, O(1) space.

def find_min(nums: list[int]) -> int:
    left, right = 0, len(nums) - 1

    while left < right:
        mid = (left + right) // 2
        # If mid > right, pivot is in the right half
        if nums[mid] > nums[right]:
            left = mid + 1
        else:
            right = mid

    return nums[left]
