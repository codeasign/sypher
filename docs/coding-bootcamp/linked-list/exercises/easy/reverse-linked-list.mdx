# Reverse Linked List

**Difficulty:** Easy

Given the `head` of a singly linked list, reverse the list and return the new head of the reversed list.

## Example 1

**Input:** `head = [1, 2, 3, 4, 5]`  
**Output:** `[5, 4, 3, 2, 1]`  
**Explanation:** The list 1 → 2 → 3 → 4 → 5 becomes 5 → 4 → 3 → 2 → 1.

## Example 2

**Input:** `head = [1, 2]`  
**Output:** `[2, 1]`

## Example 3

**Input:** `head = []`  
**Output:** `[]`

## Input Specification

The first line contains an integer `n` (the number of nodes in the linked list). The second line contains `n` space-separated integers representing the values of the nodes in order from head to tail.

## Output Specification

Print the `n` values of the reversed linked list, space-separated on a single line.

## Constraints

- `0 <= n <= 10^5`
- `-10^9 <= Node.val <= 10^9`

## Hints

<details>
<summary>Hint 1</summary>

Think about how you can change the direction of each `next` pointer. What happens to the `head` after you reverse the first link?
</details>

<details>
<summary>Hint 2</summary>

Use three pointers: `prev`, `current`, and `next_temp`. Iterate through the list, reversing one link at a time.
</details>

<details>
<summary>Hint 3</summary>

The `prev` pointer starts as `null` (or `None`). After the loop, `prev` will be the new head of the reversed list.
</details>
