# Merge Two Sorted Lists

**Difficulty:** Easy

You are given the heads of two sorted linked lists `list1` and `list2`.

Merge the two lists into one **sorted** list. The list should be made by splicing together the nodes of the first two lists.

Return the head of the merged linked list.

## Example 1

**Input:** `list1 = [1, 2, 4]`, `list2 = [1, 3, 4]`  
**Output:** `[1, 1, 2, 3, 4, 4]`  

## Example 2

**Input:** `list1 = []`, `list2 = []`  
**Output:** `[]`  

## Example 3

**Input:** `list1 = []`, `list2 = [0]`  
**Output:** `[0]`  

## Input Specification

- First line: space-separated integers of `list1` (may be empty).
- Second line: space-separated integers of `list2` (may be empty).
- `0 <= list length <= 50`

## Output Specification

Print the merged sorted list as space-separated integers.

## Constraints

- `0 <= list length <= 50`
- `-100 <= Node.val <= 100`
- Both `list1` and `list2` are sorted in non-decreasing order.

## Hints

<details>
<summary>Hint 1</summary>

Use a dummy head node to simplify the merge logic. This avoids special case handling for the first node.
</details>

<details>
<summary>Hint 2</summary>

Compare the current nodes of both lists. Append the smaller one to the result and advance that list's pointer.
</details>

<details>
<summary>Hint 3</summary>

After the loop, one list may still have remaining nodes. Append all of them directly.
</details>
