# Add Two Numbers — Solution

## Problem Overview

Add two numbers represented as reverse-order linked lists. Each node contains a single digit. This is a **linked list traversal with carry** problem.

## Approach: Iterative with Carry

**Idea:** Traverse both lists simultaneously, summing digits and tracking carry. Build the result list incrementally.

**Algorithm:**
1. Create a dummy head node for the result.
2. Initialize `carry = 0`.
3. While either list has nodes or carry > 0:
   - Sum = `carry + l1.val (if l1) + l2.val (if l2)`.
   - New digit = sum % 10.
   - Carry = sum / 10.
   - Append new digit node to result.
   - Advance l1 and l2 if they exist.
4. Return `dummy.next`.

**Complexity Analysis:**
- **Time:** O(max(n, m)) — each node is visited once.
- **Space:** O(max(n, m)) — the result list.

def add_two_numbers(l1: list[int], l2: list[int]) -> list[int]:
    result = []
    carry = 0
    i, j = 0, 0
    
    while i < len(l1) or j < len(l2) or carry:
        # Get current digits (0 if list exhausted)
        d1 = l1[i] if i < len(l1) else 0
        d2 = l2[j] if j < len(l2) else 0
        
        total = d1 + d2 + carry
        result.append(total % 10)  # Current digit
        carry = total // 10         # Carry for next step
        
        i += 1
        j += 1
    
    return result

class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0);  // Dummy head for result
        ListNode tail = dummy;
        int carry = 0;

        while (l1 != null || l2 != null || carry != 0) {
            // Get current digits, 0 if list exhausted
            int d1 = (l1 != null) ? l1.val : 0;
            int d2 = (l2 != null) ? l2.val : 0;

            int sum = d1 + d2 + carry;
            tail.next = new ListNode(sum % 10);  // Current digit
            carry = sum / 10;                     // Carry for next step

            tail = tail.next;
            if (l1 != null) l1 = l1.next;
            if (l2 != null) l2 = l2.next;
        }

        return dummy.next;
    }
}

function addTwoNumbers(l1, l2) {
    const result = [];
    let carry = 0;
    let i = 0, j = 0;

    while (i < l1.length || j < l2.length || carry) {
        const d1 = i < l1.length ? l1[i] : 0;
        const d2 = j < l2.length ? l2[j] : 0;

        const total = d1 + d2 + carry;
        result.push(total % 10);  // Current digit
        carry = Math.floor(total / 10);  // Carry for next step

        i++;
        j++;
    }

    return result;
}

## Key Takeaways

- **Carry propagation**: The carry can cause an extra digit at the end (e.g., 999 + 1 = 1000).
- **Dummy head**: Simplifies building the result list.
- **Reverse order**: The input is already in reverse order, so we process from least significant digit first.
- **Variable length lists**: Handle lists of different lengths by treating missing nodes as 0.
