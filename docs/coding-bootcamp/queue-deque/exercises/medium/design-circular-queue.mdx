# Design Circular Queue

**Difficulty:** Medium

Design your implementation of a circular queue. The circular queue is a linear data structure in which the operations are performed based on FIFO (First In First Out) principle, and the last position is connected back to the first position to make a circle.

## Example

**Input:**
```
MyCircularQueue 3
enQueue 1
enQueue 2
enQueue 3
enQueue 4
Rear
isFull
deQueue
enQueue 4
Rear
```

**Output:**
```
true
true
true
false
3
true
true
true
4
```

**Explanation:**
- `MyCircularQueue(3)` — creates a queue of size 3
- `enQueue(1)` — returns true
- `enQueue(2)` — returns true
- `enQueue(3)` — returns true
- `enQueue(4)` — returns false (queue is full)
- `Rear()` — returns 3
- `isFull()` — returns true
- `deQueue()` — returns true
- `enQueue(4)` — returns true
- `Rear()` — returns 4

## Input Specification

Each line is a command with optional arguments.

## Output Specification

For each command, output the result on a separate line.

## Constraints

- `1 <= k <= 1000`
- `0 <= value <= 1000`
- At most 3000 operations

Use an array of fixed size with head and tail pointers that wrap around using modulo arithmetic.
</details>

<summary>Hint 2</summary>

Keep track of the count of elements to distinguish between empty and full states.
</details>

<details>
