# Word Break — Solution

## Problem Overview

Determine if a string can be segmented into words from a dictionary. This is a **string DP** problem where the decision at each position depends on whether a previous position is reachable and the substring forms a valid word.

## Brute Force: Recursive Backtracking

**Idea:** Recursively try every possible word at each position. If a word matches the prefix, recurse on the remaining suffix.

**Algorithm:**
1. Convert `wordDict` to a hash set for O(1) lookup.
2. Define a recursive function `canBreak(start)` that returns true if `s[start:]` can be segmented.
3. For each end index > start, check if `s[start:end]` is in the dictionary AND `canBreak(end)` is true.
4. Base case: if `start == len(s)`, return true (empty string is segmented).

**Complexity Analysis:**
- **Time:** O(2^n) — each position branches to try all possible words.
- **Space:** O(n) — recursion depth.

## Optimal: Dynamic Programming

**Idea:** Let `dp[i]` = true if prefix `s[0:i]` can be segmented. To compute `dp[i]`, check if there exists a `j < i` such that `dp[j]` is true and `s[j:i]` is in the dictionary.

**Algorithm:**
1. Convert `wordDict` to a hash set. Find `max_len` = maximum word length (for pruning).
2. Initialize `dp[0] = true` (empty string).
3. For `i` from 1 to n:
   - For `j = max(0, i - max_len)` to `i-1`:
     - If `dp[j]` and `s[j:i]` is in the word set, set `dp[i] = true` and break.
4. Return `dp[n]`.

**Why This Works:** The string is segmentable if and only if there exists a valid segmentation point where the prefix is segmentable and the remaining substring is in the dictionary. The DP explores all possible segmentation points efficiently.

**Complexity Analysis:**
- **Time:** O(n * max_len) — each position checks at most `max_len` previous positions.
- **Space:** O(n + m) — dp array + hash set of m words.

### Edge Cases

- **Empty string**: `dp[0] = true` — an empty string is considered segmentable.
- **No matching words**: Return false.
- **Single character match**: Works if the character is a word in the dictionary.
- **Multiple valid segmentations**: The DP finds if ANY segmentation exists.
- **Word reuse**: Words can be reused any number of times (hash set lookup).
- **Large string with no match**: The DP will terminate quickly thanks to the `max_len` pruning.

### Max Length Optimization

By tracking the longest word in the dictionary (`max_len`), the inner loop only needs to check at most `max_len` positions back from `i`. Without this optimization, the inner loop would check all positions, making it O(n^2).

def word_break(s: str, word_dict: list[str]) -> bool:
    # Convert dictionary to a hash set for O(1) lookups
    word_set = set(word_dict)
    # Track the longest word for pruning the inner loop
    max_len = max(len(w) for w in word_dict) if word_dict else 0

    n = len(s)
    # dp[i] = can segment first i characters
    dp = [False] * (n + 1)
    dp[0] = True  # empty string is always segmentable

    for i in range(1, n + 1):
        # Only check at most max_len positions back from i
        # This avoids O(n^2) when max_len is small
        start = max(0, i - max_len)
        for j in range(start, i):
            if dp[j] and s[j:i] in word_set:
                dp[i] = True
                break  # no need to check further

    return dp[n]

class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        // Convert dictionary to a hash set for O(1) lookups
        Set<String> wordSet = new HashSet<>(wordDict);
        // Track the longest word for pruning
        int maxLen = 0;
        for (String w : wordDict) maxLen = Math.max(maxLen, w.length());

        int n = s.length();
        // dp[i] = can segment first i characters
        boolean[] dp = new boolean[n + 1];
        dp[0] = true;  // empty string is always segmentable

        for (int i = 1; i <= n; i++) {
            // Only check at most maxLen positions back
            int start = Math.max(0, i - maxLen);
            for (int j = start; j < i; j++) {
                if (dp[j] && wordSet.contains(s.substring(j, i))) {
                    dp[i] = true;
                    break;
                }
            }
        }

        return dp[n];
    }
}

use std::collections::HashSet;

pub fn word_break(s: String, word_dict: Vec<String>) -> bool {
    // Convert dictionary to a hash set for O(1) lookups
    let word_set: HashSet<&str> = word_dict.iter().map(|w| w.as_str()).collect();
    // Track the longest word for pruning
    let max_len = word_dict.iter().map(|w| w.len()).max().unwrap_or(0);

    let n = s.len();
    // dp[i] = can segment first i characters
    let mut dp = vec![false; n + 1];
    dp[0] = true;  // empty string is always segmentable

    for i in 1..=n {
        // Only check at most max_len positions back from i
        let start = if i > max_len { i - max_len } else { 0 };
        for j in start..i {
            if dp[j] && word_set.contains(&s[j..i]) {
                dp[i] = true;
                break;
            }
        }
    }

    dp[n]
}

// Simple hash set implementation for C
typedef struct { char** data; int size; } StringSet;

bool set_contains(StringSet* set, char* s, int len) {
    char tmp = s[len]; s[len] = '\0';
    for (int i = 0; i < set->size; i++) {
        if (strcmp(set->data[i], s) == 0) { s[len] = tmp; return true; }
    }
    s[len] = tmp;
    return false;
}

bool wordBreak(char* s, char** wordDict, int wordDictSize) {
    StringSet set = { wordDict, wordDictSize };
    int n = strlen(s);
    // Find max word length for pruning
    int maxLen = 0;
    for (int i = 0; i < wordDictSize; i++) {
        int len = strlen(wordDict[i]);
        if (len > maxLen) maxLen = len;
    }

    // dp[i] = can segment first i characters
    bool* dp = (bool*)calloc(n + 1, sizeof(bool));
    dp[0] = true;  // empty string is always segmentable

    for (int i = 1; i <= n; i++) {
        int start = i - maxLen;
        if (start < 0) start = 0;
        for (int j = start; j < i; j++) {
            if (dp[j] && set_contains(&set, s + j, i - j)) {
                dp[i] = true;
                break;
            }
        }
    }

    bool result = dp[n];
    free(dp);
    return result;
}

func wordBreak(s string, wordDict []string) bool {
    // Convert dictionary to a hash set for O(1) lookups
    wordSet := make(map[string]bool)
    maxLen := 0
    for _, w := range wordDict {
        wordSet[w] = true
        if len(w) > maxLen { maxLen = len(w) }
    }

    n := len(s)
    // dp[i] = can segment first i characters
    dp := make([]bool, n+1)
    dp[0] = true  // empty string is always segmentable

    for i := 1; i <= n; i++ {
        // Only check at most maxLen positions back from i
        start := i - maxLen
        if start < 0 { start = 0 }
        for j := start; j < i; j++ {
            if dp[j] && wordSet[s[j:i]] {
                dp[i] = true
                break
            }
        }
    }

    return dp[n]
}

## Key Takeaways

- **String DP**: The recurrence checks substring membership in a dictionary — a pattern that extends to problems like Concatenated Words and Word Break II.
- **Max-length pruning**: Bounding the inner loop by `max_word_len` is crucial for performance on large strings.
- **Hash set lookup**: Converting the dictionary to a hash set before DP avoids O(m) lookups for each substring check.
- **DP as reachability**: This problem treats DP as "can we reach position i?" — a common framing for segmentation and parsing problems.
