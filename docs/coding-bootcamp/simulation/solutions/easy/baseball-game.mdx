# Baseball Game — Solution

## Problem Overview

Simulate a baseball scoring system using stack operations: add integers, double the top, sum the top two, or invalidate the last score.

## Brute Force Approach

Use a list/array and directly simulate each operation by appending, modifying, or removing elements.

**Complexity**: O(n) time, O(n) space

## Optimal Approach

Use a stack to maintain the record. Each operation translates to a simple stack operation:

- Integer: push onto stack
- `"+"`: push sum of top two
- `"D"`: push double the top
- `"C"`: pop the top

The stack-based approach is the direct and optimal solution.

## Deep Dive

The stack is the natural data structure here because operations only affect the most recent scores. Each operation is O(1), and we process all n operations in O(n) total time. The space is O(n) in the worst case (all integer operations).

class Solution {
    public int calPoints(String[] ops) {
        Stack<Integer> stack = new Stack<>();
        for (String op : ops) {
            if (op.equals("+")) {
                int top = stack.pop();
                int newTop = top + stack.peek();
                stack.push(top);
                stack.push(newTop);
            } else if (op.equals("D")) {
                stack.push(stack.peek() * 2);
            } else if (op.equals("C")) {
                stack.pop();
            } else {
                stack.push(Integer.parseInt(op));
            }
        }
        int sum = 0;
        for (int s : stack) sum += s;
        return sum;
    }
}

int calPoints(char** ops, int opsSize) {
    int* stack = malloc(opsSize * sizeof(int));
    int top = -1;
    for (int i = 0; i < opsSize; i++) {
        if (strcmp(ops[i], "+") == 0) {
            int a = stack[top--];
            int b = stack[top];
            stack[++top] = a;
            stack[++top] = a + b;
        } else if (strcmp(ops[i], "D") == 0) {
            stack[top + 1] = stack[top] * 2;
            top++;
        } else if (strcmp(ops[i], "C") == 0) {
            top--;
        } else {
            stack[++top] = atoi(ops[i]);
        }
    }
    int sum = 0;
    for (int i = 0; i <= top; i++) sum += stack[i];
    free(stack);
    return sum;
}
