# Baseball Game

You are keeping score for a baseball game. You start with an empty record. You are given a list of strings `ops` where each operation follows these rules:

- An integer `x`: Record a new score of `x`.
- `'+'`: Record a new score that is the sum of the previous two scores.
- `'D'`: Record a new score that is double the previous score.
- `'C'`: Invalidate the previous score, removing it from the record.

Return the sum of all scores on the record after applying all operations.

## Examples

**Example 1:**
```
Input: ops = ["5","2","C","D","+"]
Output: 30
Explanation:
"5" - Add 5 → [5]
"2" - Add 2 → [5, 2]
"C" - Remove last → [5]
"D" - Double last → [5, 10]
"+" - Sum of last two → [5, 10, 15]
Sum = 5 + 10 + 15 = 30
```

**Example 2:**
```
Input: ops = ["5","-2","4","C","D","9","+","+"]
Output: 27
```

## Constraints

- `1 <= ops.length <= 1000`
- `ops[i]` is `"C"`, `"D"`, `"+"`, or a string representing an integer in the range `[-3 * 10^4, 3 * 10^4]`
- For operation `"+"`, there will always be at least two previous scores
- For `"C"` and `"D"`, there will always be at least one previous score

## Hints

<details>
<summary>Hint 1</summary>

Use a stack (list in Python) to maintain the record of scores. Each operation modifies the stack.
</details>

<details>
<summary>Hint 2</summary>

For 'C', pop from the stack. For 'D', push double the top. For '+', push the sum of the top two.
</details>
