# Longest Common Prefix

**Difficulty:** Easy

Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string `""`.

## Example 1

**Input:** `strs = ["flower","flow","flight"]`  
**Output:** `"fl"`  

## Example 2

**Input:** `strs = ["dog","racecar","car"]`  
**Output:** `""`  
**Explanation:** There is no common prefix among the input strings.

## Input Specification

A single line containing space-separated strings.

## Output Specification

Print the longest common prefix. If none exists, print an empty line.

## Constraints

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` consists only of lowercase English letters

## Hints

<details>
<summary>Hint 1</summary>

Start by comparing the first character of each string. If they all match, move to the next character.
</details>

<details>
<summary>Hint 2</summary>

You can use the first string as a reference and compare it character-by-character against all other strings.
</details>

<details>
<summary>Hint 3</summary>

A Trie approach: insert all strings into a trie, then traverse from the root following nodes that have exactly one child (and are not an end-of-word) until you reach a branch.
</details>
