# Design Add and Search Words Data Structure

**Difficulty:** Hard

Design a data structure that supports adding new words and finding if a string matches any previously added string.

Implement the `WordDictionary` class:

- `WordDictionary()` — Initializes the object.
- `void addWord(word)` — Adds `word` to the data structure, it can be matched later.
- `bool search(word)` — Returns `true` if there is any string in the data structure that matches `word`. The `word` may contain dots `'.'` where dots can be matched with any letter.

## Example

**Input:**
```
addWord bad
addWord dad
addWord mad
search pad
search bad
search .ad
search b..
```

**Output:**
```
false
true
true
true
```

**Explanation:**
- `search("pad")` — false (not in dictionary)
- `search("bad")` — true (exists)
- `search(".ad")` — true (matches "bad", "dad", "mad")
- `search("b..")` — true (matches "bad")

## Input Specification

Each line of input contains a command followed by a word (or just a command). Commands are:
- `addWord <word>` — add the word to the dictionary
- `search <word>` — check if the word matches any word in the dictionary

The search word may contain `'.'` which matches any single lowercase letter.

## Output Specification

For each `search` command, print `true` or `false` on its own line.

## Constraints

- `1 <= word.length <= 25`
- `word` in `addWord` consists only of lowercase English letters
- `word` in `search` consists of lowercase English letters and dots `'.'`
- At most `10⁴` calls will be made to `addWord` and `search`

## Hints

<details>
<summary>Hint 1</summary>

Use a Trie as the underlying data structure. Each node stores children and an end-of-word flag.
</details>

<details>
<summary>Hint 2</summary>

For `search`, when you encounter a `'.'`, you need to try all possible children. This requires a recursive or iterative backtracking search.
</details>

<details>
<summary>Hint 3</summary>

The dot wildcard `'.'` can be handled by checking all non-null children of the current node. If any child leads to a match, return `true`.
</details>

<details>
<summary>Hint 4</summary>

Consider using a helper function `searchInNode(word, node, index)` that performs the recursive search from a given node and position.
</details>
