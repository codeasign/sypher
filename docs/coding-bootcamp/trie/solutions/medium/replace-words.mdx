# Replace Words — Solution

## Problem Overview

Given a dictionary of root words and a sentence, replace every word in the sentence with its shortest root from the dictionary. If a word has no root prefix in the dictionary, keep it unchanged.

## Brute Force Approach

**Idea:** For each word in the sentence, check every root in the dictionary to see if it's a prefix.

**Algorithm:**
1. Split the sentence into words.
2. For each word, iterate through all roots in the dictionary.
3. If a root is a prefix of the word, check if it's shorter than any previously found root for this word.
4. Replace with the shortest matching root (or keep original if none found).

**Complexity:**
- **Time:** O(N × K × L) where N is the number of words, K is the dictionary size, and L is the word length
- **Space:** O(N) for the result

## Optimal Approach: Trie

**Idea:** Insert all roots into a Trie. For each word in the sentence, traverse the Trie character by character. If we reach an end-of-word node before finishing the word, we found the shortest root.

**Algorithm:**
1. Build a Trie from all roots in the dictionary.
2. Split the sentence into words.
3. For each word:
   - Traverse the Trie character by character.
   - If a child is missing, no root matches → keep the original word.
   - If we reach an end-of-word node, the path so far is the shortest root → replace.
4. Join the processed words back into a sentence.

**Why This Works:** The Trie naturally finds the shortest prefix because we stop as soon as we encounter an end-of-word marker. Since we traverse from the root downwards, the first end-of-word we encounter is necessarily the shortest root.

**Complexity Analysis:**
- **Time:** O(N + K × R + N × L) where N is the number of words, K is the dictionary size, R is average root length, and L is average word length
- **Space:** O(K × R) for the Trie

## Deep Dive

### Why the Trie is Ideal

Without a Trie, checking if a root is a prefix involves comparing characters for each root-word pair. With a Trie, the prefix check is built into the data structure: following the path character by character naturally reveals whether a prefix exists.

### Edge Cases

- **Word shorter than any root:** The traversal will hit the end of the word before finding a root. Keep the original word.
- **Multiple roots match:** The first root encountered (the shortest) is used because we stop at the first `isEnd` node.
- **Root equals the word:** The entire word is replaced by itself.
- **Empty dictionary:** No replacements happen; the sentence is returned unchanged.

class TrieNode:
    def __init__(self):
        self.children = [None] * 26
        self.is_end = False

def replace_words(dictionary: list[str], sentence: str) -> str:
    # Build Trie from dictionary roots
    root = TrieNode()
    for word in dictionary:
        node = root
        for ch in word:
            idx = ord(ch) - ord('a')
            if node.children[idx] is None:
                node.children[idx] = TrieNode()
            node = node.children[idx]
        node.is_end = True

    # Process each word in the sentence
    result = []
    for word in sentence.split():
        node = root
        replacement = ""
        found = False
        for ch in word:
            idx = ord(ch) - ord('a')
            if node.children[idx] is None:
                break  # No root matches this prefix
            replacement += ch
            node = node.children[idx]
            if node.is_end:
                found = True
                break  # Found shortest root
        if found:
            result.append(replacement)  # Replace with root
        else:
            result.append(word)         # Keep original word

    return " ".join(result)

use std::collections::HashMap;

struct TrieNode {
    children: HashMap<char, TrieNode>,
    is_end: bool,
}

impl TrieNode {
    fn new() -> Self {
        TrieNode {
            children: HashMap::new(),
            is_end: false,
        }
    }
}

fn replace_words(dictionary: Vec<String>, sentence: String) -> String {
    // Build Trie
    let mut root = TrieNode::new();
    for word in &dictionary {
        let mut node = &mut root;
        for ch in word.chars() {
            node = node.children.entry(ch).or_insert_with(TrieNode::new);
        }
        node.is_end = true;
    }

    // Process sentence
    sentence
        .split(' ')
        .map(|word| {
            let mut node = &root;
            let mut replacement = String::new();

            for ch in word.chars() {
                match node.children.get(&ch) {
                    Some(child) => {
                        replacement.push(ch);
                        if child.is_end {
                            return replacement;  // Replace with root
                        }
                        node = child;
                    }
                    None => break,  // No root matches
                }
            }

            word.to_string()  // Keep original
        })
        .collect::<Vec<_>>()
        .join(" ")
}

#include <stdlib.h>
#include <string.h>
#include <stdbool.h>

typedef struct TrieNode {
    struct TrieNode* children[26];
    bool isEnd;
} TrieNode;

TrieNode* createNode() {
    return (TrieNode*)calloc(1, sizeof(TrieNode));
}

char* replaceWords(char** dictionary, int dictionarySize, char* sentence) {
    // Build Trie
    TrieNode* root = createNode();
    for (int i = 0; i < dictionarySize; i++) {
        TrieNode* node = root;
        for (int j = 0; dictionary[i][j]; j++) {
            int idx = dictionary[i][j] - 'a';
            if (node->children[idx] == NULL) {
                node->children[idx] = createNode();
            }
            node = node->children[idx];
        }
        node->isEnd = true;
    }

    // Process sentence
    char* result = malloc(strlen(sentence) + 1);
    result[0] = '\0';
    char* word = strtok(sentence, " ");
    int first = 1;

    while (word) {
        if (!first) strcat(result, " ");
        first = 0;

        TrieNode* node = root;
        char replacement[205] = "";
        int found = 0;

        for (int j = 0; word[j]; j++) {
            int idx = word[j] - 'a';
            if (node->children[idx] == NULL) break;
            replacement[j] = word[j];
            replacement[j + 1] = '\0';
            node = node->children[idx];
            if (node->isEnd) {
                found = 1;
                break;
            }
        }

        if (found) {
            strcat(result, replacement);
        } else {
            strcat(result, word);
        }
        word = strtok(NULL, " ");
    }

    return result;
}

type TrieNode struct {
    children [26]*TrieNode
    isEnd    bool
}

func replaceWords(dictionary []string, sentence string) string {
    // Build Trie
    root := &TrieNode{}
    for _, word := range dictionary {
        node := root
        for i := 0; i < len(word); i++ {
            idx := word[i] - 'a'
            if node.children[idx] == nil {
                node.children[idx] = &TrieNode{}
            }
            node = node.children[idx]
        }
        node.isEnd = true
    }

    // Process sentence
    words := strings.Split(sentence, " ")
    for i, word := range words {
        node := root
        replacement := strings.Builder{}
        found := false

        for j := 0; j < len(word); j++ {
            idx := word[j] - 'a'
            if node.children[idx] == nil {
                break
            }
            replacement.WriteByte(word[j])
            node = node.children[idx]
            if node.isEnd {
                found = true
                break
            }
        }

        if found {
            words[i] = replacement.String()
        }
    }

    return strings.Join(words, " ")
}

## Key Takeaways

- The Trie efficiently finds the **shortest prefix match** by stopping at the first `isEnd` node encountered during traversal
- This problem showcases a simple but powerful use case for Tries: **prefix-based replacement**
- The Trie approach improves time complexity from O(N × K × L) to O(N × L + K × R)
- The same pattern can be applied to any problem involving finding the shortest matching prefix
