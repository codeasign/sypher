# Word Search II — Solution

## Problem Overview

Given an m x n board of characters and a list of words, find all words on the board. Words are constructed from adjacent cells (horizontal or vertical), and each cell can be used at most once per word.

## Brute Force Approach

**Idea:** For each word, perform a DFS from every cell on the board to check if the word exists.

**Algorithm:**
1. For each word, iterate over every cell on the board.
2. From each cell, perform DFS to match the word character by character.
3. If the word is found, add it to the result set.

**Complexity:**
- **Time:** O(N × M × 4^L) per word, where L is the word length — far too slow for 3×10⁴ words
- **Space:** O(L) for the recursion stack

**Why this is too slow:** With up to 30,000 words, even a single DFS per word becomes prohibitively expensive. We need a way to check all words simultaneously.

## Optimal Approach: Trie + Backtracking

**Idea:** Build a Trie from the word list. Traverse the board with DFS, checking the Trie at each step. When a full word is found, record it and prune it from the Trie to avoid duplicates.

**Algorithm:**
1. Build a Trie from all words in the dictionary.
2. For each cell on the board, start a DFS:
   - Check if the current character is a valid child in the Trie. If not, backtrack.
   - Mark the cell as visited (modify board or use a visited set).
   - If the current Trie node is an end-of-word, add the word to results and mark it as found (set `isEnd = false` to avoid duplicates).
   - Recurse on all four adjacent cells.
   - Restore the cell (unmark visited).
3. Return all found words.

**Why This Works:** By traversing the Trie simultaneously with the board, we check all words at once. When a prefix doesn't exist in the Trie, we prune the search immediately, saving enormous time compared to checking each word separately.

**Complexity Analysis:**
- **Time:** O(M × N × 4^L) in the worst case, but pruning via the Trie makes it much faster in practice
- **Space:** O(N × L + L) for the Trie storing N words plus the recursion stack

## Deep Dive

### Pruning with the Trie

The key insight is that the Trie acts as a **collective prefix check**. Instead of asking "does this path match word A? does it match word B?" separately, we ask "does this path match any word?" once and get the answer for all words simultaneously.

### Optimization: Removing Found Words

When we find a word, we set its Trie node's `isEnd` flag to `false`. This prevents adding the same word multiple times if it appears in different positions on the board. For even better performance, we can also delete leaf nodes from the Trie that are no longer needed, reducing future traversal overhead.

### Edge Cases

- **Same word appears multiple times on the board:** Return it only once. Handled by clearing the `isEnd` flag after finding.
- **Word is a prefix of another word:** Both should be returned. The prefix word will be found first, but its node still has children, so the longer word can still be found.
- **Empty board or empty word list:** Return an empty list.
- **Single character board and single character word:** Check that one cell.

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False
        self.word = None  # Store the actual word at leaf nodes

class Solution:
    def findWords(self, board: list[list[str]], words: list[str]) -> list[str]:
        # Build Trie from word list
        root = TrieNode()
        for word in words:
            node = root
            for ch in word:
                if ch not in node.children:
                    node.children[ch] = TrieNode()
                node = node.children[ch]
            node.is_end = True
            node.word = word  # Store word for easy retrieval

        rows, cols = len(board), len(board[0])
        result = []

        def dfs(r: int, c: int, node: TrieNode) -> None:
            # Check bounds and character match
            if r < 0 or r >= rows or c < 0 or c >= cols:
                return
            ch = board[r][c]
            if ch not in node.children:
                return  # No words with this prefix

            next_node = node.children[ch]
            if next_node.is_end:
                result.append(next_node.word)  # Found a word
                next_node.is_end = False  # Avoid duplicates

            # Mark cell as visited by replacing character
            board[r][c] = "#"

            # Explore four directions
            for dr, dc in [(-1,0), (1,0), (0,-1), (0,1)]:
                dfs(r + dr, c + dc, next_node)

            # Restore cell
            board[r][c] = ch

        # Start DFS from every cell
        for r in range(rows):
            for c in range(cols):
                dfs(r, c, root)

        return result

class TrieNode {
    TrieNode[] children = new TrieNode[26];
    boolean isEnd = false;
    String word = null;  // Store the actual word at leaf nodes
}

class Solution {
    private int rows, cols;
    private char[][] board;
    private List<String> result = new ArrayList<>();
    private static final int[][] DIRS = {{-1,0}, {1,0}, {0,-1}, {0,1}};

    public List<String> findWords(char[][] board, String[] words) {
        this.board = board;
        this.rows = board.length;
        this.cols = board[0].length;

        // Build Trie from word list
        TrieNode root = new TrieNode();
        for (String word : words) {
            TrieNode node = root;
            for (char ch : word.toCharArray()) {
                int idx = ch - 'a';
                if (node.children[idx] == null) {
                    node.children[idx] = new TrieNode();
                }
                node = node.children[idx];
            }
            node.isEnd = true;
            node.word = word;  // Store word for retrieval
        }

        // Start DFS from every cell
        for (int r = 0; r < rows; r++) {
            for (int c = 0; c < cols; c++) {
                dfs(r, c, root);
            }
        }

        return result;
    }

    private void dfs(int r, int c, TrieNode node) {
        if (r < 0 || r >= rows || c < 0 || c >= cols) return;

        char ch = board[r][c];
        int idx = ch - 'a';
        if (ch == '#' || node.children[idx] == null) return;

        TrieNode next = node.children[idx];
        if (next.isEnd) {
            result.add(next.word);  // Found a word
            next.isEnd = false;     // Avoid duplicates
        }

        board[r][c] = '#';  // Mark visited

        for (int[] dir : DIRS) {
            dfs(r + dir[0], c + dir[1], next);
        }

        board[r][c] = ch;  // Restore
    }
}

function findWords(board: string[][], words: string[]): string[] {
    const rows = board.length;
    const cols = board[0].length;
    const result: string[] = [];
    const dirs: number[][] = [[-1,0], [1,0], [0,-1], [0,1]];

    // Build Trie
    interface TrieNode {
        children: { [key: string]: TrieNode };
        isEnd: boolean;
        word: string | null;
    }
    const root: TrieNode = { children: {}, isEnd: false, word: null };

    for (const word of words) {
        let node = root;
        for (const ch of word) {
            if (!node.children[ch]) {
                node.children[ch] = { children: {}, isEnd: false, word: null };
            }
            node = node.children[ch];
        }
        node.isEnd = true;
        node.word = word;
    }

    function dfs(r: number, c: number, node: TrieNode): void {
        if (r < 0 || r >= rows || c < 0 || c >= cols) return;

        const ch = board[r][c];
        if (ch === '#' || !node.children[ch]) return;

        const next = node.children[ch];
        if (next.isEnd) {
            result.push(next.word!);
            next.isEnd = false;
        }

        board[r][c] = '#';

        for (const [dr, dc] of dirs) {
            dfs(r + dr, c + dc, next);
        }

        board[r][c] = ch;
    }

    for (let r = 0; r < rows; r++) {
        for (let c = 0; c < cols; c++) {
            dfs(r, c, root);
        }
    }

    return result;
}

use std::collections::HashMap;

struct TrieNode {
    children: HashMap<char, TrieNode>,
    is_end: bool,
    word: Option<String>,
}

impl TrieNode {
    fn new() -> Self {
        TrieNode {
            children: HashMap::new(),
            is_end: false,
            word: None,
        }
    }
}

fn find_words(board: Vec<Vec<char>>, words: Vec<String>) -> Vec<String> {
    let rows = board.len();
    let cols = board[0].len();
    let mut board = board;
    let mut result = Vec::new();
    let dirs = [(-1, 0), (1, 0), (0, -1), (0, 1)];

    // Build Trie
    let mut root = TrieNode::new();
    for word in &words {
        let mut node = &mut root;
        for ch in word.chars() {
            node = node.children.entry(ch).or_insert_with(TrieNode::new);
        }
        node.is_end = true;
        node.word = Some(word.clone());
    }

    fn dfs(
        board: &mut Vec<Vec<char>>,
        r: i32,
        c: i32,
        rows: usize,
        cols: usize,
        node: &TrieNode,
        result: &mut Vec<String>,
        dirs: &[(i32, i32); 4],
    ) {
        if r < 0 || r >= rows as i32 || c < 0 || c >= cols as i32 {
            return;
        }
        let ch = board[r as usize][c as usize];
        if ch == '#' {
            return;
        }
        let next = match node.children.get(&ch) {
            Some(n) => n,
            None => return,
        };

        if next.is_end {
            if let Some(ref w) = next.word {
                result.push(w.clone());
            }
            // We use unsafe to modify is_end through a shared ref — better design:
            // use interior mutability or restructure. For conciseness, skip mutation here.
        }

        board[r as usize][c as usize] = '#';

        for &(dr, dc) in dirs {
            dfs(board, r + dr, c + dc, rows, cols, next, result, dirs);
        }

        board[r as usize][c as usize] = ch;
    }

    for r in 0..rows {
        for c in 0..cols {
            dfs(&mut board, r as i32, c as i32, rows, cols, &root, &mut result, &dirs);
        }
    }

    result.sort();
    result.dedup();
    result
}

// The trie is built with 26-ary nodes and the board is traversed with DFS.

#include <stdbool.h>
#include <stdlib.h>
#include <string.h>

#define MAX_WORDS 1000
#define MAX_WORD_LEN 15

typedef struct TrieNode {
    struct TrieNode* children[26];
    bool isEnd;
    char* word;
} TrieNode;

TrieNode* createNode() {
    TrieNode* node = (TrieNode*)calloc(1, sizeof(TrieNode));
    node->word = NULL;
    return node;
}

void insert(TrieNode* root, char* word) {
    TrieNode* node = root;
    for (int i = 0; word[i]; i++) {
        int idx = word[i] - 'a';
        if (!node->children[idx]) node->children[idx] = createNode();
        node = node->children[idx];
    }
    node->isEnd = true;
    node->word = word;
}

int dirs[4][2] = {{-1,0}, {1,0}, {0,-1}, {0,1}};
char** result;
int resultSize;

void dfs(char** board, int rows, int cols, int r, int c, TrieNode* node) {
    if (r < 0 || r >= rows || c < 0 || c >= cols) return;
    char ch = board[r][c];
    if (ch == '#' || !node->children[ch - 'a']) return;

    TrieNode* next = node->children[ch - 'a'];
    if (next->isEnd) {
        result[resultSize++] = strdup(next->word);
        next->isEnd = false;
    }

    board[r][c] = '#';
    for (int d = 0; d < 4; d++) {
        dfs(board, rows, cols, r + dirs[d][0], c + dirs[d][1], next);
    }
    board[r][c] = ch;
}

char** findWords(char** board, int boardSize, int* boardColSize, char** words, int wordsSize, int* returnSize) {
    result = (char**)malloc(MAX_WORDS * sizeof(char*));
    resultSize = 0;

    TrieNode* root = createNode();
    for (int i = 0; i < wordsSize; i++) {
        insert(root, words[i]);
    }

    for (int r = 0; r < boardSize; r++) {
        for (int c = 0; c < boardColSize[r]; c++) {
            dfs(board, boardSize, boardColSize[r], r, c, root);
        }
    }

    *returnSize = resultSize;
    return result;
}

## Key Takeaways

- The **Trie + DFS** combination is the standard solution for word search on a board with multiple target words
- The Trie acts as a **collective prefix filter**, allowing us to prune invalid paths for all words simultaneously
- Modifying the board in-place avoids the overhead of a separate visited set
- Clearing the `isEnd` flag prevents duplicate results when the same word appears in multiple locations
- This problem demonstrates how the Trie can be combined with other algorithms (backtracking/DFS) to solve complex problems
