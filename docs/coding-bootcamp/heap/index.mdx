# Heap (Priority Queue)

A Heap is a complete binary tree where the parent is always greater (max-heap) or smaller (min-heap) than its children.

⏱ 15 minutes

## What You Will Learn

- Min-heap vs max-heap properties
- Heap operations: push, pop, heapify
- Top K and median finding patterns

## The Concept

A heap maintains the smallest (min-heap) or largest (max-heap) element at the root. Insert and extract-min/max take O(log n). Common for Top K, median, and merging problems.

## Common Applications

| Problem | Approach |
|---------|----------|
| Top K Elements | Min-heap of size K |
| Kth Largest | Min-heap of size K |
| Median Finder | Two heaps (max + min) |
| Merge K Sorted | Min-heap of smallest elements |
| Task Scheduler | Max-heap by frequency |

## Complexity

| Operation | Time |
|-----------|------|
| Push / Offer | O(log n) |
| Pop / Poll | O(log n) |
| Peek | O(1) |
| Heapify | O(n) |
