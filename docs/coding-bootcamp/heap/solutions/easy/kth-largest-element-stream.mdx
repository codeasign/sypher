# Kth Largest Element in a Stream — Solution

## Approach: Min-Heap of Size k

**Algorithm:** Maintain a min-heap of size k. The smallest element is the kth largest. When adding, push then pop if size > k.

**Complexity:** O(n log k) time, O(k) space.

## Code

<Tabs>
  <TabItem value="python" label="Python">

```python
import heapq

class KthLargest:
    def __init__(self, k: int, nums: list[int]):
        self.k = k
        self.heap = []
        for val in nums:
            self.add(val)  # Reuse add logic

    def add(self, val: int) -> int:
        heapq.heappush(self.heap, val)
        if len(self.heap) > self.k:
            heapq.heappop(self.heap)
        return self.heap[0]  # kth largest is the smallest in heap
```
