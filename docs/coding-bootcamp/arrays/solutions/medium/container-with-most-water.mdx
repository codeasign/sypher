# Container With Most Water — Solution

## Problem Overview

Find the maximum water area between any two vertical lines. Area = width × min(left_height, right_height). Solved with **two pointers**.

## Approach: Two Pointers

**Idea:** Start with the widest container (left=0, right=n-1). At each step, the area is limited by the shorter line. Moving the shorter line inward might find a taller line and increase the area.

**Algorithm:**
1. Initialize `left = 0`, `right = n-1`, `max_area = 0`.
2. While `left < right`:
   - Compute area = `min(height[left], height[right]) * (right - left)`.
   - Update `max_area = max(max_area, area)`.
   - If `height[left] < height[right]`, increment `left`. Otherwise, decrement `right`.
3. Return `max_area`.

**Complexity Analysis:**
- **Time:** O(n) — single pass.
- **Space:** O(1) — two pointers.

def max_area(height: list[int]) -> int:
    left, right = 0, len(height) - 1
    max_area = 0
    
    while left < right:
        # Compute area with the shorter line as height
        h = min(height[left], height[right])
        w = right - left
        max_area = max(max_area, h * w)
        
        # Move the pointer with the shorter height inward
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    
    return max_area

## Key Takeaways

- **Two-pointer convergence**: Start wide, move the limiting pointer inward.
- **Proof**: If `height[left] < height[right]`, then any container with `right-1, right-2, ...` as the right wall has area ≤ current area (since height is bounded by `height[left]` and width is smaller). So moving the left pointer cannot miss a better solution.
