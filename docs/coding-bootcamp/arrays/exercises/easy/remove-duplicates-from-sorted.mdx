# Remove Duplicates from Sorted Array

**Difficulty:** Easy

Given an integer array `nums` sorted in non-decreasing order, remove the duplicates **in-place** such that each unique element appears only once. The relative order of the elements should be kept the same. Return `k`, the number of unique elements.

Do **not** allocate extra space for another array. Modify `nums` in-place with O(1) extra memory.

## Example 1

**Input:** `nums = [1, 1, 2]`  
**Output:** `2` (nums becomes `[1, 2, _]`)

## Example 2

**Input:** `nums = [0, 0, 1, 1, 1, 2, 2, 3, 3, 4]`  
**Output:** `5` (nums becomes `[0, 1, 2, 3, 4, _, ...]`)

## Input Specification

- A single line of space-separated integers (sorted in non-decreasing order).
- `1 <= nums.length <= 3 * 10^4`

## Output Specification

Print a single integer: the number of unique elements `k`.

## Constraints

- `1 <= nums.length <= 3 * 10^4`
- `-100 <= nums[i] <= 100`
- `nums` is sorted in non-decreasing order.

## Hints

<details>
<summary>Hint 1</summary>

Use a slow pointer `k` that tracks where the next unique element should be placed, and a fast pointer `i` that scans through the array.
</details>

<details>
<summary>Hint 2</summary>

Whenever `nums[i] != nums[k-1]`, you've found a new unique element. Place it at `nums[k]` and increment `k`.
</details>

<details>
<summary>Hint 3</summary>

Since the array is sorted, all duplicates are adjacent. A single pass is enough.
</details>
