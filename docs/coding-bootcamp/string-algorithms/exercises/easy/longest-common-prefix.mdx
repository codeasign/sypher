# Longest Common Prefix

**Difficulty:** Easy

Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string `""`.

## Example 1

**Input:** `strs = ["flower", "flow", "flight"]`  
**Output:** `"fl"`  
**Explanation:** The first two characters "fl" are common to all three strings.

## Example 2

**Input:** `strs = ["dog", "racecar", "car"]`  
**Output:** `""`  
**Explanation:** There is no common prefix among the input strings.

## Input Specification

A single line containing comma-separated strings. Example: `flower,flow,flight`

## Output Specification

Print the longest common prefix string. If there is no common prefix, print an empty line.

## Constraints

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` consists of only lowercase English letters

## Hints

<details>
<summary>Hint 1</summary>

Start with the first string as the initial prefix candidate. Compare it character by character with each subsequent string.
</details>

<details>
<summary>Hint 2</summary>

When a mismatch is found, truncate the prefix candidate to the matching portion. If the prefix becomes empty, you can stop early.
</details>

<details>
<summary>Hint 3</summary>

You can also solve this by sorting the array and comparing only the first and last strings — the common prefix of the sorted extremes is the common prefix of all.
</details>
