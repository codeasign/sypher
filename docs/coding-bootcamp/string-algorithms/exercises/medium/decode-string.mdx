# Decode String

**Difficulty:** Medium

Given an encoded string `s`, return its decoded string.

The encoding rule is: `k[encoded_string]`, where the `encoded_string` inside the square brackets is repeated exactly `k` times. Note that `k` is guaranteed to be a positive integer.

You may assume that the input string is always valid — there are no extra white spaces, square brackets are well-formed, and `k` is always a positive integer.

## Example 1

**Input:** `s = "3[a]2[bc]"`  
**Output:** `"aaabcbc"`  
**Explanation:** "a" repeated 3 times → "aaa", "bc" repeated 2 times → "bcbc", concatenated → "aaabcbc".

## Example 2

**Input:** `s = "3[a2[c]]"`  
**Output:** `"accaccacc"`  
**Explanation:** Inner "c" repeated 2 times → "cc", then "acc" repeated 3 times → "accaccacc".

## Example 3

**Input:** `s = "2[abc]3[cd]ef"`  
**Output:** `"abcabccdcdcdef"`  
**Explanation:** "abc" twice → "abcabc", "cd" 3 times → "cdcdcd", then "ef" appended → "abcabccdcdcdef".

## Input Specification

A single line containing the encoded string `s`.

## Output Specification

Print the decoded string.

## Constraints

- `1 <= s.length <= 30`
- `s` consists of lowercase English letters, digits, and square brackets `[]`
- `s` is guaranteed to be a valid encoded string
- All integers in `s` are in the range `[1, 300]`

## Hints

<details>
<summary>Hint 1</summary>

Use two stacks: one for counts and one for the current decoded string. When you encounter a digit, parse the full number.
</details>

<details>
<summary>Hint 2</summary>

When you see `[`, push the current count and current string onto stacks, then reset them. When you see `]`, pop from the stacks and append the repeated string.
</details>

<details>
<summary>Hint 3</summary>

You can also solve this recursively: parse the string from left to right, calling the function recursively when you encounter a `[`.
</details>
