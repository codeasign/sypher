# Longest Common Prefix — Solution

## Problem Overview

Given an array of strings, find the longest common prefix shared by all strings. Return an empty string if no common prefix exists.

## Approach 1: Horizontal Scanning

**Idea:** Start with the first string as the prefix candidate. For each subsequent string, shorten the prefix until it matches the start of that string.

**Algorithm:**
1. If the array is empty, return `""`
2. Set `prefix = strs[0]`
3. For each string `s` in `strs[1..]`:
   - While `s` does not start with `prefix`, shorten `prefix` by one character from the end
   - If `prefix` becomes empty, return `""`
4. Return `prefix`

**Complexity Analysis:**
- **Time:** O(n*m) where n = number of strings, m = length of shortest string
- **Space:** O(1) — only the prefix variable

**Why this works:** The common prefix of all strings must be a prefix of every string. By iteratively narrowing the candidate, we converge to the longest possible common prefix.

## Approach 2: Vertical Scanning (Optimal)

**Idea:** Compare characters column by column across all strings simultaneously.

**Algorithm:**
1. If the array is empty, return `""`
2. For each character index `i` from 0 to `len(strs[0]) - 1`:
   - Let `c = strs[0][i]`
   - For each string `s` in `strs[1..]`:
     - If `i >= len(s)` or `s[i] != c`, return `strs[0][0..i]`
3. Return `strs[0]` (all characters matched)

**Complexity:**
- **Time:** O(n*m) worst case, but stops early on short strings
- **Space:** O(1)

class Solution {
    public String longestCommonPrefix(String[] strs) {
        if (strs == null || strs.length == 0) return "";

        // Horizontal scanning: start with first string as prefix
        String prefix = strs[0];

        for (int i = 1; i < strs.length; i++) {
            // Shrink prefix until it matches the start of strs[i]
            while (strs[i].indexOf(prefix) != 0) {
                prefix = prefix.substring(0, prefix.length() - 1);
                if (prefix.isEmpty()) return "";
            }
        }
        return prefix;
    }
}

char* longestCommonPrefix(char** strs, int strsSize) {
    if (strsSize == 0) return strdup("");

    // Start with a copy of the first string
    char* prefix = strdup(strs[0]);
    int prefixLen = strlen(prefix);

    for (int i = 1; i < strsSize; i++) {
        // Shrink prefix until strs[i] starts with it
        while (strncmp(strs[i], prefix, prefixLen) != 0) {
            prefixLen--;           // Shorten by one character
            prefix[prefixLen] = '\0';  // Null-terminate
            if (prefixLen == 0) return prefix;
        }
    }
    return prefix;
}

## Key Takeaways

- **Horizontal scanning** is intuitive and uses the first string as the anchor
- The algorithm naturally degrades to `""` as soon as any string shares no prefix
- **Time is O(n*m)** but in practice stops early due to prefix shrinkage
- An alternative **vertical scanning** approach (comparing character-by-character across all strings) can exit even earlier when a short string is encountered
