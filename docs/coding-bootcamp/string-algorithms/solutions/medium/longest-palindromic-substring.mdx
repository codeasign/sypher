# Longest Palindromic Substring — Solution

## Problem Overview

Find the longest palindrome substring in a string. The **expand-around-center** approach is the most intuitive O(n^2) solution.

## Approach: Expand Around Center

**Idea:** Every palindrome mirrors around a center. For each possible center (n single-char + n-1 between-char = 2n-1), expand outward while the outer characters match. Track the longest palindrome found.

**Algorithm:**
1. Initialize `start = 0`, `max_len = 1`.
2. For each index `i` from 0 to n-1:
   - Expand around center `i` (odd length): set `l = i, r = i`. While `l >= 0` and `r < n` and `s[l] == s[r]`, expand. If `r-l+1 > max_len`, update start and max_len.
   - Expand around center between `i, i+1` (even length): set `l = i, r = i+1`. Same expansion check.
3. Return `s[start:start+max_len]`.

**Complexity Analysis:**
- **Time:** O(n^2) — n centers × up to n expansions each.
- **Space:** O(1) — constant extra space.

```python
def longest_palindrome(s: str) -> str:
    n = len(s)
    if n < 2:
        return s

    start = 0       # Start index of the longest palindrome found
    max_len = 1     # Length of the longest palindrome found

    def expand(l: int, r: int) -> None:
        nonlocal start, max_len
        # Expand outward while characters match
        while l >= 0 and r < n and s[l] == s[r]:
            l -= 1
            r += 1
        # After expansion, palindrome is s[l+1:r]
        length = r - l - 1
        if length > max_len:
            max_len = length
            start = l + 1

    for i in range(n):
        expand(i, i)      # Odd-length palindrome (single character center)
        expand(i, i + 1)  # Even-length palindrome (between characters)

    return s[start:start + max_len]
```

```java
class Solution {
    private int start = 0, maxLen = 1;

    public String longestPalindrome(String s) {
        int n = s.length();
        if (n < 2) return s;

        for (int i = 0; i < n; i++) {
            expand(s, i, i);      // Odd-length palindrome
            expand(s, i, i + 1);  // Even-length palindrome
        }

        return s.substring(start, start + maxLen);
    }

    private void expand(String s, int l, int r) {
        // Expand outward while characters match
        while (l >= 0 && r < s.length() && s.charAt(l) == s.charAt(r)) {
            l--;
            r++;
        }
        // After expansion, palindrome is s[l+1:r]
        int len = r - l - 1;
        if (len > maxLen) {
            maxLen = len;
            start = l + 1;
        }
    }
}
```

```cpp
class Solution {
public:
    string longestPalindrome(string s) {
        int n = s.length();
        if (n < 2) return s;

        int start = 0, maxLen = 1;

        for (int i = 0; i < n; i++) {
            expand(s, i, i, start, maxLen);      // Odd
            expand(s, i, i + 1, start, maxLen);  // Even
        }

        return s.substr(start, maxLen);
    }

    void expand(const string& s, int l, int r, int& start, int& maxLen) {
        while (l >= 0 && r < s.length() && s[l] == s[r]) {
            l--; r++;
        }
        int len = r - l - 1;
        if (len > maxLen) {
            maxLen = len;
            start = l + 1;
        }
    }
};
```

## Key Takeaways

- **Center expansion**: O(n^2) time, O(1) space — best for n  5000.
- **2n-1 centers**: Don't forget even-length palindromes (center between two chars).
- **Manacher's algorithm**: O(n) time advanced solution exists but is complex.
