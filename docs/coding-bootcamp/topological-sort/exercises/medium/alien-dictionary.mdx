# Alien Dictionary

**Difficulty:** Medium

There is a new alien language that uses the English alphabet. However, the order of letters is unknown. You are given a list of strings `words` from the alien language's dictionary. The words are **sorted lexicographically** according to the rules of this alien language.

Return a string of the unique letters in the alien language in the order they appear based on the given words. If there is a cycle or the order is invalid, return an empty string.

## Example 1

**Input:** `words = ["wrt", "wrf", "er", "ett", "rftt"]`

**Output:** `"wertf"`

**Explanation:**  
From `"wrt"` and `"wrf"`: `t` comes before `f`  
From `"wrt"` and `"er"`: `w` comes before `e`  
From `"er"` and `"ett"`: `r` comes before `t`  
From `"ett"` and `"rftt"`: `e` comes before `r`  
Order: `w → e → r → t → f`

## Example 2

**Input:** `words = ["z", "x"]`

**Output:** `"zx"`

**Explanation:** `z` comes before `x`.

## Example 3

**Input:** `words = ["z", "x", "z"]`

**Output:** `""`

**Explanation:** The order is inconsistent: `z < x` from first pair, `x < z` from second pair. This creates a cycle.

## Input Specification

The first line contains an integer `n`, the number of words. The next `n` lines each contain one word.

## Output Specification

Print the alien alphabet order as a single string (no spaces). If no valid order exists, print an empty line.

## Constraints

- `1 <= n <= 100`
- `1 <= len(words[i]) <= 100`
- Words consist only of lowercase English letters

## Hints

<details>
<summary>Hint 1</summary>

Compare adjacent words pair by pair. The first position where two words differ reveals a character ordering. This character pair becomes a directed edge in a graph.
</details>

<details>
<summary>Hint 2</summary>

Watch out for the prefix rule: if word A is a prefix of word B and A comes before B, no ordering information is gained. But if B is a prefix of A (the longer word comes first), the input is invalid.
</details>

<details>
<summary>Hint 3</summary>

Build a directed graph from the character ordering pairs, then perform topological sort (Kahn's algorithm) on it. If there's a cycle, return empty string.
</details>

</ProblemEditor>
