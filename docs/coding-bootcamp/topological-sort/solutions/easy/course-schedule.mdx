# Course Schedule — Solution

## Problem Overview

Given 
umCourses courses (0 to numCourses-1) and a list of prerequisites where prerequisites[i] = [a, b] means you must take course  before course , determine if it is possible to finish all courses. This is equivalent to detecting a cycle in a directed graph.

## Approach 1: Kahn's Algorithm (BFS)

**Idea:** Use Kahn's algorithm for topological sort. Compute in-degree for each vertex, repeatedly remove vertices with in-degree 0, and check if all vertices are processed.

**Algorithm:**
1. Build an adjacency list and compute in-degree array.
2. Initialize a queue with all courses that have in-degree 0.
3. While the queue is not empty:
   - Dequeue a course, increment a counter.
   - For each neighbor, decrement its in-degree. If it becomes 0, enqueue it.
4. If the counter equals numCourses, return true (no cycle). Otherwise, return false.

**Complexity Analysis:**
- **Time:** O(V + E) — we visit each vertex and edge once.
- **Space:** O(V + E) — adjacency list and in-degree array.

## Approach 2: DFS with Cycle Detection

**Idea:** Use DFS with three node states: unvisited (0), visiting (1), and visited (2). If we encounter a node in the "visiting" state during DFS, a cycle exists.

**Algorithm:**
1. Build an adjacency list.
2. For each course, if it's unvisited, run DFS.
3. Mark the current node as "visiting" (1).
4. For each neighbor:
   - If neighbor is "visiting" (1), a cycle exists — return false.
   - If neighbor is unvisited (0), recursively DFS.
5. Mark the current node as "visited" (2).
6. If all DFS calls complete without finding a cycle, return true.

**Complexity Analysis:**
- **Time:** O(V + E) — each vertex and edge is visited once.
- **Space:** O(V) — recursion stack and visited array.

from collections import deque

def can_finish(num_courses: int, prerequisites: list[list[int]]) -> bool:
    # Build adjacency list and in-degree array
    adj = [[] for _ in range(num_courses)]
    indegree = [0] * num_courses

    for a, b in prerequisites:
        adj[b].append(a)
        indegree[a] += 1

    # Initialize queue with courses having no prerequisites
    q = deque([i for i in range(num_courses) if indegree[i] == 0])
    count = 0

    # Process courses in topological order
    while q:
        course = q.popleft()
        count += 1
        for neighbor in adj[course]:
            indegree[neighbor] -= 1
            if indegree[neighbor] == 0:
                q.append(neighbor)

    # All courses processed -> no cycle
    return count == num_courses

import java.util.*;

class Solution {
    public boolean canFinish(int numCourses, int[][] prerequisites) {
        // Build adjacency list and in-degree array
        List<Integer>[] adj = new ArrayList[numCourses];
        int[] indegree = new int[numCourses];
        for (int i = 0; i < numCourses; i++) {
            adj[i] = new ArrayList<>();
        }
        for (int[] prereq : prerequisites) {
            int a = prereq[0], b = prereq[1];
            adj[b].add(a);
            indegree[a]++;
        }

        // Initialize queue with courses having no prerequisites
        Queue<Integer> q = new LinkedList<>();
        for (int i = 0; i < numCourses; i++) {
            if (indegree[i] == 0) q.offer(i);
        }

        int count = 0;
        while (!q.isEmpty()) {
            int course = q.poll();
            count++;
            for (int neighbor : adj[course]) {
                indegree[neighbor]--;
                if (indegree[neighbor] == 0) q.offer(neighbor);
            }
        }

        return count == numCourses;
    }
}

fn can_finish(num_courses: i32, prerequisites: Vec<Vec<i32>>) -> bool {
    let n = num_courses as usize;
    let mut adj = vec![vec![]; n];
    let mut indegree = vec![0; n];

    for prereq in &prerequisites {
        let a = prereq[0] as usize;
        let b = prereq[1] as usize;
        adj[b].push(a);
        indegree[a] += 1;
    }

    let mut q = VecDeque::new();
    for i in 0..n {
        if indegree[i] == 0 {
            q.push_back(i);
        }
    }

    let mut count = 0;
    while let Some(course) = q.pop_front() {
        count += 1;
        for &neighbor in &adj[course] {
            indegree[neighbor] -= 1;
            if indegree[neighbor] == 0 {
                q.push_back(neighbor);
            }
        }
    }

    count == n
}

            int course = q.Dequeue();
            count++;
            foreach (int neighbor in adj[course]) {
                indegree[neighbor]--;
                if (indegree[neighbor] == 0) q.Enqueue(neighbor);
            }
        }

        return count == numCourses;
    }
}

func canFinish(numCourses int, prerequisites [][]int) bool {
    // Build adjacency list and in-degree array
    adj := make([][]int, numCourses)
    indegree := make([]int, numCourses)
    for i := 0; i < numCourses; i++ {
        adj[i] = []int{}
    }
    for _, prereq := range prerequisites {
        a, b := prereq[0], prereq[1]
        adj[b] = append(adj[b], a)
        indegree[a]++
    }

    // Initialize queue with courses having no prerequisites
    q := make([]int, 0)
    for i := 0; i < numCourses; i++ {
        if indegree[i] == 0 {
            q = append(q, i)
        }
    }

    count := 0
    for len(q) > 0 {
        course := q[0]
        q = q[1:]
        count++
        for _, neighbor := range adj[course] {
            indegree[neighbor]--
            if indegree[neighbor] == 0 {
                q = append(q, neighbor)
            }
        }
    }

    return count == numCourses
}

## Key Takeaways

- **Cycle detection** in a directed graph is the core of this problem — topological sort processes all nodes only if there is no cycle.
- **Kahn's algorithm** (BFS with in-degree tracking) is intuitive and efficient at O(V+E).
- **DFS with state marking** (unvisited/visiting/visited) is an alternative approach that uses recursion and detects back edges.
- Both approaches are O(V+E) time and O(V+E) space, making them suitable for large inputs.
