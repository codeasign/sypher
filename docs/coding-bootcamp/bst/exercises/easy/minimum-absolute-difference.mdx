# Minimum Absolute Difference in BST

**Difficulty:** Easy

Given the `root` of a Binary Search Tree, return the **minimum absolute difference** between the values of any two different nodes in the tree.

## Example 1

**Input:**
```
3
4 2 6
```

The BST is:
```
    4
   / \
  2   6
```

**Output:** `2`
**Explanation:** The sorted values are `[2, 4, 6]`. The minimum absolute difference is `|4-2| = 2` or `|6-4| = 2`.

## Example 2

**Input:**
```
3
1 0 48
```

The BST is:
```
    1
   / \
  0   48
```

**Output:** `1`
**Explanation:** The sorted values are `[0, 1, 48]`. The minimum absolute difference is `|1-0| = 1`.

## Example 3

**Input:**
```
5
5 3 8 1 4
```

The BST is:
```
    5
   / \
  3   8
 / \
1   4
```

**Output:** `1`
**Explanation:** The sorted values are `[1, 3, 4, 5, 8]`. The minimum absolute difference is `|4-3| = 1`.

## Input Specification

- First line: an integer `n` — number of nodes in the BST (`1 <= n <= 10^4`)
- Second line: `n` space-separated integers representing the level-order traversal of the BST (use `-1` for null nodes)

## Output Specification

Print a single integer: the minimum absolute difference between any two node values. If the tree is empty, print `0`.

## Constraints

- The number of nodes is in the range `[0, 10^4]`
- `-10^5 <= Node.val <= 10^5`
- The input tree is a valid BST

## Hints

<details>
<summary>Hint 1</summary>

An in-order traversal of a BST visits nodes in sorted order. The minimum difference between any two nodes must be the minimum difference between adjacent values in the sorted order.
</details>

<details>
<summary>Hint 2</summary>

Use a recursive in-order traversal. Keep track of the previous node's value and compute the difference with each new node visited.
</details>

<details>
<summary>Hint 3</summary>

Initialize the minimum difference to a large value (like infinity or the maximum integer value). For a single-node tree, the answer is the maximum integer value (there is no pair to compare).
</details>
