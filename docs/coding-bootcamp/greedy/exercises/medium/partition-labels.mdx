# Partition Labels

You are given a string `s`. We want to partition the string into as many parts as possible so that each letter appears in at most one part.

Return a list of integers representing the size of these parts.

## Examples

**Example 1:**
```
Input: s = "ababcbacadefegdehijhklij"
Output: [9,7,8]
Explanation:
The partition is "ababcbaca", "defegde", "hijhklij".
This is a partition so that each letter appears in at most one part.
A partition like "ababcbacadefegde", "hijhklij" is incorrect because it splits s into fewer parts.
```

**Example 2:**
```
Input: s = "eccbbbbdec"
Output: [10]
Explanation:
All letters appear across the whole string, so only one partition is possible.
```

**Example 3:**
```
Input: s = "a"
Output: [1]
```

## Constraints

- `1 <= s.length <= 500`
- `s` consists of lowercase English letters.

## Hints

<details>
<summary>Hint 1</summary>
First, record the last occurrence position of each character in the string.
</details>

<details>
<summary>Hint 2</summary>
Traverse the string, extending the current segment's end as you encounter characters whose last occurrence is further ahead. When the current index reaches the segment's end, finalize the segment.
</details>
