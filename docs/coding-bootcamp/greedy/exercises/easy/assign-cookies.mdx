# Assign Cookies

Assume you are an awesome parent and want to give your children some cookies. But you should give each child at most one cookie.

Each child `i` has a greed factor `g[i]`, which is the minimum size of a cookie that the child will be content with. Each cookie `j` has a size `s[j]`. If `s[j] >= g[i]`, we can assign cookie `j` to child `i`, and the child will be content.

Your goal is to maximize the number of content children and output the maximum number.

## Examples

**Example 1:**
```
Input: g = [1,2,3], s = [1,1]
Output: 1
Explanation: You have 3 children and 2 cookies. The greed factors are 1, 2, 3.
Even though you have two cookies, both are size 1. You can only make the child
with greed factor 1 content. You need to give each child at most one cookie.
```

**Example 2:**
```
Input: g = [1,2], s = [1,2,3]
Output: 2
Explanation: You have 2 children and 3 cookies. The greed factors are 1 and 2.
You can make both children content by giving the cookie of size 1 to the child
with greed factor 1 and the cookie of size 2 to the child with greed factor 2.
```

**Example 3:**
```
Input: g = [10,9,8,7], s = [5,6,7,8]
Output: 2
Explanation: You can make the children with greed factors 7 and 8 content.
```

## Constraints

- `1 <= g.length <= 3 * 10^4`
- `0 <= s.length <= 3 * 10^4`
- `1 <= g[i], s[j] <= 2^31 - 1`
