# Two Pointers

The Two Pointers technique uses two references (pointers) into a data
structure — usually an array or string — to process elements in a single
pass instead of nested loops.

⏱ 20 minutes

## What You Will Learn

- What the Two Pointers pattern is and when to apply it
- Opposite-direction vs. same-direction pointer patterns
- How to recognize problems solvable with two pointers
- Complexity analysis and common pitfalls

## The Problem

Many array and string problems ask you to find a pair, reverse a
sequence, or check a property. The naive approach uses nested loops
(O(n²)). Two Pointers reduces this to O(n) by eliminating redundant
comparisons.

For example, checking if an array contains a pair summing to a target:
a nested loop compares every element with every other element. With
a sorted array and two pointers, each element is visited at most once.

## The Concept

Place one pointer at the start (left) and one at the end (right). Move
them toward each other (or in the same direction) based on a condition,
processing the array in one pass.

## Two Pointer Variants

### Opposite-direction (converging)

Left starts at index 0, right at index n-1. Move them toward each
other. Use cases:

- Checking palindromes
- Two Sum in a sorted array
- Trapping Rain Water (moving the shorter wall)

### Same-direction (sliding)

Both pointers start at index 0. Left trails behind right, defining a
window. Use cases:

- Removing duplicates from a sorted array
- Moving zeros
- Subarray problems

## Complexity

| Metric | Complexity |
|--------|-----------|
| Time   | O(n) — each element visited at most once per pointer |
| Space  | O(1) — only two extra variables |

## Worked Example: Valid Palindrome

Given: `"A man, a plan, a canal: Panama"`

```
left → 'A' (alphanumeric? yes)
right → 'a' (alphanumeric? yes)
Compare lowercase: 'a' === 'a' ✓ → move both

left → ' ' (skip, not alphanumeric)
left → 'm' (alphanumeric? yes)
right → 'm' (alphanumeric? yes)
Compare: 'm' === 'm' ✓ → move both

... continue until pointers meet ...
```

The string is a palindrome.

## When to Use Two Pointers

- The input is a **sorted** array (or sortable)
- You need to find a **pair** of elements satisfying a condition
- The problem involves **comparing elements from opposite ends**
- You are working **in-place** on the array
- The brute force would be O(n²)
