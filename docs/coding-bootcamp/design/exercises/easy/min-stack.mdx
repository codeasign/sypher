# Min Stack

**Difficulty:** Easy

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

Implement the `MinStack` class:

- `MinStack()` initializes the stack object.
- `void push(int val)` pushes the element `val` onto the stack.
- `void pop()` removes the element on the top of the stack.
- `int top()` gets the top element of the stack.
- `int getMin()` retrieves the minimum element in the stack.

You must implement a solution with O(1) time complexity for each function.

## Example 1

**Input:**
```
["MinStack","push","push","push","getMin","pop","top","getMin"]
[[],[-2],[0],[-3],[],[],[],[]]
```

**Output:**
```
[null,null,null,null,-3,null,0,-2]
```

**Explanation:**
```
MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin(); // return -3
minStack.pop();
minStack.top();    // return 0
minStack.getMin(); // return -2
```

## Example 2

**Input:**
```
push 1
push 2
getMin
top
pop
getMin
```

**Output:**
```
1
2
1
```

## Input Specification

The input consists of a sequence of commands, one per line:
- `push x` — push value x onto the stack
- `pop` — pop the top element
- `top` — output the top element
- `getMin` — output the minimum element

## Output Specification

For each `top` and `getMin` command, output the result on its own line.

## Constraints

- `-2^31 <= val <= 2^31 - 1`
- Methods `pop`, `top` and `getMin` operations will always be called on non-empty stacks.
- At most `3 * 10^4` calls will be made to `push`, `pop`, `top`, and `getMin`.

## Hints

<details>
<summary>Hint 1</summary>

Use an auxiliary stack to track the minimum element at each state. When pushing a new value, compare it with the current minimum and push the smaller one.
</details>

<details>
<summary>Hint 2</summary>

You can store pairs (value, currentMin) in a single stack instead of using two separate stacks.
</details>

<details>
<summary>Hint 3</summary>

For a space-optimized solution, use a single variable to track the minimum and encode the difference when pushing smaller values.
</details>
