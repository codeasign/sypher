# Design Parking System

**Difficulty:** Easy

Design a parking system for a parking lot that has three types of parking spaces: big, medium, and small.

Implement the `ParkingSystem` class:

- `ParkingSystem(int big, int medium, int small)` — Initializes the object with the number of available spaces for each type.
- `bool addCar(int carType)` — Checks whether there is a parking space of the given type. `carType` is `1` for big, `2` for medium, and `3` for small. A car can only park in a space of its own type. Returns `true` if there is space available, `false` otherwise, and decrements the available count.

## Example 1

**Input:**
```
ParkingSystem(1, 1, 0)
addCar(1) → true
addCar(2) → true
addCar(3) → false
addCar(1) → false
```
**Explanation:** There is 1 big space, 1 medium space, 0 small spaces. After the big car parks, there are no more big spaces.

## Example 2

**Input:**
```
ParkingSystem(0, 0, 0)
addCar(1) → false
addCar(2) → false
addCar(3) → false
```
**Explanation:** No spaces at all, every call returns false.

## Input Specification

The first line contains three space-separated integers: `big medium small` (the initial capacity of each space type).

The second line contains space-separated integers representing a sequence of `addCar` calls, where each integer is the car type (1, 2, or 3).

## Output Specification

Print a single line of space-separated `true` or `false` values, one for each `addCar` call in order.

## Constraints

- `0 <= big, medium, small <= 1000`
- `carType` is `1`, `2`, or `3`
- At most `1000` calls will be made to `addCar`

## Hints

<details>
<summary>Hint 1</summary>

Store the counts for each parking space type. You need three counters — one per car type.

</details>

<details>
<summary>Hint 2</summary>

When `addCar(carType)` is called, check if the count for that type is greater than zero. If yes, decrement and return `true`; otherwise return `false`.

</details>

<details>
<summary>Hint 3</summary>

You can use an array or a map indexed by car type (1, 2, 3) to avoid a chain of if/else statements.

</details>
