# Design Browser History — Solution

## Problem Overview

Design a browser history system that supports visiting URLs, going back steps, and going forward steps. The challenge is that visiting a new URL should clear all forward history, and back/forward operations should clamp at the boundaries.

## Approach 1: Two Stacks (Back and Forward)

**Idea:** Maintain two stacks — one for back history and one for forward history. The current page is always the top of the back stack.

**Algorithm:**
1. `BrowserHistory(homepage)`: Push the homepage onto the back stack. The forward stack starts empty.
2. `visit(url)`:
   - Push the current page onto the back stack.
   - Clear the forward stack (visiting a new URL invalidates forward history).
3. `back(steps)`:
   - For each step: pop from the back stack and push the popped page onto the forward stack.
   - Stop when we've done `steps` steps or the back stack has only one element left (the homepage).
   - Return the current page (top of back stack).
4. `forward(steps)`:
   - For each step: pop from the forward stack and push onto the back stack.
   - Stop when we've done `steps` steps or the forward stack is empty.
   - Return the current page (top of back stack).

**Complexity Analysis:**
- **Time:** O(1) for `visit`, O(steps) for `back` and `forward` (or O(1) amortized)
- **Space:** O(n) where n is the total number of pages visited

## Approach 2: Doubly Linked List

**Idea:** Use a doubly linked list where each node represents a page. Maintain a pointer to the current page. This gives O(1) back/forward operations.

**Algorithm:**
1. Create a list node for the homepage. Set `current` to this node.
2. `visit(url)`: Create a new node, set it as `current.next`, update `current` to the new node. Any nodes after `current` (forward history) are discarded.
3. `back(steps)`: Follow `current.prev` pointers up to `steps` times.
4. `forward(steps)`: Follow `current.next` pointers up to `steps` times.

**Complexity Analysis:**
- **Time:** O(steps) for back/forward (or O(1) amortized). O(1) for visit.
- **Space:** O(n) where n is the total number of pages visited.

import java.util.Stack;

class BrowserHistory {
    // Stack for back navigation (current page is at the top)
    private Stack<String> backStack;
    // Stack for forward navigation
    private Stack<String> forwardStack;

    public BrowserHistory(String homepage) {
        backStack = new Stack<>();
        forwardStack = new Stack<>();
        backStack.push(homepage); // Initialize with homepage
    }

    public void visit(String url) {
        // Push the current page is implicitly on top of backStack.
        // We push the new URL as the current page.
        backStack.push(url);
        // Visiting a new URL clears all forward history
        forwardStack.clear();
    }

    public String back(int steps) {
        // Move back by moving pages from back stack to forward stack
        // We need at least 1 page in back stack (the current page)
        while (steps > 0 && backStack.size() > 1) {
            forwardStack.push(backStack.pop());
            steps--;
        }
        return backStack.peek(); // Return the current page
    }

    public String forward(int steps) {
        // Move forward by moving pages from forward stack to back stack
        while (steps > 0 && !forwardStack.isEmpty()) {
            backStack.push(forwardStack.pop());
            steps--;
        }
        return backStack.peek(); // Return the current page
    }
}

class BrowserHistory:
    def __init__(self, homepage: str):
        # Stack for back history (current page is at the top)
        self.back_stack = [homepage]
        # Stack for forward history
        self.forward_stack = []

    def visit(self, url: str) -> None:
        """Visit a new URL, clearing forward history"""
        self.back_stack.append(url)
        # Visiting a new URL invalidates forward history
        self.forward_stack.clear()

        while steps > 0 and len(self.back_stack) > 1:
            self.forward_stack.append(self.back_stack.pop())
            steps -= 1
        return self.back_stack[-1]

        # Move pages from forward stack to back stack

            self.back_stack.append(self.forward_stack.pop())
            steps -= 1
        return self.back_stack[-1]

struct BrowserHistory {
    // Stack for back history (current page is at the end)
    back_stack: Vec<String>,
    // Stack for forward history
    forward_stack: Vec<String>,

impl BrowserHistory {
    fn new(homepage: String) -> Self {
        BrowserHistory {
            back_stack: vec![homepage],
            forward_stack: Vec::new(),
        }
    }

    fn visit(&mut self, url: String) {
        // Push new URL, clear forward history
        self.back_stack.push(url);
        self.forward_stack.clear();
    }

    fn back(&mut self, steps: i32) -> String {
        let mut steps = steps;
        // Move pages from back to forward stack
        while steps > 0 && self.back_stack.len() > 1 {
            self.forward_stack.push(self.back_stack.pop().unwrap());
            steps -= 1;
        }
        self.back_stack.last().unwrap().clone()
    }

    fn forward(&mut self, steps: i32) -> String {
        let mut steps = steps;
        // Move pages from forward to back stack
        while steps > 0 && !self.forward_stack.is_empty() {
            self.back_stack.push(self.forward_stack.pop().unwrap());
            steps -= 1;
        }
        self.back_stack.last().unwrap().clone()
    }
}

#include <stdlib.h>
#include <string.h>
#include <stdio.h>

#define MAX_URL 100
#define MAX_HISTORY 5000

typedef struct {
    char backStack[MAX_HISTORY][MAX_URL]; // History for going back
    int backTop;                           // Top index of back stack
    char forwardStack[MAX_HISTORY][MAX_URL]; // History for going forward
    int forwardTop;                        // Top index of forward stack
} BrowserHistory;

BrowserHistory* browserHistoryCreate(char* homepage) {
    BrowserHistory* obj = (BrowserHistory*)malloc(sizeof(BrowserHistory));
    obj->backTop = 0;
    obj->forwardTop = -1;
    strcpy(obj->backStack[0], homepage);
    return obj;
}

void browserHistoryVisit(BrowserHistory* obj, char* url) {
    // Push the new URL onto the back stack
    obj->backTop++;
    strcpy(obj->backStack[obj->backTop], url);
    // Clear forward history
    obj->forwardTop = -1;
}

char* browserHistoryBack(BrowserHistory* obj, int steps) {
    // Move back by transferring pages to forward stack
    while (steps > 0 && obj->backTop > 0) {
        obj->forwardTop++;
        strcpy(obj->forwardStack[obj->forwardTop], obj->backStack[obj->backTop]);
        obj->backTop--;
        steps--;
    }
    return obj->backStack[obj->backTop];
}

char* browserHistoryForward(BrowserHistory* obj, int steps) {
    // Move forward by transferring pages from forward stack
    while (steps > 0 && obj->forwardTop >= 0) {
        obj->backTop++;
        strcpy(obj->backStack[obj->backTop], obj->forwardStack[obj->forwardTop]);
        obj->forwardTop--;
        steps--;
    }
    return obj->backStack[obj->backTop];
}

void browserHistoryFree(BrowserHistory* obj) {
    free(obj);
}

type BrowserHistory struct {
    backStack    []string // Stack for back navigation
    forwardStack []string // Stack for forward navigation
}

func Constructor(homepage string) BrowserHistory {
    return BrowserHistory{
        backStack:    []string{homepage},
        forwardStack: []string{},
    }
}

func (this *BrowserHistory) Visit(url string) {
    // Push the new URL and clear forward history
    this.backStack = append(this.backStack, url)
    this.forwardStack = []string{} // Clear forward history
}

func (this *BrowserHistory) Back(steps int) string {
    // Move pages from back stack to forward stack
    for steps > 0 && len(this.backStack) > 1 {
        n := len(this.backStack) - 1
        this.forwardStack = append(this.forwardStack, this.backStack[n])
        this.backStack = this.backStack[:n]
        steps--
    }
    return this.backStack[len(this.backStack)-1]
}

func (this *BrowserHistory) Forward(steps int) string {
    // Move pages from forward stack to back stack
    for steps > 0 && len(this.forwardStack) > 0 {
        n := len(this.forwardStack) - 1
        this.backStack = append(this.backStack, this.forwardStack[n])
        this.forwardStack = this.forwardStack[:n]
        steps--
    }
    return this.backStack[len(this.backStack)-1]
}

## Key Takeaways

- **Two stacks** is the simplest and most intuitive approach for browser history
- The **current page is always the top of the back stack** — this is the key invariant
- Visiting a new URL **clears forward history** because the forward path is no longer valid
- The `back` operation **moves pages from back to forward stack** (not discarding them)
- The `forward` operation **moves pages from forward to back stack** (restoring them)
- Boundary clamping is automatic — we stop when `backStack.size() > 1` (keeping at least the homepage) or when forward stack is empty
