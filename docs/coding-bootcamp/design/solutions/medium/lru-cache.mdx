# LRU Cache — Solution

## Problem Overview

Design an LRU (Least Recently Used) cache with O(1) get and put operations. When the cache reaches capacity, evict the least recently used item. The challenge is maintaining both fast lookups and a precise ordering of usage.

## Approach 1: HashMap + Doubly Linked List

**Idea:** Use a hash map for O(1) key lookup and a doubly linked list for O(1) reordering. The list keeps items in order of use: the head is the most recently used, the tail is the least recently used.

**Algorithm:**
1. Create a doubly linked list with dummy head and tail sentinel nodes. The head side is "most recently used" and the tail side is "least recently used".
2. Create a hash map mapping keys to their corresponding list nodes.
3. `get(key)`:
   - If key exists in the map: remove the node from its current position and add it to the head. Return the value.
   - If key does not exist: return -1.
4. `put(key, value)`:
   - If key exists: update the node's value, move it to the head.
   - If key does not exist: create a new node, add it to the head, add it to the map. If the size exceeds capacity, remove the tail node and delete it from the map.

**Helper methods:**
- `remove(node)`: Detach a node from the linked list by connecting its prev and next neighbors.
- `addToHead(node)`: Insert a node right after the dummy head.

**Complexity Analysis:**
- **Time:** O(1) for both get and put — hash map lookups are O(1), and linked list operations (removing a known node, adding to head) are O(1) with a doubly linked list.
- **Space:** O(capacity) — the hash map and linked list each store up to capacity nodes.

## Approach 2: Built-in LinkedHashMap / OrderedDict

Many languages provide a built-in ordered map that combines hash map and linked list behavior:
- Java: `LinkedHashMap` with access-order=true
- Python: `collections.OrderedDict` with `move_to_end()`

These are valid in an interview but you should understand the underlying implementation.

```java
import java.util.HashMap;
import java.util.Map;

class LRUCache {
    // Node class for the doubly linked list
    private static class Node {
        int key, value;
        Node prev, next;

        Node(int key, int value) {
            this.key = key;
            this.value = value;
        }
    }

    private final int capacity;
    private final Map<Integer, Node> cache; // Key → Node mapping
    private final Node head, tail; // Dummy head (MRU) and tail (LRU) sentinels

    public LRUCache(int capacity) {
        this.capacity = capacity;
        this.cache = new HashMap<>();
        // Initialize dummy sentinel nodes
        head = new Node(0, 0);
        tail = new Node(0, 0);
        head.next = tail;
        tail.prev = head;
    }

    public int get(int key) {
        Node node = cache.get(key);
        if (node == null) {
            return -1; // Key not found
        }
        // Move the accessed node to the head (most recently used)
        remove(node);
        addToHead(node);
        return node.value;
    }

    public void put(int key, int value) {
        Node node = cache.get(key);
        if (node != null) {
            // Key exists, update value and move to head
            node.value = value;
            remove(node);
            addToHead(node);
        } else {
            // Key does not exist, create new node
            node = new Node(key, value);
            cache.put(key, node);
            addToHead(node);

            // Evict LRU item if capacity exceeded
            if (cache.size() > capacity) {
                Node lru = tail.prev; // The node just before the tail dummy
                remove(lru);
                cache.remove(lru.key);
            }
        }
    }

    // Remove a node from the linked list
    private void remove(Node node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    // Add a node right after the head dummy (most recently used position)
    private void addToHead(Node node) {
        node.next = head.next;
        node.prev = head;
        head.next.prev = node;
        head.next = node;
    }
}
```

```cpp
class LRUCache {
private:
    // Node class for the doubly linked list
    struct Node {
        int key, value;
        Node* prev;
        Node* next;
        Node(int k, int v) : key(k), value(v), prev(nullptr), next(nullptr) {}
    };

    int capacity;
    unordered_map<int, Node*> cache; // Key → Node mapping
    Node* head; // Dummy head (most recently used side)
    Node* tail; // Dummy tail (least recently used side)

    // Remove a node from the linked list
    void remove(Node* node) {
        node->prev->next = node->next;
        node->next->prev = node->prev;
    }

    // Add a node right after the head (MRU position)
    void addToHead(Node* node) {
        node->next = head->next;
        node->prev = head;
        head->next->prev = node;
        head->next = node;
    }

public:
    LRUCache(int capacity) : capacity(capacity) {
        head = new Node(0, 0);
        tail = new Node(0, 0);
        head->next = tail;
        tail->prev = head;
    }

    int get(int key) {
        if (!cache.count(key)) {
            return -1;
        }
        Node* node = cache[key];
        // Move accessed node to head (most recently used)
        remove(node);
        addToHead(node);
        return node->value;
    }

    void put(int key, int value) {
        if (cache.count(key)) {
            // Key exists, update value and move to head
            Node* node = cache[key];
            node->value = value;
            remove(node);
            addToHead(node);
        } else {
            // Create new node, add to cache and list head
            Node* node = new Node(key, value);
            cache[key] = node;
            addToHead(node);

            // Evict LRU if over capacity
            if (cache.size() > capacity) {
                Node* lru = tail->prev;
                remove(lru);
                cache.erase(lru->key);
                delete lru;
            }
        }
    }
};
```

```typescript
            this.remove(node);

                this.remove(lru);
                this.cache.delete(lru.key);
            }
        }
    }
```

```javascript
class LRUCache {
    constructor(capacity) {
        this.capacity = capacity;
        this.cache = new Map(); // JavaScript Map preserves insertion order
        // Note: Using Map's natural ordering is simpler but less explicit
        // than a manual doubly linked list. Map maintains insertion order
        // which we can leverage for LRU ordering.
    }

    get(key) {
        if (!this.cache.has(key)) return -1;
        // Move the key to the end (most recently used) by re-inserting
        const value = this.cache.get(key);
        this.cache.delete(key);
        this.cache.set(key, value);
        return value;
    }

    put(key, value) {
        if (this.cache.has(key)) {
            this.cache.delete(key); // Remove existing entry
        } else if (this.cache.size >= this.capacity) {
            // Evict the least recently used (first inserted) key
            const lruKey = this.cache.keys().next().value;
            this.cache.delete(lruKey);
        }
        this.cache.set(key, value); // Insert at end (most recently used)
    }
}
```

```python
class ListNode:
    """Doubly linked list node"""
    def __init__(self, key=0, value=0):
        self.key = key
        self.value = value
        self.prev = None
        self.next = None


class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = {}  # Dictionary mapping keys to list nodes

        # Dummy head and tail sentinels
        self.head = ListNode()  # Most recently used side
        self.tail = ListNode()  # Least recently used side
        self.head.next = self.tail
        self.tail.prev = self.head

    def _remove(self, node: ListNode) -> None:
        """Remove a node from the linked list"""
        node.prev.next = node.next
        node.next.prev = node.prev

    def _add_to_head(self, node: ListNode) -> None:
        """Add a node right after the head (most recently used position)"""
        node.next = self.head.next
        node.prev = self.head
        self.head.next.prev = node
        self.head.next = node

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        node = self.cache[key]
        # Move the accessed node to the head (mark as recently used)
        self._remove(node)
        self._add_to_head(node)
        return node.value

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            # Key exists, update value and move to head
            node = self.cache[key]
            node.value = value
            self._remove(node)
            self._add_to_head(node)
```

```rust
use std::collections::HashMap;

struct LRUCache {
    capacity: i32,
    cache: HashMap<i32, (i32, i32)>, // key -> (value, timestamp)
    timestamp: i32, // Global monotonic clock for recency tracking
}

impl LRUCache {
    fn new(capacity: i32) -> Self {
        LRUCache {
            capacity,
            cache: HashMap::new(),
            timestamp: 0,
        }
    }

    fn get(&mut self, key: i32) -> i32 {
        if let Some((value, _)) = self.cache.get_mut(&key) {
            // Update the timestamp to mark as recently used
            self.timestamp += 1;
            *self.cache.get_mut(&key).unwrap() = (*value, self.timestamp);
            *value
        } else {
            -1
        }
    }

    fn put(&mut self, key: i32, value: i32) {
        self.timestamp += 1;
        // If key exists, update it
        if self.cache.contains_key(&key) {
            self.cache.insert(key, (value, self.timestamp));
            return;
        }
        // If at capacity, evict the LRU item (oldest timestamp)
        if self.cache.len() as i32 >= self.capacity {
            let lru_key = self.cache
                .iter()
                .min_by_key(|(_, &(_, ts))| ts)
                .map(|(k, _)| *k)
                .unwrap();
            self.cache.remove(&lru_key);
        }
        self.cache.insert(key, (value, self.timestamp));
    }
}
```

```c
#include <stdlib.h>
#include <stdio.h>

// Doubly linked list node
typedef struct Node {
    int key;
    int value;
    struct Node* prev;
    struct Node* next;
} Node;

// Hash map entry for chaining
typedef struct HashEntry {
    int key;
    Node* node;
    struct HashEntry* next;
} HashEntry;

typedef struct {
    int capacity;
    int size;
    Node* head; // Dummy head (MRU side)
    Node* tail; // Dummy tail (LRU side)
    HashEntry** hashTable; // Hash map for O(1) key lookup
} LRUCache;

#define HASH_SIZE 10007

// Simple hash function
int hash(int key) {
    return key % HASH_SIZE;
}

LRUCache* lRUCacheCreate(int capacity) {
    LRUCache* obj = (LRUCache*)malloc(sizeof(LRUCache));
    obj->capacity = capacity;
    obj->size = 0;
    // Initialize sentinel nodes
    obj->head = (Node*)malloc(sizeof(Node));
    obj->tail = (Node*)malloc(sizeof(Node));
    obj->head->next = obj->tail;
    obj->tail->prev = obj->head;
    // Initialize hash table
    obj->hashTable = (HashEntry**)calloc(HASH_SIZE, sizeof(HashEntry*));
    return obj;
}

// Find a node by key in the hash table
Node* findNode(LRUCache* obj, int key) {
    int idx = hash(key);
    HashEntry* entry = obj->hashTable[idx];
    while (entry) {
        if (entry->key == key) return entry->node;
        entry = entry->next;
    }
    return NULL;
}

// Insert a key-node mapping into the hash table
void insertHash(LRUCache* obj, int key, Node* node) {
    int idx = hash(key);
    HashEntry* entry = (HashEntry*)malloc(sizeof(HashEntry));
    entry->key = key;
    entry->node = node;
    entry->next = obj->hashTable[idx];
    obj->hashTable[idx] = entry;
}

// Remove a key from the hash table
void removeHash(LRUCache* obj, int key) {
    int idx = hash(key);
    HashEntry* curr = obj->hashTable[idx];
    HashEntry* prev = NULL;
    while (curr) {
        if (curr->key == key) {
            if (prev == NULL) obj->hashTable[idx] = curr->next;
            else prev->next = curr->next;
            free(curr);
            return;
        }
        prev = curr;
        curr = curr->next;
    }
}

// Remove a node from the linked list
void removeNode(Node* node) {
    node->prev->next = node->next;
    node->next->prev = node->prev;
}

// Add a node right after the head (MRU position)
void addToHead(LRUCache* obj, Node* node) {
    node->next = obj->head->next;
    node->prev = obj->head;
    obj->head->next->prev = node;
    obj->head->next = node;
}

int lRUCacheGet(LRUCache* obj, int key) {
    Node* node = findNode(obj, key);
    if (!node) return -1;
    // Move to head (mark as recently used)
    removeNode(node);
    addToHead(obj, node);
    return node->value;
}

void lRUCachePut(LRUCache* obj, int key, int value) {
    Node* node = findNode(obj, key);
    if (node) {
        // Update existing node and move to head
        node->value = value;
        removeNode(node);
        addToHead(obj, node);
    } else {
        // Create new node
        Node* newNode = (Node*)malloc(sizeof(Node));
        newNode->key = key;
        newNode->value = value;
        insertHash(obj, key, newNode);
        addToHead(obj, newNode);
        obj->size++;
        // Evict LRU if over capacity
        if (obj->size > obj->capacity) {
            Node* lru = obj->tail->prev;
            removeNode(lru);
            removeHash(obj, lru->key);
            free(lru);
            obj->size--;
        }
    }
}

void lRUCacheFree(LRUCache* obj) {
    // Free all nodes
    Node* curr = obj->head;
    while (curr) {
        Node* temp = curr;
        curr = curr->next;
        free(temp);
    }
    // Free hash table entries
    for (int i = 0; i < HASH_SIZE; i++) {
        HashEntry* entry = obj->hashTable[i];
        while (entry) {
            HashEntry* temp = entry;
            entry = entry->next;
            free(temp);
        }
    }
    free(obj->hashTable);
    free(obj);
}
```

## Key Takeaways

- **HashMap + doubly linked list** is the classic LRU Cache design pattern
- **Sentinel nodes** (dummy head/tail) eliminate null checks and simplify edge cases
- The hash map provides O(1) lookups; the linked list provides O(1) reordering
- When a key is accessed, it must be **moved to the head** — this is the "recently used" signal
- The **tail always holds the LRU item**, making eviction O(1)
- This pattern extends to other design problems like LFU Cache
