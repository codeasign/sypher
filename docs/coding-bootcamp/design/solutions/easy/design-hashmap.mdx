# Design HashMap — Solution

## Problem Overview

Design a HashMap without using built-in hash table libraries. The HashMap must support three operations: put, get, and remove. The key challenge is handling collisions efficiently while maintaining O(1) average time complexity.

## Approach 1: Array of Buckets with Chaining

**Idea:** Use a large array where each element is a bucket (linked list) that holds key-value pairs. Use a hash function to determine which bucket a key belongs to. Handle collisions by adding multiple entries to the same bucket.

**Algorithm:**
1. Initialize an array of size `capacity` (a prime number like 2069 to reduce collisions). Each bucket is an empty list.
2. `put(key, value)`: Compute `index = key % capacity`. Search the bucket at that index for the key. If found, update its value. If not, append a new (key, value) pair.
3. `get(key)`: Compute `index = key % capacity`. Search the bucket for the key. Return the value if found, -1 otherwise.
4. `remove(key)`: Compute `index = key % capacity`. Search the bucket for the key and remove it if found.

**Complexity Analysis:**
- **Time:** O(n/k) average where n is the number of keys and k is the number of buckets. With a good hash function and sufficient capacity, this is approximately O(1).
- **Space:** O(n + k) where n is the number of key-value pairs and k is the number of buckets.

## Approach 2: Open Addressing with Linear Probing

**Idea:** Store all entries directly in the array. When a collision occurs, find the next available slot by probing.

**Algorithm:**
1. Initialize an array of pairs. Use a marker like a sentinel to distinguish empty slots from occupied ones.
2. `put(key, value)`: Compute the initial index. If the slot is occupied by a different key, probe linearly until finding an empty slot or the target key.
3. `get(key)`: Compute the initial index. Probe until finding the key or hitting an empty slot.
4. `remove(key)`: Use lazy deletion (mark the slot as "deleted" rather than truly empty) to avoid breaking the probe chain.

**Complexity Analysis:**
- **Time:** O(1) average, O(n) worst case with poor hash distribution
- **Space:** O(capacity) — pre-allocated array

Chaining is generally preferred for its simplicity and predictable performance.

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;

class MyHashMap {
    // Bucket class using a linked list of (key, value) pairs
    private static class Bucket {
        private List<int[]> pairs; // Each pair is [key, value]

        Bucket() {
            pairs = new LinkedList<>();
        }

        void put(int key, int value) {
            // Search for existing key and update, or append new pair
            for (int[] pair : pairs) {
                if (pair[0] == key) {
                    pair[1] = value; // Update existing value
                    return;
                }
            }
            pairs.add(new int[]{key, value}); // Insert new key-value pair
        }

        int get(int key) {
            for (int[] pair : pairs) {
                if (pair[0] == key) {
                    return pair[1];
                }
            }
            return -1;
        }

        void remove(int key) {
            pairs.removeIf(pair -> pair[0] == key);
        }
    }

    private Bucket[] buckets;
    private static final int CAPACITY = 2069; // Prime number to reduce collisions

    public MyHashMap() {
        buckets = new Bucket[CAPACITY];
        for (int i = 0; i < CAPACITY; i++) {
            buckets[i] = new Bucket();
        }
    }

    // Simple hash function: key modulo capacity
    private int hash(int key) {
        return key % CAPACITY;
    }

    public void put(int key, int value) {
        buckets[hash(key)].put(key, value);
    }

    public int get(int key) {
        return buckets[hash(key)].get(key);
    }

    public void remove(int key) {
        buckets[hash(key)].remove(key);
    }
}
```

```cpp
#include <list>
#include <vector>

class MyHashMap {
private:
    // Each bucket is a list of (key, value) pairs
    vector<list<pair<int, int>>> buckets;
    static const int CAPACITY = 2069; // Prime capacity for better distribution

    // Hash function: key modulo capacity
    int hash(int key) {
        return key % CAPACITY;
    }

public:
    MyHashMap() : buckets(CAPACITY) {}

    void put(int key, int value) {
        int index = hash(key);
        // Search for existing key in the bucket
        for (auto& pair : buckets[index]) {
            if (pair.first == key) {
                pair.second = value; // Update existing value
                return;
            }
        }
        // Key not found, append new pair
        buckets[index].push_back({key, value});
    }

    int get(int key) {
        int index = hash(key);
        // Search for the key in its bucket
        for (const auto& pair : buckets[index]) {
            if (pair.first == key) {
                return pair.second;
            }
        }
        return -1; // Key not found
    }

    void remove(int key) {
        int index = hash(key);
        // Remove the key-value pair if it exists
        auto& bucket = buckets[index];
        bucket.remove_if([key](const pair<int, int>& p) {
            return p.first == key;
        });
    }
};
```

```python
class MyHashMap:
    def __init__(self):
        # Use a prime capacity to reduce collisions
        self.capacity = 2069
        # Each bucket is a list of (key, value) pairs
        self.buckets = [[] for _ in range(self.capacity)]

    def _hash(self, key: int) -> int:
        # Simple hash function: key modulo capacity
        return key % self.capacity

    def put(self, key: int, value: int) -> None:
        index = self._hash(key)
        bucket = self.buckets[index]
        # Search for existing key and update, or append new pair
        for i, (k, v) in enumerate(bucket):
            if k == key:
                bucket[i] = (key, value)  # Update existing value
                return
        bucket.append((key, value))  # Insert new key-value pair

    def get(self, key: int) -> int:
        index = self._hash(key)
        bucket = self.buckets[index]
        # Search for the key in its bucket
        for k, v in bucket:
            if k == key:
                return v
        return -1  # Key not found

    def remove(self, key: int) -> None:
        index = self._hash(key)
        bucket = self.buckets[index]
        # Filter out the pair with matching key
        self.buckets[index] = [(k, v) for k, v in bucket if k != key]
```

```rust
struct MyHashMap {
    // Vector of buckets, each bucket is a vector of (key, value) pairs
    buckets: Vec<Vec<(i32, i32)>>,
    capacity: i32,
}

impl MyHashMap {
    fn new() -> Self {
        let cap = 2069; // Prime capacity
        MyHashMap {
            buckets: vec![Vec::new(); cap as usize],
            capacity: cap,
        }
    }

    fn hash(&self, key: i32) -> usize {
        (key % self.capacity) as usize
    }

    fn put(&mut self, key: i32, value: i32) {
        let idx = self.hash(key);
        // Search for existing key and update, or add new pair
        let bucket = &mut self.buckets[idx];
        for pair in bucket.iter_mut() {
            if pair.0 == key {
                pair.1 = value; // Update existing value
                return;
            }
        }
        bucket.push((key, value)); // Insert new key-value pair
    }

    fn get(&self, key: i32) -> i32 {
        let idx = self.hash(key);
        let bucket = &self.buckets[idx];
        for &(k, v) in bucket {
            if k == key {
                return v;
            }
        }
        -1 // Key not found
    }

    fn remove(&mut self, key: i32) {
        let idx = self.hash(key);
        let bucket = &mut self.buckets[idx];
        // Remove the pair with matching key
        bucket.retain(|&(k, _)| k != key);
    }
}
```

```c
#include <stdlib.h>
#include <stdio.h>

#define CAPACITY 2069

// Node struct for the linked list in each bucket
typedef struct Node {
    int key;
    int value;
    struct Node* next;
} Node;

typedef struct {
    Node* buckets[CAPACITY];
} MyHashMap;

MyHashMap* myHashMapCreate() {
    MyHashMap* obj = (MyHashMap*)calloc(1, sizeof(MyHashMap));
    // buckets are zero-initialized (NULL) by calloc
    return obj;
}

int hash(int key) {
    return key % CAPACITY;
}

void myHashMapPut(MyHashMap* obj, int key, int value) {
    int index = hash(key);
    Node* curr = obj->buckets[index];
    // Search for existing key
    while (curr != NULL) {
        if (curr->key == key) {
            curr->value = value; // Update existing value
            return;
        }
        curr = curr->next;
    }
    // Key not found, insert at the head of the list
    Node* newNode = (Node*)malloc(sizeof(Node));
    newNode->key = key;
    newNode->value = value;
    newNode->next = obj->buckets[index];
    obj->buckets[index] = newNode;
}

int myHashMapGet(MyHashMap* obj, int key) {
    int index = hash(key);
    Node* curr = obj->buckets[index];
    // Search for the key in its bucket
    while (curr != NULL) {
        if (curr->key == key) {
            return curr->value;
        }
        curr = curr->next;
    }
    return -1; // Key not found
}

void myHashMapRemove(MyHashMap* obj, int key) {
    int index = hash(key);
    Node* curr = obj->buckets[index];
    Node* prev = NULL;
    // Search for the key
    while (curr != NULL) {
        if (curr->key == key) {
            if (prev == NULL) {
                obj->buckets[index] = curr->next; // Remove head
            } else {
                prev->next = curr->next; // Bypass the node
            }
            free(curr);
            return;
        }
        prev = curr;
        curr = curr->next;
    }
}

void myHashMapFree(MyHashMap* obj) {
    for (int i = 0; i < CAPACITY; i++) {
        Node* curr = obj->buckets[i];
        while (curr != NULL) {
            Node* temp = curr;
            curr = curr->next;
            free(temp);
        }
    }
    free(obj);
}
```

```csharp
using System.Collections.Generic;

public class MyHashMap {
    private static readonly int CAPACITY = 2069;
    // Each bucket is a LinkedList of (key, value) pairs
    private LinkedList<(int key, int value)>[] buckets;

    public MyHashMap() {
        buckets = new LinkedList<(int, int)>[CAPACITY];
        for (int i = 0; i < CAPACITY; i++) {
            buckets[i] = new LinkedList<(int, int)>();
        }
    }

    private int Hash(int key) {
        return key % CAPACITY;
    }

    public void Put(int key, int value) {
        int index = Hash(key);
        var bucket = buckets[index];
        // Search for existing key
        for (var node = bucket.First; node != null; node = node.Next) {
            if (node.Value.key == key) {
                // Update existing value (remove and re-add)
                bucket.Remove(node);
                bucket.AddLast((key, value));
                return;
            }
        }
        // Key not found, add new pair
        bucket.AddLast((key, value));
    }

    public int Get(int key) {
        int index = Hash(key);
        var bucket = buckets[index];
        foreach (var (k, v) in bucket) {
            if (k == key) return v;
        }
        return -1;
    }

    public void Remove(int key) {
        int index = Hash(key);
        var bucket = buckets[index];
        // Find and remove the node with matching key
        var node = bucket.First;
        while (node != null) {
            if (node.Value.key == key) {
                bucket.Remove(node);
                return;
            }
            node = node.Next;
        }
    }
}
```

```go
type pair struct {
    key   int
    value int
}

type MyHashMap struct {
    buckets [][]pair
}

const capacity = 2069 // Prime capacity for good distribution

func Constructor() MyHashMap {
    b := make([][]pair, capacity)
    for i := 0; i < capacity; i++ {
        b[i] = make([]pair, 0)
    }
    return MyHashMap{buckets: b}
}

func (this *MyHashMap) hash(key int) int {
    return key % capacity
}

func (this *MyHashMap) Put(key int, value int) {
    idx := this.hash(key)
    bucket := this.buckets[idx]
    // Search for existing key and update
    for i, p := range bucket {
        if p.key == key {
            this.buckets[idx][i].value = value // Update existing value
            return
        }
    }
    // Key not found, append new pair
    this.buckets[idx] = append(this.buckets[idx], pair{key, value})
}

func (this *MyHashMap) Get(key int) int {
    idx := this.hash(key)
    // Search for the key in its bucket
    for _, p := range this.buckets[idx] {
        if p.key == key {
            return p.value
        }
    }
    return -1 // Key not found
}

func (this *MyHashMap) Remove(key int) {
    idx := this.hash(key)
    bucket := this.buckets[idx]
    // Filter out the pair with matching key
    filtered := make([]pair, 0)
    for _, p := range bucket {
        if p.key != key {
            filtered = append(filtered, p)
        }
    }
    this.buckets[idx] = filtered
}
```

## Key Takeaways

- **Chaining with a prime-numbered array size** is the standard approach for implementing a HashMap
- The hash function should be **fast and distribute keys evenly** — modulo with a prime helps
- Bucket operations are O(n/k) average, which is effectively O(1) with a good load factor
- Memory management is important — linked lists in C require careful **freeing**
- This implementation technique is the foundation for understanding hash-based data structures
