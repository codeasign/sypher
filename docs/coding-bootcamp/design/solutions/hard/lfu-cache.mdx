# LFU Cache — Solution

## Problem Overview

Design an LFU (Least Frequently Used) cache with O(1) get and put operations. When the cache is full, evict the least frequently used key. If there is a tie (multiple keys with the same frequency), evict the least recently used key among them.

This is more complex than LRU because we need to track both frequency and recency simultaneously.

## Approach: HashMap + Frequency Buckets with Doubly Linked Lists

**Idea:** Maintain three core data structures:
1. A hash map `keyToNode` mapping keys to their list nodes (stores value and frequency).
2. A hash map `freqToList` mapping frequency counts to doubly linked lists of keys at that frequency (ordered by recency within each frequency).
3. An integer `minFreq` tracking the current minimum frequency across all keys.

**Algorithm:**
1. `get(key)`:
   - Look up the key in `keyToNode`. If not found, return -1.
   - Get the node's current frequency `f`.
   - Remove the node from the frequency `f` list.
   - If the `f` list becomes empty and `f == minFreq`, increment `minFreq`.
   - Increment the node's frequency to `f + 1` and add it to the head of the `f + 1` list.
   - Return the node's value.

2. `put(key, value)`:
   - If the key exists: update the value, then follow the same frequency-update logic as `get`.
   - If the key does not exist:
     - If the cache is at capacity, evict: remove the tail of the `minFreq` list (LRU among LFU). Delete it from `keyToNode`.
     - Create a new node with frequency 1, add it to the `freqToList[1]` list.
     - Reset `minFreq` to 1.
     - Add the key to `keyToNode`.

**Complexity Analysis:**
- **Time:** O(1) for both get and put — all operations involve hash map lookups and linked list insertions/removals.
- **Space:** O(capacity) — we store at most capacity nodes across all data structures.

```java
import java.util.HashMap;
import java.util.LinkedHashSet;
import java.util.Map;

class LFUCache {
    // Key-value node with frequency tracking
    private static class Node {
        int key, value, freq;
        Node prev, next;

        Node(int key, int value) {
            this.key = key;
            this.value = value;
            this.freq = 1; // New nodes start with frequency 1
        }
    }

    // Doubly linked list for a specific frequency bucket
    private static class FreqList {
        Node head, tail; // Dummy sentinels

        FreqList() {
            head = new Node(0, 0);
            tail = new Node(0, 0);
            head.next = tail;
            tail.prev = head;
        }

        void addToHead(Node node) {
            node.next = head.next;
            node.prev = head;
            head.next.prev = node;
            head.next = node;
        }

        void remove(Node node) {
            node.prev.next = node.next;
            node.next.prev = node.prev;
        }

        boolean isEmpty() {
            return head.next == tail;
        }

        Node removeTail() {
            if (isEmpty()) return null;
            Node lru = tail.prev;
            remove(lru);
            return lru;
        }
    }

    private final int capacity;
    private int minFreq; // Current minimum frequency
    private final Map<Integer, Node> keyToNode; // Key → Node mapping
    private final Map<Integer, FreqList> freqToList; // Freq → List of nodes at that freq

    public LFUCache(int capacity) {
        this.capacity = capacity;
        this.minFreq = 0;
        this.keyToNode = new HashMap<>();
        this.freqToList = new HashMap<>();
    }

    public int get(int key) {
        if (!keyToNode.containsKey(key)) return -1;
        Node node = keyToNode.get(key);
        updateFrequency(node); // Increment frequency and reorder
        return node.value;
    }

    public void put(int key, int value) {
        if (capacity == 0) return;

        if (keyToNode.containsKey(key)) {
            // Key exists: update value and increment frequency
            Node node = keyToNode.get(key);
            node.value = value;
            updateFrequency(node);
        } else {
            // Key does not exist: evict if needed, then insert
            if (keyToNode.size() >= capacity) {
                // Evict the LFU key (LRU among the minimum frequency bucket)
                FreqList list = freqToList.get(minFreq);
                Node toEvict = list.removeTail();
                keyToNode.remove(toEvict.key);
            }

            // Insert new node with frequency 1
            Node node = new Node(key, value);
            keyToNode.put(key, node);
            minFreq = 1; // New node is always at frequency 1
            freqToList.computeIfAbsent(1, k -> new FreqList()).addToHead(node);
        }
    }

    // Increment a node's frequency and move it to the next bucket
    private void updateFrequency(Node node) {
        int oldFreq = node.freq;
        FreqList oldList = freqToList.get(oldFreq);
        oldList.remove(node);

        // If the old list is empty and it was the min freq, update minFreq
        if (oldList.isEmpty() && oldFreq == minFreq) {
            minFreq++;
        }

        // Move node to the next frequency bucket
        node.freq++;
        freqToList.computeIfAbsent(node.freq, k -> new FreqList()).addToHead(node);
    }
}
```

```cpp
#include <unordered_map>
#include <list>

class LFUCache {
private:
    int capacity;
    int minFreq; // Current minimum frequency across all keys

    // Each entry stores key, value, and frequency
    struct Entry {
        int key, value, freq;
        Entry(int k, int v) : key(k), value(v), freq(1) {}
    };

    // Key → iterator in the list (for O(1) removal)
    unordered_map<int, list<Entry>::iterator> keyToNode;
    // Frequency → list of entries at that frequency
    unordered_map<int, list<Entry>> freqToList;

    // Update a key's frequency: remove from old list, add to new list
    void updateFrequency(int key) {
        auto it = keyToNode[key];
        int oldFreq = it->freq;
        int value = it->value;

        // Remove from old frequency list
        freqToList[oldFreq].erase(it);

        // If old list is empty and it was the minFreq, update minFreq
        if (freqToList[oldFreq].empty() && oldFreq == minFreq) {
            minFreq++;
        }

        // Add to new frequency list
        int newFreq = oldFreq + 1;
        freqToList[newFreq].push_front(Entry(key, value));
        keyToNode[key] = freqToList[newFreq].begin();
        keyToNode[key]->freq = newFreq;
    }

public:
    LFUCache(int capacity) : capacity(capacity), minFreq(0) {}

    int get(int key) {
        if (!keyToNode.count(key)) return -1;
        int value = keyToNode[key]->value;
        updateFrequency(key);
        return value;
    }

    void put(int key, int value) {
        if (capacity == 0) return;

        if (keyToNode.count(key)) {
            // Update existing key
            keyToNode[key]->value = value;
            updateFrequency(key);
        } else {
            // Evict if at capacity
            if (keyToNode.size() >= capacity) {
                // Remove the LRU entry from the minFreq list
                auto& minList = freqToList[minFreq];
                int evictKey = minList.back().key;
                keyToNode.erase(evictKey);
                minList.pop_back();
            }

            // Insert new entry with frequency 1
            freqToList[1].push_front(Entry(key, value));
            keyToNode[key] = freqToList[1].begin();
            minFreq = 1; // New entries always start at frequency 1
        }
    }
};
```

```typescript
class LFUCache {
    private capacity: number;
    private minFreq: number;
    // Key → node mapping
    private keyToNode: Map<number, { key: number; value: number; freq: number }>;
    // Frequency → Set of keys at that frequency (ordered by insertion = LRU)
    private freqToKeys: Map<number, Set<number>>;

    constructor(capacity: number) {
        this.capacity = capacity;
        this.minFreq = 0;
        this.keyToNode = new Map();
        this.freqToKeys = new Map();
    }

    get(key: number): number {
        if (!this.keyToNode.has(key)) return -1;
        const node = this.keyToNode.get(key)!;
        this.updateFrequency(key);
        return node.value;
    }

    put(key: number, value: number): void {
        if (this.capacity === 0) return;

        if (this.keyToNode.has(key)) {
            // Update existing key
            this.keyToNode.get(key)!.value = value;
            this.updateFrequency(key);
        } else {
            // Evict if at capacity
            if (this.keyToNode.size >= this.capacity) {
                // Get the LRU key from the minimum frequency bucket
                const minSet = this.freqToKeys.get(this.minFreq)!;
                const lruKey = minSet.values().next().value;
                minSet.delete(lruKey);
                this.keyToNode.delete(lruKey);
                if (minSet.size === 0) {
                    this.freqToKeys.delete(this.minFreq);
                }
            }

            // Insert new key with frequency 1
            this.keyToNode.set(key, { key, value, freq: 1 });
            if (!this.freqToKeys.has(1)) {
                this.freqToKeys.set(1, new Set());
            }
            this.freqToKeys.get(1)!.add(key);
            this.minFreq = 1;
        }
    }

    private updateFrequency(key: number): void {
        const node = this.keyToNode.get(key)!;
        const oldFreq = node.freq;

        // Remove from old frequency bucket
        const oldSet = this.freqToKeys.get(oldFreq)!;
        oldSet.delete(key);
        if (oldSet.size === 0) {
            this.freqToKeys.delete(oldFreq);
            if (oldFreq === this.minFreq) {
                this.minFreq++;
            }
        }

        // Add to new frequency bucket
        node.freq++;
        const newFreq = node.freq;
        if (!this.freqToKeys.has(newFreq)) {
            this.freqToKeys.set(newFreq, new Set());
        }
        this.freqToKeys.get(newFreq)!.add(key);
    }
}
```

```python
from collections import defaultdict, OrderedDict


class LFUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.min_freq = 0
        # Map key to [value, frequency]
        self.key_to_node = {}
        # Map frequency to OrderedDict of keys (ordered by recency)
        # Using OrderedDict gives us O(1) insertion, deletion, and LRU eviction
        self.freq_to_keys = defaultdict(OrderedDict)

    def get(self, key: int) -> int:
        if key not in self.key_to_node:
            return -1
        self._update_frequency(key)
        return self.key_to_node[key][0]

    def put(self, key: int, value: int) -> None:
        if self.capacity == 0:
            return

        if key in self.key_to_node:
            # Update existing key
            self.key_to_node[key][0] = value
            self._update_frequency(key)
        else:
            # Evict if at capacity
            if len(self.key_to_node) >= self.capacity:
                # Get the LRU key from the minimum frequency bucket
                # In OrderedDict, popitem(last=False) removes the first inserted item (LRU)
                evict_key, _ = self.freq_to_keys[self.min_freq].popitem(last=False)
                del self.key_to_node[evict_key]

            # Insert new key with frequency 1
            self.key_to_node[key] = [value, 1]
            self.freq_to_keys[1][key] = None
            self.min_freq = 1

    def _update_frequency(self, key: int) -> None:
        """Increment the frequency of a key and move it to the next bucket"""
        value, old_freq = self.key_to_node[key]

        # Remove from old frequency bucket
        del self.freq_to_keys[old_freq][key]
        if not self.freq_to_keys[old_freq]:
            del self.freq_to_keys[old_freq]
            if old_freq == self.min_freq:
                self.min_freq += 1

        # Add to new frequency bucket
        new_freq = old_freq + 1
        self.key_to_node[key] = [value, new_freq]
        self.freq_to_keys[new_freq][key] = None
```

```rust
use std::collections::{HashMap, HashSet};

struct LFUCache {
    capacity: i32,
    min_freq: i32,
    // Key → (value, frequency)
    key_to_node: HashMap<i32, (i32, i32)>,
    // Frequency → set of keys (using a simple counter for recency isn't ideal,
    // but we'll use a Vec for simplicity in this example)
    freq_to_keys: HashMap<i32, Vec<i32>>,
}

impl LFUCache {
    fn new(capacity: i32) -> Self {
        LFUCache {
            capacity,
            min_freq: 0,
            key_to_node: HashMap::new(),
            freq_to_keys: HashMap::new(),
        }
    }

    fn get(&mut self, key: i32) -> i32 {
        if !self.key_to_node.contains_key(&key) {
            return -1;
        }
        self.update_frequency(key);
        self.key_to_node[&key].0
    }

    fn put(&mut self, key: i32, value: i32) {
        if self.capacity == 0 {
            return;
        }

        if self.key_to_node.contains_key(&key) {
            // Update existing key
            self.key_to_node.get_mut(&key).unwrap().0 = value;
            self.update_frequency(key);
        } else {
            // Evict if at capacity
            if self.key_to_node.len() as i32 >= self.capacity {
                if let Some(keys) = self.freq_to_keys.get_mut(&self.min_freq) {
                    if let Some(evict_key) = keys.pop() {
                        self.key_to_node.remove(&evict_key);
                    }
                }
            }

            // Insert new key with frequency 1
            self.key_to_node.insert(key, (value, 1));
            self.freq_to_keys.entry(1).or_insert(Vec::new()).push(key);
            self.min_freq = 1;
        }
    }

    fn update_frequency(&mut self, key: i32) {
        let (value, old_freq) = self.key_to_node[&key];

        // Remove from old frequency bucket
        if let Some(keys) = self.freq_to_keys.get_mut(&old_freq) {
            keys.retain(|&k| k != key);
            if keys.is_empty() {
                self.freq_to_keys.remove(&old_freq);
                if old_freq == self.min_freq {
                    self.min_freq += 1;
                }
            }
        }

        // Add to new frequency bucket
        let new_freq = old_freq + 1;
        self.key_to_node.insert(key, (value, new_freq));
        self.freq_to_keys.entry(new_freq).or_insert(Vec::new()).push(key);
    }
}
```

```c
#include <stdlib.h>
#include <stdio.h>
#include <limits.h>

#define MAX_SIZE 100005

// Node for the doubly linked list within a frequency bucket
typedef struct Node {
    int key;
    int value;
    int freq;
    struct Node* prev;
    struct Node* next;
} Node;

// Frequency bucket: a doubly linked list of nodes at this frequency
typedef struct FreqList {
    Node* head; // Dummy head (most recently used)
    Node* tail; // Dummy tail (least recently used)
} FreqList;

// Hash table entry for key-to-node mapping
typedef struct HashEntry {
    int key;
    Node* node;
    struct HashEntry* next;
} HashEntry;

typedef struct {
    int capacity;
    int size;
    int minFreq;
    HashEntry* keyTable[MAX_SIZE]; // Key → Node mapping
    FreqList freqTable[MAX_SIZE];  // Freq → Linked list of nodes
} LFUCache;

// Initialize a frequency list with sentinel nodes
void initFreqList(FreqList* list) {
    list->head = (Node*)malloc(sizeof(Node));
    list->tail = (Node*)malloc(sizeof(Node));
    list->head->next = list->tail;
    list->tail->prev = list->head;
}

// Remove a node from its frequency list
void removeNode(Node* node) {
    node->prev->next = node->next;
    node->next->prev = node->prev;
}

// Add a node to the head of a frequency list (MRU position)
void addToHead(FreqList* list, Node* node) {
    node->next = list->head->next;
    node->prev = list->head;
    list->head->next->prev = node;
    list->head->next = node;
}

// Check if a frequency list is empty
int isListEmpty(FreqList* list) {
    return list->head->next == list->tail;
}

// Remove and return the tail node (LRU) from a frequency list
Node* removeTail(FreqList* list) {
    if (isListEmpty(list)) return NULL;
    Node* lru = list->tail->prev;
    removeNode(lru);
    return lru;
}

// Hash function
int hash(int key) {
    return key % MAX_SIZE;
}

// Find a node by key in the hash table
Node* findNode(LFUCache* obj, int key) {
    int idx = hash(key);
    HashEntry* entry = obj->keyTable[idx];
    while (entry) {
        if (entry->key == key) return entry->node;
        entry = entry->next;
    }
    return NULL;
}

// Insert a key-node mapping into the hash table
void insertKey(LFUCache* obj, int key, Node* node) {
    int idx = hash(key);
    HashEntry* entry = (HashEntry*)malloc(sizeof(HashEntry));
    entry->key = key;
    entry->node = node;
    entry->next = obj->keyTable[idx];
    obj->keyTable[idx] = entry;
}

// Remove a key from the hash table
void deleteKey(LFUCache* obj, int key) {
    int idx = hash(key);
    HashEntry* curr = obj->keyTable[idx];
    HashEntry* prev = NULL;
    while (curr) {
        if (curr->key == key) {
            if (prev == NULL) obj->keyTable[idx] = curr->next;
            else prev->next = curr->next;
            free(curr);
            return;
        }
        prev = curr;
        curr = curr->next;
    }
}

LFUCache* lFUCacheCreate(int capacity) {
    LFUCache* obj = (LFUCache*)calloc(1, sizeof(LFUCache));
    obj->capacity = capacity;
    obj->size = 0;
    obj->minFreq = 0;
    for (int i = 0; i < MAX_SIZE; i++) {
        initFreqList(&obj->freqTable[i]);
    }
    return obj;
}

int lFUCacheGet(LFUCache* obj, int key) {
    Node* node = findNode(obj, key);
    if (!node) return -1;

    // Increment frequency
    int oldFreq = node->freq;
    removeNode(node);

    // Update minFreq if the old bucket is now empty
    if (isListEmpty(&obj->freqTable[oldFreq]) && oldFreq == obj->minFreq) {
        obj->minFreq++;
    }

    // Move to next frequency bucket
    node->freq = oldFreq + 1;
    addToHead(&obj->freqTable[node->freq], node);

    return node->value;
}

void lFUCachePut(LFUCache* obj, int key, int value) {
    if (obj->capacity == 0) return;

    Node* node = findNode(obj, key);
    if (node) {
        // Update existing key
        node->value = value;

        // Same frequency update logic as get
        int oldFreq = node->freq;
        removeNode(node);
        if (isListEmpty(&obj->freqTable[oldFreq]) && oldFreq == obj->minFreq) {
            obj->minFreq++;
        }
        node->freq = oldFreq + 1;
        addToHead(&obj->freqTable[node->freq], node);
    } else {
        // Evict if at capacity
        if (obj->size >= obj->capacity) {
            Node* evict = removeTail(&obj->freqTable[obj->minFreq]);
            if (evict) {
                deleteKey(obj, evict->key);
                free(evict);
                obj->size--;
            }
        }

        // Insert new node with frequency 1
        Node* newNode = (Node*)malloc(sizeof(Node));
        newNode->key = key;
        newNode->value = value;
        newNode->freq = 1;
        insertKey(obj, key, newNode);
        addToHead(&obj->freqTable[1], newNode);
        obj->minFreq = 1;
        obj->size++;
    }
}

void lFUCacheFree(LFUCache* obj) {
    for (int i = 0; i < MAX_SIZE; i++) {
        Node* curr = obj->freqTable[i].head;
        while (curr) {
            Node* temp = curr;
            curr = curr->next;
            free(temp);
        }
        HashEntry* entry = obj->keyTable[i];
        while (entry) {
            HashEntry* temp = entry;
            entry = entry->next;
            free(temp);
        }
    }
    free(obj);
}
```

```csharp
using System.Collections.Generic;

public class LFUCache {
    private class Node {
        public int key, value, freq;
        public Node prev, next;
        public Node(int k, int v) { key = k; value = v; freq = 1; }
    }

    // Doubly linked list for a frequency bucket
    private class FreqList {
        public Node head, tail; // Sentinels

        public FreqList() {
            head = new Node(0, 0);
            tail = new Node(0, 0);
            head.next = tail;
            tail.prev = head;
        }

        public void AddToHead(Node node) {
            node.next = head.next;
            node.prev = head;
            head.next.prev = node;
            head.next = node;
        }

        public void Remove(Node node) {
            node.prev.next = node.next;
            node.next.prev = node.prev;
        }

        public bool IsEmpty() => head.next == tail;

        public Node RemoveTail() {
            if (IsEmpty()) return null;
            Node lru = tail.prev;
            Remove(lru);
            return lru;
        }
    }

    private readonly int capacity;
    private int minFreq;
    private readonly Dictionary<int, Node> keyToNode;
    private readonly Dictionary<int, FreqList> freqToList;

    public LFUCache(int capacity) {
        this.capacity = capacity;
        this.minFreq = 0;
        this.keyToNode = new Dictionary<int, Node>();
        this.freqToList = new Dictionary<int, FreqList>();
    }

    public int Get(int key) {
        if (!keyToNode.TryGetValue(key, out Node node)) return -1;
        UpdateFrequency(node);
        return node.value;
    }

    public void Put(int key, int value) {
        if (capacity == 0) return;

        if (keyToNode.TryGetValue(key, out Node node)) {
            // Update existing key
            node.value = value;
            UpdateFrequency(node);
        } else {
            // Evict if at capacity
            if (keyToNode.Count >= capacity) {
                FreqList minList = freqToList[minFreq];
                Node toEvict = minList.RemoveTail();
                if (toEvict != null) {
                    keyToNode.Remove(toEvict.key);
                }
            }

            // Insert new key with frequency 1
            Node newNode = new Node(key, value);
            keyToNode[key] = newNode;
            minFreq = 1;
            if (!freqToList.ContainsKey(1)) {
                freqToList[1] = new FreqList();
            }
            freqToList[1].AddToHead(newNode);
        }
    }

    private void UpdateFrequency(Node node) {
        int oldFreq = node.freq;
        FreqList oldList = freqToList[oldFreq];
        oldList.Remove(node);

        // Update minFreq if the old list is now empty
        if (oldList.IsEmpty() && oldFreq == minFreq) {
            minFreq++;
        }

        // Move to next frequency bucket
        node.freq++;
        if (!freqToList.ContainsKey(node.freq)) {
            freqToList[node.freq] = new FreqList();
        }
        freqToList[node.freq].AddToHead(node);
    }
}
```

```go
type Node struct {
    key, value, freq int
    prev, next       *Node
}

// FreqList is a doubly linked list for a specific frequency bucket
type FreqList struct {
    head, tail *Node // Dummy sentinels
}

func newFreqList() *FreqList {
    head := &Node{}
    tail := &Node{}
    head.next = tail
    tail.prev = head
    return &FreqList{head: head, tail: tail}
}

func (l *FreqList) addToHead(node *Node) {
    node.next = l.head.next
    node.prev = l.head
    l.head.next.prev = node
    l.head.next = node
}

func (l *FreqList) remove(node *Node) {
    node.prev.next = node.next
    node.next.prev = node.prev
}

func (l *FreqList) isEmpty() bool {
    return l.head.next == l.tail
}

func (l *FreqList) removeTail() *Node {
    if l.isEmpty() {
        return nil
    }
    lru := l.tail.prev
    l.remove(lru)
    return lru
}

type LFUCache struct {
    capacity   int
    minFreq    int
    keyToNode  map[int]*Node        // Key -> Node
    freqToList map[int]*FreqList    // Freq -> List
}

func Constructor(capacity int) LFUCache {
    return LFUCache{
        capacity:   capacity,
        minFreq:    0,
        keyToNode:  make(map[int]*Node),
        freqToList: make(map[int]*FreqList),
    }
}

func (this *LFUCache) Get(key int) int {
    node, ok := this.keyToNode[key]
    if !ok {
        return -1
    }
    this.updateFrequency(node)
    return node.value
}

func (this *LFUCache) Put(key int, value int) {
    if this.capacity == 0 {
        return
    }

    node, ok := this.keyToNode[key]
    if ok {
        // Update existing key
        node.value = value
        this.updateFrequency(node)
    } else {
        // Evict if at capacity
        if len(this.keyToNode) >= this.capacity {
            minList := this.freqToList[this.minFreq]
            if evict := minList.removeTail(); evict != nil {
                delete(this.keyToNode, evict.key)
            }
        }

        // Insert new key with frequency 1
        newNode := &Node{key: key, value: value, freq: 1}
        this.keyToNode[key] = newNode
        if _, ok := this.freqToList[1]; !ok {
            this.freqToList[1] = newFreqList()
        }
        this.freqToList[1].addToHead(newNode)
        this.minFreq = 1
    }
}

func (this *LFUCache) updateFrequency(node *Node) {
    oldFreq := node.freq
    oldList := this.freqToList[oldFreq]
    oldList.remove(node)

    // Update minFreq if the old bucket is now empty
    if oldList.isEmpty() && oldFreq == this.minFreq {
        this.minFreq++
    }

    // Move to next frequency bucket
    node.freq++
    if _, ok := this.freqToList[node.freq]; !ok {
        this.freqToList[node.freq] = newFreqList()
    }
    this.freqToList[node.freq].addToHead(node)
}
```

## Key Takeaways

- **LFU Cache** is the most challenging design problem in this section — it combines LRU's recency tracking with frequency tracking
- The key insight is using **frequency buckets**, each containing a doubly linked list ordered by recency
- The `minFreq` counter is critical for O(1) eviction — it tells us exactly which bucket to evict from
- When a frequency bucket becomes empty, we may need to increment `minFreq`
- **Tie-breaking** is handled naturally: within a frequency bucket, the tail is the LRU item, so evicting the tail handles both the frequency and recency requirements
- The C implementation requires careful **manual memory management** for nodes and hash table entries
