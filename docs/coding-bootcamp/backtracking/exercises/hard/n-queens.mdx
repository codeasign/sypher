# N-Queens

**Difficulty:** Hard

The **n-queens** puzzle is the problem of placing `n` queens on an `n x n` chessboard such that no two queens attack each other.

Given an integer `n`, return all distinct solutions to the n-queens puzzle. Each solution contains a sequence of column positions for each row.

Two queens attack each other if they share the same row, column, or diagonal.

## Example 1

**Input:**
```
4
```

**Output:**
```
1 3 0 2
2 0 3 1
```

**Explanation:** There are two distinct solutions for n=4.
- `[1, 3, 0, 2]` means: row 0 queen at col 1, row 1 queen at col 3, row 2 queen at col 0, row 3 queen at col 2
- `[2, 0, 3, 1]` means: row 0 queen at col 2, row 1 queen at col 0, row 2 queen at col 3, row 3 queen at col 1

## Example 2

**Input:**
```
1
```

**Output:**
```
0
```

**Explanation:** A single queen on a 1x1 board is trivially a valid solution.

## Input Specification

A single integer `n` (the size of the chessboard).

## Output Specification

Print each solution on its own line as `n` space-separated integers. The `i`-th integer represents the column position of the queen in row `i`. If no solution exists, print nothing (no output lines).

## Constraints

- `1 <= n <= 9`

## Hints

<details>
<summary>Hint 1</summary>

Place queens one row at a time. At each row, try every column and check if it is safe.
</details>

<details>
<summary>Hint 2</summary>

A position is safe if no other queen is in the same column, the same main diagonal (row - col), or the same anti-diagonal (row + col).
</details>

<details>
<summary>Hint 3</summary>

Use arrays to track which columns, main diagonals, and anti-diagonals are already occupied. This makes the safety check O(1).
</details>

<details>
<summary>Hint 4</summary>

The base case is when all `n` queens have been placed successfully. Add the current column configuration to the result.
</details>
