# Subsets

**Difficulty:** Easy

Given an array of **distinct** integers, return all possible subsets (the power set).

The solution set must not contain duplicate subsets. You may return the subsets in any order.

## Example 1

**Input:**
```
3
1 2 3
```

**Output:**
```

1
1 2
1 2 3
1 3
2
2 3
3
```

**Explanation:** The power set of `[1, 2, 3]` contains 8 subsets: the empty set, singletons, pairs, and the full set.

## Example 2

**Input:**
```
1
5
```

**Output:**
```

5
```

**Explanation:** The power set of `[5]` contains the empty set and `[5]`.

## Input Specification

- First line: integer `n` (the number of elements)
- Second line: `n` space-separated integers

## Output Specification

Print each subset on its own line. Each subset should be space-separated integers. Print an empty line for the empty subset.

## Constraints

- `0 <= n <= 12`
- `-10^9 <= nums[i] <= 10^9`
- All integers in the input are distinct

## Hints

<details>
<summary>Hint 1</summary>

At each step, you have a choice: either include the current element in the subset or skip it.
</details>

<details>
<summary>Hint 2</summary>

Use recursion with a base case when you have processed all elements. Add the current state to the result at each step.
</details>

<details>
<summary>Hint 3</summary>

There are exactly 2^n subsets. Use a backtracking approach where you branch into two paths at each element.
</details>

<details>
<summary>Hint 4</summary>

The standard template for backtracking involves: add, recurse, remove (the "choose-explore-unchoose" pattern).
</details>
