# Sum of Two Integers

**Difficulty:** Hard

Given two integers `a` and `b`, return the sum of the two integers **without using** the operators `+` and `-`.

You must implement the addition using bit manipulation only.

## Example 1

**Input:** `a = 1, b = 2`  
**Output:** `3`

## Example 2

**Input:** `a = 2, b = 3`  
**Output:** `5`

## Example 3

**Input:** `a = -1, b = 1`  
**Output:** `0`

## Input Specification

A single line containing two space-separated integers `a` and `b`.

## Output Specification

Print the sum `a + b`.

## Constraints

- `-1000 <= a, b <= 1000`

## Hints

<details>
<summary>Hint 1</summary>

Binary addition without carry is `a ^ b`. The carry is `(a & b) << 1`.
</details>

<details>
<summary>Hint 2</summary>

Repeat the process: XOR gives the sum without carry, AND gives the carry bits. Keep going until there is no carry.
</details>

<details>
<summary>Hint 3</summary>

This works for negative numbers too because of two's complement representation.
</details>
