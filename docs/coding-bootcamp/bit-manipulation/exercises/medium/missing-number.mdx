# Missing Number

**Difficulty:** Medium

Given an array `nums` containing `n` distinct numbers in the range `[0, n]`, return the only number in the range that is missing from the array.

## Example 1

**Input:** `nums = [3, 0, 1]`  
**Output:** `2`  
**Explanation:** `n = 3`. The set of numbers is `{0, 1, 3}` — 2 is missing.

## Example 2

**Input:** `nums = [0, 1]`  
**Output:** `2`  
**Explanation:** `n = 2`. The numbers present are 0 and 1 — 2 is missing.

## Example 3

**Input:** `nums = [9, 6, 4, 2, 3, 5, 7, 0, 1]`  
**Output:** `8`  
**Explanation:** `n = 9`. The number 8 is missing.

## Input Specification

The first line contains an integer `n`. The second line contains `n` space-separated integers (distinct, in range `[0, n]`).

## Output Specification

Print the missing number.

## Constraints

- `1 <= n <= 10⁴`
- `nums.length == n`
- All numbers in `nums` are unique and in the range `[0, n]`.

## Hints

<details>
<summary>Hint 1</summary>

Try XOR. XOR all indices from 0 to n with all numbers in nums. What cancels out?
</details>

<details>
<summary>Hint 2</summary>

Alternatively, compute the expected sum `n * (n + 1) // 2` and subtract the actual sum. But watch for overflow!
</details>

<details>
<summary>Hint 3</summary>

The XOR approach avoids overflow entirely and runs in O(1) space.
</details>

</ProblemEditor>
