# Single Number

**Difficulty:** Easy

Given a **non-empty** array of integers `nums`, every element appears **twice** except for one. Find that single one.

You must implement a solution with a linear runtime complexity and use only constant extra space.

## Example 1

**Input:** `nums = [2, 2, 1]`  
**Output:** `1`

## Example 2

**Input:** `nums = [4, 1, 2, 1, 2]`  
**Output:** `4`

## Example 3

**Input:** `nums = [1]`  
**Output:** `1`

## Input Specification

The first line contains an integer `n` (1 ≤ n ≤ 3 × 10⁴). The second line contains `n` space-separated integers.

## Output Specification

Print the single element that appears only once.

## Constraints

- `1 <= n <= 3 * 10⁴`
- `-3 * 10⁴ <= nums[i] <= 3 * 10⁴`
- Every element appears twice except for one element which appears once.

## Hints

<details>
<summary>Hint 1</summary>

Think about the XOR operator. What happens when you XOR a number with itself?
</details>

<details>
<summary>Hint 2</summary>

XOR is associative and commutative: `a ^ b ^ a = b`. All pairs cancel out.
</details>

<details>
<summary>Hint 3</summary>

Initialize a variable to 0 and XOR it with every element. The result will be the unique element.
</details>
