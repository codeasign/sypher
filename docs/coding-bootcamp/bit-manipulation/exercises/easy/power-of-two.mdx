# Power of Two

**Difficulty:** Easy

Given an integer `n`, return `true` if it is a power of two, otherwise return `false`.

An integer `n` is a power of two if there exists an integer `k` such that `n == 2^k`.

## Example 1

**Input:** `n = 1`
**Output:** `true`
**Explanation:** `2^0 = 1`

## Example 2

**Input:** `n = 16`
**Output:** `true`
**Explanation:** `2^4 = 16`

## Example 3

**Input:** `n = 3`
**Output:** `false`
**Explanation:** `3` is not expressible as `2^k` for any non-negative integer `k`.

## Input Specification

A single line containing one integer `n`.

## Output Specification

Print `true` if `n` is a power of two, otherwise print `false`.

## Constraints

- `-2^31 <= n <= 2^31 - 1`

## Hints

<details>
<summary>Hint 1</summary>

Powers of two in binary always look like `1` followed by only zeros — for example, `1`, `10`, `100`, `1000`. This means they have exactly one bit set. Can you use a bitwise operation to check this?

</details>

<details>
<summary>Hint 2</summary>

Consider the expression `n & (n - 1)`. Subtracting 1 from a power-of-two flips its single set bit to `0` and all lower bits to `1`. Performing a bitwise AND of `n` with `n - 1` clears that bit — the result is `0` for any power of two.

</details>

<details>
<summary>Hint 3</summary>

Be careful of edge cases: `n = 0` is NOT a power of two (there is no `k` such that `2^k = 0`), and negative numbers are never powers of two. Make sure your solution handles both.

</details>
