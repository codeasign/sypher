# Summary Ranges — Solution

## Approach: Linear Scan

**Algorithm:** Track start of each range. When nums[i] != nums[i-1] + 1, close the range.

**Complexity:** O(n) time, O(1) extra space.

```python
def summary_ranges(nums: list[int]) -> list[str]:
    if not nums: return []
    result = []
    start = nums[0]
    for i in range(1, len(nums)):
        if nums[i] != nums[i-1] + 1:
            result.append(str(start) if start == nums[i-1] else f'{start}->{nums[i-1]}')
            start = nums[i]
    result.append(str(start) if start == nums[-1] else f'{start}->{nums[-1]}')
    return result
```
