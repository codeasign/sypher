## Setup

Before you start this exercise, make sure you have:

- **Python 3.10 or higher** installed (`python --version`)
- **An Anthropic API key** with credits. Sign up at [console.anthropic.com](https://console.anthropic.com)
- **pip** available (`pip --version`)

### Install the Anthropic SDK

```bash
pip install anthropic
```

### Set Your API Key

```bash
export ANTHROPIC_API_KEY="sk-ant-your-key-here"
```

Or create a `.env` file:

```
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

**Never commit this file.** Add `.env` to `.gitignore`.

### Verify Installation

```bash
python -c "import anthropic; print(anthropic.__version__)"
```

Expected output (version may vary):

```
0.116.0
```

## The Build — Step by Step

### Step 1: Create the Client and Make Your First Call

Create `claude_chat.py`:

```python
import os
from anthropic import Anthropic

client = Anthropic(
    api_key=os.environ["ANTHROPIC_API_KEY"]
)

message = client.messages.create(
    model="claude-haiku-4-5-20251001",
    max_tokens=200,
    messages=[
        {"role": "user", "content": "What is the capital of Japan?"}
    ]
)

print(message.content[0].text)
```

Run it:

```bash
python claude_chat.py
```

Expected output:

```
The capital of Japan is Tokyo.
```

### Step 3: Track Token Usage

```python
import os
from anthropic import Anthropic

client = Anthropic(
    api_key=os.environ["ANTHROPIC_API_KEY"]
)

message = client.messages.create(
    model="claude-haiku-4-5-20251001",
    max_tokens=200,
    messages=[
        {"role": "user", "content": "Explain what a Python decorator is in one paragraph."}
    ]
)

print("Response:", message.content[0].text)
print()
print("--- Token Usage ---")
print(f"Input tokens:     {message.usage.input_tokens}")
print(f"Output tokens:    {message.usage.output_tokens}")
print(f"Total tokens:     {message.usage.input_tokens + message.usage.output_tokens}")

# Cost estimate
input_cost = message.usage.input_tokens / 1_000_000 * 0.25
output_cost = message.usage.output_tokens / 1_000_000 * 1.25
print(f"Estimated cost:   ${(input_cost + output_cost):.6f}")
```

### Step 4: Handle Errors

```python
import os
from anthropic import Anthropic, APIError, APIConnectionError, RateLimitError

client = Anthropic(
    api_key=os.environ["ANTHROPIC_API_KEY"]
)

try:
    message = client.messages.create(
        model="claude-haiku-4-5-20251001",
        max_tokens=200,
        messages=[{"role": "user", "content": "Hello"}]
    )
    print(message.content[0].text)

except RateLimitError:
    print("Rate limit hit. Slow down and retry.")
except APIConnectionError:
    print("Network error. Check your connection.")
except APIError as e:
    print(f"API error: {e}")
```

### Step 5: Multi-Turn Conversation

```python
import os
from anthropic import Anthropic

client = Anthropic(
    api_key=os.environ["ANTHROPIC_API_KEY"]
)

messages = [
    {"role": "user", "content": "Suggest a simple pasta dish for dinner."}
]

message = client.messages.create(
    model="claude-sonnet-5",
    max_tokens=500,
    messages=messages
)

reply = message.content[0].text
print(f"Assistant: {reply[:150]}...\n")

messages.append({"role": "assistant", "content": reply})
messages.append({"role": "user", "content": "What ingredients do I need?"})

message = client.messages.create(
    model="claude-sonnet-5",
    max_tokens=500,
    messages=messages
)

print(f"Assistant: {message.content[0].text[:150]}...")
```

### Step 6: Build a Chat Loop

Create `chat_loop.py`:

```python
import os
from anthropic import Anthropic

client = Anthropic(
    api_key=os.environ["ANTHROPIC_API_KEY"]
)

messages = []
system_prompt = "You are a helpful assistant. Keep answers brief."

print("Chat with Claude. Type 'quit' to exit.")
print("-" * 50)

while True:
    user_input = input("\nYou: ")
    if user_input.lower() == "quit":
        break

    messages.append({"role": "user", "content": user_input})

    message = client.messages.create(
        model="claude-sonnet-5",
        max_tokens=500,
        system=system_prompt,
        messages=messages
    )

    reply = message.content[0].text
    print(f"\nClaude: {reply}")

    messages.append({"role": "assistant", "content": reply})
```

## Common Errors at This Stage

### Error 1: Missing API key

```
anthropic.AuthenticationError: 401 x-api-key header is required
```

**Cause:** You did not set `ANTHROPIC_API_KEY`.
**Fix:** Set the environment variable before running: `export ANTHROPIC_API_KEY="sk-ant-..."`.

### Error 2: Missing max_tokens

```
anthropic.BadRequestError: 400 max_tokens: field is required
```

**Cause:** You omitted the required `max_tokens` parameter.
**Fix:** Always include `max_tokens` in every call.

### Error 3: Model not found

```
anthropic.BadRequestError: 404 model: "claude-opus-3" not found
```

**Cause:** The model name is wrong or deprecated.
**Fix:** Check current model names at docs.anthropic.com.

### Error 4: Insufficient credits

```
anthropic.RateLimitError: 429 You do not have sufficient credits
```

**Cause:** Your account ran out of prepaid credits.
**Fix:** Add credits at console.anthropic.com.

### Error 5: Empty messages

```
anthropic.BadRequestError: 400 messages: must contain at least one message
```

**Cause:** You passed an empty list for messages.
**Fix:** Always include at least one message with role "user".

### Error 6: Context too long

```
anthropic.BadRequestError: 400 This model's maximum context length is 200000 tokens
```

**Cause:** Your conversation exceeds the model's context window.
**Fix:** Use a model with a larger context (claude-sonnet-5 handles 1M tokens) or trim the conversation history.
