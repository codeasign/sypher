## Tier 1 — Beginner

### Exercise 1: Make a Greeting Call

**Task:** Write a script that asks Claude to "Introduce yourself in two sentences" and prints the response.

**Hint:** Use `claude-haiku-4-5-20251001` with `max_tokens=200`.

### Solution

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

message = client.messages.create(
    model="claude-haiku-4-5-20251001",
    max_tokens=200,
    messages=[
        {"role": "user", "content": "Introduce yourself in two sentences."}
    ]
)

print(message.content[0].text)
```

### Exercise 4: Switch Models

**Task:** Ask the same question to Haiku and Sonnet. Print which model gave which response.

### Solution

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

question = "What is the speed of light?"

for model in ["claude-haiku-4-5-20251001", "claude-sonnet-5"]:
    message = client.messages.create(
        model=model,
        max_tokens=100,
        messages=[{"role": "user", "content": question}]
    )

    print(message.content[0].text)

```

### Exercise 1: Document Summarizer

**Task:** Write a script that takes a long text, sends it to Claude with a system prompt to summarize in three bullet points, and prints the result.

**Hint:** Use `system="Summarize in three bullet points."` and include the text in a user message.

### Solution

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

long_text = """
Python is a high-level, general-purpose programming language created by Guido van Rossum
and first released in 1991. Its design philosophy emphasizes code readability through
significant indentation. Python is dynamically typed and garbage-collected. It supports
multiple programming paradigms including structured, object-oriented, and functional
programming. It has a large and comprehensive standard library. Python interpreters are
available for many operating systems. The language has seen significant adoption in
data science, machine learning, web development, and automation.
"""

message = client.messages.create(
    model="claude-sonnet-5",
    max_tokens=300,
    system="Summarize the following text in exactly three bullet points.",
    messages=[{"role": "user", "content": long_text}]
)

print(message.content[0].text)
```

### Exercise 2: System Prompt Comparison

**Task:** Ask the same question with three different system prompts: "Answer like a pirate", "Answer like a professor", "Answer like a five-year-old". Compare the styles.

**Hint:** Loop over system prompts, not model names.

### Solution

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

personas = [
    "Answer like a pirate.",
    "Answer like a university professor.",
    "Answer like you are explaining to a five-year-old."
]

for persona in personas:
    message = client.messages.create(
        model="claude-haiku-4-5-20251001",
        max_tokens=200,
        system=persona,
        messages=[{"role": "user", "content": "What is gravity?"}]
    )
    print(f"=== {persona} ===")
    print(message.content[0].text)
    print()
```

### Exercise 3: Temperature Exploration

**Task:** Ask the same question with temperature 0.0 five times and temperature 1.0 five times. Observe the difference in response variability.

### Solution

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

prompt = "Name a good book to read and explain why."

for temp, label in [(0.0, "Deterministic (0.0)"), (1.0, "Creative (1.0)")]:
    print(f"=== {label} ===")
    for i in range(3):
        message = client.messages.create(
            model="claude-haiku-4-5-20251001",
            max_tokens=100,
            temperature=temp,
            messages=[{"role": "user", "content": prompt}]
        )
        print(f"  {i+1}. {message.content[0].text[:80]}...")
    print()
```

## Tier 3 — Advanced

### Exercise 1: Cost-Aware Multi-Model Router

**Task:** Write a script that routes short, simple queries to Haiku and complex, long queries to Sonnet. Track the total cost across 10 sample queries.

**Hint:** Use `len(user_input) > 100` as a heuristic for complexity. Track costs in a running total.

### Solution

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

HAIKU_INPUT_COST = 0.25 / 1_000_000
HAIKU_OUTPUT_COST = 1.25 / 1_000_000
SONNET_INPUT_COST = 3.00 / 1_000_000
SONNET_OUTPUT_COST = 15.00 / 1_000_000

queries = [
    "What is 2+2?",
    "Explain the theory of general relativity in detail, covering spacetime curvature, how gravity affects light, and the experimental evidence that supports it including gravitational lensing and gravitational waves."
]

total_cost = 0.0

for query in queries:
    is_complex = len(query) > 100
    model = "claude-sonnet-5" if is_complex else "claude-haiku-4-5-20251001"

    message = client.messages.create(
        model=model,
        max_tokens=300,
        messages=[{"role": "user", "content": query}]
    )

    in_cost = message.usage.input_tokens * (SONNET_INPUT_COST if is_complex else HAIKU_INPUT_COST)
    out_cost = message.usage.output_tokens * (SONNET_OUTPUT_COST if is_complex else HAIKU_OUTPUT_COST)
    cost = in_cost + out_cost
    total_cost += cost

    print(f"Model: {model} | Cost: ${cost:.6f}")
    print(f"Response: {message.content[0].text[:80]}...\n")

print(f"Total cost: ${total_cost:.4f}")
```

### Exercise 2: Conversation with Token Budget

**Task:** Build a chat loop that tracks total tokens used across the entire conversation and stops accepting new messages when a budget of 2,000 total tokens is exceeded.

**Hint:** Sum `message.usage.input_tokens + message.usage.output_tokens` after each call.

### Solution

```python
import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

messages = []
total_tokens = 0
BUDGET = 2000

print("Chat with Claude. 'quit' to exit. Budget: 2000 tokens total.\n")

while total_tokens < BUDGET:
    user_input = input("You: ")
    if user_input.lower() == "quit":
        break

    messages.append({"role": "user", "content": user_input})

    message = client.messages.create(
        model="claude-haiku-4-5-20251001",
        max_tokens=200,
        messages=messages
    )

    reply = message.content[0].text
    print(f"Claude: {reply}\n")

    total_tokens += message.usage.input_tokens + message.usage.output_tokens
    print(f"[Tokens used: {total_tokens} / {BUDGET}]\n")

    messages.append({"role": "assistant", "content": reply})

print(f"Session ended. Total tokens: {total_tokens}")
```

### Exercise 3: Concurrent Requests with Delay

**Task:** Send 5 different questions to Claude simultaneously using threading, collect the responses, and print them in order. Add error handling for each request.

**Hint:** Use `concurrent.futures.ThreadPoolExecutor` to send requests in parallel.

### Solution

```python
import os
import concurrent.futures
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

questions = [
    "What is the capital of France?",
    "What is Python?",
    "What is the speed of light?",
    "Who wrote Hamlet?",
    "What is the largest planet?"
]

def ask_claude(question):
    try:
        message = client.messages.create(
            model="claude-haiku-4-5-20251001",
            max_tokens=100,
            messages=[{"role": "user", "content": question}]
        )
        return question, message.content[0].text, None
    except Exception as e:
        return question, None, str(e)

with concurrent.futures.ThreadPoolExecutor(max_workers=3) as executor:
    results = list(executor.map(ask_claude, questions))

for question, answer, error in results:
    if error:
        print(f"✗ {question}: ERROR - {error}")
    else:
        print(f"✓ {question[:40]}: {answer[:60]}...")
```
