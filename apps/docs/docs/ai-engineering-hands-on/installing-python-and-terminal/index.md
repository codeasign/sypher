# Installing Python and Using the Terminal

Before you can write a single line of Python or call an AI model API, you need a working development environment. This means installing Python on your computer, learning to navigate your terminal, and writing your very first script. This section covers everything a complete beginner needs to get from zero to a running Python program.

## Why It Matters

Every AI engineer's workflow starts at the terminal. You install libraries, run training scripts, launch Jupyter notebooks, and deploy services all from the command line. If you cannot reliably open a terminal, run Python, and execute a script, nothing else you learn can be put into practice. These are the mechanical foundations — once they become second nature, you can focus entirely on the AI engineering itself.

## What You Will Build

- A working Python 3.12+ installation on your operating system
- A terminal setup you can navigate with confidence
- A "Hello, AI World" script that runs from the command line
- A portable project structure ready for the next section

## Prerequisites

- A computer (Windows, macOS, or Linux)
- Administrator access to install software
- No prior programming experience required
