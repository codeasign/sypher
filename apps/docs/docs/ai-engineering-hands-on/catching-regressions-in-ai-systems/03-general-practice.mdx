## Tier 1 — Beginner

### Exercise 1.1: Detect a known regression

**Task:** You are given two sets of scores — baseline and candidate. The candidate has a known regression. Use the regression detector to confirm it. Which test catches it first? Is it the mean shift, the variance increase, or the tail degradation?

```python
# Baseline scores (production)
baseline = [0.85, 0.88, 0.82, 0.90, 0.87, 0.83, 0.89, 0.86,
            0.91, 0.84, 0.88, 0.85, 0.90, 0.87, 0.86]

# Candidate scores (known regression — tail is worse)
candidate = [0.88, 0.85, 0.70, 0.91, 0.86, 0.72, 0.89, 0.84,
             0.90, 0.83, 0.68, 0.87, 0.88, 0.71, 0.85]
```

**Hint:** Pass these arrays to `detect_regression()` and call `print_regression_report()`.

### Solution

```python
from regression_detector import detect_regression, print_regression_report

baseline = [0.85, 0.88, 0.82, 0.90, 0.87, 0.83, 0.89, 0.86,
            0.91, 0.84, 0.88, 0.85, 0.90, 0.87, 0.86]

candidate = [0.88, 0.85, 0.70, 0.91, 0.86, 0.72, 0.89, 0.84,
             0.90, 0.83, 0.68, 0.87, 0.88, 0.71, 0.85]

result = detect_regression(baseline, candidate)
print_regression_report(result)
```

The tail degradation (P5 drop) should be the first signal. The mean might only drop slightly, but the P5 drops from ~0.82 to ~0.69.

### Exercise 1.2: Set your own thresholds

**Task:** The regression detector uses default thresholds (alpha=0.05, min_effect_size=0.2). For a high-stakes application (medical advice), you want stricter thresholds. For a low-stakes application (content recommendations), you want looser thresholds. Run the same detection with different thresholds and compare the results.

**Hint:** Pass `alpha=0.01, min_effect_size=0.3` for strict, or `alpha=0.1, min_effect_size=0.1` for lenient.

### Solution

```python
from regression_detector import detect_regression

baseline = [0.85, 0.88, 0.82, 0.90, 0.87] * 3
candidate = [0.82, 0.85, 0.80, 0.88, 0.84] * 3

for label, alpha, mes in [
    ("Strict (medical)", 0.01, 0.3),
    ("Default", 0.05, 0.2),
    ("Lenient (content)", 0.1, 0.1),
]:
    result = detect_regression(
        baseline, candidate, alpha=alpha, min_effect_size=mes
    )
    status = "REGRESSION" if result["is_regression"] else "OK"
    print(f"{label:<20}: {status} "
          f"(alpha={alpha}, mes={mes})")
```

### Exercise 1.3: False positive rate estimation

**Task:** Generate 100 pairs of random score arrays from the same distribution (no actual regression). Run the regression detector on each pair. What percentage of tests flag a regression? This is your empirical false positive rate.

**Hint:** Use `numpy.random.normal()` to generate scores from the same distribution.

### Solution

```python
import numpy as np
from regression_detector import detect_regression

np.random.seed(42)
false_positives = 0
n_trials = 100

for trial in range(n_trials):
    # Generate two samples from the same distribution
    baseline = list(np.random.normal(0.85, 0.06, 20))
    candidate = list(np.random.normal(0.85, 0.06, 20))

    # Clip to valid range
    baseline = [max(0, min(1, s)) for s in baseline]
    candidate = [max(0, min(1, s)) for s in candidate]

    result = detect_regression(baseline, candidate)

    if result["is_regression"]:
        false_positives += 1

print(f"False positive rate: {false_positives}/{n_trials} "
      f"({100*false_positives/n_trials:.1f}%)")

# Expected: ~5% (matching alpha=0.05)
# If it's much higher, the tests are too sensitive for your sample size
```

## Tier 2 — Intermediate

### Exercise 2.1: Progressive regression detection

**Task:** You deploy a new prompt on Monday. Every day for 7 days, you run the eval suite and record the scores. The regression is gradual — the model's performance degrades slowly over the week. Build a system that detects the regression as early as possible (ideally by day 2-3) by comparing each day's scores against the baseline.

**Hint:** Simulate the gradual degradation by adding a small negative drift each day. Compare each day using `detect_regression()`.

### Solution

```python
import numpy as np
from regression_detector import detect_regression

# Baseline (production)
baseline = list(np.random.normal(0.87, 0.05, 30))

# Simulate gradual degradation
np.random.seed(42)
for day in range(1, 8):
    # Add daily drift: 0.01 per day
    drift = -0.01 * day
    day_scores = list(
        np.random.normal(0.87 + drift, 0.06, 30)
    )
    day_scores = [max(0, min(1, s)) for s in day_scores]

    result = detect_regression(baseline, day_scores)
    status = "⚠ REGRESSION" if result["is_regression"] else "OK"
    b_mean = result["baseline"]["mean"]
    c_mean = result["candidate"]["mean"]
    print(f"Day {day}: mean={c_mean:.3f} (vs baseline {b_mean:.3f}) "
          f"[{status}]")
```

On which day does the regression detection trigger? The P5 test should trigger first, followed by the mean shift test.

### Exercise 2.2: Regression source localization

**Task:** A regression is detected. You need to find which specific prompts in the golden dataset are the worst offenders. Write a function that, given two sets of entry-level results, returns the top 5 prompts where the score dropped the most. These are the prompts that need investigation.

**Hint:** Compare the per-entry scores between baseline and candidate. Sort by score difference (ascending) and take the top 5.

### Solution

```python
def locate_regression_sources(
    baseline_entries: list[dict],
    candidate_entries: list[dict],
    top_n: int = 5,
) -> list[dict]:
    """
    Find the prompts most affected by a regression.
    """
    # Build prompt -> score map
    b_map = {e["index"]: e["score"] for e in baseline_entries}
    c_map = {e["index"]: e["score"] for e in candidate_entries}

    diffs = []
    for idx in b_map:
        if idx in c_map:
            diff = c_map[idx] - b_map[idx]
            diffs.append({
                "index": idx,
                "prompt": next(
                    e["prompt"] for e in baseline_entries
                    if e["index"] == idx
                ),
                "category": next(
                    e["category"] for e in baseline_entries
                    if e["index"] == idx
                ),
                "baseline_score": b_map[idx],
                "candidate_score": c_map[idx],
                "difference": diff,
            })

    # Sort by worst drop first
    diffs.sort(key=lambda d: d["difference"])
    return diffs[:top_n]


# Example usage
from evaluator import run_eval_suite

baseline = run_eval_suite(
    "You are a helpful assistant. Answer accurately and concisely."
)
candidate = run_eval_suite(
    "You are a helpful assistant. Be extremely concise. "
    "Use as few words as possible."
)

sources = locate_regression_sources(
    baseline["entries"], candidate["entries"]
)

print("Top regression sources:")
for s in sources:
    print(f"  [{s['category']}] {s['prompt'][:40]}...")
    print(f"    Score: {s['baseline_score']:.3f} -> "
          f"{s['candidate_score']:.3f} ({s['difference']:+.3f})")
```

### Exercise 2.3: Multi-model regression comparison

**Task:** You have three candidate models to replace your current production model. Run the regression test suite on all three. Which model is the safest replacement? Which one has the largest regression risk?

**Hint:** Run `detect_regression()` on the scores from each model. Compare the effect sizes and the number of regression signals.

### Solution

```python
from evaluator import run_eval_suite
from regression_detector import detect_regression

# Production baseline
baseline = run_eval_suite(
    "You are a helpful assistant. Answer accurately and concisely.",
    model="gpt-4o-mini",
    temperature=0.5,
)

# Candidate models
candidates = {
    "gpt-4o-mini-v2": "gpt-4o-mini",
    "gpt-4o": "gpt-4o",
    "local-llama": "llama3.2",  # via Ollama
}

for label, model in candidates.items():
    try:
        candidate = run_eval_suite(
            "You are a helpful assistant. Answer accurately and concisely.",
            model=model,
            temperature=0.5,
        )
        result = detect_regression(
            baseline["scores"], candidate["scores"]
        )

        print(f"\n--- {label} ({model}) ---")
        print(f"  Mean: {result['candidate']['mean']:.3f} "
              f"(baseline: {result['baseline']['mean']:.3f})")
        print(f"  Effect size: {result['mean_test']['cohens_d']:.2f}")
        print(f"  Regressions: {len(result['regressions'])}")

        if result["is_regression"]:
            print(f"  ⚠ BLOCKED")
        else:
            print(f"  ✓ Approved for testing")

    except Exception as e:
        print(f"\n--- {label} ({model}) --- ERROR: {e}")
```

## Tier 3 — Advanced

### Exercise 3.1: Continuous regression monitoring dashboard

**Task:** Build a lightweight monitoring system that stores eval scores in a SQLite database, runs regression detection on each new deployment, and generates a simple HTML report showing the score trends over time with control limits.

**Solution approach:**

```python
import sqlite3
import json
import numpy as np
from datetime import datetime
from regression_detector import detect_regression, analyze_scores


class RegressionMonitor:
    """Tracks eval scores over time and detects regressions."""

    def __init__(self, db_path: str = "regression_history.db"):
        self.conn = sqlite3.connect(db_path)
        self.conn.execute("""
            CREATE TABLE IF NOT EXISTS deployments (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                version TEXT,
                timestamp TEXT,
                scores_json TEXT,
                system_prompt TEXT,
                model TEXT
            )
        """)
        self.conn.commit()

    def record_deployment(
        self,
        version: str,
        scores: list[float],
        system_prompt: str,
        model: str,
    ):
        """Record a deployment's eval scores."""
        self.conn.execute(
            """INSERT INTO deployments
               (version, timestamp, scores_json, system_prompt, model)
               VALUES (?, ?, ?, ?, ?)""",
            (
                version,
                datetime.now().isoformat(),
                json.dumps(scores),
                system_prompt,
                model,
            ),
        )
        self.conn.commit()

    def check_new_deployment(
        self,
        version: str,
        scores: list[float],
        system_prompt: str,
        model: str,
    ) -> dict:
        """
        Check a new deployment against the last known-good baseline.
        """
        cursor = self.conn.execute(
            """SELECT scores_json, version FROM deployments
               ORDER BY id DESC LIMIT 1"""
        )
        row = cursor.fetchone()

        if row is None:
            # First deployment — no baseline to compare
            self.record_deployment(
                version, scores, system_prompt, model
            )
            return {
                "status": "baseline_established",
                "message": "First deployment recorded as baseline.",
            }

        baseline_scores = json.loads(row[0])
        baseline_version = row[1]

        result = detect_regression(baseline_scores, scores)

        self.record_deployment(
            version, scores, system_prompt, model
        )

        return {
            "status": "regression" if result["is_regression"] else "ok",
            "baseline_version": baseline_version,
            "current_version": version,
            "baseline_mean": result["baseline"]["mean"],
            "current_mean": result["candidate"]["mean"],
            "effect_size": result["mean_test"]["cohens_d"],
            "regressions": result["regressions"],
            "p_value": result["mean_test"]["p_value"],
        }

    def generate_report(self) -> str:
        """Generate an HTML report of deployment history."""
        cursor = self.conn.execute(
            """SELECT version, timestamp, scores_json, model
               FROM deployments ORDER BY id"""
        )
        rows = cursor.fetchall()

        html_parts = ["<html><body><h1>Regression Monitor Report</h1>",
                      "<table border='1'>",
                      "<tr><th>Version</th><th>Date</th><th>Model</th>"
                      "<th>Mean</th><th>Std</th><th>P5</th></tr>"]

        all_means = []
        for version, ts, scores_json, model in rows:
            scores = json.loads(scores_json)
            stats = analyze_scores(scores)
            all_means.append(stats["mean"])
            html_parts.append(
                f"<tr><td>{version}</td><td>{ts[:10]}</td>"
                f"<td>{model}</td><td>{stats['mean']:.3f}</td>"
                f"<td>{stats['std']:.3f}</td>"
                f"<td>{stats['p5']:.3f}</td></tr>"
            )

        html_parts.append("</table></body></html>")
        return "\n".join(html_parts)


# Example
monitor = RegressionMonitor()

# Record first deployment
monitor.record_deployment(
    "v1.0", [0.87, 0.85, 0.90, 0.88, 0.86],
    "Original prompt", "gpt-4o-mini"
)

# Check new deployment
result = monitor.check_new_deployment(
    "v1.1", [0.85, 0.82, 0.88, 0.86, 0.84],
    "Updated prompt", "gpt-4o-mini"
)
print(f"Status: {result['status']}")
print(f"Effect size: {result['effect_size']:.2f}")

# Save report
with open("regression_report.html", "w") as f:
    f.write(monitor.generate_report())
print("Report saved to regression_report.html")
```

### Exercise 3.2: Cross-validated regression detection

**Task:** The current regression detector uses a single split of the golden dataset. Build a cross-validated version that splits the data into 5 folds, runs the regression detection on each fold, and only declares a regression if at least 4 out of 5 folds agree. This reduces false positives from noisy data.

**Solution approach:**

```python
import numpy as np
from sklearn.model_selection import KFold
from regression_detector import detect_regression


def cross_validated_regression(
    baseline_scores: list[float],
    candidate_scores: list[float],
    n_folds: int = 5,
    min_agree: int = 4,
) -> dict:
    """
    Cross-validated regression detection.

    Splits data into n_folds, runs detection on each fold,
    and only declares regression if min_agree folds agree.
    """
    baseline = np.array(baseline_scores)
    candidate = np.array(candidate_scores)

    if len(baseline) != len(candidate):
        raise ValueError("Baseline and candidate must have same length")

    kf = KFold(n_splits=n_folds, shuffle=True, random_state=42)
    fold_results = []

    for fold, (train_idx, test_idx) in enumerate(kf.split(baseline)):
        b_train = baseline[train_idx].tolist()
        c_train = candidate[train_idx].tolist()

        result = detect_regression(b_train, c_train)
        fold_results.append({
            "fold": fold,
            "is_regression": result["is_regression"],
            "mean_diff": result["mean_test"]["mean_difference"],
            "cohens_d": result["mean_test"]["cohens_d"],
            "regressions": result["regressions"],
        })

    regression_count = sum(
        1 for r in fold_results if r["is_regression"]
    )

    return {
        "is_regression": regression_count >= min_agree,
        "regression_folds": regression_count,
        "total_folds": n_folds,
        "mean_difference_across_folds": np.mean([
            r["mean_diff"] for r in fold_results
        ]),
        "fold_results": fold_results,
    }


# Example with known regression
baseline = [0.88, 0.85, 0.90, 0.87, 0.86, 0.89, 0.84, 0.91,
            0.83, 0.88, 0.86, 0.90, 0.87, 0.85, 0.89]
candidate = [0.85, 0.82, 0.88, 0.84, 0.83, 0.86, 0.80, 0.88,
             0.79, 0.85, 0.82, 0.87, 0.84, 0.81, 0.86]

result = cross_validated_regression(baseline, candidate)

print(f"Regression detected: {result['is_regression']}")
print(f"Folds flagging regression: "
      f"{result['regression_folds']}/{result['total_folds']}")
print(f"Mean difference: {result['mean_difference_across_folds']:.3f}")
```

### Exercise 3.3: Automated rollback decision system

**Task:** Build a system that, when a regression is detected, automatically determines whether to rollback. The rollback decision should consider: (1) the severity of the regression (effect size), (2) the number of dimensions affected, (3) whether safety-related dimensions are affected, and (4) the cost of rollback (service disruption, re-deployment time).

**Solution approach:**

```python
from regression_detector import detect_regression


class RollbackDecider:
    """
    Determines whether to rollback based on regression severity.
    """

    SEVERITY_WEIGHTS = {
        "safety": 3.0,
        "factuality": 2.0,
        "relevance": 1.0,
        "tone": 0.75,
        "conciseness": 0.5,
    }

    def __init__(
        self,
        safety_dimensions: list[str] = None,
        rollback_cost: float = 1.0,
    ):
        self.safety_dimensions = safety_dimensions or ["safety"]
        self.rollback_cost = rollback_cost

    def evaluate_rollback(
        self,
        dimension_results: dict[str, dict],
    ) -> dict:
        """
        Evaluate whether to rollback.

        Args:
            dimension_results: Dict mapping dimension names to
                regression detection results.

        Returns:
            Dict with rollback decision and reasoning.
        """
        severity_score = 0.0
        regression_details = []
        safety_affected = False

        for dim, result in dimension_results.items():
            if not result["is_regression"]:
                continue

            # Base weight
            weight = self.SEVERITY_WEIGHTS.get(dim, 1.0)

            # Amplify by effect size
            effect = abs(result["mean_test"]["cohens_d"])
            weighted = weight * effect

            # Safety multiplier
            if dim in self.safety_dimensions:
                weighted *= self.SEVERITY_WEIGHTS["safety"]
                safety_affected = True

            severity_score += weighted
            regression_details.append({
                "dimension": dim,
                "effect_size": effect,
                "weighted_score": weighted,
                "mean_drop": result["mean_test"]["mean_difference"],
            })

        # Decision logic
        ROLLBACK_THRESHOLD = 2.0
        should_rollback = severity_score >= ROLLBACK_THRESHOLD

        if safety_affected and severity_score >= 1.0:
            should_rollback = True  # Safety always wins

        if not should_rollback and self.rollback_cost > 2.0:
            # High cost of rollback — require stronger evidence
            should_rollback = severity_score >= ROLLBACK_THRESHOLD * 1.5

        return {
            "should_rollback": should_rollback,
            "severity_score": severity_score,
            "safety_affected": safety_affected,
            "regression_count": len(regression_details),
            "details": regression_details,
            "reason": self._build_reason(
                should_rollback, severity_score,
                safety_affected, regression_details,
            ),
        }

    def _build_reason(
        self, should_rollback, severity, safety, details
    ) -> str:
        if should_rollback and safety:
            return (
                f"Rollback required: safety dimension affected "
                f"(severity={severity:.1f})"
            )
        if should_rollback:
            dims = [d["dimension"] for d in details]
            return (
                f"Rollback recommended: regression in "
                f"{', '.join(dims)} (severity={severity:.1f})"
            )
        return (
            f"No rollback: severity {severity:.1f} below threshold"
        )


# Example
decider = RollbackDecider(safety_dimensions=["safety"])

# Simulate dimension results
dimension_results = {
    "factuality": detect_regression(
        [0.88, 0.85, 0.90] * 5,
        [0.85, 0.82, 0.88] * 5,
    ),
    "safety": detect_regression(
        [0.95, 0.97, 0.96] * 5,
        [0.88, 0.92, 0.90] * 5,
    ),
    "relevance": detect_regression(
        [0.90, 0.88, 0.92] * 5,
        [0.89, 0.87, 0.90] * 5,
    ),
}

decision = decider.evaluate_rollback(dimension_results)
print(f"Rollback: {decision['should_rollback']}")
print(f"Severity: {decision['severity_score']:.1f}")
print(f"Safety affected: {decision['safety_affected']}")
print(f"Reason: {decision['reason']}")
```

**Discussion:** In a production system, the rollback decider would be integrated into your CI/CD pipeline. When a deployment triggers a regression alert, the pipeline runs the rollback evaluation and automatically reverts the change if the severity exceeds the threshold. For safety-related regressions, the threshold is intentionally lower — even a small safety regression is unacceptable.
