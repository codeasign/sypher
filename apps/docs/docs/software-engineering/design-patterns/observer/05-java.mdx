## Implementation

Below is a complete Java implementation of the Observer pattern using a stock market domain. A `StockExchange` (Subject) tracks prices for AAPL, GOOGL, and MSFT. Three observers — `PriceDisplay`, `AlertSystem`, and `PortfolioTracker` — register with the exchange and react to price changes. The implementation uses Java's interface system for the Observer contract and demonstrates modern Java 17+ features.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────┐
//   │   interface Observer              │
//   ├──────────────────────────────────┤
//   │   void update(String, double)    │ ← OOP: Abstraction via interface
//   └────────────────┬─────────────────┘    ISP: single-method contract
//                    ▲ implements
//   ┌────────────────┴─────────────────┐
//   │                │                 │
//   ┌────────────┐ ┌──────────┐ ┌──────────────┐
//   │PriceDisplay│ │AlertSys  │ │Portfolio     │ ← OOP: Polymorphism
//   ├────────────┤ ├──────────┤ ├──────────────┤    LSP: all implement
//   │ update()   │ │ update() │ │ update()     │     Observer
//   └────────────┘ └──────────┘ └──────────────┘
//
//   ┌──────────────────────────────────┐
//   │  StockExchange                   │ ← OOP: Encapsulation
//   ├──────────────────────────────────┤    SRP: prices + observers
//   │  attach(Observer)                │     OCP: new observers
//   │  detach(Observer)                │     via attach() only
//   │  setPrice(String, double)        │    DIP: depends on Observer
//   │  getPrice(String): double        │     interface
//   └──────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — The `Observer` interface defines the notification contract. `StockExchange` depends on `Observer`, not on concrete observer classes.
2. **Encapsulation** — `StockExchange` hides its `List<Observer>` and `Map<String, Double>` behind private access modifiers. Each observer encapsulates its own state.
3. **Inheritance** — Java's interface inheritance: `PriceDisplay`, `AlertSystem`, and `PortfolioTracker` all implement `Observer` and provide the `update()` method.
4. **Polymorphism** — The `for` loop in `notifyObservers()` iterates over `Observer` references and calls `update()` polymorphically.

#### SOLID Principles Applied

5. **SRP** — `StockExchange` manages only prices and observer registration. Each observer has a single responsibility.
6. **OCP** — New observer types implement `Observer` and register via `attach()`. The `StockExchange` class is never modified.
7. **LSP** — Any `Observer` implementation can substitute for the interface type in the notification loop.
8. **ISP** — The `Observer` interface has a single method. No class is forced to depend on methods it does not use.
9. **DIP** — `StockExchange` depends on the `Observer` abstraction. Concrete observers are injected via `attach()`.

### Observer Interface

```java
// Observer Pattern — Java — Stock Market example
// ===============================================
// Participants: StockExchange (Subject), PriceDisplay,
// AlertSystem, PortfolioTracker (Concrete Observers)
// OOP: abstraction, encapsulation, inheritance, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// ===============================================

import java.util.*;

// ── Observer Interface ──
interface Observer {
    void update(String ticker, double price);
}
```

### StockExchange (Subject)

```java
// ── Subject ──
class StockExchange {
    private final List<Observer> observers = new ArrayList<>();
    private final Map<String, Double> prices = new HashMap<>();

    /** Register an observer. */
    public void attach(Observer observer) {
        observers.add(observer);
    }

    /** Unregister an observer. */
    public void detach(Observer observer) {
        observers.remove(observer);
    }

    /** Notify all registered observers of a price change. */
    private void notifyObservers(String ticker, double price) {
        for (Observer obs : observers) {
            obs.update(ticker, price);
        }
    }

    /** Set a new price and broadcast the change. */
    public void setPrice(String ticker, double price) {
        prices.put(ticker, price);
        notifyObservers(ticker, price);
    }

    /** Get the current price of a ticker. */
    public double getPrice(String ticker) {
        return prices.getOrDefault(ticker, 0.0);
    }
}
```

### PriceDisplay (Concrete Observer)

```java
// ── Concrete Observer 1: PriceDisplay ──
class PriceDisplay implements Observer {
    @Override
    public void update(String ticker, double price) {
        System.out.printf("DISPLAY: %s is now $%.2f%n", ticker, price);
    }
}
```

### AlertSystem (Concrete Observer)

```java
// ── Concrete Observer 2: AlertSystem ──
class AlertSystem implements Observer {
    private final Map<String, Double> previousPrices = new HashMap<>();
    private final double threshold;

    public AlertSystem(double thresholdPercent) {
        this.threshold = thresholdPercent;
    }

    @Override
    public void update(String ticker, double price) {
        if (previousPrices.containsKey(ticker)) {
            double oldPrice = previousPrices.get(ticker);
            double change = Math.abs(price - oldPrice) / oldPrice * 100.0;
            if (change >= threshold) {
                String direction = price >= oldPrice ? "up" : "down";
                System.out.printf(
                    "ALERT: %s moved %s %.1f%% ($%.2f -> $%.2f)%n",
                    ticker, direction, change, oldPrice, price
                );
            }
        }
        previousPrices.put(ticker, price);
    }
}
```

### PortfolioTracker (Concrete Observer)

```java
// ── Concrete Observer 3: PortfolioTracker ──
class PortfolioTracker implements Observer {
    private final Map<String, Integer> holdings = new HashMap<>();
    private double totalValue = 0.0;
    private final StockExchange exchange;

    public PortfolioTracker(StockExchange exchange) {
        this.exchange = exchange;
    }

    /** Add a stock holding to the portfolio. */
    public void addHolding(String ticker, int shares) {
        holdings.put(ticker, shares);
    }

    /** Recalculate total portfolio value from all holdings. */
    private void recalculate() {
        totalValue = 0.0;
        for (Map.Entry<String, Integer> entry : holdings.entrySet()) {
            double price = exchange.getPrice(entry.getKey());
            totalValue += price * entry.getValue();
        }
    }

    @Override
    public void update(String ticker, double price) {
        if (holdings.containsKey(ticker)) {
            recalculate();
            System.out.printf("PORTFOLIO: Updated value is $%.2f%n", totalValue);
        }
    }

    public double getTotalValue() {
        return totalValue;
    }
}
```

### Main / Driver

```java
// ── Main / Driver ──
public class ObserverDemo {
    public static void main(String[] args) {
        StockExchange exchange = new StockExchange();

        PriceDisplay display = new PriceDisplay();
        AlertSystem alerts = new AlertSystem(5.0);
        PortfolioTracker portfolio = new PortfolioTracker(exchange);

        // Register observers
        exchange.attach(display);
        exchange.attach(alerts);
        exchange.attach(portfolio);

        // Set up portfolio holdings
        portfolio.addHolding("AAPL", 10);
        portfolio.addHolding("GOOGL", 5);
        portfolio.addHolding("MSFT", 8);

        System.out.println("=== Stock Market Observer Demo ===");
        System.out.println();

        exchange.setPrice("AAPL", 150.00);
        exchange.setPrice("GOOGL", 2800.00);
        exchange.setPrice("MSFT", 300.00);

        System.out.println();

        // Detach display before a large price swing
        exchange.detach(display);
        exchange.setPrice("AAPL", 162.00);  // 8% change — alert fires, no display
        exchange.attach(display);

        System.out.println();

        exchange.setPrice("GOOGL", 2750.00);  // < 5% — no alert
        exchange.setPrice("MSFT", 315.00);     // 5% change — alert fires
    }
}
```

### Language-Specific Notes

- **Interface contract**: Java's `interface` keyword creates a formal contract that all observer implementations must satisfy. The `@Override` annotation ensures the compiler catches mismatched signatures.
- **Collections API**: `ArrayList` and `HashMap` are the standard Java collections for the observer list and price map. Using `List<Observer>` and `Map<String, Double>` provides type safety and standard iteration patterns.
- **`printf` for formatted output**: Java's `System.out.printf` with `%s` (string), `%.2f` (2 decimal places), and `%n` (platform-independent newline) provides precise control over output formatting.
- **`getOrDefault`**: `Map.getOrDefault(key, defaultValue)` avoids null checks when querying prices that may not exist yet. This is a Java 8+ addition that simplifies the code.
- **Single public class per file**: Java requires the public class (`ObserverDemo`) to match the filename. The other classes are package-private (no access modifier) and can coexist in the same file.
