## Implementation

Below is a complete TypeScript implementation of the Observer pattern using a stock market domain. A `StockExchange` (Subject) manages prices for AAPL, GOOGL, and MSFT. Three observers — `PriceDisplay`, `AlertSystem`, and `PortfolioTracker` — register with the exchange and react to price changes. The implementation uses TypeScript's strict typing, interfaces, and generics to provide type-safe observer registration and notification.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────┐
//   │   interface Observer<T>           │
//   ├──────────────────────────────────┤
//   │ + update(ticker: T, price: num)  │ ← OOP: Abstraction via interface
//   └────────────────┬─────────────────┘    ISP: single-method interface
//                    ▲ implements
//   ┌────────────────┴─────────────────┐
//   │                │                 │
//   ┌────────────┐ ┌──────────┐ ┌──────────────┐
//   │PriceDisplay│ │AlertSys  │ │Portfolio     │ ← OOP: Polymorphism
//   ├────────────┤ ├──────────┤ ├──────────────┤    LSP: all implement
//   │ update()   │ │ update() │ │ update()     │     Observer<string>
//   └────────────┘ └──────────┘ └──────────────┘
//
//   ┌──────────────────────────────────┐
//   │  StockExchange                   │ ← OOP: Encapsulation
//   ├──────────────────────────────────┤    SRP: one responsibility
//   │  + attach(obs: Observer<string>) │    OCP: new observers via
//   │  + detach(obs: Observer<string>) │     attach() without modifying
//   │  + setPrice(t: string, p: num)   │     StockExchange
//   │  + getPrice(t: string): num      │    DIP: depends on Observer
//   └──────────────────────────────────┘     interface
```

#### OOP Concepts Applied

1. **Abstraction** — The `Observer<T>` interface defines the notification contract. StockExchange depends on this interface, not on concrete observer classes.
2. **Encapsulation** — StockExchange hides its internal observer array and price map. Each observer encapsulates its own state — AlertSystem tracks previous prices, PortfolioTracker holds its share quantities.
3. **Inheritance** — TypeScript interfaces support implementation inheritance. Each concrete observer class implements the `Observer<string>` interface.
4. **Polymorphism** — StockExchange iterates over `Observer<string>[]` and calls `update()` on each reference. The runtime type of each observer determines the behavior.

#### SOLID Principles Applied

5. **SRP** — StockExchange manages prices and observer registration only. Each observer has a single domain responsibility.
6. **OCP** — New observer types (e.g., `ChartRenderer`, `TradeExecutor`) can be added by implementing `Observer<string>` and calling `attach()`. StockExchange is never modified.
7. **LSP** — Every `Observer<string>` implementation is substitutable for the interface type. The notification loop treats all observers identically.
8. **ISP** — The `Observer<T>` interface has a single method. No class implements methods it does not use.
9. **DIP** — StockExchange depends on the `Observer<string>` abstraction. Concrete observers are injected via `attach()`.

### Observer Interface

```typescript
// Observer Pattern — TypeScript — Stock Market example
// =====================================================
// Participants: StockExchange (Subject), PriceDisplay,
// AlertSystem, PortfolioTracker (Concrete Observers)
// OOP: abstraction, encapsulation, inheritance, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// =====================================================

// ── Observer Interface ──
interface Observer<T extends string> {
  update(ticker: T, price: number): void;
}
```

### StockExchange (Subject)

```typescript
// ── Subject ──
class StockExchange<T extends string> {
  private observers: Observer<T>[] = [];
  private prices: Map<T, number> = new Map();

  /** Register an observer */
  attach(observer: Observer<T>): void {
    this.observers.push(observer);
  }

  /** Unregister an observer */
  detach(observer: Observer<T>): void {
    const idx = this.observers.indexOf(observer);
    if (idx !== -1) {
      this.observers.splice(idx, 1);
    }
  }

  /** Notify all registered observers */
  private notifyObservers(ticker: T, price: number): void {
    for (const obs of this.observers) {
      obs.update(ticker, price);
    }
  }

  /** Set a new price and broadcast the change */
  setPrice(ticker: T, price: number): void {
    this.prices.set(ticker, price);
    this.notifyObservers(ticker, price);
  }

  /** Get the current price of a ticker */
  getPrice(ticker: T): number {
    return this.prices.get(ticker) ?? 0;
  }
}
```

### PriceDisplay (Concrete Observer)

```typescript
// ── Concrete Observer 1: PriceDisplay ──
class PriceDisplay<T extends string> implements Observer<T> {
  update(ticker: T, price: number): void {
    console.log(`DISPLAY: ${ticker} is now $${price.toFixed(2)}`);
  }
}
```

### AlertSystem (Concrete Observer)

```typescript
// ── Concrete Observer 2: AlertSystem ──
class AlertSystem<T extends string> implements Observer<T> {
  private previousPrices: Map<T, number> = new Map();
  private readonly threshold: number;

  constructor(thresholdPercent: number) {
    this.threshold = thresholdPercent;
  }

  update(ticker: T, price: number): void {
    if (this.previousPrices.has(ticker)) {
      const oldPrice = this.previousPrices.get(ticker)!;
      const change = Math.abs(price - oldPrice) / oldPrice * 100;
      if (change >= this.threshold) {
        const direction = price >= oldPrice ? 'up' : 'down';
        console.log(
          `ALERT: ${ticker} moved ${direction} ${change.toFixed(1)}% ` +
          `($${oldPrice.toFixed(2)} → $${price.toFixed(2)})`
        );
      }
    }
    this.previousPrices.set(ticker, price);
  }
}
```

### PortfolioTracker (Concrete Observer)

```typescript
// ── Concrete Observer 3: PortfolioTracker ──
class PortfolioTracker<T extends string> implements Observer<T> {
  private holdings: Map<T, number> = new Map();
  private totalValue: number = 0;
  private readonly exchange: StockExchange<T>;

  constructor(exchange: StockExchange<T>) {
    this.exchange = exchange;
  }

  /** Add a stock holding to the portfolio */
  addHolding(ticker: T, shares: number): void {
    this.holdings.set(ticker, shares);
  }

  /** Recalculate total value of all holdings */
  private recalculate(): void {
    this.totalValue = 0;
    for (const [ticker, shares] of this.holdings) {
      const price = this.exchange.getPrice(ticker);
      this.totalValue += price * shares;
    }
  }

  update(ticker: T, _price: number): void {
    if (this.holdings.has(ticker)) {
      this.recalculate();
      console.log(`PORTFOLIO: Updated value is $${this.totalValue.toFixed(2)}`);
    }
  }

  getTotalValue(): number {
    return this.totalValue;
  }
}
```

### Main / Driver

```typescript
// ── Main / Driver ──
type Ticker = 'AAPL' | 'GOOGL' | 'MSFT';

const exchange = new StockExchange<Ticker>();

const display = new PriceDisplay<Ticker>();
const alerts = new AlertSystem<Ticker>(5.0);
const portfolio = new PortfolioTracker<Ticker>(exchange);

// Register observers
exchange.attach(display);
exchange.attach(alerts);
exchange.attach(portfolio);

// Set up portfolio holdings
portfolio.addHolding('AAPL', 10);
portfolio.addHolding('GOOGL', 5);
portfolio.addHolding('MSFT', 8);

console.log('=== Stock Market Observer Demo ===');
console.log('');

exchange.setPrice('AAPL', 150.00);
exchange.setPrice('GOOGL', 2800.00);
exchange.setPrice('MSFT', 300.00);

console.log('');

// Detach display before a large price swing
exchange.detach(display);
exchange.setPrice('AAPL', 162.00);  // 8% change — alert fires, no display
exchange.attach(display);

console.log('');

exchange.setPrice('GOOGL', 2750.00);  // < 5% — no alert
exchange.setPrice('MSFT', 315.00);     // 5% change — alert fires
```

### Language-Specific Notes

- **Generic Observer interface**: `Observer<T extends string>` uses TypeScript generics to constrain the ticker type to string literals (`'AAPL' | 'GOOGL' | 'MSFT'`). This provides compile-time type safety when calling `setPrice()` — passing an invalid ticker causes a type error.
- **Strict mode**: The code uses TypeScript's strict mode with no implicit `any`. The `private` keyword enforces encapsulation at the type level, and the `readonly` modifier prevents reassignment of threshold and exchange reference.
- **Discriminated union for tickers**: The `Ticker` type alias (`'AAPL' | 'GOOGL' | 'MSFT'`) creates a closed set of valid ticker symbols. This is idiomatic TypeScript for modeling a fixed set of string values.
- **Map with generic keys**: `Map<T, number>` uses the generic type parameter as the key type, ensuring that only valid ticker strings can be used as map keys.
- **Non-null assertion**: The `!` operator in `this.previousPrices.get(ticker)!` is safe because the `has()` check immediately precedes the `get()` call.
