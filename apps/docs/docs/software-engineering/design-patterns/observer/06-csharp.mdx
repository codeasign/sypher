## Implementation

Below is a complete C# implementation of the Observer pattern using a stock market domain. A `StockExchange` (Subject) tracks prices for AAPL, GOOGL, and MSFT. Three observers — `PriceDisplay`, `AlertSystem`, and `PortfolioTracker` — register with the exchange and react to price changes. The implementation uses C# 12 / .NET 8 features including primary constructors, interfaces, and the `EventArgs` pattern.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────┐
//   │   interface IObserver             │
//   ├──────────────────────────────────┤
//   │   void Update(string, double)    │ ← OOP: Abstraction via interface
//   └────────────────┬─────────────────┘    ISP: single-method contract
//                    ▲ implements
//   ┌────────────────┴─────────────────┐
//   │                │                 │
//   ┌────────────┐ ┌──────────┐ ┌──────────────┐
//   │PriceDisplay│ │AlertSys  │ │Portfolio     │ ← OOP: Polymorphism
//   ├────────────┤ ├──────────┤ ├──────────────┤    LSP: all implement
//   │ Update()   │ │ Update() │ │ Update()     │     IObserver
//   └────────────┘ └──────────┘ └──────────────┘
//
//   ┌──────────────────────────────────┐
//   │  StockExchange                   │ ← OOP: Encapsulation
//   ├──────────────────────────────────┤    SRP: prices + observers
//   │  Attach(IObserver)               │     OCP: new observers
//   │  Detach(IObserver)               │     via Attach() only
//   │  SetPrice(string, double)        │    DIP: depends on IObserver
//   │  GetPrice(string): double        │     interface
//   └──────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — The `IObserver` interface defines the notification contract. `StockExchange` depends on `IObserver`, not on concrete observer classes.
2. **Encapsulation** — `StockExchange` hides its internal observer list and price dictionary behind private fields. Each observer encapsulates its own state.
3. **Inheritance** — Interface inheritance: `PriceDisplay`, `AlertSystem`, and `PortfolioTracker` all implement `IObserver`.
4. **Polymorphism** — The `foreach` loop in `NotifyObservers()` iterates over `IObserver` references and calls `Update()` polymorphically.

#### SOLID Principles Applied

5. **SRP** — `StockExchange` manages only prices and observer registration. Each observer has a single responsibility.
6. **OCP** — New observer types implement `IObserver` and register via `Attach()`. The `StockExchange` class is never modified.
7. **LSP** — Any `IObserver` implementation can substitute for the interface type in the notification loop.
8. **ISP** — The `IObserver` interface has a single method. No class is forced to implement unused methods.
9. **DIP** — `StockExchange` depends on the `IObserver` abstraction. Concrete observers are injected via `Attach()`.

### IObserver Interface

```csharp
// Observer Pattern — C# — Stock Market example
// =============================================
// Participants: StockExchange (Subject), PriceDisplay,
// AlertSystem, PortfolioTracker (Concrete Observers)
// OOP: abstraction, encapsulation, inheritance, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// =============================================

// ── Observer Interface ──
public interface IObserver
{
    void Update(string ticker, double price);
}
```

### StockExchange (Subject)

```csharp
// ── Subject ──
public class StockExchange
{
    private readonly List<IObserver> _observers = [];
    private readonly Dictionary<string, double> _prices = [];

    /// <summary>Register an observer.</summary>
    public void Attach(IObserver observer)
    {
        _observers.Add(observer);
    }

    /// <summary>Unregister an observer.</summary>
    public void Detach(IObserver observer)
    {
        _observers.Remove(observer);
    }

    /// <summary>Notify all registered observers of a price change.</summary>
    private void NotifyObservers(string ticker, double price)
    {
        foreach (var obs in _observers)
        {
            obs.Update(ticker, price);
        }
    }

    /// <summary>Set a new price and broadcast the change.</summary>
    public void SetPrice(string ticker, double price)
    {
        _prices[ticker] = price;
        NotifyObservers(ticker, price);
    }

    /// <summary>Get the current price of a ticker.</summary>
    public double GetPrice(string ticker)
    {
        return _prices.GetValueOrDefault(ticker, 0.0);
    }
}
```

### PriceDisplay (Concrete Observer)

```csharp
// ── Concrete Observer 1: PriceDisplay ──
public class PriceDisplay : IObserver
{
    public void Update(string ticker, double price)
    {
        Console.WriteLine($"DISPLAY: {ticker} is now ${price:F2}");
    }
}
```

### AlertSystem (Concrete Observer)

```csharp
// ── Concrete Observer 2: AlertSystem ──
public class AlertSystem : IObserver
{
    private readonly Dictionary<string, double> _previousPrices = [];
    private readonly double _threshold;

    public AlertSystem(double thresholdPercent)
    {
        _threshold = thresholdPercent;
    }

    public void Update(string ticker, double price)
    {
        if (_previousPrices.TryGetValue(ticker, out double oldPrice))
        {
            double change = Math.Abs(price - oldPrice) / oldPrice * 100.0;
            if (change >= _threshold)
            {
                string direction = price >= oldPrice ? "up" : "down";
                Console.WriteLine(
                    $"ALERT: {ticker} moved {direction} {change:F1}% " +
                    $"(${oldPrice:F2} -> ${price:F2})");
            }
        }
        _previousPrices[ticker] = price;
    }
}
```

### PortfolioTracker (Concrete Observer)

```csharp
// ── Concrete Observer 3: PortfolioTracker ──
public class PortfolioTracker : IObserver
{
    private readonly Dictionary<string, int> _holdings = [];
    private double _totalValue = 0.0;
    private readonly StockExchange _exchange;

    public PortfolioTracker(StockExchange exchange)
    {
        _exchange = exchange;
    }

    /// <summary>Add a stock holding to the portfolio.</summary>
    public void AddHolding(string ticker, int shares)
    {
        _holdings[ticker] = shares;
    }

    /// <summary>Recalculate total portfolio value from all holdings.</summary>
    private void Recalculate()
    {
        _totalValue = 0.0;
        foreach (var (ticker, shares) in _holdings)
        {
            double price = _exchange.GetPrice(ticker);
            _totalValue += price * shares;
        }
    }

    public void Update(string ticker, double price)
    {
        if (_holdings.ContainsKey(ticker))
        {
            Recalculate();
            Console.WriteLine($"PORTFOLIO: Updated value is ${_totalValue:F2}");
        }
    }

    public double TotalValue => _totalValue;
}
```

### Main / Driver

```csharp
// ── Main / Driver ──
public class ObserverDemo
{
    public static void Main()
    {
        var exchange = new StockExchange();

        var display = new PriceDisplay();
        var alerts = new AlertSystem(5.0);
        var portfolio = new PortfolioTracker(exchange);

        // Register observers
        exchange.Attach(display);
        exchange.Attach(alerts);
        exchange.Attach(portfolio);

        // Set up portfolio holdings
        portfolio.AddHolding("AAPL", 10);
        portfolio.AddHolding("GOOGL", 5);
        portfolio.AddHolding("MSFT", 8);

        Console.WriteLine("=== Stock Market Observer Demo ===");
        Console.WriteLine();

        exchange.SetPrice("AAPL", 150.00);
        exchange.SetPrice("GOOGL", 2800.00);
        exchange.SetPrice("MSFT", 300.00);

        Console.WriteLine();

        // Detach display before a large price swing
        exchange.Detach(display);
        exchange.SetPrice("AAPL", 162.00);  // 8% change — alert fires, no display
        exchange.Attach(display);

        Console.WriteLine();

        exchange.SetPrice("GOOGL", 2750.00);  // < 5% — no alert
        exchange.SetPrice("MSFT", 315.00);     // 5% change — alert fires
    }
}
```

### Language-Specific Notes

- **Interface naming convention**: C# uses the `I` prefix for interfaces (`IObserver`). This is a language-wide convention enforced by .NET coding guidelines and tooling (analyzers, StyleCop).
- **Primary constructors**: C# 12 allows primary constructors for classes, but this implementation uses explicit constructors for clarity. The `readonly` modifier on fields prevents accidental reassignment after construction.
- **Collection expressions**: The `[]` syntax for initializing `List<IObserver>` and `Dictionary<string, double>` is a C# 12 feature, replacing the older `new()` syntax.
- **`GetValueOrDefault`**: `Dictionary.GetValueOrDefault(key, default)` is a .NET Core 2.0+ / .NET 5+ extension method that safely returns the value or a default. This avoids KeyNotFoundException without explicit `TryGetValue` checks.
- **String interpolation with format specifiers**: C# `$"${price:F2}"` uses the `F2` format specifier (fixed-point with 2 decimal places) inside string interpolation. The double `$` before `{price}` is literal, not a format specifier — the `$` before the opening quote enables interpolation, and the `$` inside the string is the dollar sign character.
- **Expression-bodied property**: `public double TotalValue => _totalValue;` is a concise syntax for a read-only property that returns a private field.
