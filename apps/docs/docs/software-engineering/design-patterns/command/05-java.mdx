## Implementation

Below is a complete Java implementation of the Command pattern using a stock market trading domain. A `Broker` accepts buy and sell orders for stocks, executes them, supports cancellation (undo), and maintains a complete order history. Java's interface system provides the command contract, and records make the concrete command implementations concise and immutable.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────────────┐
//   │   interface Order                        │
//   │   ────────────────────────────────────── │
//   │   void execute()                         │ ← OOP: Abstraction
//   │   void undo()                            │    ISP: minimal contract
//   └──────────────────┬───────────────────────┘
//                      ▲ implements
//   ┌──────────────────┴───────────────────────┐
//   │                                          │
//   ┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
//   │   BuyOrder       │  │   SellOrder      │  │  CancelOrder     │
//   ├──────────────────┤  ├──────────────────┤  ├──────────────────┤
//   │ + execute()      │  │ + execute()      │  │ + execute()      │ ← OOP: Polymorphism
//   │   calls buy()    │  │   calls sell()   │  │   calls undo()   │    LSP: substitutable
//   │ + undo()         │  │ + undo()         │  │ + undo()         │    OCP: new order types
//   │   calls sell()   │  │   calls buy()    │  │   calls exec()   │
//   └────────┬─────────┘  └────────┬─────────┘  └────────┬─────────┘
//            │                     │                     │
//            ▼                     ▼                     ▼
//   ┌────────────────────────────────────────────────────────┐
//   │   class Stock (Receiver)                               │
//   ├────────────────────────────────────────────────────────┤
//   │   void buy(int quantity, double price)                 │ ← SRP: one responsibility
//   │   void sell(int quantity, double price)                │
//   │   String getInfo()                                     │
//   └────────────────────────────────────────────────────────┘
//
//   ┌────────────────────────────────────────────────────────┐
//   │   class Broker (Invoker)                               │
//   ├────────────────────────────────────────────────────────┤
//   │   List<Order> orderHistory                             │ ← SRP: manages order lifecycle
//   │   void placeOrder(Order order)                         │    DIP: depends on Order
//   │   void cancelLastOrder()                               │     interface, not concrete
//   │   void showHistory()                                   │     command types
//   └────────────────────────────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — The `Order` interface defines the contract. The `Broker` depends only on `Order`, never on concrete command classes.
2. **Encapsulation** — Each command hides its internal state. The `Stock` class encapsulates position data behind `buy()`, `sell()`, and `getInfo()` methods.
3. **Inheritance** — `BuyOrder`, `SellOrder`, and `CancelOrder` all implement the `Order` interface.
4. **Polymorphism** — The `Broker` calls `order.execute()` on any `Order` reference. The correct implementation dispatches per type at runtime.

#### SOLID Principles Applied

5. **SRP** — `Broker` only manages the order lifecycle. `Stock` only tracks position. Each command class encapsulates one trade type.
6. **OCP** — Adding a new `LimitOrder` command requires zero changes to `Broker` or `Stock`. Just implement the `Order` interface.
7. **LSP** — Every concrete command type is substitutable for `Order` without breaking the `Broker`.
8. **ISP** — The `Order` interface has only two methods. No class implements methods it does not need.
9. **DIP** — `Broker.placeOrder` accepts `Order` — an abstraction, not a concrete type.

### Order (Command Interface)

```java
interface Order {
    void execute();
    void undo();
}
```

### Stock (Receiver)

```java
class Stock {
    private final String ticker;
    private double price;
    private int sharesOutstanding;

    public Stock(String ticker, double price, int sharesOutstanding) {
        this.ticker = ticker;
        this.price = price;
        this.sharesOutstanding = sharesOutstanding;
    }

    public void buy(int quantity, double price) {
        this.sharesOutstanding += quantity;
        this.price = price;
        System.out.printf("Bought %d shares of %s at $%.2f%n",
            quantity, ticker, price);
    }

    public void sell(int quantity, double price) {
        if (this.sharesOutstanding >= quantity) {
            this.sharesOutstanding -= quantity;
            this.price = price;
            System.out.printf("Sold %d shares of %s at $%.2f%n",
                quantity, ticker, price);
        } else {
            System.out.printf("Insufficient shares of %s to sell %d%n",
                ticker, quantity);
        }
    }

    public String getInfo() {
        return String.format("%s: $%.2f/share, %d shares outstanding",
            ticker, price, sharesOutstanding);
    }

    public String getTicker() { return ticker; }
}
```

### BuyOrder (ConcreteCommand)

```java
class BuyOrder implements Order {
    private final Stock stock;
    private final int quantity;
    private final double price;

    public BuyOrder(Stock stock, int quantity, double price) {
        this.stock = stock;
        this.quantity = quantity;
        this.price = price;
    }

    @Override
    public void execute() {
        stock.buy(quantity, price);
    }

    @Override
    public void undo() {
        stock.sell(quantity, price);
    }

    public Stock getStock() { return stock; }
    public int getQuantity() { return quantity; }
    public double getPrice() { return price; }
}
```

### SellOrder (ConcreteCommand)

```java
class SellOrder implements Order {
    private final Stock stock;
    private final int quantity;
    private final double price;

    public SellOrder(Stock stock, int quantity, double price) {
        this.stock = stock;
        this.quantity = quantity;
        this.price = price;
    }

    @Override
    public void execute() {
        stock.sell(quantity, price);
    }

    @Override
    public void undo() {
        stock.buy(quantity, price);
    }

    public Stock getStock() { return stock; }
    public int getQuantity() { return quantity; }
    public double getPrice() { return price; }
}
```

### CancelOrder (ConcreteCommand)

```java
class CancelOrder implements Order {
    private final Order targetOrder;
    private boolean wasExecuted = false;

    public CancelOrder(Order targetOrder) {
        this.targetOrder = targetOrder;
    }

    @Override
    public void execute() {
        wasExecuted = true;
        targetOrder.undo();
        System.out.println("CancelOrder executed - original order undone");
    }

    @Override
    public void undo() {
        if (wasExecuted) {
            targetOrder.execute();
            System.out.println("CancelOrder undone - original order re-applied");
        }
    }

    public Order getTargetOrder() { return targetOrder; }
}
```

### Broker (Invoker)

```java
class Broker {
    private final List<Order> orderHistory = new ArrayList<>();

    public void placeOrder(Order order) {
        order.execute();
        orderHistory.add(order);
        System.out.println("Order recorded in history\n");
    }

    public void cancelLastOrder() {
        if (orderHistory.isEmpty()) {
            System.out.println("No orders to cancel\n");
            return;
        }
        Order lastOrder = orderHistory.remove(orderHistory.size() - 1);
        CancelOrder cancelCmd = new CancelOrder(lastOrder);
        cancelCmd.execute();
        orderHistory.add(cancelCmd);
        System.out.println("Cancellation recorded in history\n");
    }

    public void showHistory() {
        System.out.println("=== Order History ===");
        if (orderHistory.isEmpty()) {
            System.out.println("No orders in history");
            return;
        }
        for (int i = 0; i < orderHistory.size(); i++) {
            Order order = orderHistory.get(i);
            String typeName = order.getClass().getSimpleName();
            String details;
            if (order instanceof CancelOrder cmd) {
                details = "CancelOrder targeting "
                    + cmd.getTargetOrder().getClass().getSimpleName();
            } else if (order instanceof BuyOrder cmd) {
                details = cmd.getStock().getTicker() + " x"
                    + cmd.getQuantity() + " @ $" + String.format("%.2f", cmd.getPrice());
            } else if (order instanceof SellOrder cmd) {
                details = cmd.getStock().getTicker() + " x"
                    + cmd.getQuantity() + " @ $" + String.format("%.2f", cmd.getPrice());
            } else {
                details = "Unknown order type";
            }
            System.out.printf("  %d. %s: %s%n", i + 1, typeName, details);
        }
        System.out.println();
    }
}
```

### Main / Driver

```java
public class CommandDemo {
    public static void main(String[] args) {
        System.out.println("=== Trading Session Started ===\n");

        Stock aapl = new Stock("AAPL", 175.50, 1000);
        Stock googl = new Stock("GOOGL", 141.25, 800);
        Stock msft = new Stock("MSFT", 378.90, 1200);

        Broker broker = new Broker();

        BuyOrder buy1 = new BuyOrder(aapl, 100, 175.50);
        broker.placeOrder(buy1);

        SellOrder sell1 = new SellOrder(googl, 50, 141.25);
        broker.placeOrder(sell1);

        BuyOrder buy2 = new BuyOrder(msft, 200, 378.90);
        broker.placeOrder(buy2);

        broker.showHistory();

        System.out.println("=== Cancelling Last Order ===\n");
        broker.cancelLastOrder();

        broker.showHistory();

        System.out.println("=== Current Stock Positions ===");
        System.out.println(aapl.getInfo());
        System.out.println(googl.getInfo());
        System.out.println(msft.getInfo());
    }
}
```

### Language-Specific Notes

- **Interface as contract** — Java's `interface Order` defines the command contract. All command classes implement `Order`, and the `Broker` depends only on the interface. This is enforced at compile time.
- **Records (Java 16+)** — This example uses traditional classes for clarity, but Java 16+ records could replace `BuyOrder` and `SellOrder` with `record BuyOrder(Stock stock, int quantity, double price) implements Order`. Records auto-generate constructors, accessors, `equals()`, `hashCode()`, and `toString()`.
- **Pattern matching `instanceof`** — Java 17+ pattern matching for `instanceof` (e.g., `if (order instanceof BuyOrder cmd)`) eliminates explicit casts and makes the code more readable.
- **`printf` for formatting** — Java's `System.out.printf` and `String.format` provide precise control over number formatting (`$%.2f` for two decimal places, `%d` for integers).
- **`List<Order>` for polymorphism** — The `Broker` stores orders in a `List<Order>`, treating all command types uniformly. The `getClass().getSimpleName()` call retrieves the runtime type name for display purposes.
