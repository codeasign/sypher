This page implements the Proxy design pattern in Java 21 using a Cars domain. The Proxy pattern provides a surrogate object that controls access to another object. Here, a `Car` interface is implemented by `SportsCar` (the real subject), while `LazyCarModelProxy` defers expensive 3D model loading and `ValetCarProxy` restricts speed and access for valet parking. Java's interface system and access modifiers enforce the pattern contract.

## How the Implementation Works

### Class Diagram (Pseudocode)

### OOP Concepts Applied

- **Abstraction** — The `Car` interface defines the contract that all car representations must implement. Java enforces this contract at compile time — a class that claims to implement `Car` but omits any method will not compile.
- **Encapsulation** — `SportsCar` hides its internal state behind `private` fields and methods. The `load3DModel()` method is private to the car class. The proxies encapsulate their access control logic.
- **Inheritance and Interface Implementation** — `SportsCar`, `LazyCarModelProxy`, and `ValetCarProxy` implement the `Car` interface using the `implements` keyword.
- **Polymorphism** — The driver calls methods on `Car` references without knowing the concrete type. Method dispatch is resolved at runtime via Java's virtual method table.

### SOLID Principles Applied

- **Single Responsibility Principle** — `SportsCar` handles real car behavior. `ValetCarProxy` handles access control. `LazyCarModelProxy` handles lazy initialization.
- **Open/Closed Principle** — Adding a new proxy type requires creating a new class that implements `Car`. Existing classes remain unchanged.
- **Liskov Substitution Principle** — Every proxy can replace `Car` in client code without breaking correctness.
- **Interface Segregation Principle** — The `Car` interface exposes only the methods every car representation needs.
- **Dependency Inversion Principle** — High-level driver code depends on the `Car` abstraction, not on concrete classes.

## Implementation

The complete Java program below demonstrates the Proxy pattern in a Cars context. It is runnable with Java 21 and uses sealed interfaces, records, and modern Java features.

### Utility

```java
import java.util.*;

/** Generate a random 17-character vehicle identification number. */
private static String generateVin() {
    String chars = "ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789";
    Random random = new Random();
    StringBuilder vin = new StringBuilder();
    for (int i = 0; i < 17; i++) {
        vin.append(chars.charAt(random.nextInt(chars.length())));
    }
    return vin.toString();
}
```

### RealSubject: SportsCar

```java
class SportsCar implements Car {
    private final String vin;
    private final String make;
    private final String model;
    private final int year;
    private final int horsepower = 750;
    private final String engineType = "5.2L V10 Twin-Turbo";
    private final double acceleration0to60 = 2.8;
    private final double maxSpeed = 200.0;
    private Map<String, Object> full3DModel;
    private final List<String> trunkContents = List.of(
        "First aid kit", "Tool set", "Owners manual"
    );
    private final List<String> gloveboxContents = List.of(
        "Registration", "Insurance card", "Service record"
    );
    private boolean isEngineRunning = false;
    private double currentSpeed = 0.0;

    public SportsCar(String vin, String make, String model, int year) {
        this.vin = vin;
        this.make = make;
        this.model = model;
        this.year = year;
        load3DModel();
    }

    /** Simulate expensive 3D model loading. */
    private void load3DModel() {
        System.out.println(
            "  [SportsCar] Loading 3D model for " + vin + "... " +
            "200 MB of polygon data loaded."
        );
        full3DModel = new HashMap<>();
        full3DModel.put("polygons", 450_000);
        full3DModel.put("textures", List.of(
            "carbon_fiber", "leather_black", "aluminum_brushed"
        ));
        full3DModel.put("materials", List.of(
            "forged_carbon", "alcantara", "titanium"
        ));
    }

    @Override
    public String getVin() { return vin; }

    @Override
    public String getMake() { return make; }

    @Override
    public String getModel() { return model; }

    @Override
    public int getYear() { return year; }

    @Override
    public String getShortDescription() {
        return year + " " + make + " " + model + " (" + vin + ")";
    }

    @Override
    @SuppressWarnings("unchecked")
    public String displayDetails() {
        int polygons = (int) full3DModel.get("polygons");
        List<String> textures = (List<String>) full3DModel.get("textures");
        String trunk = String.join(", ", trunkContents);
        String glovebox = String.join(", ", gloveboxContents);
        return String.format(
            "--- %s ---%n" +
            "  Engine: %s%n" +
            "  Horsepower: %d hp%n" +
            "  0-60 mph: %.1fs%n" +
            "  Top Speed: %.1f mph%n" +
            "  3D Model: %,d polygons, %d textures%n" +
            "  Trunk: %s%n" +
            "  Glovebox: %s",
            getShortDescription(), engineType, horsepower,
            acceleration0to60, maxSpeed,
            polygons, textures.size(),
            trunk, glovebox
        );
    }

    @Override
    public String drive(double speed) {
        double actualSpeed = Math.min(speed, maxSpeed);
        isEngineRunning = true;
        currentSpeed = actualSpeed;
        return String.format(
            "  [SportsCar] ROAR! V10 screams through the rev range.%n" +
            "  Speed: %.1f mph | Gear: 7th | RPM: 8200",
            actualSpeed
        );
    }

    @Override
    public double getMaxSpeed() { return maxSpeed; }

    @Override
    public String accessTrunk() {
        String contents = String.join(", ", trunkContents);
        return "  [SportsCar] Trunk opened. Contents: " + contents;
    }
}
```

### Virtual Proxy: LazyCarModelProxy

```java
class LazyCarModelProxy implements Car {
    private SportsCar realCar = null;
    private final String vin;
    private final String make;
    private final String model;
    private final int year;

    public LazyCarModelProxy(String vin, String make, String model, int year) {
        this.vin = vin;
        this.make = make;
        this.model = model;
        this.year = year;
    }

    /**
     * Initialize the real car on first access.
     */
    private SportsCar getRealCar() {
        if (realCar == null) {
            System.out.println(
                "  [LazyCarModelProxy] Initializing real SportsCar for " +
                getShortDescription() + "..."
            );
            realCar = new SportsCar(vin, make, model, year);
        } else {
            System.out.println("  [LazyCarModelProxy] Reusing existing SportsCar instance.");
        }
        return realCar;
    }

    @Override
    public String getVin() { return vin; }

    @Override
    public String getMake() { return make; }

    @Override
    public String getModel() { return model; }

    @Override
    public int getYear() { return year; }

    @Override
    public String getShortDescription() {
        return year + " " + make + " " + model + " (" + vin + ")";
    }

    @Override
    public String displayDetails() {
        return getRealCar().displayDetails();
    }

    @Override
    public String drive(double speed) {
        return getRealCar().drive(speed);
    }

    @Override
    public double getMaxSpeed() {
        return getRealCar().getMaxSpeed();
    }

    @Override
    public String accessTrunk() {
        return getRealCar().accessTrunk();
    }
}
```

### Protection Proxy: ValetCarProxy

```java
class ValetCarProxy implements Car {
    private final Car realCar;
    private final double maxValetSpeed;
    private final String trunkDeniedMessage;

    public ValetCarProxy(Car realCar) {
        this(realCar, 30.0);
    }

    public ValetCarProxy(Car realCar, double maxValetSpeed) {
        this.realCar = realCar;
        this.maxValetSpeed = maxValetSpeed;
        this.trunkDeniedMessage =
            "  [ValetCarProxy] ACCESS DENIED: Valet key does not permit trunk access.";
    }

    @Override
    public String getVin() { return realCar.getVin(); }

    @Override
    public String getMake() { return realCar.getMake(); }

    @Override
    public String getModel() { return realCar.getModel(); }

    @Override
    public int getYear() { return realCar.getYear(); }

    @Override
    public String getShortDescription() {
        return realCar.getShortDescription();
    }

    @Override
    public String displayDetails() {
        return String.format(
            "--- Valet Card ---%n" +
            "  VIN: %s%n" +
            "  Max Permitted Speed: %.1f mph%n" +
            "  Trunk Access: DENIED%n" +
            "  Glovebox Access: DENIED",
            getVin(), maxValetSpeed
        );
    }

    @Override
    public String drive(double speed) {
        double limitedSpeed = Math.min(speed, maxValetSpeed);
        if (limitedSpeed < speed) {
            System.out.println(
                "  [ValetCarProxy] Speed limited to " + maxValetSpeed + " mph."
            );
        }
        return realCar.drive(limitedSpeed);
    }

    @Override
    public double getMaxSpeed() { return maxValetSpeed; }

    @Override
    public String accessTrunk() { return trunkDeniedMessage; }
}
```

### Main / Driver

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("=".repeat(72));
        System.out.println("  PROXY PATTERN — Cars Domain (Java 21)");
        System.out.println("=".repeat(72));
        System.out.println();

        // ── Scenario 1: Virtual Proxy (Lazy Loading) ──────────────────────
        System.out.println("--- SCENARIO 1: Virtual Proxy (Lazy Loading) ---");
        Car lazyCar = new LazyCarModelProxy(
            generateVin(), "Sypher", "GT", 2026
        );
        System.out.println(
            "  Showroom: Created LazyCarModelProxy. No 3D model loaded yet."
        );
        System.out.println();

        System.out.println("  Customer browses thumbnail:");
        System.out.println("  " + lazyCar.getShortDescription());
        System.out.println();

        System.out.println("  Customer clicks \"View Full Details\":");
        String details = lazyCar.displayDetails();
        System.out.println(details);
        System.out.println();

        // ── Scenario 2: Protection Proxy (Valet Key) ───────────────────────
        System.out.println("--- SCENARIO 2: Protection Proxy (Valet Key) ---");
        Car realCar = new SportsCar(generateVin(), "Sypher", "GT-S", 2026);
        Car valetKey = new ValetCarProxy(realCar);
        System.out.println();

        System.out.println("  Owner hands the valet key to the parking attendant.");
        System.out.println();

        System.out.println("  Valet: Displays info card:");
        System.out.println(valetKey.displayDetails());
        System.out.println();

        System.out.println("  Valet: Attempts to drive at 80 mph:");
        System.out.println(valetKey.drive(80.0));
        System.out.println();

        System.out.println("  Valet: Attempts to access the trunk:");
        System.out.println(valetKey.accessTrunk());
        System.out.println();

        // ── Scenario 3: Owner Returns — Full Access ───────────────────────
        System.out.println("--- SCENARIO 3: Owner Returns --- Full Access ---");
        System.out.println("  Owner uses the real car key directly:");
        System.out.println(realCar.drive(120.0));
        System.out.println(realCar.accessTrunk());
        System.out.println();

        // ── Scenario 4: Polymorphic Iteration ──────────────────────────────
        System.out.println("--- SCENARIO 4: Polymorphic Car Display ---");
        List<Car> cars = List.of(lazyCar, valetKey);
        for (Car car : cars) {
            System.out.printf(
                "  [%s] Max Speed: %.0f mph%n",
                car.getVin(), car.getMaxSpeed()
            );
        }
        System.out.println();

        // ── Scenario 5: Demonstrating Lazy Loading Benefit ────────────────
        System.out.println("--- SCENARIO 5: Lazy Loading Benefit ---");
        LazyCarModelProxy[] showroomCars = {
            new LazyCarModelProxy(generateVin(), "Sypher", "GT", 2026),
            new LazyCarModelProxy(generateVin(), "Sypher", "GT-S", 2026),
            new LazyCarModelProxy(generateVin(), "Falcon", "X1", 2026),
        };
        System.out.println("  All 3 proxies created. Memory usage: minimal.");
        System.out.println("  Showing thumbnails...");
        for (LazyCarModelProxy car : showroomCars) {
            System.out.println("    " + car.getShortDescription());
        }
        System.out.println();

        System.out.println("  Customer examines one car in detail:");
        System.out.println(showroomCars[0].displayDetails());
        System.out.println();

        System.out.println("=".repeat(72));
        System.out.println("  Proxy pattern demonstration complete.");
        System.out.println("=".repeat(72));
    }
}
```

## Language-Specific Notes

- **`interface` for clean abstraction** — Java's `interface` keyword provides a pure abstraction contract that all car representations must implement. Unlike abstract classes, interfaces in Java 21 can include `default` methods but are primarily used for contract definition. The `Car` interface declares the method signatures without any implementation, enforcing the DIP at compile time.

- **Constructor overloading for flexible configuration** — `ValetCarProxy` provides two constructors: a default constructor with a 30 mph speed limit and a parameterized constructor with a custom limit. Java's constructor overloading allows callers to choose the level of configuration without sacrificing clarity. This follows the principle of providing sensible defaults while allowing customization.

- **`List.of()` for immutable collections** — The `trunkContents` and `gloveboxContents` fields use `List.of()` to create immutable lists. This prevents accidental modification of the car's contents from outside the class and aligns with the Encapsulation principle by making the lists unmodifiable.

- **`String.format()` for structured output** — Java's `String.format()` method provides printf-style string formatting. The `%n` format specifier produces the platform-specific line separator, making the output portable across Windows, macOS, and Linux. The `%,d` specifier formats the polygon count with thousand separators.

- **`Math.min()` for safe speed limiting** — Both `SportsCar.drive()` and `ValetCarProxy.drive()` use `Math.min()` to clamp the requested speed to the permitted maximum. This ensures that even if the caller passes an unreasonably high speed, the car never exceeds its limit. The protection proxy uses the same technique but with a lower bound.

- **`java.util.Random` for VIN generation** — The `generateVin()` utility method uses `java.util.Random` to produce a random 17-character alphanumeric VIN. The `StringBuilder` class is used for efficient string construction inside the loop, avoiding the overhead of repeated string concatenation.
