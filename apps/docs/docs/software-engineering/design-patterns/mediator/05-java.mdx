The Mediator pattern decouples collaborating objects by routing all communication through a central mediator. In a stock market domain, the `StockExchange` interface defines the contract for registering agents, placing bids and asks, and broadcasting messages. Concrete exchanges coordinate `TradingAgent` implementations without the agents ever holding direct references to one another.

## How the Implementation Works

This Java implementation leverages modern language features introduced in Java 17+ to build a clean mediator structure:

1. **`StockExchange` (interface)** — declares the mediator contract with four methods: `registerAgent`, `placeBid`, `placeAsk`, and `broadcast`. Interfaces in Java provide full abstraction; any class that `implements StockExchange` must provide bodies for all declared methods.

2. **`TradingAgent` (interface)** — the colleague contract with `getName`, `getSymbol`, and `onMessage` methods. Agents hold no reference to each other, only to the mediator.

3. **`NasdaqExchange`** — the concrete mediator that maintains a `Map<String, TradingAgent>` for agent registry and a `List<Order>` for the order book. Each bid or ask appends an order and triggers `matchOrders()`, which scans pending orders for compatible buy/sell pairs (same symbol, bid price >= ask price), executes trades at the bid price, and notifies both parties.

4. **`BaseTradingAgent`** — an abstract class implementing `TradingAgent` to hold common state (`name`, `symbol`, `quantity`, `priceLimit`). Concrete subclasses `BuyAgent` and `SellAgent` extend it, providing their own `send` implementation that delegates to `exchange.placeBid(...)` or `exchange.placeAsk(...)`.

5. **Records** — `Order` and `Trade` are declared as Java records, giving them a compact syntax with automatically generated constructor, accessors, `equals()`, `hashCode()`, and `toString()`.

6. **SOLID principles**: Single Responsibility (agents manage trading intent, the exchange handles matching), Open/Closed (new agent types extend `BaseTradingAgent` without modifying `NasdaqExchange`), Liskov Substitution (any `TradingAgent` implementation works with any `StockExchange` implementation), Interface Segregation (two focused interfaces), Dependency Inversion (both layers depend on interfaces, not concrete classes).

### NasdaqExchange (Concrete Mediator)
```java
// ---------------------------------------------------------------------------
// Concrete mediator
// ---------------------------------------------------------------------------

class NasdaqExchange implements StockExchange {
    private final Map<String, TradingAgent> agents = new HashMap<>();
    private final List<Order> orderBook = new ArrayList<>();

    @Override
    public void registerAgent(TradingAgent agent) {
        agents.put(agent.getName(), agent);
        broadcast(agent.getName() + " has joined the exchange.");
    }

    @Override
    public void placeBid(TradingAgent agent, String symbol, int quantity, double price) {
        Order order = new Order(agent.getName(), symbol, quantity, price, "buy");
        orderBook.add(order);
        broadcast("NEW BID: " + agent.getName() + " wants to buy " + quantity
                + " shares of " + symbol + " @ $" + String.format("%.2f", price));
        processMatches();
    }

    @Override
    public void placeAsk(TradingAgent agent, String symbol, int quantity, double price) {
        Order order = new Order(agent.getName(), symbol, quantity, price, "sell");
        orderBook.add(order);
        broadcast("NEW ASK: " + agent.getName() + " wants to sell " + quantity
                + " shares of " + symbol + " @ $" + String.format("%.2f", price));
        processMatches();
    }

    @Override
    public void broadcast(String message) {
        System.out.println("[EXCHANGE] " + message);
    }

    private void processMatches() {
        List<Order> remaining = new ArrayList<>(orderBook);
        List<Order> newOrders = new ArrayList<>();

        for (Order order : orderBook) {
            if ("sell".equals(order.type())) {
                for (Order buy : new ArrayList<>(remaining)) {
                    if ("buy".equals(buy.type())
                            && buy.symbol().equals(order.symbol())
                            && buy.price() >= order.price()) {

                        int fillQty = Math.min(buy.quantity(), order.quantity());
                        double tradePrice = buy.price();

                        Trade trade = new Trade(buy.agentName(), order.agentName(),
                                order.symbol(), fillQty, tradePrice);

                        TradingAgent buyer = agents.get(trade.buyer());
                        TradingAgent seller = agents.get(trade.seller());

                        if (buyer != null) {
                            buyer.onMessage("BOUGHT " + fillQty + " shares of "
                                    + trade.symbol() + " from " + trade.seller()
                                    + " @ $" + String.format("%.2f", tradePrice));
                        }
                        if (seller != null) {
                            seller.onMessage("SOLD " + fillQty + " shares of "
                                    + trade.symbol() + " to " + trade.buyer()
                                    + " @ $" + String.format("%.2f", tradePrice));
                        }

                        remaining.remove(buy);
                        remaining.remove(order);

                        int buyRemainder = buy.quantity() - fillQty;
                        int sellRemainder = order.quantity() - fillQty;
                        if (buyRemainder > 0) {
                            newOrders.add(new Order(buy.agentName(), buy.symbol(),
                                    buyRemainder, buy.price(), "buy"));
                        }
                        if (sellRemainder > 0) {
                            newOrders.add(new Order(order.agentName(), order.symbol(),
                                    sellRemainder, order.price(), "sell"));
                        }
                        break;
                    }
                }
            }
        }

        orderBook.clear();
        orderBook.addAll(remaining);
        orderBook.addAll(newOrders);
    }
}
```

### BaseTradingAgent (Abstract Base Class)
```java
// ---------------------------------------------------------------------------
// Base colleague (abstract class)
// ---------------------------------------------------------------------------

abstract class BaseTradingAgent implements TradingAgent {
    private final String name;
    private final String symbol;
    private final int quantity;
    private final double priceLimit;

    protected BaseTradingAgent(String name, String symbol, int quantity, double priceLimit) {
        this.name = name;
        this.symbol = symbol;
        this.quantity = quantity;
        this.priceLimit = priceLimit;
    }

    @Override
    public String getName() {
        return name;
    }

    @Override
    public String getSymbol() {
        return symbol;
    }

    protected int getQuantity() {
        return quantity;
    }

    protected double getPriceLimit() {
        return priceLimit;
    }

    @Override
    public void onMessage(String message) {
        System.out.println("[" + name + "] " + message);
    }

    public abstract void send(StockExchange exchange);
}
```

### Main / Driver
```java
// ---------------------------------------------------------------------------
// Driver
// ---------------------------------------------------------------------------

public class MediatorDemo {
    public static void main(String[] args) {
        NasdaqExchange nasdaq = new NasdaqExchange();

        BuyAgent buyerAapl = new BuyAgent("Alice", "AAPL", 100, 152.50);
        SellAgent sellerAapl = new SellAgent("Bob", "AAPL", 100, 151.00);
        BuyAgent buyerGoogl = new BuyAgent("Carol", "GOOGL", 50, 190.35);
        SellAgent sellerGoogl = new SellAgent("Dave", "GOOGL", 50, 189.30);

        nasdaq.registerAgent(buyerAapl);
        nasdaq.registerAgent(sellerAapl);
        nasdaq.registerAgent(buyerGoogl);
        nasdaq.registerAgent(sellerGoogl);

        System.out.println("\n--- Placing AAPL orders ---");
        buyerAapl.send(nasdaq);
        sellerAapl.send(nasdaq);

        System.out.println("\n--- Placing GOOGL orders ---");
        buyerGoogl.send(nasdaq);
        sellerGoogl.send(nasdaq);
    }
}
```

## Language-Specific Notes — Java

- **Records (`record Order(...)`, `record Trade(...)`)** — a Java 17+ feature that replaces verbose POJOs with a single-line declaration. The compiler auto-generates the constructor, `equals()`, `hashCode()`, `toString()`, and accessor methods (`order.agentName()` not `order.getAgentName()`). All fields are `private final` by default.
- **`interface` vs `abstract class`** — `StockExchange` and `TradingAgent` are pure interfaces (all methods implicitly `public abstract`). `BaseTradingAgent` is an `abstract class` that provides shared field storage and a default `onMessage` implementation, leaving only `send()` for subclasses to implement. This avoids repeating field declarations in every agent type.
- **`HashMap` and `ArrayList`** — the go-to standard library collections. `HashMap<String, TradingAgent>` provides O(1) agent lookup by name, and `ArrayList<Order>` maintains insertion order for the order book backlog.
- **`instanceof` pattern matching** — not used in this example but available in Java 17+: `if (obj instanceof BuyAgent ba)` simultaneously checks the type and binds the variable. The code above uses string comparison on `order.type()` instead, which is equally explicit and avoids coupling to the agent class hierarchy.
- **`String.format("%.2f", price)`** — used for consistent two-decimal-place price output. Java's `System.out.printf` is an alternative, but explicit formatting keeps each `println` self-contained.
- **Method encapsulation** — `processMatches()` is a `private` method in `NasdaqExchange`, visible only within the class. This follows the principle of minimum access and keeps the public API clean. Helper methods like `getQuantity()` and `getPriceLimit()` in `BaseTradingAgent` are `protected` to allow subclass access while hiding them from external callers.
