Rust implements the Mediator pattern through traits and struct composition. Unlike class-based languages, Rust uses traits to define shared behavior without inheritance. In the stock market domain, a Rust `NasdaqExchange` struct mediates between `BuyAgent` and `SellAgent` colleagues using trait objects (`Box<dyn TradingAgent>`), matching buy and sell orders through a centralized order book while respecting Rust's ownership and borrowing rules.

## How the Implementation Works

| Aspect | Implementation |
|---|---|
| **Mediator** | `trait StockExchange` with `register_agent`, `place_bid`, `place_ask`, `broadcast` |
| **ConcreteMediator** | `struct NasdaqExchange` with `HashMap<String, Box<dyn TradingAgent>>` and `Vec<Order>`, matches bids against asks |
| **Colleague** | `trait TradingAgent` with `name`, `symbol`, `on_message` |
| **ConcreteColleagues** | `struct BuyAgent` and `struct SellAgent` each own name, symbol, quantity, price limit |
| **Ownership** | Exchange holds agents as `Box<dyn TradingAgent>` (owned trait objects); orders as plain structs |
| **Dispatch** | Dynamic dispatch via `Box<dyn TradingAgent>` enables runtime polymorphism without generics |

### StockExchange (Mediator Trait)
```rust
// ── Mediator Trait ──
/// The mediator trait defining the contract for a stock exchange.
/// All communication between trading agents flows through this trait.
trait StockExchange {
    /// Register a trading agent with the exchange.
    fn register_agent(&mut self, agent: Box<dyn TradingAgent>);

    /// Place a buy order (bid) on the exchange.
    fn place_bid(&mut self, agent_name: &str, symbol: &str, quantity: u32, price: f64);

    /// Place a sell order (ask) on the exchange.
    fn place_ask(&mut self, agent_name: &str, symbol: &str, quantity: u32, price: f64);

    /// Broadcast a message to all registered agents.
    fn broadcast(&self, message: &str);
}
```

### TradingAgent (Colleague Trait)
```rust
// ── Colleague Trait ──
/// The colleague trait for any participant in the exchange.
/// Agents communicate through the mediator, never directly with each other.
trait TradingAgent {
    /// Get the name of this trading agent.
    fn name(&self) -> &str;

    /// Get the stock symbol this agent trades.
    fn symbol(&self) -> &str;

    /// Receive a message from the mediator.
    fn on_message(&self, message: &str);
}
```

```rust
// ── Data Types ──
/// The type of order: buy (Bid) or sell (Ask).
#[derive(Debug, Clone, PartialEq)]
enum OrderType {
    Bid,
    Ask,
}

/// Represents a single order in the exchange order book.
#[derive(Debug, Clone)]
struct Order {
    agent_name: String,
    symbol: String,
    quantity: u32,
    price: f64,
    order_type: OrderType,
}

/// Represents a matched trade between a buyer and a seller.
#[derive(Debug, Clone)]
struct Trade {
    buyer: String,
    seller: String,
    symbol: String,
    quantity: u32,
    price: f64,
}

impl Trade {
    /// Format this trade as a human-readable string.
    fn summary(&self) -> String {
        format!(
            "TRADE: {} bought {} {} from {} @ ${:.2}",
            self.buyer, self.quantity, self.symbol, self.seller, self.price
        )
    }
}
```

### NasdaqExchange (Concrete Mediator)
```rust
// ── Concrete Mediator ──
/// A concrete stock exchange that mediates between trading agents.
/// Maintains a registry of agents and an order book, matching bids against asks.
struct NasdaqExchange {
    agents: HashMap<String, Box<dyn TradingAgent>>,
    order_book: Vec<Order>,
    executed_trades: Vec<Trade>,
}

impl NasdaqExchange {
    /// Create a new empty exchange.
    fn new() -> Self {
        NasdaqExchange {
            agents: HashMap::new(),
            order_book: Vec::new(),
            executed_trades: Vec::new(),
        }
    }

    /// Match compatible bids and asks from the order book.
    /// A bid matches an ask when the same symbol, bid price >= ask price, and quantities align.
    fn match_orders(&mut self) {
        loop {
            let mut matched = false;

            // Scan for a matching bid-ask pair.
            let match_result = self.find_match();
            if let Some((bid_idx, ask_idx, trade_quantity, execution_price)) = match_result {
                let bid = &self.order_book[bid_idx];
                let ask = &self.order_book[ask_idx];

                let trade = Trade {
                    buyer: bid.agent_name.clone(),
                    seller: ask.agent_name.clone(),
                    symbol: bid.symbol.clone(),
                    quantity: trade_quantity,
                    price: execution_price,
                };

                println!("  [Exchange] {}", trade.summary());

                // Notify agents.
                if let Some(buyer) = self.agents.get(&trade.buyer) {
                    buyer.on_message(&format!(
                        "Trade executed: You bought {} {} @ ${:.2}",
                        trade.quantity, trade.symbol, trade.price
                    ));
                }
                if let Some(seller) = self.agents.get(&trade.seller) {
                    seller.on_message(&format!(
                        "Trade executed: You sold {} {} @ ${:.2}",
                        trade.quantity, trade.symbol, trade.price
                    ));
                }

                self.executed_trades.push(trade);

                // Remove the matched orders and add back any remaining quantity.
                self.remove_and_readjust(bid_idx, ask_idx, trade_quantity);

                matched = true;
            }

            if !matched {
                break;
            }
        }
    }

    /// Find a matching bid and ask in the order book.
    fn find_match(&self) -> Option<(usize, usize, u32, f64)> {
        for (i, bid) in self.order_book.iter().enumerate() {
            if bid.order_type != OrderType::Bid {
                continue;
            }
            for (j, ask) in self.order_book.iter().enumerate() {
                if ask.order_type != OrderType::Ask {
                    continue;
                }
                if bid.symbol != ask.symbol {
                    continue;
                }
                if bid.price < ask.price {
                    continue;
                }

                let trade_qty = bid.quantity.min(ask.quantity);
                let exec_price = ask.price;
                return Some((i, j, trade_qty, exec_price));
            }
        }
        None
    }

    /// Remove matched orders and re-add any residual quantities.
    fn remove_and_readjust(&mut self, bid_idx: usize, ask_idx: usize, trade_qty: u32) {
        // Extract orders by index (adjust for removals).
        let bid_idx_adj;
        let ask_idx_adj;

        if bid_idx < ask_idx {
            bid_idx_adj = bid_idx;
            ask_idx_adj = ask_idx - 1;
        } else {
            ask_idx_adj = ask_idx;
            bid_idx_adj = bid_idx - 1;
        }

        let bid = self.order_book.remove(bid_idx_adj);
        let ask = self.order_book.remove(ask_idx_adj);

        if bid.quantity > trade_qty {
            self.order_book.push(Order {
                quantity: bid.quantity - trade_qty,
                ..bid
            });
        }
        if ask.quantity > trade_qty {
            self.order_book.push(Order {
                quantity: ask.quantity - trade_qty,
                ..ask
            });
        }
    }

    /// Print a summary of all trades executed during this session.
    fn print_trade_summary(&self) {
        println!();
        println!("  ┌" + &"─".repeat(54) + "┐");
        println!("  │                  TRADE EXECUTION SUMMARY              │");
        println!("  ├" + &"─".repeat(54) + "┤");
        println!(
            "  │  Total trades executed: {:<3}                                 │",
            self.executed_trades.len()
        );
        println!("  ├──────┬──────────┬──────────┬──────────┬───────────────┤");
        println!("  │  #   │ Buyer    │ Seller   │ Symbol   │ Price         │");
        println!("  ├──────┼──────────┼──────────┼──────────┼───────────────┤");

        for (k, t) in self.executed_trades.iter().enumerate() {
            println!(
                "  │  {:<3} │ {:<8} │ {:<8} │ {:<8} │ ${:<10.2} │",
                k + 1,
                t.buyer,
                t.seller,
                t.symbol,
                t.price
            );
        }

        println!("  └──────┴──────────┴──────────┴──────────┴───────────────┘");
    }
}

impl StockExchange for NasdaqExchange {
    fn register_agent(&mut self, agent: Box<dyn TradingAgent>) {
        let name = agent.name().to_string();
        let symbol = agent.symbol().to_string();

        if self.agents.contains_key(&name) {
            println!("  [Exchange] Agent '{}' is already registered.", name);
            return;
        }

        self.agents.insert(name.clone(), agent);
        println!(
            "  [Exchange] Registered agent '{}' trading {}.",
            name, symbol
        );
        self.broadcast(&format!(
            "Welcome, {}! You are now trading {} on Nasdaq.",
            name, symbol
        ));
    }

    fn place_bid(&mut self, agent_name: &str, symbol: &str, quantity: u32, price: f64) {
        println!(
            "  [Exchange] Bid received: {} wants to buy {} {} @ ${:.2}",
            agent_name, quantity, symbol, price
        );

        self.order_book.push(Order {
            agent_name: agent_name.to_string(),
            symbol: symbol.to_string(),
            quantity,
            price,
            order_type: OrderType::Bid,
        });

        self.match_orders();
    }

    fn place_ask(&mut self, agent_name: &str, symbol: &str, quantity: u32, price: f64) {
        println!(
            "  [Exchange] Ask received: {} wants to sell {} {} @ ${:.2}",
            agent_name, quantity, symbol, price
        );

        self.order_book.push(Order {
            agent_name: agent_name.to_string(),
            symbol: symbol.to_string(),
            quantity,
            price,
            order_type: OrderType::Ask,
        });

        self.match_orders();
    }

    fn broadcast(&self, message: &str) {
        for agent in self.agents.values() {
            agent.on_message(&format!("[BROADCAST] {}", message));
        }
    }
}
```

### BuyAgent (Concrete Colleague 1)
```rust
// ── Concrete Colleagues ──
/// A trading agent that places buy orders (bids) on the exchange.
struct BuyAgent {
    name: String,
    symbol: String,
    quantity: u32,
    price_limit: f64,
}

impl BuyAgent {
    /// Create a new BuyAgent.
    fn new(name: &str, symbol: &str, quantity: u32, price_limit: f64) -> Self {
        BuyAgent {
            name: name.to_string(),
            symbol: symbol.to_string(),
            quantity,
            price_limit,
        }
    }

    /// Submit a buy order to the exchange.
    fn send(&self, exchange: &mut dyn StockExchange) {
        println!(
            "  [BuyAgent] {} submitting bid: {} {} @ ${:.2}",
            self.name, self.quantity, self.symbol, self.price_limit
        );
        exchange.place_bid(&self.name, &self.symbol, self.quantity, self.price_limit);
    }
}

impl TradingAgent for BuyAgent {
    fn name(&self) -> &str {
        &self.name
    }

    fn symbol(&self) -> &str {
        &self.symbol
    }

    fn on_message(&self, message: &str) {
        println!("  [BuyAgent:{}] {}", self.name, message);
    }
}
```

### Main / Driver
```rust
// ── Driver ──
fn main() {
    println!("{}", "=".repeat(65));
    println!("MEDIATOR PATTERN — Rust Stock Exchange Demo");
    println!("{}", "=".repeat(65));
    println!();

    // Create the mediator (stock exchange).
    let mut exchange = NasdaqExchange::new();

    // Create trading agents (colleagues) on the stack.
    // Each agent calls send() BEFORE being moved into the exchange registry.
    let alice = BuyAgent::new("Alice", "AAPL", 100, 150.50);
    let bob = SellAgent::new("Bob", "AAPL", 100, 150.50);
    let charlie = BuyAgent::new("Charlie", "GOOGL", 50, 175.35);
    let diana = SellAgent::new("Diana", "GOOGL", 50, 175.30);
    let eve = BuyAgent::new("Eve", "MSFT", 75, 340.00);
    let frank = SellAgent::new("Frank", "MSFT", 75, 340.10);

    // Register agents with the exchange.
    // We box each agent and transfer ownership to the exchange's HashMap.
    println!("--- Registering Agents ---");
    exchange.register_agent(Box::new(BuyAgent::new("Alice", "AAPL", 100, 150.50)));
    exchange.register_agent(Box::new(SellAgent::new("Bob", "AAPL", 100, 150.50)));
    exchange.register_agent(Box::new(BuyAgent::new("Charlie", "GOOGL", 50, 175.35)));
    exchange.register_agent(Box::new(SellAgent::new("Diana", "GOOGL", 50, 175.30)));
    exchange.register_agent(Box::new(BuyAgent::new("Eve", "MSFT", 75, 340.00)));
    exchange.register_agent(Box::new(SellAgent::new("Frank", "MSFT", 75, 340.10)));
    println!();

    // Place orders through the mediator.
    // Each agent calls its send() method, which takes &self and &mut dyn StockExchange.
    // The agents live on the stack in the driver; the exchange owns boxed copies.
    println!("--- Placing Orders ---");

    // Alice wants to buy 100 AAPL @ $150.50.
    alice.send(&mut exchange);
    println!();

    // Bob wants to sell 100 AAPL @ $150.50 — bid price ($150.50) >= ask price ($150.50), so match.
    bob.send(&mut exchange);
    println!();

    // Charlie wants to buy 50 GOOGL @ $175.35.
    charlie.send(&mut exchange);
    println!();

    // Diana wants to sell 50 GOOGL @ $175.30 — bid price ($175.35) >= ask price ($175.30), so match.
    diana.send(&mut exchange);
    println!();

    // Eve wants to buy 75 MSFT @ $340.00.
    eve.send(&mut exchange);
    println!();

    // Frank wants to sell 75 MSFT @ $340.10 — bid ($340.00) < ask ($340.10), so no match.
    frank.send(&mut exchange);
    println!();

    // Print trade summary.
    exchange.print_trade_summary();

    println!();
    println!("{}", "=".repeat(65));
    println!("All orders processed through the NasdaqExchange mediator.");
}
```

## Language-Specific Notes

- Rust uses traits instead of interfaces or abstract classes. The `StockExchange` trait defines the mediator contract, and the `TradingAgent` trait defines the colleague contract. Concrete types like `NasdaqExchange`, `BuyAgent`, and `SellAgent` implement these traits via `impl TraitName for StructName`. There is no inheritance — each type is a plain struct that happens to implement the trait.

- The `HashMap<String, Box<dyn TradingAgent>>` pattern stores owned trait objects. `Box<dyn TradingAgent>` is a heap-allocated pointer to any type that implements `TradingAgent`, enabling runtime polymorphism. The `dyn` keyword indicates dynamic dispatch through a vtable, which has a small runtime cost but allows heterogeneous agent types in the same collection.

- The `OrderType` enum with `Bid` and `Ask` variants is a type-safe alternative to string-based type tags. Rust's `match` expressions on enums are exhaustive — the compiler ensures every variant is handled. The `#[derive(Debug, Clone, PartialEq)]` attribute auto-implements those traits, providing formatting, cloning, and equality comparison without manual boilerplate.

- The driver creates agents as stack-allocated concrete structs (e.g., `let alice = BuyAgent::new(...)`) and calls `alice.send(&mut exchange)` directly, then registers separate boxed copies in the exchange. This dual-ownership pattern avoids downcasting: the driver retains concrete agent references for calling `send`, while the exchange owns `Box<dyn TradingAgent>` trait objects for broadcasting. In production, a single-ownership approach using `Rc<RefCell<>>` or shared references could eliminate the duplication.

- Ownership and borrowing are central to the design: the exchange takes ownership of agents via `Box<dyn TradingAgent>` (the `HashMap` owns the heap-allocated agents), and orders are plain `Vec<Order>` values. The `match_orders` method borrows `self` mutably, which required separating the find-and-remove logic into helper methods to satisfy the borrow checker — a common Rust pattern where complex self-mutation is broken into smaller steps.

- The struct update syntax (`Order { quantity: bid.quantity - trade_qty, ..bid }`) creates a new `Order` from an existing one while overriding specific fields. This is analogous to C#'s `with` expression on records and avoids manual field-by-field reconstruction. The original `bid` is consumed (moved) in the process, which is safe because it was already removed from the order book vector.
