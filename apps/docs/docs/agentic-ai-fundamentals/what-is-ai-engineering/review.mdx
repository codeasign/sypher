# What is AI Engineering — Review

*Reading time: 3 minutes*

## Key Takeaways

- AI engineering combines software engineering, ML operations, and product thinking to build production systems with LLMs
- The AI stack has four layers: infrastructure, model, orchestration, and application
- LLM APIs are stateless — you must send the full conversation history on every call
- API calls return token usage metadata that is essential for cost tracking and monitoring
- Error handling with retries and exponential backoff is mandatory for production use

## Recap in 60 Seconds

1. LLMs are probabilistic software components accessed through APIs — they are powerful but not deterministic
2. You integrate them behind abstraction layers (a wrapper function, a service class) rather than calling them directly from UI code
3. Always track tokens, handle errors, and validate outputs — the same rigor you apply to any production API
