# Loop Safety and Recovery — Review

*Reading time: 3 minutes*

## Key Takeaways

- Three safety layers: hard limits (unconditional), recovery strategies (retry, fallback), circuit breakers (pattern-based)
- Hard limits are the last line of defense — always set them even if you have recovery strategies
- Exponential backoff with jitter prevents retry storms
- Circuit breakers track error rates and halt execution when patterns of failure emerge
- Always return structured error messages when safety systems trigger

## Recap in 60 Seconds

1. Hard limits prevent runaway costs; recovery strategies handle transient failures; circuit breakers detect systemic failures
2. Add jitter to retry delays so multiple agents do not retry in sync
3. Test safety systems by forcing failures — the system that has never been tested will fail when you need it most
