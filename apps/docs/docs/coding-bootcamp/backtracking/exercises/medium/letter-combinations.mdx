# Letter Combinations of a Phone Number

**Difficulty:** Medium

Given a string containing digits from `2-9` inclusive, return all possible letter combinations that the number could represent. Return the answer in any order consistent with the mapping below.

A mapping of digits to letters (just like on the telephone buttons) is given:

| Digit | Letters |
|-------|---------|
| 2     | abc     |
| 3     | def     |
| 4     | ghi     |
| 5     | jkl     |
| 6     | mno     |
| 7     | pqrs    |
| 8     | tuv     |
| 9     | wxyz    |

Note that `1` does not map to any letters.

## Example 1

**Input:** `digits = "23"`  
**Output:** `["ad","ae","af","bd","be","bf","cd","ce","cf"]`  
**Explanation:** Digit `2` maps to `abc`, digit `3` maps to `def`. All combinations are formed by choosing one letter from each digit's group.

## Example 2

**Input:** `digits = ""`  
**Output:** `[]`  
**Explanation:** An empty string produces no combinations.

## Example 3

**Input:** `digits = "2"`  
**Output:** `["a","b","c"]`  
**Explanation:** A single digit returns only its mapped letters.

## Input Specification

A single line containing a string of digits from `2-9`. May be empty.

## Output Specification

Print all combinations as space-separated strings on one line, in the order they are generated (left-to-right, digit by digit). If the input is empty, print an empty line.

## Constraints

- `0 <= digits.length <= 4`
- `digits[i]` is a digit in the range `['2', '9']`

## Hints

<details>
<summary>Hint 1</summary>

Think of this as building combinations one digit at a time. For each digit you process, extend every existing partial combination with each letter mapped to that digit. This is the classic backtracking structure: pick a letter, recurse on the remaining digits, then undo the choice.

</details>

<details>
<summary>Hint 2</summary>

Start with a base case: if the current index equals the length of `digits`, the current path is a complete combination — add it to your results. Otherwise, iterate over every letter for `digits[index]` and recurse with `index + 1`.

</details>

<details>
<summary>Hint 3</summary>

You can also solve this iteratively. Maintain a list of current combinations (start with `[""]`). For each digit, expand the list: for every existing combination, append each letter for that digit and collect the new expanded list.

</details>

</ProblemEditor>
