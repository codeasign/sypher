# Container With Most Water

**Difficulty:** Medium

You are given an integer array `height` of length `n`. There are `n` vertical lines drawn such that the two endpoints of the i-th line are `(i, 0)` and `(i, height[i])`.

Find two lines that together with the x-axis form a container, such that the container contains the most water.

Return the maximum amount of water a container can store.

**Note:** You may not slant the container.

## Example 1

**Input:** `height = [1, 8, 6, 2, 5, 4, 8, 3, 7]`  
**Output:** `49`  

## Example 2

**Input:** `height = [1, 1]`  
**Output:** `1`  

## Input Specification

- A single line of space-separated integers representing `height[i]`.
- `2 <= height.length <= 10^5`

## Output Specification

Print a single integer: the maximum water the container can hold.

## Constraints

- `2 <= height.length <= 10^5`
- `0 <= height[i] <= 10^4`

## Hints

<details>
<summary>Hint 1</summary>

The water area is determined by the shorter line. Moving the taller line inward cannot increase the area, but moving the shorter line might.
</details>

<details>
<summary>Hint 2</summary>

Use two pointers starting at both ends. At each step, compute the area and move the pointer with the shorter height inward.
</details>

<details>
<summary>Hint 3</summary>

This greedy-like two-pointer approach works because the width decreases each step, but the height can only increase by moving the shorter line.
</details>

</ProblemEditor>
