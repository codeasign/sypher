# Merge Sorted Array

**Difficulty:** Easy

You are given two integer arrays `nums1` and `nums2`, sorted in **non-decreasing order**, and two integers `m` and `n`, representing the number of elements in `nums1` and `nums2` respectively.

`nums1` has a length of `m + n`, where the last `n` elements are set to `0` and should be ignored. Merge `nums1` and `nums2` into a single array sorted in **non-decreasing order**.

The merge must be done **in-place** on `nums1` — modify the array directly, do not return a new array.

## Example 1

**Input:** `nums1 = [1, 2, 3, 0, 0, 0]`, `m = 3`, `nums2 = [2, 5, 6]`, `n = 3`  
**Output:** `[1, 2, 2, 3, 5, 6]`  
**Explanation:** The arrays merged in sorted order produce `[1, 2, 2, 3, 5, 6]`.

## Example 2

**Input:** `nums1 = [1]`, `m = 1`, `nums2 = []`, `n = 0`  
**Output:** `[1]`  
**Explanation:** The second array is empty, so `nums1` is unchanged.

## Example 3

**Input:** `nums1 = [0]`, `m = 0`, `nums2 = [1]`, `n = 1`  
**Output:** `[1]`  
**Explanation:** The first array is empty, so the result is just the second array placed into `nums1`.

## Input Specification

The first line contains two space-separated integers `m` and `n` — the number of elements in `nums1` and `nums2` respectively.
The second line contains `m + n` space-separated integers representing `nums1`. The first `m` values are the sorted elements; the last `n` values are zeros (placeholders).
The third line contains `n` space-separated integers representing the sorted array `nums2`. If `n = 0`, this line is empty.

## Output Specification

Print `m + n` space-separated integers representing the merged sorted array.

## Constraints

- `0 <= m, n <= 200`
- `1 <= m + n <= 200`
- `-10^9 <= nums1[i], nums2[i] <= 10^9`
- Both `nums1` and `nums2` are sorted in **non-decreasing order**.

## Hints

<details>
<summary>Hint 1</summary>

Work backwards from the end of `nums1` to avoid overwriting elements that have not been placed yet.

</details>

<details>
<summary>Hint 2</summary>

Use three pointers: one at the last actual element of `nums1` (index `m - 1`), one at the last element of `nums2` (index `n - 1`), and one at the very end of `nums1` (index `m + n - 1`). Compare and place the larger element at the end.

</details>

<details>
<summary>Hint 3</summary>

After one array is exhausted, any remaining elements from the other array are already in their correct position — you do not need to move them.

</details>
