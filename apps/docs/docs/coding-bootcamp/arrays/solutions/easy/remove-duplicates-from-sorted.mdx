# Remove Duplicates from Sorted Array — Solution

## Problem Overview

Return the count of unique elements in a sorted array, removing duplicates in-place. Solved with the **two-pointer (slow/fast) technique**.

## Approach: Two Pointers

**Idea:** Keep a slow pointer `k` pointing to the position where the next unique element should go. Scan with a fast pointer `i`. When `nums[i]` differs from `nums[k-1]`, copy it to `nums[k]` and advance `k`.

**Algorithm:**
1. If `nums` is empty, return 0.
2. Initialize `k = 1` (first element is always unique).
3. For `i` from 1 to `n-1`: if `nums[i] != nums[k-1]`, set `nums[k] = nums[i]`, `k++`.
4. Return `k`.

**Complexity:**
- **Time:** O(n) — single pass.
- **Space:** O(1) — in-place, no extra allocation.

def remove_duplicates(nums: list[int]) -> int:
    if not nums:
        return 0
    k = 1
    for i in range(1, len(nums)):
        if nums[i] != nums[k - 1]:
            nums[k] = nums[i]
            k += 1
    return k

## Key Takeaways

- **Two-pointer in-place**: The slow pointer `k` marks the boundary of the "cleaned" prefix. The fast pointer scans for the next unique value.
- **Sorted property**: Since duplicates are adjacent in a sorted array, comparing `nums[i]` to `nums[k-1]` (the last written unique value) is sufficient.
- **O(1) space**: No auxiliary array is created — the input array is modified in place.
