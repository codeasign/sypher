# Power of Two — Solution

## Problem Overview

Given an integer `n`, return `true` if it is a power of two, otherwise `false`. An integer is a power of two if there exists an integer `x` such that `n == 2^x`.

## Approach: Bit Manipulation

**Idea:** A power of two in binary has exactly one `1` bit. Use `n & (n - 1)` to check: this operation flips the lowest set bit to 0. If the result is 0, `n` had exactly one `1` bit.

**Algorithm:**
1. If `n <= 0`, return `false` (negative numbers and zero are not powers of two)
2. Return `(n & (n - 1)) == 0`

**Complexity Analysis:**
- **Time:** O(1) — constant-time bit operation
- **Space:** O(1) — constant extra space

## Key Takeaways

- **Bit manipulation** (`n & (n - 1)`) is the most efficient way to check for powers of two
- Powers of two have exactly one bit set in binary representation
- Handle `n <= 0` as an early exit for edge cases
