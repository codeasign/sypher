# Rotate Image

**Difficulty:** Medium

You are given an `n x n` 2D `matrix` representing an image. Rotate the image by **90 degrees (clockwise)** in-place.

You must modify the matrix in-place — do not allocate another 2D matrix for the rotation.

## Example 1

**Input:** `matrix = [[1,2,3],[4,5,6],[7,8,9]]`  
**Output:** `[[7,4,1],[8,5,2],[9,6,3]]`

## Example 2

**Input:** `matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]`  
**Output:** `[[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]`

## Input Specification

First line: integer `n`.  
Next `n` lines: each contains `n` integers representing the matrix.

## Output Specification

Print the rotated matrix in the same format — one row per line, elements separated by spaces.

## Constraints

- `n == matrix.length == matrix[i].length`
- `1 <= n <= 20`
- `-1000 <= matrix[i][j] <= 1000`

## Hints

<details>
<summary>Hint 1</summary>

Try transposing the matrix first (swap `matrix[i][j]` with `matrix[j][i]`), then reverse each row.
</details>

<details>
<summary>Hint 2</summary>

Alternatively, rotate groups of 4 cells at a time in each layer (top → right → bottom → left).
</details>

<details>
<summary>Hint 3</summary>

For the rotate-groups approach, work from the outermost layer inward, iterating only through the first `n/2` rows and `(n+1)/2` columns.
</details>

</ProblemEditor>
