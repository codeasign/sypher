# Set Matrix Zeroes

**Difficulty:** Medium

Given an `m x n` integer matrix `matrix`, if an element is `0`, set its entire row and column to `0`'s.

You must do it **in place**.

## Example 1

**Input:** `matrix = [[1,1,1],[1,0,1],[1,1,1]]`  
**Output:** `[[1,0,1],[0,0,0],[1,0,1]]`  

## Example 2

**Input:** `matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]`  
**Output:** `[[0,0,0,0],[0,4,5,0],[0,3,1,0]]`  

## Input Specification

- First two ints: `m n`, followed by `m*n` matrix entries.
- `1 <= m, n <= 200`

## Output Specification

Print the resulting matrix, each row on its own line.

## Constraints

- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

## Hints

<details>
<summary>Hint 1</summary>

Use the first row and first column as markers. Scan the matrix, and when you find a zero, mark its row and column in the first row/column.
</details>

<details>
<summary>Hint 2</summary>

You need a separate flag for whether the first row or first column itself should be zeroed.
</details>

<details>
<summary>Hint 3</summary>

After marking, iterate through the matrix again and set cells to zero based on the markers, except handle the first row/column separately.
</details>
