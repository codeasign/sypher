# Rotate Image — Solution

## Problem Overview

Rotate an `n x n` matrix by 90 degrees clockwise **in-place** (no extra matrix).

## Approach 1: Transpose + Reverse

**Idea:** A 90-degree clockwise rotation is equivalent to transposing the matrix (rows become columns) and then reversing each row.

**Algorithm:**
1. **Transpose**: For `i` from `0` to `n-1`, for `j` from `i+1` to `n-1`, swap `matrix[i][j]` with `matrix[j][i]`.
2. **Reverse each row**: For each row, reverse its elements.

**Complexity Analysis:**
- **Time:** O(n²) — transpose takes n²/2 swaps, reverse takes n * n/2 = n²/2 swaps
- **Space:** O(1) — in-place

## Approach 2: Rotate Groups of Four

**Idea:** Process the matrix in concentric layers. For each layer, rotate groups of 4 cells at a time. The top cell moves to the right, the right cell moves to the bottom, the bottom cell moves to the left, and the left cell moves to the top.

**Algorithm:**
1. For each layer from outermost inward:
   - For each cell in the first `n-1` cells of the top row of the layer:
     - Save top-left value
     - Move bottom-left to top-left
     - Move bottom-right to bottom-left
     - Move top-right to bottom-right
     - Move saved top-left to top-right

**Complexity Analysis:**
- **Time:** O(n²)
- **Space:** O(1)

## Deep Dive: Visual Walkthrough

### Transpose + Reverse

For `matrix = [[1,2,3],[4,5,6],[7,8,9]]`:

```
Original:        Transpose:       Reverse rows:
1 2 3            1 4 7            7 4 1
4 5 6    →       2 5 8    →      8 5 2
7 8 9            3 6 9            9 6 3
```

### Edge Cases to Consider

- **Single element**: No rotation needed.
- **All same values**: The matrix is unchanged regardless.
- **Negative values**: Swap logic works with any integer values.

class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        // Transpose: swap matrix[i][j] with matrix[j][i]
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        // Reverse each row
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n / 2; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[i][n - 1 - j];
                matrix[i][n - 1 - j] = temp;
            }
        }
    }
}

## Key Takeaways

- **Transpose + reverse** is an intuitive way to implement 90-degree rotation.
- For 90-degree **counter-clockwise**: transpose + reverse each column instead.
- For 180-degree rotation: reverse each row, then reverse each column (or vice versa).
- The pattern generalizes to other rotation problems.
