# Spiral Matrix — Solution

## Problem Overview

Given an `m x n` matrix, return all elements in spiral order: starting from the top-left corner, move right across the top row, down the right column, left across the bottom row, up the left column, then repeat inward.

## Boundary Shrinking Approach

**Idea:** Maintain four pointers (`top`, `bottom`, `left`, `right`) that define the current layer. Traverse the perimeter, then shrink the boundaries inward.

**Algorithm:**
1. Initialize `top = 0`, `bottom = m - 1`, `left = 0`, `right = n - 1`.
2. While `top <= bottom` and `left <= right`:
   - Traverse left-to-right along `top` row (col from `left` to `right`). Increment `top`.
   - Traverse top-to-bottom along `right` col (row from `top` to `bottom`). Decrement `right`.
   - If `top <= bottom`: traverse right-to-left along `bottom` row (col from `right` to `left`). Decrement `bottom`.
   - If `left <= right`: traverse bottom-to-top along `left` col (row from `bottom` to `top`). Increment `left`.
3. Return the collected elements.

**Complexity Analysis:**
- **Time:** O(m × n) — each element visited exactly once
- **Space:** O(1) extra space (excluding output)

## Deep Dive: How the Boundaries Work

### Visual Walkthrough

For `matrix = [[1,2,3],[4,5,6],[7,8,9]]`:

```
Initial: top=0, bottom=2, left=0, right=2

Pass 1 → (right): Collect row 0 [1,2,3]. top=1
Pass 2 → (down):  Collect col 2 [6,9]. right=1
Pass 3 → (left):  Collect row 2 [8,7]. bottom=1
Pass 4 → (up):    Collect col 0 [4]. left=1
Pass 5 → (right): Collect row 1 [5]. top=2 → loop ends

Result: [1,2,3,6,9,8,7,4,5]
```

### Edge Cases to Consider

- **Single row** (m == 1): Only the left-to-right traversal happens.
- **Single column** (n == 1): Only the top-to-bottom traversal happens.
- **Single element** (m == n == 1): One element collected, loop exits.
- **Rectangle, not square**: Works correctly because each pass checks the loop condition independently.

```java
class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        List<Integer> result = new ArrayList<>();
        int top = 0, bottom = matrix.length - 1;
        int left = 0, right = matrix[0].length - 1;

        while (top <= bottom && left <= right) {
            // Traverse right across top row
            for (int col = left; col <= right; col++)
                result.add(matrix[top][col]);
            top++;

            // Traverse down right column
            for (int row = top; row <= bottom; row++)
                result.add(matrix[row][right]);
            right--;

            // Traverse left across bottom row (if still valid)
            if (top <= bottom) {
                for (int col = right; col >= left; col--)
                    result.add(matrix[bottom][col]);
                bottom--;
            }

            // Traverse up left column (if still valid)
            if (left <= right) {
                for (int row = bottom; row >= top; row--)
                    result.add(matrix[row][left]);
                left++;
            }
        }
        return result;
    }
}
```

```python
def spiral_order(matrix: list[list[int]]) -> list[int]:
    result = []
    top, bottom = 0, len(matrix) - 1
    left, right = 0, len(matrix[0]) - 1

    while top <= bottom and left <= right:
        # Traverse right across top row
        for col in range(left, right + 1):
            result.append(matrix[top][col])
        top += 1

        # Traverse down right column
        for row in range(top, bottom + 1):
            result.append(matrix[row][right])
        right -= 1

        # Traverse left across bottom row (if still valid)
        if top <= bottom:
            for col in range(right, left - 1, -1):
                result.append(matrix[bottom][col])
            bottom -= 1

        # Traverse up left column (if still valid)
        if left <= right:
            for row in range(bottom, top - 1, -1):
                result.append(matrix[row][left])
            left += 1

    return result
```

```rust
fn spiral_order(matrix: Vec<Vec<i32>>) -> Vec<i32> {
    let mut result = Vec::new();
    if matrix.is_empty() { return result; }

    let (mut top, mut bottom) = (0, matrix.len() as i32 - 1);
    let (mut left, mut right) = (0, matrix[0].len() as i32 - 1);

    while top <= bottom && left <= right {
        for col in left..=right {
            result.push(matrix[top as usize][col as usize]);
        }
        top += 1;

        for row in top..=bottom {
            result.push(matrix[row as usize][right as usize]);
        }
        right -= 1;

        if top <= bottom {
            for col in (left..=right).rev() {
                result.push(matrix[bottom as usize][col as usize]);
            }
            bottom -= 1;
        }

        if left <= right {
            for row in (top..=bottom).rev() {
                result.push(matrix[row as usize][left as usize]);
            }
            left += 1;
        }
    }
    result
}
```

## Key Takeaways

- **Boundary technique** (`top`, `bottom`, `left`, `right`) elegantly handles spiral traversal.
- **Direction order** is always: right → down → left → up → repeat.
- **Guard checks** (`if top <= bottom`, `if left <= right`) prevent revisiting elements in non-square matrices.
- **O(m×n) time, O(1) space** is optimal — you must visit every element at least once.
