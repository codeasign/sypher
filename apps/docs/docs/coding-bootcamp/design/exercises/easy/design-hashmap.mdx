# Design HashMap

**Difficulty:** Easy

Design a HashMap without using any built-in hash table libraries.

Implement the `MyHashMap` class:

- `MyHashMap()` initializes the object with an empty map.
- `void put(int key, int value)` inserts a `(key, value)` pair into the HashMap. If the key already exists in the map, update the corresponding value.
- `int get(int key)` returns the value to which the specified key is mapped, or `-1` if this map contains no mapping for the key.
- `void remove(int key)` removes the key and its corresponding value if the map contains the mapping for the key.

## Example 1

**Input:**
```
["MyHashMap","put","put","get","get","put","get","remove","get"]
[[],[1,1],[2,2],[1],[3],[2,1],[2],[2],[2]]
```

**Output:**
```
[null,null,null,1,-1,null,1,null,-1]
```

**Explanation:**
```
MyHashMap myHashMap = new MyHashMap();
myHashMap.put(1, 1);   // The map is now [[1,1]]
myHashMap.put(2, 2);   // The map is now [[1,1],[2,2]]
myHashMap.get(1);      // return 1
myHashMap.get(3);      // return -1 (not found)
myHashMap.put(2, 1);   // update the existing value
myHashMap.get(2);      // return 1
myHashMap.remove(2);   // remove the mapping for 2
myHashMap.get(2);      // return -1 (not found)
```

## Input Specification

The input consists of a sequence of commands:
- `put key value` — insert or update key with value
- `get key` — output the value for key or -1
- `remove key` — delete the key-value mapping

## Output Specification

For each `get` command, output the result on its own line.

## Constraints

- `0 <= key, value <= 10^6`
- At most `10^4` calls will be made to `put`, `get`, and `remove`.

## Hints

<details>
<summary>Hint 1</summary>

Use an array of linked lists (buckets). Compute a hash function to determine which bucket a key belongs to, then handle collisions with a linked list.
</details>

<details>
<summary>Hint 2</summary>

A simple hash function: `index = key % capacity`. Choose a prime number for the capacity, like 1009 or 2069, to reduce collisions.
</details>

<details>
<summary>Hint 3</summary>

Each bucket stores a list of (key, value) pairs. For put, check if the key exists in the bucket and update it, or append a new pair. For get, search the bucket. For remove, delete the pair from the bucket.
</details>

</ProblemEditor>
