# Merge Two Sorted Lists — Solution

## Problem Overview

Merge two sorted linked lists into one sorted list by splicing nodes. This is a classic **two-pointer** linked list problem.

## Approach: Iterative with Dummy Head

**Idea:** Use a dummy head node to simplify edge cases. Compare nodes from both lists, appending the smaller one to the result.

**Algorithm:**
1. Create a dummy head node and a `tail` pointer starting at dummy.
2. While both lists are non-empty:
   - Compare `list1.val` and `list2.val`.
   - Append the smaller node to `tail.next` and advance that list.
   - Advance `tail`.
3. Append any remaining nodes from the non-empty list.
4. Return `dummy.next`.

**Complexity Analysis:**
- **Time:** O(n + m) — each node is visited once.
- **Space:** O(1) — only pointers used.

def merge_two_lists(list1: list[int], list2: list[int]) -> list[int]:
    # Simple array merge for the array-based harness
    result = []
    i = j = 0
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            result.append(list1[i])
            i += 1
        else:
            result.append(list2[j])
            j += 1
    # Append remaining elements
    result.extend(list1[i:])
    result.extend(list2[j:])
    return result

class Solution {
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        // Dummy head avoids special case for the first node
        ListNode dummy = new ListNode(0);
        ListNode tail = dummy;

        // Compare and append the smaller node
        while (list1 != null && list2 != null) {
            if (list1.val <= list2.val) {
                tail.next = list1;
                list1 = list1.next;
            } else {
                tail.next = list2;
                list2 = list2.next;
            }
            tail = tail.next;
        }

        // Append remaining nodes from the non-empty list
        tail.next = (list1 != null) ? list1 : list2;

        return dummy.next;  // head of the merged list
    }
}

## Key Takeaways

- **Dummy head pattern**: Always use a dummy node when building a linked list to avoid null-checking the head.
- **Two-pointer merge**: The standard merge algorithm from Merge Sort, applied to linked lists.
- **Recursive solution**: This can also be solved recursively: `merge(l1, l2) = l1.next = merge(l1.next, l2)` if `l1.val < l2.val`.
