# Linked List

Linked list problems test pointer manipulation and in-place modification skills.

⏱ 20 minutes

## What You Will Learn

- Traversing, reversing, and merging linked lists
- Fast & slow pointer technique for cycle detection
- Dealing with dummy heads and recursion

## The Problem

Linked lists are linear data structures where each node points to the next. Common operations include reversal, merging, detecting cycles, and removing elements.

## Key Techniques

- **Dummy head**: Use a sentinel node to simplify edge cases
- **Two pointers**: Fast/slow for cycle detection, finding middle
- **Recursion**: Natural for reversal and merge operations
- **In-place**: Modify pointers rather than creating new nodes

## Complexity

| Operation | Time | Space |
|-----------|------|-------|
| Traversal | O(n) | O(1) |
| Reversal | O(n) | O(1) or O(n) recursive |
| Cycle detection | O(n) | O(1) |
