# Reorder List

**Difficulty:** Medium

You are given the head of a singly linked-list. The list can be represented as:

`L0 -> L1 -> ... -> Ln-1 -> Ln`

Reorder the list to the following form:

`L0 -> Ln -> L1 -> Ln-1 -> L2 -> Ln-2 -> ...`

You may not modify the values in the nodes, only the nodes themselves may be changed.

## Example 1

**Input:** `head = [1, 2, 3, 4]`  
**Output:** `[1, 4, 2, 3]`  

## Example 2

**Input:** `head = [1, 2, 3, 4, 5]`  
**Output:** `[1, 5, 2, 4, 3]`  

## Input Specification

- A single line of space-separated integers.
- `1 <= list length <= 5 * 10^4`

## Output Specification

Print the reordered list as space-separated integers.

## Constraints

- `1 <= list length <= 5 * 10^4`
- `0 <= Node.val <= 1000`

## Hints

<details>
<summary>Hint 1</summary>

Find the middle of the linked list using slow and fast pointers.
</details>

<details>
<summary>Hint 2</summary>

Reverse the second half of the list.
</details>

<details>
<summary>Hint 3</summary>

Merge the two halves by alternating nodes from each.
</details>
