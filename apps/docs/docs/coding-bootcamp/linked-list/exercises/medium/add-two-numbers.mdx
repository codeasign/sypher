# Add Two Numbers

**Difficulty:** Medium

You are given two **non-empty** linked lists representing two non-negative integers. The digits are stored in **reverse order**, and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

## Example 1

**Input:** `l1 = [2, 4, 3]`, `l2 = [5, 6, 4]`  
**Output:** `[7, 0, 8]`  
**Explanation:** 342 + 465 = 807.

## Example 2

**Input:** `l1 = [0]`, `l2 = [0]`  
**Output:** `[0]`  

## Example 3

**Input:** `l1 = [9, 9, 9, 9, 9, 9, 9]`, `l2 = [9, 9, 9, 9]`  
**Output:** `[8, 9, 9, 9, 0, 0, 0, 1]`  

## Input Specification

- First line: space-separated digits of `l1` (least significant first).
- Second line: space-separated digits of `l2` (least significant first).
- `1 <= list length <= 100`

## Output Specification

Print the sum as space-separated digits (least significant first).

## Constraints

- `1 <= list length <= 100`
- `0 <= Node.val <= 9`
- The digits are stored in reverse order.

## Hints

<details>
<summary>Hint 1</summary>

Traverse both lists simultaneously, adding corresponding digits along with any carry from the previous step.
</details>

<details>
<summary>Hint 2</summary>

The carry is always either 0 or 1 because 9 + 9 + 1 = 19, which is at most 19.
</details>

<details>
<summary>Hint 3</summary>

After the loop, if carry is 1, append a new node with value 1 at the end.
</details>
