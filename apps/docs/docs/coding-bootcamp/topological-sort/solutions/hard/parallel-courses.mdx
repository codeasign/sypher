# Parallel Courses — Solution

## Problem Overview

Given 
 courses and a list of prerequisites where prerequisites[i] = [a, b] means you must take course  before course , determine the minimum number of semesters needed to take all courses. In one semester, you can take any number of courses as long as all prerequisites are satisfied. If impossible (cycle exists), return -1.

## Approach: Kahn's Algorithm with Level-Based BFS

**Idea:** Use Kahn's algorithm but process the queue in levels (batches). Each level represents one semester. All courses that have in-degree 0 at the start of a semester can be taken simultaneously in that semester.

**Algorithm:**
1. Build an adjacency list and compute in-degree for each course.
2. Initialize a queue with all courses that have in-degree 0.
3. Initialize semesters = 0 and processed = 0.
4. While the queue is not empty:
   - Increment semesters.
   - Let size = q.size() (courses available this semester).
   - For i from 0 to size - 1:
     - Dequeue a course, increment processed.
     - For each neighbor, decrement its in-degree. If it becomes 0, enqueue it.
5. If processed == n, return semesters. Otherwise, return -1.

**Why This Works:** Processing the queue in levels ensures that all courses that become available in the same semester are taken together. This gives the minimum number of semesters because we never delay taking a course when it's available.

**Complexity Analysis:**
- **Time:** O(V + E) — each vertex and edge is visited once.
- **Space:** O(V + E) — adjacency list, in-degree array, and queue.

### Edge Cases

- **No prerequisites**: All courses in one semester, return 1.
- **Chain dependencies**: Each semester takes exactly one course (e.g., linear chain of n courses → n semesters).
- **Cycle**: Queue empties before processing all courses → return -1.
- **Single course**: Return 1.

from collections import deque

def min_semesters(n: int, prerequisites: list[list[int]]) -> int:
    # Build adjacency list and in-degree array
    adj = [[] for _ in range(n)]
    indegree = [0] * n

    for a, b in prerequisites:
        adj[b].append(a)
        indegree[a] += 1

    # Initialize queue with courses having no prerequisites
    q = deque([i for i in range(n) if indegree[i] == 0])
    semesters = 0
    processed = 0

    # Process courses semester by semester
    while q:
        semesters += 1
        size = len(q)
        for _ in range(size):
            course = q.popleft()
            processed += 1
            for neighbor in adj[course]:
                indegree[neighbor] -= 1
                if indegree[neighbor] == 0:
                    q.append(neighbor)

    return semesters if processed == n else -1

import java.util.*;

class Solution {
    public int minSemesters(int n, int[][] prerequisites) {
        // Build adjacency list and in-degree array
        List<Integer>[] adj = new ArrayList[n];
        int[] indegree = new int[n];
        for (int i = 0; i < n; i++) {
            adj[i] = new ArrayList<>();
        }
        for (int[] prereq : prerequisites) {
            int a = prereq[0], b = prereq[1];
            adj[b].add(a);
            indegree[a]++;
        }

        // Initialize queue with courses having no prerequisites
        Queue<Integer> q = new LinkedList<>();
        for (int i = 0; i < n; i++) {
            if (indegree[i] == 0) q.offer(i);
        }

        int semesters = 0;
        int processed = 0;

        // Process courses semester by semester
        while (!q.isEmpty()) {
            semesters++;
            int size = q.size();
            for (int i = 0; i < size; i++) {
                int course = q.poll();
                processed++;
                for (int neighbor : adj[course]) {
                    indegree[neighbor]--;
                    if (indegree[neighbor] == 0) q.offer(neighbor);
                }
            }
        }

        return processed == n ? semesters : -1;
    }
}

fn min_semesters(n: i32, prerequisites: Vec<Vec<i32>>) -> i32 {
    let n = n as usize;
    let mut adj = vec![vec![]; n];
    let mut indegree = vec![0; n];

    for prereq in &prerequisites {
        let a = prereq[0] as usize;
        let b = prereq[1] as usize;
        adj[b].push(a);
        indegree[a] += 1;
    }

    // Initialize queue with courses having no prerequisites
    let mut q = VecDeque::new();
    for i in 0..n {
        if indegree[i] == 0 {
            q.push_back(i);
        }
    }

    let mut semesters = 0;
    let mut processed = 0;

    // Process courses semester by semester
    while !q.is_empty() {
        semesters += 1;
        let size = q.len();
        for _ in 0..size {
            let course = q.pop_front().unwrap();
            processed += 1;
            for &neighbor in &adj[course] {
                indegree[neighbor] -= 1;
                if indegree[neighbor] == 0 {
                    q.push_back(neighbor);
                }
            }
        }
    }

    if processed == n { semesters } else { -1 }
}

#include <stdlib.h>

#define MAX_NODES 2000

int minSemesters(int n, int** prerequisites, int prerequisitesSize, int* prerequisitesColSize) {
    int adj[MAX_NODES][MAX_NODES];
    int adjSize[MAX_NODES] = {0};
    int indegree[MAX_NODES] = {0};

    for (int i = 0; i < prerequisitesSize; i++) {
        int a = prerequisites[i][0], b = prerequisites[i][1];
        adj[b][adjSize[b]++] = a;
        indegree[a]++;
    }

    int q[MAX_NODES], front = 0, rear = 0;
    for (int i = 0; i < n; i++) {
        if (indegree[i] == 0) q[rear++] = i;
    }

    int semesters = 0;
    int processed = 0;

    while (front < rear) {
        semesters++;
        int size = rear - front;
        for (int i = 0; i < size; i++) {
            int course = q[front++];
            processed++;
            for (int j = 0; j < adjSize[course]; j++) {
                int neighbor = adj[course][j];
                indegree[neighbor]--;
                if (indegree[neighbor] == 0) q[rear++] = neighbor;
            }
        }
    }

    return processed == n ? semesters : -1;
}

using System.Collections.Generic;

class Solution {
    public int MinSemesters(int n, int[][] prerequisites) {
        // Build adjacency list and in-degree array
        List<int>[] adj = new List<int>[n];
        int[] indegree = new int[n];
        for (int i = 0; i < n; i++) {
            adj[i] = new List<int>();
        }
        foreach (var prereq in prerequisites) {
            int a = prereq[0], b = prereq[1];
            adj[b].Add(a);
            indegree[a]++;
        }

        // Initialize queue with courses having no prerequisites
        Queue<int> q = new Queue<int>();
        for (int i = 0; i < n; i++) {
            if (indegree[i] == 0) q.Enqueue(i);
        }

        int semesters = 0;
        int processed = 0;

        // Process courses semester by semester
        while (q.Count > 0) {
            semesters++;
            int size = q.Count;
            for (int i = 0; i < size; i++) {
                int course = q.Dequeue();
                processed++;
                foreach (int neighbor in adj[course]) {
                    indegree[neighbor]--;
                    if (indegree[neighbor] == 0) q.Enqueue(neighbor);
                }
            }
        }

        return processed == n ? semesters : -1;
    }
}

func minSemesters(n int, prerequisites [][]int) int {
    // Build adjacency list and in-degree array
    adj := make([][]int, n)
    indegree := make([]int, n)
    for i := 0; i < n; i++ {
        adj[i] = []int{}
    }
    for _, prereq := range prerequisites {
        a, b := prereq[0], prereq[1]
        adj[b] = append(adj[b], a)
        indegree[a]++
    }

    // Initialize queue with courses having no prerequisites
    q := make([]int, 0)
    for i := 0; i < n; i++ {
        if indegree[i] == 0 {
            q = append(q, i)
        }
    }

    semesters := 0
    processed := 0

    // Process courses semester by semester
    for len(q) > 0 {
        semesters++
        size := len(q)
        for i := 0; i < size; i++ {
            course := q[0]
            q = q[1:]
            processed++
            for _, neighbor := range adj[course] {
                indegree[neighbor]--
                if indegree[neighbor] == 0 {
                    q = append(q, neighbor)
                }
            }
        }
    }

    if processed == n {
        return semesters
    }
    return -1
}

## Key Takeaways

- **Level-based BFS** in Kahn's algorithm naturally models the semester structure.
- All available courses are taken in the same semester, which guarantees the **minimum number of semesters**.
- The standard **cycle detection** applies: if processed < n, the graph has a cycle.
- The algorithm runs in **O(V + E)** time, making it efficient for large course catalogs.
- This is a classic example of **parallel task scheduling** with dependencies.
