# Course Schedule II — Solution

## Problem Overview

Given 
umCourses courses (0 to numCourses-1) and a list of prerequisites where prerequisites[i] = [a, b] means you must take course  before course , return the ordering of courses to take to finish all courses. If it's impossible (there is a cycle), return an empty array.

## Approach: Kahn's Algorithm

**Idea:** Perform a topological sort using Kahn's algorithm. The result of a valid topological sort gives a valid course ordering.

**Algorithm:**
1. Build an adjacency list and compute in-degree for each course.
2. Initialize a queue with all courses that have in-degree 0.
3. While the queue is not empty:
   - Dequeue a course and add it to the result order.
   - For each neighbor (course that depends on this one), decrement its in-degree.
   - If a neighbor's in-degree becomes 0, enqueue it.
4. If the result contains all courses, return it. Otherwise, a cycle exists — return an empty array.

**Why This Works:** Kahn's algorithm processes courses in topological order. Courses with no prerequisites are taken first. When a course is taken, it may unlock other courses whose prerequisites are now satisfied.

**Complexity Analysis:**
- **Time:** O(V + E) — each vertex and edge is visited once.
- **Space:** O(V + E) — adjacency list, in-degree array, and queue.

```
        q = [2]

Step 3: Dequeue 2, result = [0, 1, 2]
        Neighbors of 2: 3 (indeg 1->0)
        q = [3]

Step 4: Dequeue 3, result = [0, 1, 2, 3]
        q = []

Result: [0, 1, 2, 3] (or [0, 2, 1, 3])
```

### Edge Cases

- **No prerequisites**: All courses can be taken in any order.
- **Cycle**: If the queue empties before processing all courses, a cycle exists.
- **Single course**: Always possible, result is just [0].
- **Disconnected graph**: Multiple dependencies chains can be interleaved.

```python
from collections import deque

def find_order(num_courses: int, prerequisites: list[list[int]]) -> list[int]:
    # Build adjacency list and in-degree array
    adj = [[] for _ in range(num_courses)]
    indegree = [0] * num_courses

    for a, b in prerequisites:
        adj[b].append(a)
        indegree[a] += 1

    # Initialize queue with courses having no prerequisites
    q = deque([i for i in range(num_courses) if indegree[i] == 0])
    result = []

    # Process courses in topological order
    while q:
        course = q.popleft()
        result.append(course)
        for neighbor in adj[course]:
            indegree[neighbor] -= 1
            if indegree[neighbor] == 0:
                q.append(neighbor)

    return result if len(result) == num_courses else []
```

```java
import java.util.*;

class Solution {
    public int[] findOrder(int numCourses, int[][] prerequisites) {
        // Build adjacency list and in-degree array
        List<Integer>[] adj = new ArrayList[numCourses];
        int[] indegree = new int[numCourses];
        for (int i = 0; i < numCourses; i++) {
            adj[i] = new ArrayList<>();
        }
        for (int[] prereq : prerequisites) {
            int a = prereq[0], b = prereq[1];
            adj[b].add(a);
            indegree[a]++;
        }

        // Initialize queue with courses having no prerequisites
        Queue<Integer> q = new LinkedList<>();
        for (int i = 0; i < numCourses; i++) {
            if (indegree[i] == 0) q.offer(i);
        }

        int[] result = new int[numCourses];
        int idx = 0;

        while (!q.isEmpty()) {
            int course = q.poll();
            result[idx++] = course;
            for (int neighbor : adj[course]) {
                indegree[neighbor]--;
                if (indegree[neighbor] == 0) q.offer(neighbor);
            }
        }

        return idx == numCourses ? result : new int[0];
    }
}
```

```rust
    let n = num_courses as usize;
    let mut adj = vec![vec![]; n];
    let mut indegree = vec![0; n];

    for prereq in &prerequisites {
        let a = prereq[0] as usize;
        let b = prereq[1] as usize;
        adj[b].push(a);
        indegree[a] += 1;
    }

    // Initialize queue with courses having no prerequisites
    let mut q = VecDeque::new();
    for i in 0..n {
        if indegree[i] == 0 {
            q.push_back(i);
        }
    }

    let mut result = Vec::new();
    while let Some(course) = q.pop_front() {
        result.push(course as i32);

            indegree[neighbor] -= 1;
            if indegree[neighbor] == 0 {
                q.push_back(neighbor);
            }
        }
```

```c
#include <stdlib.h>

#define MAX_NODES 2000

int* findOrder(int numCourses, int** prerequisites, int prerequisitesSize, int* prerequisitesColSize, int* returnSize) {
    int adj[MAX_NODES][MAX_NODES];
    int adjSize[MAX_NODES] = {0};
    int indegree[MAX_NODES] = {0};

    for (int i = 0; i < prerequisitesSize; i++) {
        int a = prerequisites[i][0], b = prerequisites[i][1];
        adj[b][adjSize[b]++] = a;
        indegree[a]++;
    }

    int q[MAX_NODES], front = 0, rear = 0;
    for (int i = 0; i < numCourses; i++) {
        if (indegree[i] == 0) q[rear++] = i;
    }

    int* result = (int*)malloc(numCourses * sizeof(int));
    int idx = 0;

    while (front < rear) {
        int course = q[front++];
        result[idx++] = course;
        for (int j = 0; j < adjSize[course]; j++) {
            int neighbor = adj[course][j];
            indegree[neighbor]--;
            if (indegree[neighbor] == 0) q[rear++] = neighbor;
        }
    }

    if (idx == numCourses) {
        *returnSize = numCourses;
        return result;
    }

    free(result);
    *returnSize = 0;
    return NULL;
}
```

```csharp
using System.Collections.Generic;

class Solution {
    public int[] FindOrder(int numCourses, int[][] prerequisites) {
        // Build adjacency list and in-degree array
        List<int>[] adj = new List<int>[numCourses];
        int[] indegree = new int[numCourses];
        for (int i = 0; i < numCourses; i++) {
            adj[i] = new List<int>();
        }
        foreach (var prereq in prerequisites) {
            int a = prereq[0], b = prereq[1];
            adj[b].Add(a);
            indegree[a]++;
        }

        // Initialize queue with courses having no prerequisites
        Queue<int> q = new Queue<int>();
        for (int i = 0; i < numCourses; i++) {
            if (indegree[i] == 0) q.Enqueue(i);
        }

        List<int> result = new List<int>();
        while (q.Count > 0) {
            int course = q.Dequeue();
            result.Add(course);
            foreach (int neighbor in adj[course]) {
                indegree[neighbor]--;
                if (indegree[neighbor] == 0) q.Enqueue(neighbor);
            }
        }

        return result.Count == numCourses ? result.ToArray() : new int[0];
    }
}
```

```go
func findOrder(numCourses int, prerequisites [][]int) []int {
    // Build adjacency list and in-degree array
    adj := make([][]int, numCourses)
    indegree := make([]int, numCourses)
    for i := 0; i < numCourses; i++ {
        adj[i] = []int{}
    }
    for _, prereq := range prerequisites {
        a, b := prereq[0], prereq[1]
        adj[b] = append(adj[b], a)
        indegree[a]++
    }

    // Initialize queue with courses having no prerequisites
    q := make([]int, 0)
    for i := 0; i < numCourses; i++ {
        if indegree[i] == 0 {
            q = append(q, i)
        }
    }

    result := make([]int, 0)
    for len(q) > 0 {
        course := q[0]
        q = q[1:]
        result = append(result, course)
        for _, neighbor := range adj[course] {
            indegree[neighbor]--
            if indegree[neighbor] == 0 {
                q = append(q, neighbor)
            }
        }
    }

    if len(result) == numCourses {
        return result
    }
    return []int{}
}
```

## Key Takeaways

- **Kahn's algorithm** directly produces a topological ordering when one exists.
- The algorithm detects cycles naturally — if the result is missing courses, a cycle exists.
- **Multiple valid orderings** often exist; any valid topological sort is acceptable.
- The O(V+E) time complexity makes this efficient for large course catalogs.
