# Design Add and Search Words Data Structure — Solution

## Problem Overview

Design a data structure that supports adding words and searching for words with wildcard support. The `'.'` character in a search query matches any single lowercase letter.

## Approach: Trie with DFS for Wildcard Search

**Idea:** Use a Trie as the underlying storage. For normal characters, traverse as usual. For `'.'` wildcards, recursively try all possible children.

**Algorithm:**

1. **addWord(word):** Insert the word into a standard Trie, character by character.

2. **search(word):** Traverse the Trie with a recursive helper function `searchInNode(word, node, index)`:
   - If `index == word.length`, return `node.isEnd`.
   - If `word[index]` is a lowercase letter, check if the child exists. If yes, recurse; if no, return `false`.
   - If `word[index]` is `'.'`, try all 26 children (or all entries in a map-based Trie). If any child leads to a match, return `true`. Otherwise return `false`.

**Why This Works:** The `'.'` wildcard forces a backtracking search over all possible paths at that position. Since the word length is bounded by 25, the branching factor is manageable.

**Complexity Analysis:**
- **addWord:** O(L) time, O(L) space
- **search:** O(26^W × L) in the worst case, where W is the number of `'.'` wildcards. In practice, much faster because most nodes have few children.
- **Space:** O(N × L) for N words of average length L

## Deep Dive

### Handling the Wildcard

The `'.'` wildcard is what makes this problem "Hard." Without it, a standard Trie `search` would suffice. With wildcards, we need backtracking:

```
search("b.."):
  root → 'b'          (exact match)
        → 'a' or 'd'  (wildcard '.' matches both)
        → any child   (wildcard '.' matches all remaining children)

For each path: check if the node is an end-of-word
```

### Optimization: Pruning with Word Length Tracking

If many words are short and searches are long, we can store a set of word lengths per node to quickly reject searches when no word of the target length exists. This is an optional optimization.

### Edge Cases

- **Empty string search (`""`):** Return `true` if the root's `isEnd` is `true`.
- **All wildcards (`"..."`):** Return `true` if any word of length 3 exists.
- **Multiple consecutive wildcards (`"a..b"`):** The recursion handles each `'.'` independently.
- **No words added, then search:** Return `false`.
- **Duplicate word additions:** Should not cause errors. The `isEnd` flag remains `true`.

```python
class TrieNode:
    def __init__(self):
        self.children = [None] * 26
        self.is_end = False

class WordDictionary:
    def __init__(self):
        self.root = TrieNode()

    def addWord(self, word: str) -> None:
        node = self.root
        for ch in word:
            idx = ord(ch) - ord('a')
            if node.children[idx] is None:
                node.children[idx] = TrieNode()
            node = node.children[idx]
        node.is_end = True

    def search(self, word: str) -> bool:
        return self._search_in_node(word, self.root, 0)

    def _search_in_node(self, word: str, node: TrieNode, index: int) -> bool:
        if index == len(word):
            return node.is_end  # Reached end of search word

        ch = word[index]
        if ch == '.':
            # Try all 26 children for the wildcard
            for child in node.children:
                if child is not None and self._search_in_node(word, child, index + 1):
                    return True
            return False  # No child matched
        else:
            # Exact character match
            idx = ord(ch) - ord('a')
            if node.children[idx] is None:
                return False  # Character not found
            return self._search_in_node(word, node.children[idx], index + 1)
```

```java
class WordDictionary {
    private TrieNode root;

        root = new TrieNode();
    }

    public void addWord(String word) {
        TrieNode node = root;
        for (char ch : word.toCharArray()) {
            int idx = ch - 'a';
            if (node.children[idx] == null) {
                node.children[idx] = new TrieNode();
            }
            node = node.children[idx];
        }
        node.isEnd = true;
    }

    public boolean search(String word) {
        return searchInNode(word, root, 0);
    }

    private boolean searchInNode(String word, TrieNode node, int index) {
        if (index == word.length()) {
            return node.isEnd;  // Reached end of search word
        }

        char ch = word.charAt(index);
        if (ch == '.') {
            // Try all 26 children for the wildcard
            for (TrieNode child : node.children) {
                if (child != null && searchInNode(word, child, index + 1)) {
                    return true;  // Found a match through this child
                }
            }
            return false;  // No child matched
        } else {
            // Exact character match
            int idx = ch - 'a';
            if (node.children[idx] == null) {
                return false;  // Character not found
            }
            return searchInNode(word, node.children[idx], index + 1);
        }
    }
}
```

```rust
use std::collections::HashMap;

struct TrieNode {
    children: HashMap<char, TrieNode>,
    is_end: bool,
}

struct WordDictionary {
    root: TrieNode,
}

impl WordDictionary {
    fn new() -> Self {
        WordDictionary {
            root: TrieNode {
                children: HashMap::new(),
                is_end: false,
            },
        }
    }

    fn add_word(&mut self, word: &str) {
        let mut node = &mut self.root;
        for ch in word.chars() {
            node = node.children.entry(ch).or_insert_with(|| TrieNode {
                children: HashMap::new(),
                is_end: false,
            });
        }
        node.is_end = true;
    }

    fn search(&self, word: &str) -> bool {
        self.search_in_node(word, &self.root, 0)
    }

    fn search_in_node(&self, word: &str, node: &TrieNode, index: usize) -> bool {
        if index == word.len() {
            return node.is_end;
        }

        let ch = word.chars().nth(index).unwrap();
        if ch == '.' {
            // Try all children for the wildcard
            for child in node.children.values() {
                if self.search_in_node(word, child, index + 1) {
                    return true;
                }
            }
            false
        } else {
            // Exact character match
            match node.children.get(&ch) {
                Some(child) => self.search_in_node(word, child, index + 1),
                None => false,
            }
        }
    }
}
```

```c
#include <stdbool.h>
#include <stdlib.h>
#include <string.h>

typedef struct TrieNode {
    struct TrieNode* children[26];
    bool isEnd;
} TrieNode;

typedef struct {
    TrieNode* root;
} WordDictionary;

TrieNode* createNode() {
    return (TrieNode*)calloc(1, sizeof(TrieNode));
}

WordDictionary* wordDictionaryCreate() {
    WordDictionary* obj = (WordDictionary*)malloc(sizeof(WordDictionary));
    obj->root = createNode();
    return obj;
}

void wordDictionaryAddWord(WordDictionary* obj, char* word) {
    TrieNode* node = obj->root;
    for (int i = 0; word[i]; i++) {
        int idx = word[i] - 'a';
        if (node->children[idx] == NULL) {
            node->children[idx] = createNode();
        }
        node = node->children[idx];
    }
    node->isEnd = true;
}

bool searchInNode(char* word, TrieNode* node, int index) {
    if (word[index] == '\0') {
        return node->isEnd;  // Reached end of search word
    }

    if (word[index] == '.') {
        // Try all 26 children for the wildcard
        for (int i = 0; i < 26; i++) {
            if (node->children[i] != NULL &&
                searchInNode(word, node->children[i], index + 1)) {
                return true;  // Found a match through this child
            }
        }
        return false;  // No child matched
    } else {
        // Exact character match
        int idx = word[index] - 'a';
        if (node->children[idx] == NULL) return false;
        return searchInNode(word, node->children[idx], index + 1);
    }
}

bool wordDictionarySearch(WordDictionary* obj, char* word) {
    return searchInNode(word, obj->root, 0);
}

void freeNode(TrieNode* node) {
    if (!node) return;
    for (int i = 0; i < 26; i++) {
        freeNode(node->children[i]);
    }
    free(node);
}

void wordDictionaryFree(WordDictionary* obj) {
    freeNode(obj->root);
    free(obj);
}
```

## Key Takeaways

- The **wildcard `'.'`** forces a DFS backtracking search at that position, trying all possible children
- Without wildcards, search is O(L); with wildcards, it can be O(26^W × L) but is much faster in practice
- The recursive `searchInNode` helper keeps the search logic clean and composable
- This problem extends the standard Trie to handle **pattern matching**, not just exact matching
- The Trie's prefix-based structure makes wildcard search feasible by exploring only existing paths, not all possible strings
