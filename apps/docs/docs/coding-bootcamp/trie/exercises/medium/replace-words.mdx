# Replace Words

**Difficulty:** Medium

In English, we have a concept called **root**, which can be followed by some other word to form a longer word — let's call this word a **successor**. For example, when the root `"an"` is followed by the successor `"other"`, we can form a new word `"another"`.

Given a `dictionary` consisting of many roots and a `sentence`, replace all successors in the sentence with the root forming it. If a successor can be replaced by more than one root, replace it with the root with the **shortest length**.

Return the sentence after the replacement.

## Example 1

**Input:**  
```
dictionary = ["cat","bat","rat"]
sentence = "the cattle was rattled by the battery"
```
**Output:** `"the cat was rat by the bat"`

## Example 2

**Input:**
```
dictionary = ["a","b","c"]
sentence = "aadsf bdsf cdsf"
```
**Output:** `"a b c"`

## Input Specification

First line: space-separated list of root words (the dictionary).  
Second line: the sentence to process.

## Output Specification

Print the sentence with successors replaced by their shortest matching roots.

## Constraints

- `1 <= dictionary.length <= 1000`
- `1 <= dictionary[i].length <= 100`
- `1 <= sentence.length <= 10⁶`
- `dictionary[i]` and `sentence` consist only of lowercase English letters
- `sentence` contains space-separated words
- All `dictionary[i]` are unique

## Hints

<details>
<summary>Hint 1</summary>

Build a Trie from the dictionary roots. For each word in the sentence, traverse the Trie character by character. If you find a node that is an end-of-word, that's the shortest root.
</details>

<details>
<summary>Hint 2</summary>

If no root is found in the Trie for a word, keep the original word unchanged.
</details>

<details>
<summary>Hint 3</summary>

Be careful with the sentence length constraint — an O(N * L) solution per word is acceptable, but you should avoid O(N * K) where K is the dictionary size.
</details>
