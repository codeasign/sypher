# Word Ladder — Solution

## Problem Overview

Given `beginWord`, `endWord`, and a dictionary `wordList`, find the length of the shortest transformation sequence from `beginWord` to `endWord` where only one letter changes at each step and every intermediate word must exist in `wordList`. Return `0` if no such sequence exists.

This is fundamentally a **shortest path problem on an unweighted graph**, where each word is a node and an edge exists between two words that differ by exactly one character.

## Brute Force Approach

**Idea:** Generate all possible transformation sequences by exploring every possible word at each step, checking which words in the wordList differ by exactly one character from the current word.

**Algorithm:**
1. Start with `beginWord` as the current word
2. For each position in the current word, try replacing it with each letter `'a'` to `'z'`
3. If the resulting word is in `wordList`, recursively explore from that word
4. Keep track of visited words to avoid cycles
5. Return the length when `endWord` is reached, or `0` after exhausting all paths

**Complexity Analysis:**
- **Time:** O(M * 26^L) in the worst case, where M is the word length and L is the depth of the search — completely impractical
- **Space:** O(N) for the recursion stack and visited set

**Why this is suboptimal:** The brute force explores all possible paths without any pruning. The branching factor (up to 26 letters per position) leads to exponential explosion. BFS provides a principled way to find the shortest path without exploring all paths.

## Optimal Approach: Breadth-First Search (BFS)

**Idea:** Use BFS to explore the graph level by level. Since all edges have equal weight, the first time we encounter `endWord`, we have found the shortest path. Use a set for O(1) word lookup and a queue to track the current word and its transformation length.

**Algorithm:**
1. Convert `wordList` to a set for O(1) membership checks
2. If `endWord` is not in the set, return `0` immediately
3. Initialize a queue with `(beginWord, 1)` — the pair represents (current word, sequence length so far)
4. Remove `beginWord` from the set (mark as visited)
5. While the queue is not empty:
   - Dequeue the front element `(word, length)`
   - For each character position in `word`:
     - Replace the character with each letter `'a'` to `'z'`
     - If the new word equals `endWord`, return `length + 1`
     - If the new word is in the set, add `(newWord, length + 1)` to the queue and remove it from the set
6. Return `0` if the queue empties without reaching `endWord`

**Why This Works:** BFS explores all nodes at distance 1 from the start before distance 2, and so on. Since all edges have equal weight (each transformation counts as 1 step), the first time we reach `endWord` is guaranteed to be the shortest path. The visited set ensures we never revisit a word, preventing infinite loops.

**Complexity Analysis:**
- **Time:** O(M^2 * N), where M is the length of each word and N is the size of the wordList. For each word in the queue (up to N), we generate O(M * 26) possible transformations, and each transformation involves creating a new string of length M (O(M)).
- **Space:** O(M * N) — the queue and set store up to N words, each of length M.

**Optimization — Bidirectional BFS:** A common optimization is to run BFS from both the start and the end simultaneously, alternating between the two frontiers. Since BFS expands outward in all directions, the two searches meet in the middle, reducing the effective branching factor. This can improve the time complexity in practice, though the worst-case remains the same.

## Deep Dive: How BFS Solves Word Ladder

### Visual Walkthrough

For `beginWord = "hit"`, `endWord = "cog"`, `wordList = ["hot","dot","dog","lot","log","cog"]`:

```
Level 1:              "hit"           (length = 1)
                        |
                      "hot"           (hit → hot: change 'i' to 'o')
Level 2:              "hot"           (length = 2)
                      /   \
                   "dot"   "lot"      (hot → dot: change 'h' to 'd')
Level 3:           /         \        (hot → lot: change 'h' to 'l')
                "dot"       "lot"     (length = 3)
                 |           |
               "dog"       "log"      (dot → dog: change 't' to 'g')
Level 4:        /             \       (lot → log: change 't' to 'g')
             "dog"          "log"     (length = 4)
                 \           /
                  "cog"    "cog"      (dog → cog: change 'd' to 'c')
Level 5:              "cog"           (length = 5)
                ✓ Found! Return 5
```

The BFS queue processes level by level:
- Level 1: `["hit"]` — beginWord, length 1
- Level 2: `["hot"]` — length 2, found by changing 'i' → 'o'
- Level 3: `["dot", "lot"]` — length 3, both neighbors of "hot"
- Level 4: `["dog", "log"]` — length 4, neighbors of "dot" and "lot"
- Level 5: `["cog"]` — length 5, neighbor of "dog" or "log", matches endWord

### Edge Cases to Consider

- **endWord not in wordList**: Return 0 immediately — no transformation is possible.
- **beginWord equals endWord**: The sequence length is 1 (just the starting word).
- **Empty word list**: Return 0 — there are no intermediate words to use.
- **No valid path**: BFS exhausts all reachable words without finding endWord; return 0.
- **beginWord in wordList**: Remove it from the set to avoid cycles, but it doesn't affect the path length.
- **Single-step transformation**: If beginWord differs from endWord by one letter and endWord is in wordList, return 2.

### Why BFS Over DFS for This Problem

DFS would find a path, but not necessarily the shortest one. You would need to explore all paths to find the minimum, which is far less efficient. BFS explores level by level, so the first time you encounter the target, you are guaranteed to have the shortest path. This property makes BFS the natural choice for unweighted shortest path problems.

### Bidirectional BFS Optimization

Instead of expanding from `beginWord` only, bidirectional BFS expands from both `beginWord` and `endWord` simultaneously. When the two frontiers meet, the shortest path is found. This reduces the search space from O(b^d) to approximately O(b^(d/2) + b^(d/2)), where b is the branching factor and d is the depth.

```
Standard BFS:       ○ → ○ → ○ → ○ → ○ → ○ → ○ → ●
                    ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                    Expands exponentially to depth d

Bidirectional:      ○ → ○ → ○ → ●
                    ^^^^^^^^^  ←  ^^^^^^^^^
                    Left BFS    Right BFS
                    meets at depth d/2
```

class Solution {
    public int ladderLength(String beginWord, String endWord, List<String> wordList) {
        // Convert wordList to a set for O(1) membership checks
        Set<String> wordSet = new HashSet<>(wordList);

        // If endWord is not in the set, transformation is impossible
        if (!wordSet.contains(endWord)) return 0;

        // BFS queue stores (currentWord, sequenceLength)
        Queue<Pair<String, Integer>> queue = new LinkedList<>();
        queue.offer(new Pair<>(beginWord, 1));

        // Remove beginWord from set to avoid revisiting it
        wordSet.remove(beginWord);

        while (!queue.isEmpty()) {
            Pair<String, Integer> current = queue.poll();
            String word = current.getKey();
            int length = current.getValue();

            // Try changing each character position
            for (int i = 0; i < word.length(); i++) {
                char[] chars = word.toCharArray();
                // Try every lowercase letter at position i
                for (char c = 'a'; c <= 'z'; c++) {
                    chars[i] = c;
                    String newWord = new String(chars);

                    // If this word is endWord, return the sequence length
                    if (newWord.equals(endWord)) return length + 1;

                    // If this word is in the set, add to queue and remove from set
                    if (wordSet.contains(newWord)) {
                        queue.offer(new Pair<>(newWord, length + 1));
                        wordSet.remove(newWord);  // Mark visited immediately
                    }
                }
            }
        }

        // No path found
        return 0;
    }
}

class Solution {
public:
    int ladderLength(string beginWord, string endWord, vector<string>& wordList) {
        // Convert wordList to unordered_set for O(1) lookups
        unordered_set<string> wordSet(wordList.begin(), wordList.end());

        // If endWord is not in the set, no transformation is possible
        if (wordSet.find(endWord) == wordSet.end()) return 0;

        // BFS queue: each element is (currentWord, sequenceLength)
        queue<pair<string, int>> q;
        q.push({beginWord, 1});

        // Remove beginWord from set to prevent revisiting
        wordSet.erase(beginWord);

        while (!q.empty()) {
            auto [word, length] = q.front();
            q.pop();

            // Try changing each position in the word
            for (int i = 0; i < word.length(); i++) {
                char original = word[i];
                // Try every lowercase letter at position i
                for (char c = 'a'; c <= 'z'; c++) {
                    if (c == original) continue;  // Skip the same character
                    string newWord = word;
                    newWord[i] = c;

                    // Found the target word — return the path length
                    if (newWord == endWord) return length + 1;

                    // If newWord is in the set, add to queue and remove from set
                    if (wordSet.find(newWord) != wordSet.end()) {
                        q.push({newWord, length + 1});
                        wordSet.erase(newWord);  // Mark as visited
                    }
                }
            }
        }

        // Exhausted all reachable words without finding endWord
        return 0;
    }
};

from collections import deque
from typing import List

def ladderLength(beginWord: str, endWord: str, wordList: List[str]) -> int:
    # Convert wordList to a set for O(1) membership checks
    word_set = set(wordList)

    # If endWord is not in the set, no valid transformation exists
    if endWord not in word_set:
        return 0

    # BFS queue: each element is (currentWord, sequenceLength)
    queue = deque([(beginWord, 1)])

    # Remove beginWord from set to prevent revisiting it
    word_set.discard(beginWord)

    while queue:
        word, length = queue.popleft()

        # Try changing each character position in the word
        for i in range(len(word)):
            # Try every lowercase letter at position i
            for c in 'abcdefghijklmnopqrstuvwxyz':
                if c == word[i]:
                    continue  # Skip the same character
                new_word = word[:i] + c + word[i + 1:]

                # If this is the target word, return the sequence length
                if new_word == endWord:
                    return length + 1

                # If new_word is in the set, add to queue and mark visited
                if new_word in word_set:
                    queue.append((new_word, length + 1))
                    word_set.remove(new_word)  # Remove prevents revisiting

    # No transformation sequence found
    return 0

use std::collections::{HashSet, VecDeque};

fn ladder_length(begin_word: &str, end_word: &str, word_list: Vec<&str>) -> i32 {
    // Convert wordList to a HashSet for O(1) membership checks
    let mut word_set: HashSet<String> = word_list.iter().map(|&w| w.to_string()).collect();

    // If endWord is not in the set, transformation is impossible
    if !word_set.contains(end_word) {
        return 0;
    }

    // BFS queue: each element is (currentWord, sequenceLength)
    let mut queue = VecDeque::new();
    queue.push_back((begin_word.to_string(), 1));

    // Remove beginWord from set to avoid revisiting
    word_set.remove(begin_word);

    while let Some((word, length)) = queue.pop_front() {
        // Try changing each character position in the word
        for i in 0..word.len() {
            let mut chars: Vec<char> = word.chars().collect();
            // Try every lowercase letter at position i
            for c in b'a'..=b'z' {
                if chars[i] == c as char {
                    continue;  // Skip the same character
                }
                chars[i] = c as char;
                let new_word: String = chars.iter().collect();

                // Found the target word — return the sequence length
                if new_word == end_word {
                    return length + 1;
                }

                // If newWord is in the set, add to queue and remove from set
                if word_set.contains(&new_word) {
                    queue.push_back((new_word.clone(), length + 1));
                    word_set.remove(&new_word);  // Mark as visited
                }
            }
        }
    }

    // No path found — return 0
    0
}

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <stdbool.h>

// Simple queue node for BFS
typedef struct {
    char word[15];
    int length;
} QueueNode;

int ladderLength(char* beginWord, char* endWord, char** wordList, int wordListSize) {
    // Use a simple visited array (1 = in use/visited)
    int* visited = (int*)calloc(wordListSize, sizeof(int));
    int wordLen = strlen(beginWord);

    // Check if endWord is in wordList
    bool endFound = false;
    for (int i = 0; i < wordListSize; i++) {
        if (strcmp(wordList[i], endWord) == 0) {
            endFound = true;
            break;
        }
    }
    if (!endFound) { free(visited); return 0; }

    // BFS queue (simple array-based queue)
    QueueNode* queue = (QueueNode*)malloc((wordListSize + 1) * sizeof(QueueNode));
    int front = 0, rear = 0;

    // Start with beginWord
    strcpy(queue[rear].word, beginWord);
    queue[rear].length = 1;
    rear++;

    // Mark words in wordList that match beginWord as visited
    for (int i = 0; i < wordListSize; i++) {
        if (strcmp(wordList[i], beginWord) == 0) {
            visited[i] = 1;
            break;
        }
    }

    char temp[15];
    while (front < rear) {
        QueueNode cur = queue[front++];

        // Try changing each character position
        for (int i = 0; i < wordLen; i++) {
            for (char c = 'a'; c <= 'z'; c++) {
                if (c == cur.word[i]) continue;  // Skip same character
                strcpy(temp, cur.word);
                temp[i] = c;
                temp[wordLen] = '\0';

                // Check if this is the endWord
                if (strcmp(temp, endWord) == 0) {
                    free(queue);
                    free(visited);
                    return cur.length + 1;
                }

                // Check if temp is in wordList and not visited
                for (int j = 0; j < wordListSize; j++) {
                    if (!visited[j] && strcmp(wordList[j], temp) == 0) {
                        strcpy(queue[rear].word, temp);
                        queue[rear].length = cur.length + 1;
                        rear++;
                        visited[j] = 1;  // Mark visited
                        break;
                    }
                }
            }
        }
    }

    free(queue);
    free(visited);
    return 0;  // No path found
}

using System;
using System.Collections.Generic;

class Solution {
    public int LadderLength(string beginWord, string endWord, IList<string> wordList) {
        // Convert wordList to a HashSet for O(1) membership checks
        HashSet<string> wordSet = new HashSet<string>(wordList);

        // If endWord is not in the set, transformation is impossible
        if (!wordSet.Contains(endWord)) return 0;

        // BFS queue stores tuples of (currentWord, sequenceLength)
        Queue<Tuple<string, int>> queue = new Queue<Tuple<string, int>>();
        queue.Enqueue(Tuple.Create(beginWord, 1));

        // Remove beginWord from set to avoid revisiting it
        wordSet.Remove(beginWord);

        while (queue.Count > 0) {
            var current = queue.Dequeue();
            string word = current.Item1;
            int length = current.Item2;

            char[] chars = word.ToCharArray();

            // Try changing each character position in the word
            for (int i = 0; i < word.Length; i++) {
                char original = chars[i];
                // Try every lowercase letter at position i
                for (char c = 'a'; c <= 'z'; c++) {
                    if (c == original) continue;  // Skip the same character
                    chars[i] = c;
                    string newWord = new string(chars);

                    // Found the target word — return the sequence length
                    if (newWord == endWord) return length + 1;

                    // If newWord is in the set, add to queue and remove from set
                    if (wordSet.Contains(newWord)) {
                        queue.Enqueue(Tuple.Create(newWord, length + 1));
                        wordSet.Remove(newWord);  // Mark visited immediately
                    }
                }
                chars[i] = original;  // Restore original character
            }
        }

        // No path found — return 0
        return 0;
    }

## Key Takeaways

- **BFS is the natural choice** for shortest path problems on unweighted graphs — it explores level by level, guaranteeing the shortest path the first time the target is reached.
- **Model the problem as a graph**: Each word is a node, and an edge exists between words that differ by exactly one character.
- **Generate neighbors on the fly**: Instead of precomputing all pairwise differences, generate all possible words that differ by one character (O(M * 26) per word) and check membership in the word set.
- **Use a set for O(1) lookups**: Converting the word list to a set is essential for efficient neighbor checking and cycle prevention.
- **Remove words from the set when visited** to avoid revisiting, which is more efficient than maintaining a separate visited set.
- **Time complexity O(M^2 * N)** comes from generating O(M * 26) new strings per word, each requiring O(M) to construct.
- **Bidirectional BFS** is a powerful optimization that can reduce the search space from O(b^d) to O(b^(d/2)) by searching from both ends simultaneously.
