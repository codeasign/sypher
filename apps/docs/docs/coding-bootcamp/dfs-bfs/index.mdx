# DFS & BFS

Depth-First Search (DFS) and Breadth-First Search (BFS) are fundamental graph traversal algorithms.

⏱ 25 minutes

## What You Will Learn

- DFS (recursive and iterative) and BFS approaches
- When to use DFS vs BFS for trees and graphs
- Common graph patterns: connected components, shortest path

## The Concept

DFS explores as far as possible along each branch before backtracking. BFS explores all neighbors at the current level before going deeper.

## When to Use Each

| Use Case | Algorithm |
|----------|-----------|
| Find if path exists | DFS |
| Shortest path in unweighted graph | BFS |
| Topological sort | DFS |
| Connected components | Either |
| Tree traversals | DFS (inorder/preorder/postorder) |
| Level-order traversal | BFS |
| Detect cycles | DFS |

## Complexity

| Metric | DFS | BFS |
|--------|-----|-----|
| Time   | O(V + E) | O(V + E) |
| Space  | O(H) recursion stack | O(W) queue width |
