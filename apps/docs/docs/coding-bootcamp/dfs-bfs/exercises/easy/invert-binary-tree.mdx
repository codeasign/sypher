# Invert Binary Tree

**Difficulty:** Easy

Given the `root` of a binary tree, **invert** the tree and return its root.

Inverting a binary tree means swapping the left and right children of every node in the tree. This is also known as creating a **mirror** of the tree.

## Example 1

```
Original:
    4
   / \
  2   7
 / \ / \
1  3 6  9

Inverted:
    4
   / \
  7   2
 / \ / \
9  6 3  1
```

**Input:**
```
7
4 2 7 1 3 6 9
```

**Output:**
```
4 7 2 9 6 3 1
```

**Explanation:** Every node's left and right children are swapped, producing the mirror image.

## Example 2

```
Original:    Inverted:
    1           1
   /             \
  2               2
```

**Input:**
