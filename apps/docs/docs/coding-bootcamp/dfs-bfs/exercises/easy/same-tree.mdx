# Same Tree

**Difficulty:** Easy

Given the roots of two binary trees `p` and `q`, write a function to check if they are the same or not.

Two binary trees are considered the same if they are **structurally identical**, and the nodes have the **same value**.

## Example 1

```
Tree p:        Tree q:
    1             1
   / \           / \
  2   3         2   3
```

**Input:**
```
3
1 2 3
3
1 2 3
```

**Output:** `true`
**Explanation:** Both trees are structurally identical and have the same values at every node.

## Example 2

```
Tree p:        Tree q:
    1             1
   /               \
  2                 2
```

**Input:**
```
3
1 2 -1
3
1 -1 2
```

**Output:** `false`
**Explanation:** The trees have different structures. Tree `p` has a left child, while tree `q` has a right child.

## Example 3

```
Tree p:        Tree q:
    1             1
   / \           / \
  2   1         1   2
```

**Input:**
```
3
1 2 1
3
1 1 2
```

**Output:** `false`
**Explanation:** The trees have the same structure but different values.

## Input Specification

- First line: an integer `n1` — number of nodes in tree `p` (`0 <= n1 <= 10^4`)
- Second line: `n1` space-separated integers representing the level-order traversal of tree `p` (use `-1` for null nodes)
- Third line: an integer `n2` — number of nodes in tree `q` (`0 <= n2 <= 10^4`)
- Fourth line: `n2` space-separated integers representing the level-order traversal of tree `q` (use `-1` for null nodes)

## Output Specification

Print `true` if the two trees are the same, otherwise `false`.

## Constraints

- The number of nodes in each tree is in the range `[0, 100]`
- `-10^4 <= Node.val <= 10^4`
- Both trees are binary trees

## Hints

<details>
<summary>Hint 1</summary>

Use recursion to traverse both trees simultaneously. Check the current node's value, then recurse on the left and right subtrees.
</details>

<details>
<summary>Hint 2</summary>

The base case: if both nodes are `null`, the subtrees are identical at this point. If one is `null` and the other is not, they differ.
</details>

<details>
<summary>Hint 3</summary>

The recursion pattern is: check current values, then recursively check left children AND right children. Both must match for the trees to be the same.
</details>
