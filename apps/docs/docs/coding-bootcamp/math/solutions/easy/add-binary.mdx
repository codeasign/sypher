# Add Binary — Solution

## Problem Recap

Given two binary strings `a` and `b`, return their sum as a binary string.

## Approach: Bit-by-Bit Addition with Carry

Simulate the manual binary addition process:

1. Start from the rightmost (least significant) bit of both strings.
2. At each step compute `digit_sum = bit_a + bit_b + carry`.
3. Append `digit_sum % 2` to the result, set `carry = digit_sum // 2`.
4. Continue until both strings are consumed and carry is 0.
5. Reverse the accumulated result.

**Complexity:** O(max(n, m)) time where n and m are the lengths of the two strings. O(max(n, m)) space for the output.

```python
def add_binary(a: str, b: str) -> str:
    i, j = len(a) - 1, len(b) - 1
    carry = 0
    result = []
    while i >= 0 or j >= 0 or carry:
        bit_a = int(a[i]) if i >= 0 else 0
        bit_b = int(b[j]) if j >= 0 else 0
        total = bit_a + bit_b + carry
        result.append(str(total % 2))
        carry = total // 2
        i -= 1
        j -= 1
    return ''.join(reversed(result))
```

## Alternative: Built-in Conversion (Python)

```python
def add_binary(a: str, b: str) -> str:
    return bin(int(a, 2) + int(b, 2))[2:]
```

This is concise for Python but may not be allowed in interviews that require a manual implementation.

## Walkthrough: `a = "11"`, `b = "1"`

```
Step 1: i=1, j=0  → bit_a=1, bit_b=1, carry=0 → total=2 → result=['0'], carry=1
Step 2: i=0, j=-1 → bit_a=1, bit_b=0, carry=1 → total=2 → result=['0','0'], carry=1
Step 3: i=-1,j=-1 → bit_a=0, bit_b=0, carry=1 → total=1 → result=['0','0','1'], carry=0
Reversed: "100"
```
