# Plus One — Solution

## Approach

We iterate from the rightmost digit (least significant) to the leftmost. We add 1 to the last digit, and if it becomes 10, we set it to 0 and carry 1 to the next digit. If the carry propagates past the first digit, we need to insert a 1 at the beginning.

**Time Complexity:** O(n) where n is the length of the array
**Space Complexity:** O(1) extra (excluding the output array)

## Steps

1. Start from the last index and move left
2. Add 1 to the current digit
3. If the digit is now 10, set it to 0 and continue carrying
4. If the digit is less than 10, return the array immediately
5. If we exit the loop, all digits were 9 — prepend 1

## Visual

```
Input: [1, 2, 9]

Step 1: [1, 2, 9]  → add 1 to last: 9+1=10 → set to 0, carry 1
Step 2: [1, 2, 0]  → add carry: 2+1=3 → no more carry
Step 3: [1, 3, 0]  → done

Result: [1, 3, 0]
```

def plus_one(digits: list[int]) -> list[int]:
    # Start from the rightmost digit
    for i in range(len(digits) - 1, -1, -1):
        # Add 1 to current digit
        digits[i] += 1
        # If it becomes 10, carry 1 to next digit
        if digits[i] == 10:
            digits[i] = 0
        else:
            # No carry, we're done
            return digits
    # All digits were 9, need to prepend 1
    return [1] + digits

fn plus_one(mut digits: Vec<i32>) -> Vec<i32> {
    let n = digits.len();
    // Start from the rightmost digit
    for i in (0..n).rev() {
        // Add 1 to current digit
        digits[i] += 1;
        // If it becomes 10, carry 1 to next digit
        if digits[i] == 10 {

        } else {

            return digits;
        }

    result

int* plusOne(int* digits, int digitsSize, int* returnSize) {

    for (int i = digitsSize - 1; i >= 0; i--) {

        digits[i]++;

        if (digits[i] == 10) {

    *returnSize = digitsSize + 1;

    return result;

## Key Takeaways

- Process from right to left (least significant digit first)
- Once a digit is less than 10 after increment, we can stop immediately
- The only case requiring a new array is when all original digits were 9
- The worst case [9, 9, 9, ..., 9] is the only scenario where we need O(n) extra space
