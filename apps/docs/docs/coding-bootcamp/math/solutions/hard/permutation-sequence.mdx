# Permutation Sequence — Solution

## Approach

Use the **factorial number system**. For a given `n`, there are `(n-1)!` permutations starting with each digit. By dividing `(k-1)` by `(n-1)!`, we determine which digit comes first. We then remove that digit, update `k`, and repeat for the remaining positions.

**Time Complexity:** O(n²) — we remove elements from a list in O(n) each iteration
**Space Complexity:** O(n) for the list of digits

## Steps

1. Precompute factorials up to `n`
2. Create a list of available digits `[1, 2, ..., n]`
3. Convert `k` to 0-indexed: `k = k - 1`
4. For each position from `n` down to `1`:
   - Compute the index: `idx = k / (n-1)!`
   - Select the digit at that index from the available list
   - Remove the selected digit from the list
   - Update `k = k % (n-1)!`
5. Return the concatenated result

## Visual

```
n = 4, k = 9

Factorials: 0! = 1, 1! = 1, 2! = 2, 3! = 6, 4! = 24

Available digits: [1, 2, 3, 4]
k = 9 - 1 = 8 (0-indexed)

Position 1 (n=4): idx = 8 / 3! = 8 / 6 = 1
  Select digits[1] = 2, remove 2 → remaining: [1, 3, 4]
  k = 8 % 6 = 2

Position 2 (n=3): idx = 2 / 2! = 2 / 2 = 1
  Select digits[1] = 3, remove 3 → remaining: [1, 4]
  k = 2 % 2 = 0

Position 3 (n=2): idx = 0 / 1! = 0 / 1 = 0
  Select digits[0] = 1, remove 1 → remaining: [4]
  k = 0 % 1 = 0

Position 4 (n=1): idx = 0 / 0! = 0 / 1 = 0
  Select digits[0] = 4
```

```python
def get_permutation(n: int, k: int) -> str:
    # Precompute factorials
    fact = [1] * (n + 1)
    for i in range(1, n + 1):
        fact[i] = fact[i - 1] * i

    # Available digits
    digits = list(range(1, n + 1))

    # Convert to 0-indexed
    k -= 1

    result = []
    for i in range(n, 0, -1):
        # Determine which digit comes next
        idx = k // fact[i - 1]
        k %= fact[i - 1]
        # Select and remove the digit
        result.append(str(digits.pop(idx)))

    return ''.join(result)
```

```rust
fn get_permutation(n: i32, k: i32) -> String {
    let n = n as usize;
    let mut k = k as usize - 1;  // Convert to 0-indexed

    // Precompute factorials
    let mut fact = vec![1usize; n + 1];
    for i in 1..=n {
        fact[i] = fact[i - 1] * i;
    }

    // Available digits
    let mut digits: Vec<usize> = (1..=n).collect();

    let mut result = String::new();
    for i in (1..=n).rev() {
        // Determine which digit comes next
        let idx = k / fact[i - 1];
        k %= fact[i - 1];
        // Select and remove the digit
        result.push(char::from_digit(digits[idx] as u32, 10).unwrap());
        digits.remove(idx);
    }

    result
}
```

```c
char* getPermutation(int n, int k) {
    // Precompute factorials
    int* fact = malloc((n + 1) * sizeof(int));
    fact[0] = 1;
    for (int i = 1; i <= n; i++) {
        fact[i] = fact[i - 1] * i;
    }

    // Available digits
    int* digits = malloc(n * sizeof(int));
    for (int i = 0; i < n; i++) {
        digits[i] = i + 1;
    }

    // Convert to 0-indexed
    k--;

    char* result = malloc((n + 1) * sizeof(char));
    int resultIdx = 0;
    int remaining = n;

    for (int i = n; i > 0; i--) {
        // Determine which digit comes next
        int idx = k / fact[i - 1];
        k %= fact[i - 1];
        // Select the digit
        result[resultIdx++] = '0' + digits[idx];
        // Remove the digit by shifting
        for (int j = idx; j < remaining - 1; j++) {
            digits[j] = digits[j + 1];
        }
        remaining--;
    }
    result[resultIdx] = '\0';

    free(fact);
    free(digits);
    return result;
}
```

```go
func getPermutation(n int, k int) string {
    // Precompute factorials
    fact := make([]int, n+1)
    fact[0] = 1
    for i := 1; i <= n; i++ {
        fact[i] = fact[i-1] * i
    }

    // Available digits
    digits := make([]int, n)
    for i := 0; i < n; i++ {
        digits[i] = i + 1
    }

    // Convert to 0-indexed
    k--

    result := make([]byte, 0, n)
    for i := n; i > 0; i-- {
        // Determine which digit comes next
        idx := k / fact[i-1]
        k %= fact[i-1]
        // Select the digit
        result = append(result, byte('0'+digits[idx]))
        // Remove the digit
        digits = append(digits[:idx], digits[idx+1:]...)
    }

    return string(result)
}
```

## Key Takeaways

- The factorial number system maps an integer to a permutation directly — no need to generate all permutations
- Converting to 0-indexed (`k-1`) makes the division and modulo work cleanly
- Since `n <= 9`, factorials fit in 32-bit integers (9! = 362880)
- The algorithm is O(n²) due to list removal, but this is fine for `n <= 9`
- Alternative: use a Fenwick tree (BIT) to reduce removal to O(log n), but unnecessary for small n
