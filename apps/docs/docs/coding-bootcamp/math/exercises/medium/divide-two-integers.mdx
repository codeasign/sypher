# Divide Two Integers

Given two integers `dividend` and `divisor`, divide two integers **without** using multiplication, division, or mod operators.

Return the quotient after dividing `dividend` by `divisor`.

**Truncation:** The integer division should truncate toward zero, meaning the fractional part is discarded.

**Overflow handling:** If the result is outside the 32-bit signed integer range `[-2³¹, 2³¹ - 1]`, return `2³¹ - 1` (2147483647).

## Examples

**Example 1:**
```
Input: dividend = 10, divisor = 3
Output: 3
Explanation: 10/3 = 3.333... truncated to 3.
```

**Example 2:**
```
Input: dividend = 7, divisor = -3
Output: -2
Explanation: 7/(-3) = -2.333... truncated to -2.
```

**Example 3:**
```
Input: dividend = 0, divisor = 1
Output: 0
```

## Constraints

- `-2³¹ <= dividend, divisor <= 2³¹ - 1`
- `divisor != 0`

## Hints

<details>
<summary>Hint 1</summary>

Use subtraction and bit shifting. Instead of subtracting divisor one at a time, double the divisor repeatedly using bit shifts until it exceeds the dividend.

</details>

<details>
<summary>Hint 2</summary>

Handle the sign separately. Convert both numbers to positive (or negative) and apply the sign at the end.

</details>

<details>
<summary>Hint 3</summary>

Be careful with the edge case `dividend = -2³¹` and `divisor = -1` which would overflow if negated. Clamp to `2³¹ - 1`.

</details>

</ProblemEditor>
