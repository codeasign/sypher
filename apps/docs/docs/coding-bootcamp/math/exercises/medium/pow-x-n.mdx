# Pow(x, n)

Implement `pow(x, n)`, which calculates `x` raised to the power `n` (i.e., `x^n`).

**Important:** You must implement this in O(log n) time using exponentiation by squaring.

## Examples

**Example 1:**
```
Input: x = 2.0, n = 10
Output: 1024.00000
```

**Example 2:**
```
Input: x = 2.0, n = -2
Output: 0.25000
Explanation: 2⁻² = 1/4 = 0.25
```

**Example 3:**
```
Input: x = 2.0, n = 0
Output: 1.00000
```

## Constraints

- `-100.0 < x < 100.0`
- `-2³¹ <= n <= 2³¹ - 1`
- `n` is a 32-bit signed integer
- Either `x != 0` or `n > 0` (no zero to a negative power)
- Answer within `10⁻⁵` of the true value is accepted

## Hints

<details>
<summary>Hint 1</summary>

Use exponentiation by squaring: x^n = (x^(n/2))² if n is even, or x * (x^((n-1)/2))² if n is odd.

</details>

<details>
<summary>Hint 2</summary>

Handle negative exponents by computing x^(-n) = 1 / x^n. Be careful about the integer overflow when n = -2³¹.

</details>

<details>
<summary>Hint 3</summary>

Use a 64-bit integer (long) to store the absolute value of n to avoid overflow when negating Integer.MIN_VALUE.

</details>
