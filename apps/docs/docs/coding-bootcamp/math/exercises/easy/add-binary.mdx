# Add Binary

**Difficulty:** Easy

Given two binary strings `a` and `b`, return their **sum as a binary string**.

Both input strings consist only of characters `'0'` and `'1'`. The result must also be a binary string with no leading zeros (except when the result is `"0"`).

## Example 1

**Input:** `a = "11"`, `b = "1"`
**Output:** `"100"`
**Explanation:** `3 + 1 = 4`, which is `100` in binary.

## Example 2

**Input:** `a = "1010"`, `b = "1011"`
**Output:** `"10101"`
**Explanation:** `10 + 11 = 21`, which is `10101` in binary.

## Example 3

**Input:** `a = "0"`, `b = "0"`
**Output:** `"0"`
**Explanation:** `0 + 0 = 0`.

## Input Specification

A single line containing two space-separated binary strings `a` and `b`.

## Output Specification

Print the binary sum of `a` and `b` as a single string with no leading zeros (output `"0"` if the result is zero).

## Constraints

- `1 <= a.length, b.length <= 10^4`
- `a` and `b` consist only of `'0'` or `'1'` characters.
- Each string does not contain leading zeros except for the string `"0"` itself.

## Hints

<details>
<summary>Hint 1</summary>

Process the two strings from right to left (least significant bit to most significant bit), keeping track of a carry value. This mirrors how you would add two numbers by hand.

</details>

<details>
<summary>Hint 2</summary>

At each position, the digit sum is `bit_a + bit_b + carry`. The result bit is `sum % 2`, and the carry for the next position is `sum // 2`. Continue until both strings are exhausted and carry is 0.

</details>

<details>
<summary>Hint 3</summary>

Build the result in reverse order and flip it at the end. Alternatively, use Python's built-in `int(x, 2)` to convert binary strings to integers, add them, then convert back with `bin()` — though the manual approach is better practice for interviews.

</details>
