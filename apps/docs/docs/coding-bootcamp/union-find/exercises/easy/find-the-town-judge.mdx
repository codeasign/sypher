# Find the Town Judge

**Difficulty:** Easy

In a town of `n` people (labeled 1 to `n`), there is a rumor that one person
is the town judge. The town judge trusts nobody. Everybody else (except the
judge) trusts the judge. There is exactly one person that satisfies these
properties.

You are given an integer `n` and a 2D integer array `trust` where
`trust[i] = [a_i, b_i]` means person `a_i` trusts person `b_i`.

Return the label of the town judge if they exist, or `-1` if no judge exists.

## Example 1

**Input:** `n = 2`, `trust = [[1,2]]`
**Output:** `2`
**Explanation:** Person 1 trusts person 2. Person 2 trusts nobody. So person 2
is the judge.

## Example 2

**Input:** `n = 3`, `trust = [[1,3],[2,3]]`
**Output:** `3`
**Explanation:** Both person 1 and person 2 trust person 3. Person 3 trusts
nobody. So person 3 is the judge.

## Example 3

**Input:** `n = 3`, `trust = [[1,3],[2,3],[3,1]]`
**Output:** `-1`
**Explanation:** Person 3 trusts person 1, so person 3 cannot be the judge.
Person 1 trusts person 3, so person 1 cannot be the judge. Person 2 trusts
person 3, so person 2 cannot be the judge. No one qualifies.

## Input Specification

The first line contains an integer `n` (the number of people).
The second line contains an integer `t` (the number of trust relationships).
The next `t` lines each contain two space-separated integers `a` and `b`,
meaning person `a` trusts person `b`. If `t = 0`, there are no trust lines.

## Output Specification

Print a single integer: the label of the town judge, or `-1` if no judge
exists.

## Constraints

- `1 <= n <= 1000`
- `0 <= t <= 10^4`
- `a_i != b_i` (no self-trust)
- All pairs `(a_i, b_i)` are unique

## Hints

<details>
<summary>Hint 1</summary>

Track the in-degree (trust received) and out-degree (trust given) for each
person. The judge has in-degree = n - 1 and out-degree = 0.
</details>

<details>
<summary>Hint 2</summary>

Use two arrays of size n+1 (1-indexed). For each trust pair [a, b]:
increment indegree[b] and decrement outdegree[a] (or just track net
balance).
</details>

<details>
<summary>Hint 3</summary>

After processing all trust pairs, scan from 1 to n. If any person has
out-degree 0 and in-degree n-1, that person is the judge.
</details>

</ProblemEditor>
