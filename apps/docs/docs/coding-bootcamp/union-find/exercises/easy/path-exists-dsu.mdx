# Find if Path Exists (DSU)

**Difficulty:** Easy

Given an undirected graph with `n` nodes labeled from `0` to `n - 1`, and a list of edges, determine whether a valid path exists from node `source` to node `destination`.

Use a **Disjoint Set Union (DSU)** data structure (also called Union-Find) to solve this problem efficiently.

## Example 1

**Input:**
```
n = 3
edges = [[0, 1], [1, 2]]
source = 0
destination = 2
```
**Output:** `true`

**Explanation:** There is a path `0 → 1 → 2` from source `0` to destination `2`.

## Example 2

**Input:**
```
n = 6
edges = [[0, 1], [0, 2], [3, 5], [5, 4], [4, 3]]
source = 0
destination = 5
```
**Output:** `false`

**Explanation:** Nodes `0, 1, 2` are in one component and nodes `3, 4, 5` are in another. No path connects source `0` to destination `5`.

## Example 3

**Input:**
```
n = 1
edges = []
source = 0
destination = 0
```
**Output:** `true`

**Explanation:** The source and destination are the same node, so a path trivially exists.

## Input Specification

- Line 1: a single integer `n` (number of nodes)
- Line 2: a single integer `m` (number of edges)
- Next `m` lines: two space-separated integers `u v` representing an undirected edge
- Next line: integer `source`
- Next line: integer `destination`

## Output Specification

Print `true` if a path from `source` to `destination` exists, otherwise print `false`.

## Constraints

- `1 <= n <= 2 * 10^5`
- `0 <= m <= 2 * 10^5`
- `0 <= edges[i][0], edges[i][1] <= n - 1`
- `edges[i][0] != edges[i][1]`
- `0 <= source, destination <= n - 1`
- There are no duplicate edges.

## Hints

<details>
<summary>Hint 1</summary>

Create a DSU with `n` nodes. Initially, each node is its own parent. For each edge, union the two endpoints together.

</details>

<details>
<summary>Hint 2</summary>

The `find` operation returns the root (representative) of a node's component. Two nodes are connected if and only if they share the same root. Use **path compression** in `find` to keep the tree flat and operations near O(1).

</details>

<details>
<summary>Hint 3</summary>

After processing all edges, simply call `find(source)` and `find(destination)`. If they return the same root, a path exists — return `true`. Otherwise return `false`. No graph traversal (BFS/DFS) is needed.

</details>
