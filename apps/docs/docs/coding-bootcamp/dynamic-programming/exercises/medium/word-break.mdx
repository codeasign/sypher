# Word Break

**Difficulty:** Medium

Given a string `s` and a dictionary of strings `wordDict`, return `true` if `s` can be segmented into a space-separated sequence of one or more dictionary words.

Note that the same word in the dictionary may be reused multiple times in the segmentation.

## Example 1

**Input:** `s = "leetcode"`, `wordDict = ["leet", "code"]`  
**Output:** `true`  
**Explanation:** Return true because "leetcode" can be segmented as "leet code".

## Example 2

**Input:** `s = "applepenapple"`, `wordDict = ["apple", "pen"]`  
**Output:** `true`  
**Explanation:** Return true because "applepenapple" can be segmented as "apple pen apple". Note that you are allowed to reuse a dictionary word.

## Example 3

**Input:** `s = "catsandog"`, `wordDict = ["cats", "dog", "sand", "and", "cat"]`  
**Output:** `false`  

## Input Specification

- First line: the string `s` (without quotes).
- Second line: space-separated words in the dictionary (may be empty).
- `1 <= s.length <= 300`
- `1 <= wordDict.length <= 1000`

## Output Specification

Print `true` if `s` can be segmented, `false` otherwise.

## Constraints

- `1 <= s.length <= 300`
- `1 <= wordDict.length <= 1000`
- `1 <= wordDict[i].length <= 20`
- `s` and `wordDict[i]` consist of only lowercase English letters.

## Hints

<details>
<summary>Hint 1</summary>

Define `dp[i]` as whether the prefix `s[0:i]` can be segmented. How can you compute `dp[i]` from smaller subproblems?
</details>

<details>
<summary>Hint 2</summary>

For each `j < i`, if `dp[j]` is true and `s[j:i]` is in the dictionary, then `dp[i]` is true. This is a classic DP with string segmentation.
</details>

<details>
<summary>Hint 3</summary>

Store the dictionary in a hash set for O(1) lookups. Bound the inner loop by the maximum word length in the dictionary for efficiency.
</details>
