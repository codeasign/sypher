# House Robber II

**Difficulty:** Medium

You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed. **All houses at this place are arranged in a circle.** The first house is adjacent to the last house.

Given an integer array `nums` representing the amount of money at each house, return the maximum amount you can rob tonight without alerting the police.

## Example 1

**Input:** `nums = [2, 3, 2]`  
**Output:** `3`  

## Example 2

**Input:** `nums = [1, 2, 3, 1]`  
**Output:** `4`  

## Example 3

**Input:** `nums = [1, 2, 3]`  
**Output:** `3`  

## Input Specification

- A single line of space-separated integers.
- `1 <= nums.length <= 100`

## Output Specification

Print a single integer.

## Constraints

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 1000`

## Hints

<details>
<summary>Hint 1</summary>

Since houses are arranged in a circle, you cannot rob both the first and last house. Break the circle by considering two cases.
</details>

<details>
<summary>Hint 2</summary>

Run House Robber I on two subarrays: nums[0:n-1] (exclude last) and nums[1:n] (exclude first). Take the max.
</details>

<details>
<summary>Hint 3</summary>

The House Robber I recurrence: dp[i] = max(dp[i-1], dp[i-2] + nums[i]).
</details>
