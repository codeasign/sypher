# Decode Ways — Solution

## Approach: Dynamic Programming

**Algorithm:** dp[i] = ways to decode prefix of length i. dp[0] = 1. For i from 1 to n: if s[i-1] != '0', add dp[i-1]. If two-digit 10-26, add dp[i-2].

**Complexity:** O(n) time, O(1) space.

## Code

<Tabs>
  <TabItem value="python" label="Python">

```python
def num_decodings(s: str) -> int:
    if not s or s[0] == '0': return 0
    prev, curr = 1, 1  # dp[i-2], dp[i-1]
    for i in range(1, len(s)):
        tmp = 0
        if s[i] != '0': tmp += curr
        if 10 <= int(s[i-1:i+1]) <= 26: tmp += prev
        prev, curr = curr, tmp
    return curr
```
