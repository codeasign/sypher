# Asteroid Collision

We are given an array `asteroids` of integers representing asteroids in a row. For each asteroid, the absolute value represents its size, and the sign represents its direction (positive = right, negative = left). Each asteroid moves at the same speed.

Find out the state of the asteroids after all collisions. If two asteroids meet, the larger one destroys the smaller one. If they are the same size, both are destroyed. Two asteroids moving in the same direction never meet.

## Examples

**Example 1:**
```
Input: asteroids = [5, 10, -5]
Output: [5, 10]
Explanation: 10 and -5 collide, 10 > 5 so -5 is destroyed. 5 and 10 never collide (same direction).
```

**Example 2:**
```
Input: asteroids = [8, -8]
Output: []
Explanation: Both asteroids are the same size and destroy each other.
```

**Example 3:**
```
Input: asteroids = [10, 2, -5]
Output: [10]
Explanation: 2 and -5 collide, -5 > 2 so 2 is destroyed. Then 10 and -5 collide, 10 > 5 so -5 is destroyed.
```

## Constraints

- `2 <= asteroids.length <= 10^4`
- `-1000 <= asteroids[i] <= 1000`
- `asteroids[i] != 0`

## Hints

<details>
<summary>Hint 1</summary>

Use a stack. Compare each asteroid with the top of the stack. They only collide if the top is moving right (+) and the new asteroid is moving left (-).
</details>

<details>
<summary>Hint 2</summary>

When a moving-right asteroid is smaller than the moving-left asteroid, pop it and continue checking. This is a simulation of cascading collisions.
</details>
