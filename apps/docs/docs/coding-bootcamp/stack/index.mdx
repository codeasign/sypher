# Stack

The Stack pattern uses LIFO (Last-In-First-Out) order to solve problems involving nested structures, monotonic sequences, and expression evaluation.

⏱ 20 minutes

## What You Will Learn

- Basic stack operations and applications
- Monotonic stack for next greater/smaller element patterns
- Expression evaluation with stacks

## The Concept

A stack stores elements in LIFO order. Push adds to the top, pop removes from the top. The monotonic stack variant maintains elements in sorted order (increasing or decreasing).

## Common Applications

| Problem | Approach |
|---------|----------|
| Valid Parentheses | Push opening, match closing |
| Next Greater Element | Monotonic decreasing stack |
| Min Stack | Maintain min alongside each element |
| Expression Evaluation | Two stacks (operands + operators) |
| Histogram | Monotonic stack for boundaries |

## Complexity

| Operation | Time | Space |
|-----------|------|-------|
| Push/Pop | O(1) | O(1) |
| Monotonic walk | O(n) | O(n) |
