# Remove Duplicate Letters

**Difficulty:** Medium

Given a string `s`, remove duplicate letters so that every letter appears exactly once. You must make your result the **smallest lexicographical** subsequence possible while preserving the original order of the letters.

In other words, find the lexicographically smallest string that contains every unique character from `s` exactly once, in the same relative order as in `s`.

## Example 1

**Input:** `s = "bcabc"`  
**Output:** `"abc"`  
**Explanation:** Both `"abc"` and `"bca"` contain each letter once, but `"abc"` is lexicographically smaller.

## Example 2

**Input:** `s = "cbacdcbc"`  
**Output:** `"acdb"`  
**Explanation:** The unique letters are a, b, c, d. The lexicographically smallest valid subsequence is "acdb".

## Example 3

**Input:** `s = "abcd"`  
**Output:** `"abcd"`  

## Input Specification

A single line containing the string `s`. The string contains only lowercase English letters.

## Output Specification

Print the lexicographically smallest subsequence containing each unique character exactly once.

## Constraints

- `1 <= s.length <= 10^4`
- `s` consists of lowercase English letters

## Hints

<details>
<summary>Hint 1</summary>

Count the frequency of each character first. Use a monotonic stack to build the result greedily.
</details>

<details>
<summary>Hint 2</summary>

When deciding whether to keep or remove a character from the stack, check if the character can appear later in the string (frequency > 0). If it can, and the current character is smaller, pop the larger character.
</details>

<details>
<summary>Hint 3</summary>

Use a boolean array or set to track which characters are already in the result. Only add a character if it is not already present.
</details>
