# Daily Temperatures

**Difficulty:** Medium

Given an array of integers `temperatures` that represents the daily temperatures, return an array `answer` such that `answer[i]` is the number of days you have to wait after the `i`th day to get a warmer temperature. If there is no future day for which this is possible, keep `answer[i] == 0`.

## Example 1

**Input:** `temperatures = [73, 74, 75, 71, 69, 72, 76, 73]`  
**Output:** `[1, 1, 4, 2, 1, 1, 0, 0]`  
**Explanation:**  
- Day 1 (73): Next warmer is day 2 (74), wait 1 day
- Day 2 (74): Next warmer is day 3 (75), wait 1 day
- Day 3 (75): Next warmer is day 7 (76), wait 4 days
- Day 4 (71): Next warmer is day 6 (72), wait 2 days
- Day 5 (69): Next warmer is day 6 (72), wait 1 day
- Day 6 (72): Next warmer is day 7 (76), wait 1 day
- Day 7 (76): No warmer day, answer is 0
- Day 8 (73): No warmer day, answer is 0

## Example 2

**Input:** `temperatures = [30, 40, 50, 60]`  
**Output:** `[1, 1, 1, 0]`  

## Example 3

**Input:** `temperatures = [30, 60, 90]`  
**Output:** `[1, 1, 0]`  

## Input Specification

The first line contains an integer `n` (the length of the array). The second line contains `n` space-separated integers representing the daily temperatures.

## Output Specification

Print `n` space-separated integers where the `i`th integer is the number of days until a warmer temperature (or 0 if none exists).

## Constraints

- `1 <= temperatures.length <= 10^5`
- `30 <= temperatures[i] <= 100`

## Hints

<details>
<summary>Hint 1</summary>

Try processing the array from right to left, keeping track of warmer temperatures you have seen.
</details>

<details>
<summary>Hint 2</summary>

A monotonic decreasing stack is the classic solution. Store indices, not temperatures, so you can compute the day difference.
</details>

<details>
<summary>Hint 3</summary>

When you find a temperature warmer than the top of the stack, repeatedly pop indices until the stack is empty or the top is warmer.
</details>
