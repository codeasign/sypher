# Next Greater Element

**Difficulty:** Easy

Given an array of integers `nums`, return an array `result` where `result[i]` is the next greater element for `nums[i]`. The next greater element is the first larger element to its right in the array. If no greater element exists, use `-1`.

## Example 1

**Input:** `nums = [4, 5, 2, 25]`  
**Output:** `[5, 25, 25, -1]`  
**Explanation:**  
- 4 → next greater is 5
- 5 → next greater is 25
- 2 → next greater is 25
- 25 → no greater element to the right, so -1

## Example 2

**Input:** `nums = [13, 7, 6, 12]`  
**Output:** `[-1, 12, 12, -1]`  

## Example 3

**Input:** `nums = [1, 2, 3, 4]`  
**Output:** `[2, 3, 4, -1]`  

## Input Specification

The first line contains an integer `n` (the length of the array). The second line contains `n` space-separated integers representing the array elements.

## Output Specification

Print `n` space-separated integers where the `i`th integer is the next greater element of `nums[i]` (or `-1` if none exists).

## Constraints

- `1 <= nums.length <= 10^5`
- `0 <= nums[i] <= 10^9`

## Hints

<details>
<summary>Hint 1</summary>

Try iterating from right to left, keeping track of elements in a stack to quickly find the next greater element.
</details>

<details>
<summary>Hint 2</summary>

A monotonic decreasing stack is the classic solution. When you encounter a new element, pop smaller elements from the stack — the new element is their next greater element.
</details>

<details>
<summary>Hint 3</summary>

If you iterate from left to right, push indices onto the stack. When a larger value appears, pop indices whose next greater element has been found.
</details>
