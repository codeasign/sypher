# Backspace String Compare

**Difficulty:** Easy

Given two strings `s` and `t`, return `true` if they are equal when both are typed into empty text editors. In this problem, `#` means a **backspace** character — it deletes the previous character. If there is no previous character, the backspace has no effect.

## Example 1

**Input:** `s = "ab#c"`, `t = "ad#c"`  
**Output:** `true`  
**Explanation:** Both strings become `"ac"` after processing backspaces.

## Example 2

**Input:** `s = "ab##"`, `t = "c#d#"`  
**Output:** `true`  
**Explanation:** Both strings become `""` after processing backspaces.

## Example 3

**Input:** `s = "a#c"`, `t = "b"`  
**Output:** `false`  
**Explanation:** `s` becomes `"c"` but `t` becomes `"b"`.

## Input Specification

The first line contains the string `s`.  
The second line contains the string `t`. An empty second line means `t` is an empty string.

## Output Specification

Print `true` if the two processed strings are equal, otherwise print `false`.

## Constraints

- `1 <= s.length, t.length <= 200`
- `s` and `t` contain only lowercase English letters and `#` characters.
- `0 <= number of '#' in s, t <= 200`

## Hints

<details>
<summary>Hint 1</summary>

Use a **stack** to simulate the typing process for each string. When you encounter a regular character, push it onto the stack. When you encounter `#`, pop from the stack if it is non-empty.

</details>

<details>
<summary>Hint 2</summary>

After processing both strings with their respective stacks, compare the two resulting stacks (or the strings built from them). If they are equal, return `true`.

</details>

<details>
<summary>Hint 3</summary>

For an O(1) space solution, use two pointers starting from the **end** of each string. Traverse backwards, keeping a count of pending backspaces. Skip a character if backspace count is positive; otherwise compare the characters at each pointer.

</details>
