# Decode String — Solution

## Problem Overview

Decode a string encoded as `k[encoded_string]`, where the encoded string inside brackets is repeated `k` times. Brackets can be nested.

## Approach: Stack-Based Iterative

**Idea:** Use two stacks — one for counts and one for the current decoded string. As we scan, we push state when entering a bracket and pop when exiting, building the result incrementally.

**Algorithm:**
1. Initialize an empty count stack and a string stack. Also maintain a `currentString` and `currentCount`.
2. For each character `c` in `s`:
   - If `c` is a digit: accumulate into `currentCount` (handles multi-digit numbers)
   - If `c` is `[`: push `currentCount` and `currentString` onto their stacks, then reset both
   - If `c` is `]`: pop `count` and `prevString` from stacks. Repeat `currentString` `count` times and append to `prevString`. Set `currentString` to the result.
   - Otherwise (letter): append `c` to `currentString`
3. Return `currentString`

**Complexity Analysis:**
- **Time:** O(n) — each character is processed once, and each appended character is part of the final output
- **Space:** O(n) — stacks and output require linear space

**Why this works:** The stack preserves the outer context when entering a nested bracket. Each `]` restores the previous context and applies the multiplication, naturally handling arbitrary nesting depth.

```java
class Solution {
    public String decodeString(String s) {
        Deque<Integer> countStack = new ArrayDeque<>();  // Stack for repeat counts
        Deque<String> strStack = new ArrayDeque<>();     // Stack for previous strings
        StringBuilder current = new StringBuilder();      // Building current segment
        int k = 0;                                        // Current repeat count

        for (char c : s.toCharArray()) {
            if (Character.isDigit(c)) {
                k = k * 10 + (c - '0');  // Build multi-digit number
            } else if (c == '[') {
                countStack.push(k);       // Save current count
                strStack.push(current.toString());  // Save current string
                k = 0;                              // Reset for inner content
                current = new StringBuilder();       // Start fresh inside brackets
            } else if (c == ']') {
                int repeat = countStack.pop();       // Get repeat count
                StringBuilder temp = new StringBuilder(strStack.pop());  // Previous string
                temp.append(current.toString().repeat(repeat));  // Append repeated content
                current = temp;                     // Restore as current segment
            } else {
                current.append(c);  // Regular character — append
            }
        }
        return current.toString();
    }
}
```

```rust
    let mut str_stack: Vec<String> = Vec::new(); // Stack for previous strings

            current.push(c);  // Regular character — append
        }
    }
```

```c
#include <stdlib.h>
#include <string.h>
#include <ctype.h>

char* decodeString(char* s) {
    // Use dynamic arrays as stacks
    int* countStack = malloc(100 * sizeof(int));
    char** strStack = malloc(100 * sizeof(char*));
    int sp = 0;  // Stack pointer
    char* current = malloc(10000);
    current[0] = '\0';
    int k = 0;

    for (int i = 0; s[i]; i++) {
        if (isdigit(s[i])) {
            k = k * 10 + (s[i] - '0');  // Build multi-digit number
        } else if (s[i] == '[') {
            countStack[sp] = k;             // Save current count
            strStack[sp] = strdup(current); // Save current string
            sp++;
            k = 0;
            current[0] = '\0';              // Reset for inner content
        } else if (s[i] == ']') {
            sp--;
            int repeat = countStack[sp];                            // Get repeat count
            char* prev = strStack[sp];                              // Get previous
            char* temp = malloc(strlen(prev) + strlen(current) * repeat + 1);
            strcpy(temp, prev);
            for (int j = 0; j < repeat; j++) strcat(temp, current); // Append repeated
            free(current);
            current = temp;
            free(prev);
        } else {
            char tmp[2] = {s[i], '\0'};
            strcat(current, tmp);  // Regular character — append
        }
    }
    free(countStack);
    free(strStack);
    return current;
}
```

```go
import "strings"

func decodeString(s string) string {
    type frame struct {
        count int
        str   string
    }
    stack := make([]frame, 0)  // Stack of frames (count + previous string)
    current := ""               // Building current segment
    k := 0                      // Current repeat count

    for _, c := range s {
        if c >= '0' && c <= '9' {
            k = k*10 + int(c-'0')  // Build multi-digit number
        } else if c == '[' {
            stack = append(stack, frame{k, current})  // Save state
            k = 0
            current = ""                               // Reset for inner content
        } else if c == ']' {
            f := stack[len(stack)-1]                          // Pop frame
            stack = stack[:len(stack)-1]
            current = f.str + strings.Repeat(current, f.count) // Append repeated
        } else {
            current += string(c)  // Regular character — append
        }
    }
    return current
}
```

## Key Takeaways

- **Two stacks** (count and string) handle nested bracket structures naturally
- Multi-digit numbers require **accumulating digits** until `[` is encountered
- Each `]` pops the previous context and produces the decoded substring for that level
- The same approach works **recursively** by calling the function on bracket contents
- **O(n) time** since every character (input and output) is processed once
