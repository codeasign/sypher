# Is Subsequence

**Difficulty:** Easy

Given two strings `s` and `t`, return `true` if `s` is a **subsequence** of `t`, or `false` otherwise.

A **subsequence** of a string is a new string formed from the original string by deleting some (or no) characters without disturbing the relative order of the remaining characters.

## Example 1

**Input:** `s = "abc"`, `t = "ahbgdc"`  
**Output:** `true`  
**Explanation:** The characters `a`, `b`, `c` all appear in `t` in order (`a` at index 0, `b` at index 2, `c` at index 5).

## Example 2

**Input:** `s = "axc"`, `t = "ahbgdc"`  
**Output:** `false`  
**Explanation:** After matching `a`, we need `x` next, but no `x` appears in the remaining portion of `t`.

## Example 3

**Input:** `s = ""`, `t = "ahbgdc"`  
**Output:** `true`  
**Explanation:** The empty string is a subsequence of every string.

## Input Specification

The first line contains the string `s`.  
The second line contains the string `t`.  
Either or both strings may be empty (empty line).

## Output Specification

Print `true` if `s` is a subsequence of `t`, otherwise print `false`.

## Constraints

- `0 <= s.length <= 100`
- `0 <= t.length <= 10^4`
- Both `s` and `t` consist only of lowercase English letters.

## Hints

<details>
<summary>Hint 1</summary>

Use two pointers — one for `s` and one for `t`. Advance the `s` pointer only when you find a match in `t`; always advance the `t` pointer.

</details>

<details>
<summary>Hint 2</summary>

Walk through `t` character by character. Whenever `t[j] == s[i]`, increment `i`. At the end, check if `i == len(s)`.

</details>

<details>
<summary>Hint 3</summary>

The empty string is always a subsequence of any string. Make sure your solution handles the case where `s` is empty without any special-case logic — the two-pointer loop will never enter its body and `i` will already equal `len(s)` (which is 0).

</details>
