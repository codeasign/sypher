# Valid Palindrome II

**Difficulty:** Easy

Given a string `s`, return `true` if the string can be made a palindrome by deleting **at most one** character from it.

## Example 1

**Input:** `s = "aba"`  
**Output:** `true`  
**Explanation:** The string is already a palindrome.

## Example 2

**Input:** `s = "abca"`  
**Output:** `true`  
**Explanation:** Delete the character 'c' at index 2 — the resulting string "aba" is a palindrome.

## Example 3

**Input:** `s = "abc"`  
**Output:** `false`  
**Explanation:** No single deletion can make "abc" a palindrome.

## Input Specification

A single line containing the string `s`.

## Output Specification

Print `true` if the string can become a palindrome with at most one deletion, otherwise `false`.

## Constraints

- `0 <= s.length <= 10^5`
- `s` consists of lowercase English letters only

## Hints

<details>
<summary>Hint 1</summary>

Use two pointers starting from the left and right ends. Move them inward as long as the characters match.
</details>

<details>
<summary>Hint 2</summary>

When you encounter a mismatch, you have two choices: delete the character at the left pointer or delete the character at the right pointer. Check both possibilities.
</details>

<details>
<summary>Hint 3</summary>

Write a helper function `isPalindrome(s, left, right)` that checks if a substring of `s` is a palindrome using two pointers.
</details>
