# String Compression

**Difficulty:** Medium

Given an array of characters `chars`, compress it in-place using the following algorithm:

Begin with an empty string `s`. For each group of **consecutive repeating characters** in `chars`:
- If the group's length is 1, append the character to `s`.
- Otherwise, append the character followed by the group's length.

The compressed string `s` should be stored **in-place** in the input array `chars`. Return the new length of the compressed array.

**Note:** Group lengths that are 10 or longer must be split into multiple characters (e.g., a count of 12 becomes `"12"` — two characters: '1' and '2').

## Example 1

**Input:** `chars = ["a","a","b","b","c","c","c"]`  
**Output:** Return 6, and the first 6 characters of the input array should be: `["a","2","b","2","c","3"]`  
**Explanation:** The groups are "aa", "bb", and "ccc". This compresses to "a2b2c3".

## Example 2

**Input:** `chars = ["a"]`  
**Output:** Return 1, and the first character of the input array should be: `["a"]`  
**Explanation:** The only group is "a", which remains uncompressed since its length is 1.

## Example 3

**Input:** `chars = ["a","b","b","b","b","b","b","b","b","b","b","b","b"]`  
**Output:** Return 4, and the first 4 characters should be: `["a","b","1","2"]`  
**Explanation:** The groups are "a" and "bbbbbbbbbbbb". "a" stays as-is, "bbbbbbbbbbbb" compresses to "b12".

## Input Specification

A single line containing the string to compress (each character is a character in the array).

## Output Specification

Print the compressed string (the first `n` characters of the modified array).

## Constraints

- `1 <= chars.length <= 2000`
- `chars[i]` is a lowercase English letter

## Hints

<details>
<summary>Hint 1</summary>

Use two pointers: one to read (`i`) and one to write (`write`). Iterate through the array, tracking the start of each group.
</details>

<details>
<summary>Hint 2</summary>

When a group ends (chars[i] != chars[i+1] or at the end), write the character to the write position, then write the count (if > 1) by converting each digit individually.
</details>

<details>
<summary>Hint 3</summary>

For multi-digit counts, use a small temporary buffer or write the digits in reverse and then reverse them in-place.
</details>
