# Longest Palindromic Substring

**Difficulty:** Medium

Given a string `s`, return the **longest palindromic substring** in `s`.

## Example 1

**Input:** `s = "babad"`  
**Output:** `"bab"`  
**Explanation:** "aba" is also a valid answer.

## Example 2

**Input:** `s = "cbbd"`  
**Output:** `"bb"`  

## Input Specification

- A single line containing the string `s`.
- `1 <= s.length <= 1000`

## Output Specification

Print the longest palindromic substring.

## Constraints

- `1 <= s.length <= 1000`
- `s` consists of only lowercase English letters.

## Hints

<details>
<summary>Hint 1</summary>

A palindrome mirrors around its center. For each position, try expanding outward while the characters match.
</details>

<details>
<summary>Hint 2</summary>

There are 2n-1 possible centers: n single-character centers (odd length) and n-1 between-character centers (even length).
</details>

<details>
<summary>Hint 3</summary>

The expand-around-center approach takes O(n^2) time and O(1) space, which is optimal for n  1000.
</details>
