# Implement strStr (KMP)

**Difficulty:** Hard

Implement `strStr()`, which finds the first occurrence of a substring (`needle`) in a string (`haystack`). Return the index of the first occurrence, or `-1` if the `needle` is not part of the `haystack`.

**Clarification:** For this problem, implement the **Knuth-Morris-Pratt (KMP)** algorithm, which runs in O(n+m) time by preprocessing the pattern to avoid redundant comparisons.

## Example 1

**Input:** `haystack = "sadbutsad", needle = "sad"`  
**Output:** `0`  
**Explanation:** "sad" occurs at index 0 and 6. The first occurrence is at index 0.

## Example 2

**Input:** `haystack = "leetcode", needle = "leeto"`  
**Output:** `-1`  
**Explanation:** "leeto" did not occur in "leetcode".

## Input Specification

A single line containing the haystack and needle separated by a comma: `haystack,needle`

## Output Specification

Print the index of the first occurrence, or `-1` if not found.

## Constraints

- `0 <= haystack.length, needle.length <= 10^5`
- `haystack` and `needle` consist of only lowercase English letters

## Hints

<details>
<summary>Hint 1</summary>

Build the LPS (Longest Prefix Suffix) array for the needle. LPS[i] is the length of the longest proper prefix of needle[0..i] that is also a suffix.
</details>

<details>
<summary>Hint 2</summary>

During matching, when a mismatch occurs at position j in the needle, set j = LPS[j-1] instead of resetting to 0 — this avoids re-scanning already-matched characters.
</details>

<details>
<summary>Hint 3</summary>

Empty needle is a special case: return 0 by convention (an empty string is always found at the start).
</details>
