# Continuous Subarray Sum

**Difficulty:** Medium

Given an integer array `nums` and an integer `k`, return `true` if `nums` has a **good subarray** or `false` otherwise.

A **good subarray** is defined as a subarray that:
- Has a length of **at least two**, and
- The sum of its elements is a **multiple of `k`** (i.e., `sum % k == 0`).

**Note:** A subarray is a contiguous part of the array. For `k = 0`, a good subarray is one whose sum is `0`.

## Example 1

**Input:** `nums = [23, 2, 4, 6, 7]`, `k = 6`

**Output:** `true`

**Explanation:** Subarray `[2, 4]` has sum 6, which is a multiple of 6.

## Example 2

**Input:** `nums = [23, 2, 6, 4, 7]`, `k = 6`

**Output:** `true`

**Explanation:** Subarray `[23, 2, 6, 4, 7]` has sum 42, which is a multiple of 6.

## Example 3

**Input:** `nums = [23, 2, 6, 4, 7]`, `k = 13`

**Output:** `false`

**Explanation:** No subarray of length at least 2 has a sum divisible by 13.

## Input Specification

- First line: integer `n` (length of `nums`)
- Second line: `n` space-separated integers `nums[i]`
- Third line: integer `k`

## Output Specification

Print `true` if a good subarray exists, otherwise `false`.

## Constraints

- `1 <= n <= 10^5`
- `0 <= nums[i] <= 10^9`
- `0 <= k <= 2^31 - 1`

## Hints

<details>
<summary>Hint 1</summary>

If `prefix[j] % k == prefix[i] % k`, then the subarray `(i+1..j)` has a sum divisible by `k`. Use a hash map to store the earliest index where each remainder appears.

</details>

<details>
<summary>Hint 2</summary>

Check that the distance between the two indices (`j - i`) is at least 2. Only store the **first** occurrence of each remainder to maximize the distance for later matches.

</details>

<details>
<summary>Hint 3</summary>

Handle `k = 0` as a special case: the modulo operation is undefined, so instead check for equal prefix sums (i.e., remainders without modulo) at a distance of at least 2.

</details>

</ProblemEditor>
