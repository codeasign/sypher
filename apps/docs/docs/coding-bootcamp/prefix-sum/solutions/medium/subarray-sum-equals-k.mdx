# Subarray Sum Equals K — Solution

## Problem Overview

Given an array `nums` and an integer `k`, count the number of contiguous subarrays whose sum equals `k`.

## Brute Force Approach

**Idea:** Check every possible subarray and compute its sum.

**Algorithm:**
1. Initialize `count = 0`
2. For each starting index `i` from `0` to `n-1`:
   - Initialize `sum = 0`
   - For each ending index `j` from `i` to `n-1`:
     - Add `nums[j]` to `sum`
     - If `sum == k`, increment `count`
3. Return `count`

**Complexity Analysis:**
- **Time:** O(n^2) — two nested loops
- **Space:** O(1) — no extra space

**Why this could be better:** O(n^2) is too slow for n = 10^4. We can use prefix sums with a hash map for O(n).

## Optimal Approach: Prefix Sum + Hash Map

**Idea:** The sum of subarray `(i..j)` is `prefix[j] - prefix[i-1]`. If this equals `k`, then `prefix[i-1] == prefix[j] - k`. As we iterate, we store prefix sum frequencies in a hash map and check how many previous prefix sums equal `current_prefix - k`.

**Algorithm:**
1. Initialize a hash map `count_map = {0: 1}` (prefix sum 0 before any element)
2. Initialize `current_sum = 0` and `total_count = 0`
3. For each `num` in `nums`:
   - `current_sum += num`
   - If `current_sum - k` is in `count_map`, add its frequency to `total_count`
   - Increment `count_map[current_sum]`
4. Return `total_count`

**Why This Works:** Each time we see a prefix sum `s`, we check if `s - k` has appeared before. If it has, then every subarray ending at the current position whose prefix sum was `s - k` now has sum `k`. The hash map stores how many times each prefix sum has occurred.

**Complexity Analysis:**
- **Time:** O(n) — single pass through the array
- **Space:** O(n) — hash map storing up to n prefix sums

import java.util.HashMap;
import java.util.Map;

class Solution {
    public int subarraySum(int[] nums, int k) {
        Map<Integer, Integer> prefixCount = new HashMap<>();
        prefixCount.put(0, 1);  // Base case: prefix sum 0 before any element

        int currentSum = 0;
        int totalCount = 0;

        for (int num : nums) {
            currentSum += num;
            // Check if there is a prefix sum that, when subtracted, gives k
            totalCount += prefixCount.getOrDefault(currentSum - k, 0);
            // Record the current prefix sum
            prefixCount.put(currentSum, prefixCount.getOrDefault(currentSum, 0) + 1);
        }
        return totalCount;
    }
}

use std::collections::HashMap;

fn subarray_sum(nums: &Vec<i32>, k: i32) -> i32 {
    let mut prefix_count = HashMap::new();
    prefix_count.insert(0, 1);  // Base case

    let mut current_sum = 0;
    let mut total_count = 0;

    for &num in nums {
        current_sum += num;
        // Check if current_sum - k exists as a previous prefix sum
        total_count += prefix_count.get(&(current_sum - k)).unwrap_or(&0);
        // Record the current prefix sum
        *prefix_count.entry(current_sum).or_insert(0) += 1;
    }
    total_count
}

#define MAX 20000

int subarraySum(int* nums, int numsSize, int k) {
    // Simple approach using direct array for small value range
    // For production, use a proper hash table
    long long prefixCount[MAX * 2] = {0};
    int offset = MAX;
    prefixCount[offset] = 1;  // Base case

    long long currentSum = 0;
    int totalCount = 0;

    for (int i = 0; i < numsSize; i++) {
        currentSum += nums[i];
        int idx = currentSum - k + offset;
        if (idx >= 0 && idx < MAX * 2) {
            totalCount += prefixCount[idx];
        }
        int storeIdx = currentSum + offset;
        if (storeIdx >= 0 && storeIdx < MAX * 2) {
            prefixCount[storeIdx]++;
        }
    }
    return totalCount;
}

## Key Takeaways

- **Prefix sum + hash map** is the classic pattern for counting subarrays with a given sum
- The base case `{0: 1}` handles subarrays that start at index 0
- Each match represents all subarrays ending at the current position with the target sum
