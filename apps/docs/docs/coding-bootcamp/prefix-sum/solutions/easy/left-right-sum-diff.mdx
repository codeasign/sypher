# Left and Right Sum Differences — Solution

## Problem Overview

Given an array of integers `nums`, return a new array where each element at index `i` is the absolute difference between the sum of elements to the left of `i` and the sum of elements to the right of `i`. For index `0`, the left sum is `0`. For the last index, the right sum is `0`.

## Approach: One Pass with Total Sum

**Idea:** Compute the total sum of the array in one pass. Then iterate through the array, maintaining a running `left_sum`. At each index `i`, the right sum is `total_sum - left_sum - nums[i]`. The result is `abs(left_sum - right_sum)`.

**Algorithm:**
1. Compute `total_sum = sum(nums)`
2. Initialize `left_sum = 0` and an empty result array
3. For each index `i`:
   - `right_sum = total_sum - left_sum - nums[i]`
   - Append `abs(left_sum - right_sum)` to result
   - Update `left_sum += nums[i]`
4. Return the result array

**Complexity Analysis:**
- **Time:** O(n) — two passes (one for total sum, one for computing differences)
- **Space:** O(n) — for the result array (not counting the input)

class Solution {
    public int[] leftRightSumDiff(int[] nums) {
        int totalSum = 0;
        for (int num : nums) {
            totalSum += num;
        }

        int[] result = new int[nums.length];

            int rightSum = totalSum - leftSum - nums[i];
            result[i] = Math.abs(leftSum - rightSum);
            leftSum += nums[i];
        }

        return result;
    }
}

#include <stdlib.h>

int* leftRightSumDiff(int* nums, int n) {
    int totalSum = 0;
    for (int i = 0; i < n; i++) {
        totalSum += nums[i];
    }

    int* result = malloc(n * sizeof(int));
    int leftSum = 0;

    for (int i = 0; i < n; i++) {
        int rightSum = totalSum - leftSum - nums[i];
        int diff = leftSum - rightSum;
        result[i] = diff < 0 ? -diff : diff;
        leftSum += nums[i];
    }

    return result;
}

func leftRightSumDiff(nums []int) []int {
    totalSum := 0
    for _, num := range nums {
        totalSum += num
    }

    result := make([]int, len(nums))
    leftSum := 0

    for i, num := range nums {
        rightSum := totalSum - leftSum - num
        diff := leftSum - rightSum
        if diff < 0 {
            diff = -diff
        }
        result[i] = diff
        leftSum += num
    }

    return result
}

## Key Takeaways

- **Prefix/suffix sums** can be computed in a single pass by tracking the running total
- The absolute difference between left and right sums gives a measure of "balance" at each position
- This is a building block for more complex prefix sum problems like pivot index
