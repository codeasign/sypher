# Range Sum Query — Solution

## Problem Overview

Given an integer array `nums`, implement a class that supports initializing with the array and answering range sum queries `sumRange(left, right)` in O(1) time.

## Brute Force Approach

**Idea:** For each query, iterate from `left` to `right` and sum the elements.

**Algorithm:**
1. Store the original array in the constructor
2. For `sumRange(l, r)`, loop from `l` to `r` and accumulate the sum

**Complexity Analysis:**
- **Constructor:** O(1) — just store the reference
- **Query:** O(n) — each query traverses the range
- **Space:** O(1) — only the original array

**Why this could be better:** For `q` queries, this is O(n*q) — too slow for large inputs. The optimal approach precomputes prefix sums for O(1) queries.

## Optimal Approach: Prefix Sum

**Idea:** Precompute a prefix sum array where `prefix[i]` stores the sum of `nums[0..i]`. Then `sumRange(l, r) = prefix[r] - prefix[l-1]`, or simply `prefix[r]` if `l == 0`.

**Algorithm:**
1. In the constructor, build `prefix` where `prefix[i] = prefix[i-1] + nums[i]`
2. `sumRange(l, r)`: return `prefix[r] - (prefix[l-1] if l > 0 else 0)`

**Why This Works:** The prefix sum at any index represents the cumulative sum from the start. Subtracting the prefix sum just before `l` gives the sum of the subarray from `l` to `r`.

**Complexity Analysis:**
- **Constructor:** O(n) — one pass to build prefix array
- **Query:** O(1) — single subtraction
- **Space:** O(n) — for the prefix array

class NumArray {
    private int[] prefix;

    public NumArray(int[] nums) {
        prefix = new int[nums.length];
        prefix[0] = nums[0];
        for (int i = 1; i < nums.length; i++) {
            prefix[i] = prefix[i - 1] + nums[i];
        }
    }

    public int sumRange(int left, int right) {
        if (left == 0) return prefix[right];
        return prefix[right] - prefix[left - 1];
    }
}

impl NumArray {
    fn new(nums: Vec<i32>) -> Self {
        let mut prefix = vec![0; nums.len()];
        prefix[0] = nums[0];
        for i in 1..nums.len() {
            prefix[i] = prefix[i - 1] + nums[i];
        }
        NumArray { prefix }
    }

    fn sum_range(&self, left: i32, right: i32) -> i32 {
        let l = left as usize;
        let r = right as usize;
        if l == 0 { self.prefix[r] }
        else { self.prefix[r] - self.prefix[l - 1] }
    }
}

## Key Takeaways

- **Prefix sums** enable O(1) range sum queries after O(n) preprocessing
- The formula `sum(l, r) = prefix[r] - prefix[l-1]` is the core insight
- This is optimal for static arrays with many queries
