# Non-overlapping Intervals — Solution

## Problem Overview

Find the minimum number of intervals to remove so all remaining intervals are non-overlapping. This is the classic **interval scheduling** greedy problem.

## Approach: Greedy by End Time

**Idea:** Sort intervals by end time. Greedily pick the interval that ends earliest — this leaves the most room for remaining intervals. Count overlaps as removals.

**Algorithm:**
1. Sort intervals by end time (ascending).
2. Initialize `end = intervals[0][1]`, `removals = 0`.
3. For each remaining interval:
   - If `interval[0] < end`, it overlaps — increment removals.
   - Otherwise, update `end = interval[1]`.
4. Return `removals`.

**Complexity Analysis:**
- **Time:** O(n log n) — sorting.
- **Space:** O(1) — constant extra space.

def erase_overlap_intervals(intervals: list[list[int]]) -> int:
    # Sort by end time: greedy choice for maximum remaining space
    intervals.sort(key=lambda x: x[1])

    removals = 0
    end = intervals[0][1]  # End time of the last kept interval

    for i in range(1, len(intervals)):
        if intervals[i][0] < end:
            # This interval overlaps — remove it
            removals += 1
        else:
            # No overlap — keep this interval, update end time
            end = intervals[i][1]

    return removals

## Key Takeaways

- **Sort by end time**: The optimal greedy choice for interval scheduling.
- **Activity selection**: This is the classic "maximum number of non-overlapping activities" problem in reverse.
- **Greedy proof**: The earliest-finishing interval always leaves the most room for remaining intervals.
