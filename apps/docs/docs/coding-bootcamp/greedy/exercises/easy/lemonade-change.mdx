# Lemonade Change

At a lemonade stand, each lemonade costs **$5**. Customers are standing in a queue to buy from you and order one at a time (in the order specified by `bills`). Each customer will only buy one lemonade and pay with either a **$5**, **$10**, or **$20** bill. You must provide the correct change to each customer so that the net transaction is that the customer pays $5.

Note that you do not have any change in hand at first.

Return `true` if you can provide every customer with correct change, otherwise return `false`.

## Examples

**Example 1:**
```
Input: bills = [5,5,5,10,20]
Output: true
Explanation:
From the first 3 customers, we collect three $5 bills in order.
From the fourth customer, we collect a $10 bill and give back a $5.
From the fifth customer, we give a $10 bill and a $5 bill (since we don't have two $5 bills left, we use the $10).
Since all customers got correct change, we output true.
```

**Example 2:**
```
Input: bills = [5,5,10,10,20]
Output: false
Explanation:
From the first two customers, we collect two $5 bills.
From the third customer, we collect a $10 bill and give back a $5.
From the fourth customer, we collect a $10 bill and give back a $5.
From the fifth customer, we need to give back $15 change. We have two $10 bills and no $5 bills left.
We cannot give correct change, so we output false.
```

**Example 3:**
```
Input: bills = [5,5,5,5,5]
Output: true
```

## Constraints

- `1 <= bills.length <= 10^5`
- `bills[i]` is either `5`, `10`, or `20`.

## Hints

<details>
<summary>Hint 1</summary>
Track the count of $5 and $10 bills you have. You never need to track $20 bills since you never give them as change.
</details>

<details>
<summary>Hint 2</summary>
When a customer pays with $20, always try to give one $10 and one $5 first. Only use three $5 bills if you have no $10 bills.
</details>
