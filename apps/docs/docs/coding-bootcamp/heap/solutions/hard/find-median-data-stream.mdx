# Find Median from Data Stream — Solution

## Problem Overview

Design a class `MedianFinder` that supports:
- **`addNum(num)`** — adds an integer to the stream
- **`findMedian()`** — returns the median of all numbers seen so far as a float

The median is the middle value in an ordered list: if the count is odd, it's the middle element; if even, it's the average of the two middle elements.

## Brute Force Approach

**Idea:** Maintain a sorted list. Insert each new number in sorted order.

**Algorithm:**
1. Keep a dynamic array or list of numbers
2. On `addNum(num)`, insert `num` in the correct position using binary search and list insertion
3. On `findMedian()`, if the size is odd, return the middle element; if even, return the average of the two middle elements

**Complexity Analysis:**
- **Time:** O(n) per insertion — binary search is O(log n) but list insertion is O(n) due to shifting
- **Space:** O(n) — storing all elements

**Why this could be better:** Each insertion is O(n), which is too slow for large streams. The optimal approach uses two heaps for O(log n) insertion and O(1) median retrieval.

## Optimal Approach: Two Heaps

**Idea:** Use two heaps to split the numbers into two halves:
- A **max-heap** (using a min-heap with negated values) for the **smaller half** of numbers
- A **min-heap** for the **larger half** of numbers

The max-heap's top gives us the largest number in the smaller half; the min-heap's top gives us the smallest number in the larger half. By keeping the heaps balanced (size difference ≤ 1), the median is always at the top of one or both heaps.

**Algorithm:**
1. **`addNum(num)`**:
   - If the max-heap is empty or `num` ≤ top of max-heap, push to the max-heap (smaller half)
   - Otherwise, push to the min-heap (larger half)
   - Rebalance: if one heap has more than 1 extra element, move the top to the other heap
2. **`findMedian()`**:
   - If the total count is odd, return the top of the heap with more elements
   - If even, return the average of both heap tops

**Why This Works:** The max-heap always contains the smaller numbers, and the min-heap always contains the larger numbers. With balanced sizes, the two middle elements are always at the tops of the two heaps.

**Complexity Analysis:**
- **Time:** O(log n) per `addNum()` (two heap operations), O(1) for `findMedian()`
- **Space:** O(n) — storing all elements across two heaps

## Deep Dive: Tracing the Two-Heap Approach

### Visual Walkthrough

Let's trace adding `[5, 15, 1, 3]` one at a time.

**Initial State:**

```
maxHeap (smaller half): []
minHeap (larger half):  []
```

**addNum(5):**

```
Compare: maxHeap is empty, so push to maxHeap

maxHeap: [5]   (top = 5)
minHeap: []

Sizes: max=1, min=0. Difference = 1. Balanced.

findMedian: maxHeap has more → return 5.0
```

**addNum(15):**

```
Compare: 15 > maxHeap.top (5), so push to minHeap

maxHeap: [5]        (top = 5)
minHeap: [15]       (top = 15)

Sizes: max=1, min=1. Balanced.

findMedian: equal sizes → return (5 + 15) / 2 = 10.0
```

**addNum(1):**

```
Compare: 1 ≤ maxHeap.top (5), so push to maxHeap

maxHeap: [1, 5]     (top = 5)  — actually max-heap, so 5 is top
minHeap: [15]       (top = 15)

Sizes: max=2, min=1. Difference = 1. Balanced.

findMedian: max has more → return 5.0
```

Wait — in a max-heap implemented with a min-heap and negated values, the top is the *largest* in the smaller half:

```
maxHeap (stored as negatives for max behavior): [-5, -1]  → top = -(-5) = 5
minHeap (normal min-heap):                       [15]

5 is the largest value in the smaller half. Since max has 2 elements vs min's 1,
max has the extra element, and the median is its top: 5.0.
```

**addNum(3):**

```
Compare: 3 ≤ maxHeap.top (5), so push to maxHeap

maxHeap: [-5, -1, -3]  → top = -(-5) = 5
minHeap: [15]

Sizes: max=3, min=1. Difference = 2. Rebalance needed!

Move top of maxHeap (5) to minHeap:
maxHeap: [-3, -1]  → top = 3
minHeap: [5, 15]   → top = 5

Sizes: max=2, min=2. Balanced.

findMedian: equal sizes → return (3 + 5) / 2 = 4.0
```

### Rebalancing Logic

After every insertion, check if one heap has grown too large:

```
if maxHeap.size() > minHeap.size() + 1:
    move maxHeap.top() to minHeap

if minHeap.size() > maxHeap.size() + 1:
    move minHeap.top() to maxHeap
```

This ensures the median is always accessible in O(1).

### Edge Cases

- **Single element:** The max-heap has 1 element, min-heap is empty. The median is just that element.
- **All equal values:** They all go to the max-heap. The heaps still balance correctly.
- **Large stream of 10^5 elements:** Each insertion is O(log n), easily handling the constraint.
- **Negative numbers:** Handled naturally — the heap comparisons work for all integers.

```java
import java.util.PriorityQueue;
import java.util.Collections;

class MedianFinder {
    private PriorityQueue<Integer> maxHeap;  // Smaller half (max-heap)
    private PriorityQueue<Integer> minHeap;  // Larger half (min-heap)

    public MedianFinder() {
        maxHeap = new PriorityQueue<>(Collections.reverseOrder());  // Max-heap
        minHeap = new PriorityQueue<>();                             // Min-heap
    }

    public void addNum(int num) {
        // Step 1: Decide which heap to push to
        if (maxHeap.isEmpty() || num <= maxHeap.peek()) {
            maxHeap.offer(num);
        } else {
            minHeap.offer(num);
        }

        // Step 2: Rebalance — keep size difference ≤ 1
        if (maxHeap.size() > minHeap.size() + 1) {
            minHeap.offer(maxHeap.poll());
        } else if (minHeap.size() > maxHeap.size() + 1) {
            maxHeap.offer(minHeap.poll());
        }
    }

    public double findMedian() {
        if (maxHeap.size() > minHeap.size()) {
            return maxHeap.peek();                        // Odd count, max-heap has extra
        } else if (minHeap.size() > maxHeap.size()) {
            return minHeap.peek();                        // Odd count, min-heap has extra
        } else {
            return (maxHeap.peek() + minHeap.peek()) / 2.0; // Even count, average
        }
    }
}
```

```typescript
class MedianFinder {
    private maxHeap: number[];  // Smaller half (max-heap, stored as negatives)
    private minHeap: number[];  // Larger half (min-heap)

    constructor() {
        this.maxHeap = [];
        this.minHeap = [];
    }

    addNum(num: number): void {
        // Push to appropriate heap
        if (this.maxHeap.length === 0 || num <= -this.maxHeap[0]) {
            this._pushMax(-num);
        } else {
            this._pushMin(num);
        }

        // Rebalance
        if (this.maxHeap.length > this.minHeap.length + 1) {
            this._pushMin(-this._popMax());
        } else if (this.minHeap.length > this.maxHeap.length + 1) {
            this._pushMax(-this._popMin());
        }
    }

    findMedian(): number {
        if (this.maxHeap.length > this.minHeap.length) {
            return -this.maxHeap[0];
        } else if (this.minHeap.length > this.maxHeap.length) {
            return this.minHeap[0];
        } else {
            return (-this.maxHeap[0] + this.minHeap[0]) / 2;
        }
    }

    private _pushMax(val: number): void {
        this.maxHeap.push(val);
        let i = this.maxHeap.length - 1;
        while (i > 0) {
            const p = Math.floor((i - 1) / 2);
            if (this.maxHeap[p] <= this.maxHeap[i]) break;
            [this.maxHeap[p], this.maxHeap[i]] = [this.maxHeap[i], this.maxHeap[p]];
            i = p;
        }
    }

    private _popMax(): number {
        const top = this.maxHeap[0];
        const last = this.maxHeap.pop()!;
        if (this.maxHeap.length > 0) {
            this.maxHeap[0] = last;
            let i = 0, n = this.maxHeap.length;
            while (true) {
                let smallest = i;
                const l = 2 * i + 1, r = 2 * i + 2;
                if (l < n && this.maxHeap[l] < this.maxHeap[smallest]) smallest = l;
                if (r < n && this.maxHeap[r] < this.maxHeap[smallest]) smallest = r;
                if (smallest === i) break;
                [this.maxHeap[i], this.maxHeap[smallest]] = [this.maxHeap[smallest], this.maxHeap[i]];
                i = smallest;
            }
        }
        return top;
    }

    private _pushMin(val: number): void {
        this.minHeap.push(val);
        let i = this.minHeap.length - 1;
        while (i > 0) {
            const p = Math.floor((i - 1) / 2);
            if (this.minHeap[p] <= this.minHeap[i]) break;
            [this.minHeap[p], this.minHeap[i]] = [this.minHeap[i], this.minHeap[p]];
            i = p;
        }
    }

    private _popMin(): number {
        const top = this.minHeap[0];
        const last = this.minHeap.pop()!;
        if (this.minHeap.length > 0) {
            this.minHeap[0] = last;
            let i = 0, n = this.minHeap.length;
            while (true) {
                let smallest = i;
                const l = 2 * i + 1, r = 2 * i + 2;
                if (l < n && this.minHeap[l] < this.minHeap[smallest]) smallest = l;
                if (r < n && this.minHeap[r] < this.minHeap[smallest]) smallest = r;
                if (smallest === i) break;
                [this.minHeap[i], this.minHeap[smallest]] = [this.minHeap[smallest], this.minHeap[i]];
                i = smallest;
            }
        }
        return top;
    }
}
```

```python
import heapq

class MedianFinder:
    def __init__(self):
        self.max_heap = []   # Smaller half (max-heap using negatives)
        self.min_heap = []   # Larger half (min-heap)

    def addNum(self, num: int) -> None:
        # Step 1: Decide which heap to push to
        if not self.max_heap or num <= -self.max_heap[0]:
            heapq.heappush(self.max_heap, -num)  # Negate for max-heap behavior
        else:
            heapq.heappush(self.min_heap, num)

        # Step 2: Rebalance — keep size difference ≤ 1
        if len(self.max_heap) > len(self.min_heap) + 1:
            heapq.heappush(self.min_heap, -heapq.heappop(self.max_heap))
        elif len(self.min_heap) > len(self.max_heap) + 1:
            heapq.heappush(self.max_heap, -heapq.heappop(self.min_heap))

    def findMedian(self) -> float:
        if len(self.max_heap) > len(self.min_heap):
            return -self.max_heap[0]                   # Odd count, max-heap has extra
        elif len(self.min_heap) > len(self.max_heap):
            return self.min_heap[0]                    # Odd count, min-heap has extra
        else:
            return (-self.max_heap[0] + self.min_heap[0]) / 2.0  # Even count, average
```

```rust
use std::collections::BinaryHeap;
use std::cmp::Reverse;

struct MedianFinder {
    max_heap: BinaryHeap<i32>,          // Smaller half (max-heap)
    min_heap: BinaryHeap<Reverse<i32>>, // Larger half (min-heap via Reverse)
}

impl MedianFinder {
    fn new() -> Self {
        MedianFinder {
            max_heap: BinaryHeap::new(),
            min_heap: BinaryHeap::new(),
        }
    }

    fn add_num(&mut self, num: i32) {
        // Step 1: Decide which heap
        if self.max_heap.is_empty() || num <= *self.max_heap.peek().unwrap() {
            self.max_heap.push(num);
        } else {
            self.min_heap.push(Reverse(num));
        }

        // Step 2: Rebalance
        if self.max_heap.len() > self.min_heap.len() + 1 {
            self.min_heap.push(Reverse(self.max_heap.pop().unwrap()));
        } else if self.min_heap.len() > self.max_heap.len() + 1 {
            self.max_heap.push(self.min_heap.pop().unwrap().0);
        }
    }

    fn find_median(&self) -> f64 {
        if self.max_heap.len() > self.min_heap.len() {
            *self.max_heap.peek().unwrap() as f64
        } else if self.min_heap.len() > self.max_heap.len() {
            self.min_heap.peek().unwrap().0 as f64
        } else {
            (*self.max_heap.peek().unwrap() as f64
                + self.min_heap.peek().unwrap().0 as f64) / 2.0
        }
    }
}
```

```c
#include <stdlib.h>

typedef struct {
    int* data;
    int size;
    int capacity;
} Heap;

typedef struct {
    Heap* maxHeap;  // Smaller half (max-heap)
    Heap* minHeap;  // Larger half (min-heap)
} MedianFinder;

void swap(int* a, int* b) { int t = *a; *a = *b; *b = t; }

Heap* heapCreate(int capacity) {
    Heap* h = (Heap*)malloc(sizeof(Heap));
    h->data = (int*)malloc(capacity * sizeof(int));
    h->size = 0;
    h->capacity = capacity;
    return h;
}

void bubbleUpMax(Heap* h, int idx) {
    while (idx > 0) {
        int p = (idx - 1) / 2;
        if (h->data[p] >= h->data[idx]) break;
        swap(&h->data[p], &h->data[idx]);
        idx = p;
    }
}

void bubbleDownMax(Heap* h, int idx) {
    int n = h->size;
    while (1) {
        int largest = idx;
        int l = 2 * idx + 1, r = 2 * idx + 2;
        if (l < n && h->data[l] > h->data[largest]) largest = l;
        if (r < n && h->data[r] > h->data[largest]) largest = r;
        if (largest == idx) break;
        swap(&h->data[idx], &h->data[largest]);
        idx = largest;
    }
}

void bubbleUpMin(Heap* h, int idx) {
    while (idx > 0) {
        int p = (idx - 1) / 2;
        if (h->data[p] <= h->data[idx]) break;
        swap(&h->data[p], &h->data[idx]);
        idx = p;
    }
}

void bubbleDownMin(Heap* h, int idx) {
    int n = h->size;
    while (1) {
        int smallest = idx;
        int l = 2 * idx + 1, r = 2 * idx + 2;
        if (l < n && h->data[l] < h->data[smallest]) smallest = l;
        if (r < n && h->data[r] < h->data[smallest]) smallest = r;
        if (smallest == idx) break;
        swap(&h->data[idx], &h->data[smallest]);
        idx = smallest;
    }
}

void heapPushMax(Heap* h, int val) {
    h->data[h->size++] = val;
    bubbleUpMax(h, h->size - 1);
}

int heapPopMax(Heap* h) {
    int top = h->data[0];
    h->data[0] = h->data[--h->size];
    bubbleDownMax(h, 0);
    return top;
}

void heapPushMin(Heap* h, int val) {
    h->data[h->size++] = val;
    bubbleUpMin(h, h->size - 1);
}

int heapPopMin(Heap* h) {
    int top = h->data[0];
    h->data[0] = h->data[--h->size];
    bubbleDownMin(h, 0);
    return top;
}

MedianFinder* medianFinderCreate() {
    MedianFinder* mf = (MedianFinder*)malloc(sizeof(MedianFinder));
    mf->maxHeap = heapCreate(100005);
    mf->minHeap = heapCreate(100005);
    return mf;
}

void medianFinderAddNum(MedianFinder* obj, int num) {
    if (obj->maxHeap->size == 0 || num <= obj->maxHeap->data[0]) {
        heapPushMax(obj->maxHeap, num);
    } else {
        heapPushMin(obj->minHeap, num);
    }

    if (obj->maxHeap->size > obj->minHeap->size + 1) {
        heapPushMin(obj->minHeap, heapPopMax(obj->maxHeap));
    } else if (obj->minHeap->size > obj->maxHeap->size + 1) {
        heapPushMax(obj->maxHeap, heapPopMin(obj->minHeap));
    }
}

double medianFinderFindMedian(MedianFinder* obj) {
    if (obj->maxHeap->size > obj->minHeap->size) {
        return (double)obj->maxHeap->data[0];
    } else if (obj->minHeap->size > obj->maxHeap->size) {
        return (double)obj->minHeap->data[0];
    } else {
        return (obj->maxHeap->data[0] + obj->minHeap->data[0]) / 2.0;
    }
}

void medianFinderFree(MedianFinder* obj) {
    free(obj->maxHeap->data);
    free(obj->maxHeap);
    free(obj->minHeap->data);
    free(obj->minHeap);
    free(obj);
}
```

```csharp
using System;
using System.Collections.Generic;

class MedianFinder {
    private List<int> maxHeap;  // Smaller half (max-heap)
    private List<int> minHeap;  // Larger half (min-heap)

    public MedianFinder() {
        maxHeap = new List<int>();
        minHeap = new List<int>();
    }

    public void AddNum(int num) {
        if (maxHeap.Count == 0 || num <= maxHeap[0]) {
            PushMax(num);
        } else {
            PushMin(num);
        }

        if (maxHeap.Count > minHeap.Count + 1) {
            PushMin(PopMax());
        } else if (minHeap.Count > maxHeap.Count + 1) {
            PushMax(PopMin());
        }
    }

    public double FindMedian() {
        if (maxHeap.Count > minHeap.Count) return maxHeap[0];
        if (minHeap.Count > maxHeap.Count) return minHeap[0];
        return (maxHeap[0] + minHeap[0]) / 2.0;
    }

    private void PushMax(int val) {
        maxHeap.Add(val);
        int i = maxHeap.Count - 1;
        while (i > 0) {
            int p = (i - 1) / 2;
            if (maxHeap[p] >= maxHeap[i]) break;
            Swap(maxHeap, p, i);
            i = p;
        }
    }

    private int PopMax() {
        int top = maxHeap[0];
        maxHeap[0] = maxHeap[maxHeap.Count - 1];
        maxHeap.RemoveAt(maxHeap.Count - 1);
        int i = 0, n = maxHeap.Count;
        while (true) {
            int largest = i;
            int l = 2 * i + 1, r = 2 * i + 2;
            if (l < n && maxHeap[l] > maxHeap[largest]) largest = l;
            if (r < n && maxHeap[r] > maxHeap[largest]) largest = r;
            if (largest == i) break;
            Swap(maxHeap, i, largest);
            i = largest;
        }
        return top;
    }

    private void PushMin(int val) {
        minHeap.Add(val);
        int i = minHeap.Count - 1;
        while (i > 0) {
            int p = (i - 1) / 2;
            if (minHeap[p] <= minHeap[i]) break;
            Swap(minHeap, p, i);
            i = p;
        }
    }

    private int PopMin() {
        int top = minHeap[0];
        minHeap[0] = minHeap[minHeap.Count - 1];
        minHeap.RemoveAt(minHeap.Count - 1);
        int i = 0, n = minHeap.Count;
        while (true) {
            int smallest = i;
            int l = 2 * i + 1, r = 2 * i + 2;
            if (l < n && minHeap[l] < minHeap[smallest]) smallest = l;
            if (r < n && minHeap[r] < minHeap[smallest]) smallest = r;
            if (smallest == i) break;
            Swap(minHeap, i, smallest);
            i = smallest;
        }
        return top;
    }

    private void Swap(List<int> list, int i, int j) {
        int t = list[i]; list[i] = list[j]; list[j] = t;
    }
}
```

## Key Takeaways

- **Two heaps** (max-heap for the smaller half, min-heap for the larger half) is the classic solution for streaming median problems
- **Balancing the heap sizes** ensures O(1) median retrieval — the median is always at the top of one or both heaps
- The approach achieves **O(log n) insertion** and **O(1) median lookup**
- This pattern generalizes to other sliding-window or streaming percentile problems
- In languages without a built-in max-heap (Python, JavaScript/TypeScript), negate values or use `Reverse` wrappers to achieve max-heap behavior from a min-heap
