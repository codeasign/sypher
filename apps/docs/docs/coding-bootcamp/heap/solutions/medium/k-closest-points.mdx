# K Closest Points to Origin — Solution

## Approach: Max-Heap

**Algorithm:** Push (distance, point) into a max-heap. If heap size > k, pop the farthest.

**Complexity:** O(n log k) time, O(k) space.

## Code

<Tabs>
  <TabItem value="python" label="Python">

```python
import heapq

def k_closest(points: list[list[int]], k: int) -> list[list[int]]:
    heap = []
    for x, y in points:
        dist = x*x + y*y  # no sqrt needed for comparison
        heapq.heappush(heap, (-dist, x, y))  # max-heap via negative
        if len(heap) > k:
            heapq.heappop(heap)
    return [[x, y] for _, x, y in heap]
```
