# Top K Frequent Elements — Solution

## Problem Overview

Given an integer array `nums` and an integer `k`, return the `k` most frequently occurring elements. The answer can be in any order and is guaranteed to be unique.

## Brute Force Approach

**Idea:** Count frequencies, sort by frequency, then take the top k.

**Algorithm:**
1. Build a frequency map (hash map) of each element
2. Convert the map entries to a list
3. Sort the list by frequency in descending order
4. Take the first k elements

**Complexity Analysis:**
- **Time:** O(n log n) — dominated by sorting
- **Space:** O(n) — storing the frequency map and the sorted list

**Why this could be better:** Sorting all elements is unnecessary when we only need the top k. We can avoid the log n factor by using a heap of size k.

## Optimal Approach: Min-Heap of Size k

**Idea:** After building a frequency map, iterate through the entries and maintain a **min-heap of size k** keyed by frequency. The heap keeps the k entries with the highest frequencies — any entry with a frequency higher than the minimum in the heap pushes the minimum out.

**Algorithm:**
1. Count frequencies using a hash map
2. Iterate over the frequency map entries:
   - Push the entry (frequency, element) into a min-heap
   - If the heap exceeds size k, pop the entry with the smallest frequency
3. Extract the elements from the heap into the result array

**Why This Works:** The min-heap always contains the k entries with the highest frequencies seen so far. When a new entry has a frequency higher than the smallest in the heap, the smallest is evicted. At the end, the heap holds exactly the k most frequent elements.

**Complexity Analysis:**
- **Time:** O(n log k) — building the frequency map is O(n), each heap operation is O(log k)
- **Space:** O(n) — the frequency map stores up to n unique entries

## Bucket Sort Alternative (O(n))

**Idea:** Use the fact that frequencies are bounded by n. Create an array of buckets where index `i` holds elements that appear `i` times.

**Algorithm:**
1. Count frequencies with a hash map
2. Create an array of buckets sized n + 1, where bucket[i] is a list of elements with frequency i
3. Iterate from the highest frequency down, collecting elements until we have k

**Complexity:**
- **Time:** O(n) — no comparison-based sorting needed
- **Space:** O(n) — buckets array and frequency map

This is the most efficient approach but trades space for speed and only works when frequencies are bounded by the input size.

## Deep Dive: Tracing the Min-Heap Approach

### Visual Walkthrough

Let's trace `nums = [1, 1, 1, 2, 2, 3]` and `k = 2`.

**Step 1: Build frequency map**

```
nums = [1, 1, 1, 2, 2, 3]

Frequency map:
  1 → 3
  2 → 2
  3 → 1
```

**Step 2: Process entries with min-heap (k = 2)**

```
Initialize: min-heap = []

Process entry (1, 3): push → heap = [(3, 1)]        size=1 ≤ 2, keep
Process entry (2, 2): push → heap = [(2, 2), (3, 1)] size=2 ≤ 2, keep
Process entry (3, 1): push → heap = [(1, 3), (3, 1), (2, 2)]
                       size=3 > 2, pop → remove (1, 3)
                       heap = [(2, 2), (3, 1)]
```

Wait — if we push (frequency, element) into a min-heap keyed by frequency, then after processing all:

```
heap content (min-heap by frequency):
  (2, 2)   → element 2, frequency 2
  (3, 1)   → element 1, frequency 3

Result: [2, 1] (or [1, 2] — order doesn't matter)
```

**Step 3: Extract elements**

The heap contains entries for elements 1 and 2 — the two most frequent elements. We extract the elements and return them.

### How the Heap Filters

```
Entry:  (freq=3, val=1) → heap = [(3, 1)]              ✓ stays (size < k)
Entry:  (freq=2, val=2) → heap = [(2, 2), (3, 1)]      ✓ stays (size = k)
Entry:  (freq=1, val=3) → heap = [(1, 3), (3, 1), (2, 2)]
                          pop → heap = [(2, 2), (3, 1)]  ✗ evicted (freq too low)
```

The element with frequency 1 was evicted because its frequency was lower than the current minimum in the heap (frequency 2).

### Edge Cases

- **All elements the same:** The frequency map has one entry with frequency n. The heap will hold just that one element (or fewer if k > 1 and there are fewer unique elements than k — but the problem guarantees k ≤ n).
- **k = n:** Every unique element is returned. The heap grows to hold all entries.
- **Negative numbers:** Frequencies are still non-negative, and the hash map handles negative keys without issue.

```java
import java.util.*;

class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        // Step 1: Count frequencies
        Map<Integer, Integer> freq = new HashMap<>();
        for (int num : nums) {
            freq.put(num, freq.getOrDefault(num, 0) + 1);
        }

        // Step 2: Min-heap of size k by frequency
        PriorityQueue<Map.Entry<Integer, Integer>> minHeap = new PriorityQueue<>(
            (a, b) -> a.getValue() - b.getValue()
        );

        for (Map.Entry<Integer, Integer> entry : freq.entrySet()) {
            minHeap.offer(entry);
            if (minHeap.size() > k) minHeap.poll();
        }

        // Step 3: Extract elements
        int[] result = new int[k];
        for (int i = 0; i < k; i++) {
            result[i] = minHeap.poll().getKey();
        }
        return result;
    }
}
```

```cpp
#include <vector>
#include <unordered_map>
#include <queue>
using namespace std;

class Solution {
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        // Step 1: Count frequencies
        unordered_map<int, int> freq;
        for (int num : nums) freq[num]++;

        // Step 2: Min-heap of size k by frequency
        using Entry = pair<int, int>;  // (frequency, element)
        priority_queue<Entry, vector<Entry>, greater<Entry>> minHeap;

        for (auto& [num, count] : freq) {
            minHeap.push({count, num});
            if (minHeap.size() > k) minHeap.pop();
        }

        // Step 3: Extract elements
        vector<int> result;
        while (!minHeap.empty()) {
            result.push_back(minHeap.top().second);
            minHeap.pop();
        }
        return result;
    }
};
```

```typescript
function topKFrequent(nums: number[], k: number): number[] {
    // Step 1: Count frequencies
    const freq = new Map<number, number>();
    for (const num of nums) {
        freq.set(num, (freq.get(num) || 0) + 1);
    }

    // Step 2: Min-heap using array, keyed by frequency
    const heap: [number, number][] = [];  // [frequency, element]

    function push(entry: [number, number]): void {
        heap.push(entry);
        let i = heap.length - 1;
        while (i > 0) {
            const p = Math.floor((i - 1) / 2);
            if (heap[p][0] <= heap[i][0]) break;
            [heap[p], heap[i]] = [heap[i], heap[p]];
            i = p;
        }
    }

    function pop(): void {
        heap[0] = heap[heap.length - 1];
        heap.pop();
        let i = 0, n = heap.length;
        while (true) {
            let smallest = i;
            const l = 2 * i + 1, r = 2 * i + 2;
            if (l < n && heap[l][0] < heap[smallest][0]) smallest = l;
            if (r < n && heap[r][0] < heap[smallest][0]) smallest = r;
            if (smallest === i) break;
            [heap[i], heap[smallest]] = [heap[smallest], heap[i]];
            i = smallest;
        }
    }

    for (const [num, count] of freq.entries()) {
        push([count, num]);
        if (heap.length > k) pop();
    }

    return heap.map(e => e[1]);
}
```

```python
import heapq
from collections import Counter

def top_k_frequent(nums: list[int], k: int) -> list[int]:
    # Step 1: Count frequencies using Counter
    freq = Counter(nums)              # {element: frequency}

    # Step 2: Min-heap of size k by frequency
    heap = []                          # Store (frequency, element)
    for num, count in freq.items():
        heapq.heappush(heap, (count, num))
        if len(heap) > k:
            heapq.heappop(heap)        # Evict smallest frequency

    # Step 3: Extract elements from heap
    return [num for _, num in heap]
```

```rust
use std::collections::HashMap;
use std::collections::BinaryHeap;
use std::cmp::Reverse;

fn top_k_frequent(nums: Vec<i32>, k: i32) -> Vec<i32> {
    let k = k as usize;

    // Step 1: Count frequencies
    let mut freq = HashMap::new();
    for num in nums {
        *freq.entry(num).or_insert(0) += 1;
    }

    // Step 2: Min-heap of size k using Reverse for min-heap behavior
    let mut heap = BinaryHeap::with_capacity(k + 1);
    for (&num, &count) in freq.iter() {
        heap.push(Reverse((count, num)));
        if heap.len() > k {
            heap.pop();
        }
    }

    // Step 3: Extract elements
    heap.into_iter().map(|Reverse((_, num))| num).collect()
}
```

```c
#include <stdlib.h>

typedef struct {
    int num;
    int freq;
} Entry;

void swap(Entry* a, Entry* b) { Entry t = *a; *a = *b; *b = t; }

void bubbleUp(Entry* heap, int idx) {
    while (idx > 0) {
        int parent = (idx - 1) / 2;
        if (heap[parent].freq <= heap[idx].freq) break;
        swap(&heap[parent], &heap[idx]);
        idx = parent;
    }
}

void bubbleDown(Entry* heap, int n, int idx) {
    while (1) {
        int smallest = idx;
        int l = 2 * idx + 1, r = 2 * idx + 2;
        if (l < n && heap[l].freq < heap[smallest].freq) smallest = l;
        if (r < n && heap[r].freq < heap[smallest].freq) smallest = r;
        if (smallest == idx) break;
        swap(&heap[idx], &heap[smallest]);
        idx = smallest;
    }
}

int* topKFrequent(int* nums, int numsSize, int k, int* returnSize) {
    // Step 1: Build frequency map
    typedef struct { int key; int val; } MapEntry;
    MapEntry* map = (MapEntry*)calloc(numsSize, sizeof(MapEntry));
    int mapSize = 0;

    for (int i = 0; i < numsSize; i++) {
        int found = 0;
        for (int j = 0; j < mapSize; j++) {
            if (map[j].key == nums[i]) {
                map[j].val++;
                found = 1;
                break;
            }
        }
        if (!found) {
            map[mapSize].key = nums[i];
            map[mapSize].val = 1;
            mapSize++;
        }
    }

    // Step 2: Min-heap of size k
    Entry* heap = (Entry*)malloc((k + 1) * sizeof(Entry));
    int heapSize = 0;

    for (int i = 0; i < mapSize; i++) {
        heap[heapSize].num = map[i].key;
        heap[heapSize].freq = map[i].val;
        heapSize++;
        bubbleUp(heap, heapSize - 1);
        if (heapSize > k) {
            heap[0] = heap[--heapSize];
            bubbleDown(heap, heapSize, 0);
        }
    }

    // Step 3: Extract results
    *returnSize = heapSize;
    int* result = (int*)malloc(heapSize * sizeof(int));
    for (int i = 0; i < heapSize; i++) {
        result[i] = heap[i].num;
    }

    free(heap);
    free(map);
    return result;
}
```

```csharp
using System;
using System.Collections.Generic;

class Solution {
    public int[] TopKFrequent(int[] nums, int k) {
        // Step 1: Count frequencies
        var freq = new Dictionary<int, int>();
        foreach (int num in nums) {
            if (freq.ContainsKey(num)) freq[num]++;
            else freq[num] = 1;
        }

        // Step 2: Min-heap of size k by frequency
        var heap = new List<(int freq, int num)>();

        void BubbleUp(int idx) {
            while (idx > 0) {
                int p = (idx - 1) / 2;
                if (heap[p].freq <= heap[idx].freq) break;
                var t = heap[p]; heap[p] = heap[idx]; heap[idx] = t;
                idx = p;
            }
        }

        void BubbleDown(int idx) {
            int n = heap.Count;
            while (true) {
                int smallest = idx;
                int l = 2 * idx + 1, r = 2 * idx + 2;
                if (l < n && heap[l].freq < heap[smallest].freq) smallest = l;
                if (r < n && heap[r].freq < heap[smallest].freq) smallest = r;
                if (smallest == idx) break;
                var t = heap[idx]; heap[idx] = heap[smallest]; heap[smallest] = t;
                idx = smallest;
            }
        }

        foreach (var kvp in freq) {
            heap.Add((kvp.Value, kvp.Key));
            BubbleUp(heap.Count - 1);
            if (heap.Count > k) {
                heap[0] = heap[heap.Count - 1];
                heap.RemoveAt(heap.Count - 1);
                BubbleDown(0);
            }
        }

        int[] result = new int[heap.Count];
        for (int i = 0; i < heap.Count; i++) result[i] = heap[i].num;
        return result;
    }
}
```

## Key Takeaways

- **Frequency map + min-heap of size k** is the standard pattern for "top k" problems
- The heap approach avoids sorting all n elements, reducing the complexity from **O(n log n) to O(n log k)**
- When k is much smaller than n, the savings are significant
- **Bucket sort** offers an O(n) alternative when frequencies are bounded by n
- This pattern generalizes to many "top k by some metric" problems (top k frequent words, top k closest points, etc.)
