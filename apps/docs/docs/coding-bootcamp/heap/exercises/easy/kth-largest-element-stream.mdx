# Kth Largest Element in a Stream

**Difficulty:** Easy

Design a class to find the **kth largest element** in a stream of numbers. The class should have:
- `KthLargest(k, nums)` — Initializes the object with `k` and the initial stream `nums`.
- `add(val)` — Appends `val` to the stream and returns the kth largest element.

## Example

**Input:** `k = 3, nums = [4, 5, 8, 2]`  
**Calls:** `add(3)`, `add(5)`, `add(10)`, `add(9)`, `add(4)`  
**Output:** `[4, 5, 5, 8, 8]`  

## Input Specification

- First line: integer `k`.
- Second line: space-separated stream values.
- `1 <= k <= 10^4`

## Output Specification

Print the result of each add call.

## Constraints

- `1 <= k <= 10^4`
- `0 <= nums.length <= 10^4`
- `-10^4 <= val <= 10^4`

## Hints

<details>
<summary>Hint 1</summary>

Use a min-heap of size k. The smallest element in the heap is the kth largest.
</details>

<details>
<summary>Hint 2</summary>

When adding, push the value then pop the smallest if heap exceeds size k. Return the heap's min.
</details>
