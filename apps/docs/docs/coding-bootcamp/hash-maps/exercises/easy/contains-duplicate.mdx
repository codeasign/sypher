# Contains Duplicate

Given an integer array `nums`, return `true` if any value appears **at least twice** in the array, and return `false` if every element is distinct.

## Examples

**Example 1:**
```
Input: nums = [1,2,3,1]
Output: true
```

**Example 2:**
```
Input: nums = [1,2,3,4]
Output: false
```

**Example 3:**
```
Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true
```

## Constraints

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## Hints

<details>
<summary>Hint 1</summary>

A brute-force approach checks every pair, which is O(n^2). Can you do better?
</details>

<details>
<summary>Hint 2</summary>

Use a hash set to track numbers you've already seen. If you encounter a number already in the set, return true.
</details>

<details>
<summary>Hint 3</summary>

If you reach the end of the array without finding a duplicate, return false.
</details>
