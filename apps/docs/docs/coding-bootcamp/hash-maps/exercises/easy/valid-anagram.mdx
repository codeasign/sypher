# Valid Anagram

Given two strings `s` and `t`, return `true` if `t` is an **anagram** of `s`, and `false` otherwise.

An **anagram** is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

## Examples

**Example 1:**
```
Input: s = "anagram", t = "nagaram"
Output: true
```

**Example 2:**
```
Input: s = "rat", t = "car"
Output: false
```

**Example 3:**
```
Input: s = "", t = ""
Output: true
```

## Constraints

- `0 <= s.length, t.length <= 5 * 10^4`
- `s` and `t` consist of lowercase English letters.

## Hints

<details>
<summary>Hint 1</summary>

An anagram has the same character counts as the original string. Try counting the frequency of each character.
</details>

<details>
<summary>Hint 2</summary>

Use a hash map to store character frequencies. Increment counts for `s` and decrement for `t`. If all counts are zero at the end, they're anagrams.
</details>

<details>
<summary>Hint 3</summary>

If the two strings have different lengths, they cannot be anagrams — check this first for an early exit.
</details>
