# Valid Palindrome

**Difficulty:** Easy

A phrase is a **palindrome** if, after converting all uppercase letters
into lowercase letters and removing all non-alphanumeric characters, it
reads the same forward and backward. Alphanumeric characters include
letters and numbers.

Given a string `s`, return `true` if it is a palindrome, or `false`
otherwise.

## Example 1

**Input:** `s = "A man, a plan, a canal: Panama"`  
**Output:** `true`  
**Explanation:** After ignoring non-alphanumeric chars and case,
`"amanaplanacanalpanama"` reads the same forward and backward.

## Example 2

**Input:** `s = "race a car"`  
**Output:** `false`  
**Explanation:** After cleanup, `"raceacar"` does not read the same
backward as forward (`"raceacar"` vs `"racacear"`).

## Input Specification

A single line containing the string `s`. The string may contain spaces
and printable ASCII characters.

## Output Specification

Print `true` if the string is a palindrome, otherwise `false`.

## Constraints

- `1 <= s.length <= 2 * 10⁵`
- `s` consists only of printable ASCII characters

## Hints

<details>
<summary>Hint 1</summary>

Use two pointers starting at opposite ends of the string.
</details>

<details>
<summary>Hint 2</summary>

Skip non-alphanumeric characters with `isalnum()` or
`Character.isLetterOrDigit()`.
</details>

<details>
<summary>Hint 3</summary>

Compare characters case-insensitively using `.lower()` or
`tolower()`.
</details>
