# Sort Colors (Dutch National Flag)

**Difficulty:** Medium

Given an array `nums` containing only the integers `0`, `1`, and `2`, sort it **in-place** so that all `0`s come first, followed by all `1`s, and then all `2`s.

You must solve this problem **without using a built-in sort function** and in a **single pass** (O(n) time, O(1) space).

This is the classic **Dutch National Flag** problem.

## Example 1

**Input:** `nums = [2, 0, 2, 1, 1, 0]`  
**Output:** `[0, 0, 1, 1, 2, 2]`

## Example 2

**Input:** `nums = [2, 0, 1]`  
**Output:** `[0, 1, 2]`

## Example 3

**Input:** `nums = [0, 0, 0]`  
**Output:** `[0, 0, 0]`

## Input Specification

A single line containing space-separated integers. Each integer is `0`, `1`, or `2`.

## Output Specification

Print the sorted integers as space-separated values on a single line.

## Constraints

- `1 <= nums.length <= 3 * 10^4`
- `nums[i]` is either `0`, `1`, or `2`

## Hints

<details>
<summary>Hint 1</summary>

Use three pointers: `low`, `mid`, and `high`. Maintain the invariant that elements before `low` are all `0`s, elements after `high` are all `2`s, and elements between `low` and `mid` are all `1`s.

</details>

<details>
<summary>Hint 2</summary>

Walk `mid` from left to right. When `nums[mid] == 0`, swap it with `nums[low]` and advance both `low` and `mid`. When `nums[mid] == 2`, swap it with `nums[high]` and decrement `high` (do not advance `mid` — the swapped value needs to be examined). When `nums[mid] == 1`, just advance `mid`.

</details>

<details>
<summary>Hint 3</summary>

The loop runs while `mid <= high`. Because the `2`-swap does not advance `mid`, every element is processed at most once — guaranteeing a single pass with O(n) time and O(1) extra space.

</details>

</ProblemEditor>
