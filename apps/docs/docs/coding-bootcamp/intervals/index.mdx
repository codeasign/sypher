# Intervals

Interval problems involve ranges [start, end] and operations like merging, inserting, or finding overlaps.

⏱ 15 minutes

## What You Will Learn

- Sorting intervals by start time
- Merging overlapping intervals
- Detecting conflicts and scheduling

## The Concept

Intervals represent ranges with a start and end. Most problems become tractable after sorting by start (or end) time, then scanning linearly.

## Key Operations

| Operation | Approach |
|-----------|----------|
| Merge Overlapping | Sort by start, merge when overlap detected |
| Insert Interval | Find insertion point, merge as needed |
| Count Conflicts | Sweep line: increment on start, decrement on end |
| Meeting Rooms | Check if any overlaps exist |

## Complexity

| Operation | Time | Space |
|-----------|------|-------|
| Merge | O(n log n) | O(n) |
| Insert | O(n) | O(n) |
| Conflict count | O(n log n) | O(n) |
