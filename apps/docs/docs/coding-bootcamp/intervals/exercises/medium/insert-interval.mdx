# Insert Interval

**Difficulty:** Medium

You are given an array of non-overlapping intervals `intervals` where `intervals[i] = [start_i, end_i]` represent the start and end of the i-th interval, and `intervals` is sorted in ascending order by `start_i`. You are also given an interval `newInterval = [start, end]`.

Insert `newInterval` into `intervals` such that `intervals` is still sorted in ascending order and still has no overlapping intervals (merge overlapping intervals if necessary).

Return `intervals` after the insertion.

## Example 1

**Input:** `intervals = [[1,3],[6,9]]`, `newInterval = [2,5]`  
**Output:** `[[1,5],[6,9]]`  

## Example 2

**Input:** `intervals = [[1,2],[3,5],[6,7],[8,10],[12,16]]`, `newInterval = [4,8]`  
**Output:** `[[1,2],[3,10],[12,16]]`  

## Input Specification

- First line: space-separated interval pairs (may be empty).
- Second line: two integers for the new interval.
- `0 <= intervals.length <= 10^4`

## Output Specification

Print the resulting intervals as space-separated pairs.

## Constraints

- `0 <= intervals.length <= 10^4`
- `-10^5 <= start_i, end_i <= 10^5`

## Hints

<details>
<summary>Hint 1</summary>

Intervals before the new interval that don't overlap can be added directly. Find the first interval that overlaps.
</details>

<details>
<summary>Hint 2</summary>

Merge the new interval with all overlapping intervals by extending its start and end.
</details>

<details>
<summary>Hint 3</summary>

Add remaining intervals after the merged interval.
</details>

</ProblemEditor>
