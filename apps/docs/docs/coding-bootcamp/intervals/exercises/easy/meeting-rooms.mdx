# Meeting Rooms

**Difficulty:** Easy

Given an array of meeting time intervals consisting of start and end times `[start, end]`, determine if a person could attend all meetings.

## Example 1

**Input:** `intervals = [[0, 30], [5, 10], [15, 20]]`  
**Output:** `false`  
**Explanation:** The meeting at time 0-30 overlaps with the meeting at 5-10 and 15-20.

## Example 2

**Input:** `intervals = [[7, 10], [2, 4]]`  
**Output:** `true`  
**Explanation:** The meetings do not overlap. The person can attend both.

## Input Specification

The first line contains an integer `n`, the number of intervals.  
Each of the next `n` lines contains two integers `start` and `end` representing a meeting interval.

## Output Specification

Print `true` if the person can attend all meetings, otherwise `false`.

## Constraints

- `0 <= n <= 10⁴`
- `-10⁶ <= start < end <= 10⁶`

## Hints

<details>
<summary>Hint 1</summary>

Sort the intervals by start time. Then check if any interval overlaps with the next one.
</details>

<details>
<summary>Hint 2</summary>

Two intervals `[a, b]` and `[c, d]` overlap if `c < b` (when sorted by start time). If the end of one interval is greater than the start of the next, they conflict.
</details>

<details>
<summary>Hint 3</summary>

After sorting, iterate through the intervals and check if `intervals[i][1] > intervals[i+1][0]`. If true, they overlap and the person cannot attend all meetings.
</details>

</ProblemEditor>
