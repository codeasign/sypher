# Moving Average from Data Stream

**Difficulty:** Easy

Given a stream of integers and a window size, calculate the moving average of all integers in the sliding window.

## Example

**Input:**
```
3
1 10 3 5
```

**Output:**
```
1.00000
5.50000
4.66667
5.00000
```

**Explanation:**
- `next(1)` = 1 / 1 = 1.00000
- `next(10)` = (1 + 10) / 2 = 5.50000
- `next(3)` = (1 + 10 + 3) / 3 = 4.66667
- `next(5)` = (10 + 3 + 5) / 3 = 5.00000

## Input Specification

First line: integer `size` (window size), followed by a space-separated list of integers to process.

## Output Specification

For each integer, output the moving average after processing it, formatted to 5 decimal places.

## Constraints

- `1 <= size <= 1000`
- `-10^5 <= val <= 10^5`
- At most 10^4 operations

## Hints

<details>
<summary>Hint 1</summary>

Use a queue to maintain the sliding window. When the queue exceeds the size, remove the oldest element.
</details>

<details>
<summary>Hint 2</summary>

Maintain a running sum to avoid recalculating the sum from scratch each time.
</details>

<details>
<summary>Hint 3</summary>

The average is the running sum divided by the current queue size (or the window size, whichever is smaller).
</details>
