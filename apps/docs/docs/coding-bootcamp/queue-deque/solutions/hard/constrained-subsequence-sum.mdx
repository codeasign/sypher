# Constrained Subsequence Sum — Solution

## Problem Overview

Given an array `nums` and an integer `k`, find the maximum sum of a non-empty subsequence where consecutive elements in the subsequence have indices at most `k` apart.

## Approach 1: DP with Deque Optimization

**Idea:** This is a DP problem where `dp[i]` represents the maximum sum of a subsequence ending at index `i` (including `nums[i]`). The recurrence is:
- `dp[i] = nums[i] + max(0, max(dp[j] for j in [i-k, i-1]))`

We use a **monotonic deque** to maintain the maximum `dp[j]` in the sliding window `[i-k, i-1]` in O(1) time.

**Algorithm:**
1. Initialize `dp[0] = nums[0]`, a deque, and a variable to track the maximum sum.
2. For each index `i`:
   - Remove indices from the front of the deque that are outside the window `[i-k, i-1]`.
   - Get the best previous dp value from the front of the deque (or 0 if the deque is empty), but only take it if it's positive.
   - Compute `dp[i] = nums[i] + max(0, best)`.
   - Remove indices from the back of the deque while `dp[dq.back()] <= dp[i]` (maintain decreasing order).
   - Add `i` to the back of the deque.
   - Update the global maximum sum.
3. Return the global maximum.

**Why This Works:** The deque maintains dp values in decreasing order, so the front always has the maximum dp value in the window. The constraint `max(0, best)` allows us to start a new subsequence at any point.

**Complexity Analysis:**
- **Time:** O(n) — each element is pushed and popped from the deque at most once.
- **Space:** O(n) — dp array of size n, plus the deque.

### Edge Cases

- **All negative numbers**: The best previous dp is always 0, so each `dp[i] = nums[i]`. The answer is the maximum element.
- **Increasing numbers**: Each element adds to the previous best, creating a growing sum.
- **k=1**: Each element can only be adjacent to the previous one.
- **k=n**: The entire array can be one subsequence.

```java
import java.util.*;

class Solution {
    public int constrainedSubsetSum(int[] nums, int k) {
        int n = nums.length;
        int[] dp = new int[n];  // dp[i] = max sum ending at i
        Deque<Integer> dq = new ArrayDeque<>();  // stores indices in decreasing dp order
        int maxVal = Integer.MIN_VALUE;

        for (int i = 0; i < n; i++) {
            // Remove indices that are outside the window [i-k, i-1]
            while (!dq.isEmpty() && dq.peekFirst() < i - k) {
                dq.pollFirst();
            }

            // Get the best dp value in the window (or 0 if none or negative)
            int best = dq.isEmpty() ? 0 : Math.max(0, dp[dq.peekFirst()]);
            dp[i] = nums[i] + best;

            // Maintain decreasing order of dp values in the deque
            // Remove indices from the back with dp <= dp[i]
            while (!dq.isEmpty() && dp[dq.peekLast()] <= dp[i]) {
                dq.pollLast();
            }

            dq.offerLast(i);
            maxVal = Math.max(maxVal, dp[i]);
        }
        return maxVal;
    }
}
```

```python
from collections import deque

def constrained_subset_sum(nums, k):
    n = len(nums)
    dp = [0] * n  # dp[i] = max sum ending at i
    dq = deque()  # stores indices in decreasing dp order
    max_val = float('-inf')

    for i, nv in enumerate(nums):
        # Remove indices that are outside the window [i-k, i-1]
        while dq and dq[0] < i - k:
            dq.popleft()

        # Get the best dp value in the window (or 0 if none or negative)
        best = max(0, dp[dq[0]]) if dq else 0
        dp[i] = nv + best

        # Maintain decreasing order of dp values in the deque
        # Remove indices from the back with dp <= dp[i]
        while dq and dp[dq[-1]] <= dp[i]:
            dq.pop()

        dq.append(i)
        max_val = max(max_val, dp[i])

    return max_val
```

```rust
use std::collections::VecDeque;

fn constrained_subset_sum(nums: &[i32], k: i32) -> i32 {
    let n = nums.len();
    let k = k as usize;
    let mut dp = vec![0; n];  // dp[i] = max sum ending at i
    let mut dq: VecDeque<usize> = VecDeque::new();  // stores indices in decreasing dp order
    let mut max_val = i32::MIN;

    for i in 0..n {
        // Remove indices that are outside the window [i-k, i-1]
        while let Some(&front) = dq.front() {
            if front + k < i {
                dq.pop_front();
            } else {
                break;
            }
        }

        // Get the best dp value in the window (or 0 if none or negative)
        let best = if dq.is_empty() {
            0
        } else {
            0.max(dp[dq.front().unwrap()])
        };
        dp[i] = nums[i] + best;

        // Maintain decreasing order of dp values in the deque
        while let Some(&back) = dq.back() {
            if dp[back] <= dp[i] {
                dq.pop_back();
            } else {
                break;
            }
        }

        dq.push_back(i);
        max_val = max_val.max(dp[i]);
    }
    max_val
}
```

```c
#include <stdlib.h>
#include <limits.h>

int constrainedSubsetSum(int* nums, int numsSize, int k) {
    int* dp = (int*)malloc(numsSize * sizeof(int));  // dp[i] = max sum ending at i
    int* dq = (int*)malloc(numsSize * sizeof(int));  // stores indices in decreasing dp order
    int front = 0, back = 0;  // deque pointers
    int maxVal = INT_MIN;

    for (int i = 0; i < numsSize; i++) {
        // Remove indices that are outside the window [i-k, i-1]
        while (back > front && dq[front] < i - k) {
            front++;
        }

        // Get the best dp value in the window (or 0 if none or negative)
        int best = (back > front) ? (0 > dp[dq[front]] ? 0 : dp[dq[front]]) : 0;
        dp[i] = nums[i] + best;

        // Maintain decreasing order of dp values in the deque
        while (back > front && dp[dq[back - 1]] <= dp[i]) {
            back--;
        }

        dq[back++] = i;
        if (dp[i] > maxVal) maxVal = dp[i];
    }

    free(dp);
    free(dq);
    return maxVal;
}
```

```csharp
using System.Collections.Generic;

public class Solution {
    public int ConstrainedSubsetSum(int[] nums, int k) {
        int n = nums.Length;
        int[] dp = new int[n];  // dp[i] = max sum ending at i
        LinkedList<int> dq = new LinkedList<int>();  // stores indices in decreasing dp order
        int maxVal = int.MinValue;

        for (int i = 0; i < n; i++) {
            // Remove indices that are outside the window [i-k, i-1]
            while (dq.Count > 0 && dq.First.Value < i - k) {
                dq.RemoveFirst();
            }

            // Get the best dp value in the window (or 0 if none or negative)
            int best = dq.Count > 0 ? System.Math.Max(0, dp[dq.First.Value]) : 0;
            dp[i] = nums[i] + best;

            // Maintain decreasing order of dp values in the deque
            while (dq.Count > 0 && dp[dq.Last.Value] <= dp[i]) {
                dq.RemoveLast();
            }

            dq.AddLast(i);
            maxVal = System.Math.Max(maxVal, dp[i]);
        }
        return maxVal;
    }
}
```

```go
func constrainedSubsetSum(nums []int, k int) int {
    n := len(nums)
    dp := make([]int, n)  // dp[i] = max sum ending at i
    dq := make([]int, 0)  // stores indices in decreasing dp order
    maxVal := nums[0]

    for i, nv := range nums {
        // Remove indices that are outside the window [i-k, i-1]
        for len(dq) > 0 && dq[0] < i-k {
            dq = dq[1:]
        }

        // Get the best dp value in the window (or 0 if none or negative)
        best := 0
        if len(dq) > 0 {
            if dp[dq[0]] > 0 {
                best = dp[dq[0]]
            }
        }
        dp[i] = nv + best

        // Maintain decreasing order of dp values in the deque
        for len(dq) > 0 && dp[dq[len(dq)-1]] <= dp[i] {
            dq = dq[:len(dq)-1]
        }

        dq = append(dq, i)
        if dp[i] > maxVal {
            maxVal = dp[i]
        }
    }
    return maxVal
}
```

## Key Takeaways

- **DP + Monotonic Deque** is a powerful combination for problems with a sliding window constraint on subsequences.
- The **deque maintains the maximum dp value** in the window, enabling O(1) transition.
- The `max(0, best)` pattern allows starting a new subsequence at any point.
- **O(n) time complexity** — each element is processed once.
