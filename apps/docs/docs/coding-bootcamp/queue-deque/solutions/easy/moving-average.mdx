# Moving Average from Data Stream — Solution

## Problem Overview

Given a stream of integers and a window size, calculate the moving average of all integers in the sliding window. After each new value, output the average of the current window (up to `size` most recent values).

## Approach: Queue with Running Sum

**Idea:** Use a queue to maintain the sliding window. Keep a running sum to avoid recalculating from scratch each time.

**Algorithm:**
1. Maintain a queue, a running sum, and the window size.
2. On each new value:
   - Add the value to the queue and the running sum.
   - If the queue exceeds the window size, remove the oldest element from the queue and subtract it from the sum.
   - Return the sum divided by the current queue size.

**Why This Works:** The queue naturally maintains FIFO order, so the oldest element is always at the front. The running sum eliminates the need to iterate through all elements each time.

**Complexity Analysis:**
- **Time:** O(1) per operation — enqueue, dequeue, sum update, and division are all O(1).
- **Space:** O(k) — the queue stores at most k elements, where k is the window size.

## Deep Dive

### Visual Walkthrough

For `size = 3` and values `[1, 10, 3, 5]`:

```
next(1):  queue=[1],       sum=1,   avg=1/1=1.00000
next(10): queue=[1,10],    sum=11,  avg=11/2=5.50000
next(3):  queue=[1,10,3],  sum=14,  avg=14/3=4.66667
next(5):  queue=[10,3,5],  sum=18,  avg=18/3=5.00000
          (1 removed from front since queue exceeded size 3)
```

### Edge Cases

- **First element**: Queue has only one element; average is just that element.
- **Window not yet full**: Before reaching `size` elements, the average is computed over fewer elements.
- **All same values**: The moving average remains constant.
- **Negative values**: Works correctly with negative numbers in the sum.

import java.util.LinkedList;
import java.util.Queue;

class MovingAverage {
    private int size;          // window size
    private Queue<Integer> queue;  // stores elements in the window
    private double sum;        // running sum of all elements in the window

    public MovingAverage(int size) {
        this.size = size;
        this.queue = new LinkedList<>();
        this.sum = 0.0;
    }

    public double next(int val) {
        // Add the new value to the queue and update the sum
        queue.offer(val);
        sum += val;

        // If the queue exceeds the window size, remove the oldest element
        if (queue.size() > size) {
            sum -= queue.poll();
        }

        // Return the average of the current window
        return sum / queue.size();
    }
}

from collections import deque

    def __init__(self, size: int):
        # window size
        self.size = size
        # stores elements in the window

        # running sum of all elements in the window

#include <stdlib.h>
#include <stdbool.h>

typedef struct Node {
    int val;
    struct Node* next;
} Node;

typedef struct {
    Node* front;  // front of the queue (oldest element)
    Node* rear;   // rear of the queue (newest element)
    int size;     // window size
    int count;    // current number of elements
    double sum;   // running sum of all elements in the window
} MovingAverage;

MovingAverage* movingAverageCreate(int size) {
    MovingAverage* obj = (MovingAverage*)malloc(sizeof(MovingAverage));
    obj->front = NULL;
    obj->rear = NULL;
    obj->size = size;
    obj->count = 0;
    obj->sum = 0.0;
    return obj;
}

double movingAverageNext(MovingAverage* obj, int val) {
    // Create a new node for the value
    Node* node = (Node*)malloc(sizeof(Node));
    node->val = val;
    node->next = NULL;

    // Add the new node to the rear of the queue
    if (obj->rear) {
        obj->rear->next = node;
    } else {
        obj->front = node;
    }
    obj->rear = node;

    // Update the running sum and count
    obj->sum += val;
    obj->count++;

    // If the queue exceeds the window size, remove the oldest element
    if (obj->count > obj->size) {
        Node* tmp = obj->front;
        obj->sum -= tmp->val;
        obj->front = tmp->next;
        free(tmp);
        obj->count--;
    }

    // Return the average of the current window
    return obj->sum / obj->count;
}

void movingAverageFree(MovingAverage* obj) {
    Node* cur = obj->front;
    while (cur) {
        Node* tmp = cur;
        cur = cur->next;
        free(tmp);
    }
    free(obj);
}

using System.Collections.Generic;

public class MovingAverage {

    private Queue<int> queue;      // stores elements in the window

    public MovingAverage(int size) {

        this.queue = new Queue<int>();
        this.sum = 0.0;

        // Add the new value to the queue and update the sum
        queue.Enqueue(val);
        sum += val;

        // If the queue exceeds the window size, remove the oldest element
        if (queue.Count > size) {
            sum -= queue.Dequeue();
        }

        // Return the average of the current window
        return sum / queue.Count;
    }
}

    if len(this.queue) > this.size {

        this.queue = this.queue[1:]
    }

    // Return the average of the current window

## Key Takeaways

- **Queue** is the natural data structure for sliding window problems due to FIFO order.
- A **running sum** avoids O(k) recomputation per operation, making each `next()` call O(1).
- The window size limits the queue's memory usage to O(k).
- Works with **negative numbers** and **edge cases** (first element, not-yet-full window).
