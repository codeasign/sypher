# Time Needed to Inform All Employees

**Difficulty:** Easy

A company has `n` employees numbered from `0` to `n - 1`. The company head is identified by `headID`. Every employee has exactly one direct manager, except the head whose manager is `-1`. The `manager` array stores the direct manager of each employee: `manager[i]` is the manager of employee `i`.

Before the head can inform their direct reports, it takes `informTime[headID]` minutes. More generally, it takes `informTime[i]` minutes for employee `i` to notify all their direct reports simultaneously. Leaf employees have `informTime[i] = 0`.

Return the **total number of minutes** it takes to inform all `n` employees of the news.

## Example 1

**Input:** `n = 1`, `headID = 0`, `manager = [-1]`, `informTime = [0]`  
**Output:** `0`  
**Explanation:** There is only one employee — no one needs to be informed.

## Example 2

**Input:** `n = 6`, `headID = 2`, `manager = [2, 2, -1, 3, 3, 3]`, `informTime = [0, 0, 4, 0, 0, 3]`  
**Output:** `4`  
**Explanation:** Employee 2 is the head. They take 4 minutes to notify employees 0 and 1. Employees 0 and 1 have no direct reports. Total time = 4.

## Example 3

**Input:** `n = 4`, `headID = 0`, `manager = [-1, 0, 1, 2]`, `informTime = [1, 2, 3, 0]`  
**Output:** `6`  
**Explanation:** Linear chain 0 → 1 → 2 → 3. Time = 1 + 2 + 3 = 6.

## Input Specification

The first line contains two integers `n` and `headID`.  
The second line contains `n` space-separated integers representing the `manager` array.  
The third line contains `n` space-separated integers representing the `informTime` array.

## Output Specification

Print a single integer — the total minutes needed to inform all employees.

## Constraints

- `1 <= n <= 10^5`
- `0 <= headID < n`
- `manager.length == n`
- `0 <= manager[i] < n`
- `manager[headID] == -1`
- `informTime.length == n`
- `0 <= informTime[i] <= 1000`
- `informTime[i] == 0` if employee `i` has no subordinates

## Hints

<details>
<summary>Hint 1</summary>

Build an adjacency list (tree) from the `manager` array where each node contains its direct reports. The answer is the longest path from `headID` to any leaf, summing `informTime` along the way.

</details>

<details>
<summary>Hint 2</summary>

Use DFS from `headID`. For each employee, the time to inform their entire branch is `informTime[employee] + max(dfs(direct_report))` over all direct reports. A leaf returns 0.

</details>

<details>
<summary>Hint 3</summary>

You can also use BFS. Keep track of the cumulative time each employee receives the message (parent's cumulative time + parent's `informTime`). The answer is the maximum cumulative time across all employees.

</details>

</ProblemEditor>
