# Time Needed to Inform Employees — Solution

## Problem Overview

Given a company hierarchy (tree) and the time each manager takes to inform their direct reports, find the total time needed to inform all employees.

## Approach: DFS on Tree

**Idea:** Build an adjacency list (tree) from the `manager` array where each node points to its direct reports. Then run DFS from `headID` to find the maximum cumulative time along any root-to-leaf path.

**Algorithm:**
1. Build a tree: for each employee `i`, if `manager[i] != -1`, add `i` as a child of `manager[i]`
2. Run DFS from `headID`:
   - If the current employee has no reports, return 0
   - Otherwise, return `informTime[employee] + max(dfs(child))` over all children
3. The DFS result is the total time needed

**Complexity Analysis:**
- **Time:** O(n) — each node visited once
- **Space:** O(n) — tree adjacency list and recursion stack

import java.util.*;

class Solution {
    public int numOfMinutes(int n, int headID, int[] manager, int[] informTime) {
        // Build tree: each manager -> list of direct reports
        List<List<Integer>> tree = new ArrayList<>();
        for (int i = 0; i < n; i++) tree.add(new ArrayList<>());
        for (int i = 0; i < n; i++) {
            if (manager[i] != -1) {
                tree.get(manager[i]).add(i);
            }
        }
        return dfs(headID, tree, informTime);
    }

    private int dfs(int emp, List<List<Integer>> tree, int[] informTime) {
        int maxTime = 0;
        for (int report : tree.get(emp)) {
            maxTime = Math.max(maxTime, dfs(report, tree, informTime));
        }
        return informTime[emp] + maxTime;
    }
}

fn num_of_minutes(n: i32, head_id: i32, manager: Vec<i32>, inform_time: Vec<i32>) -> i32 {
    let n = n as usize;
    let head_id = head_id as usize;
    // Build tree: each manager -> list of direct reports
    let mut tree: Vec<Vec<usize>> = vec![vec![]; n];
    for (i, &mgr) in manager.iter().enumerate() {
        if mgr != -1 {
            tree[mgr as usize].push(i);
        }
    }

    fn dfs(emp: usize, tree: &Vec<Vec<usize>>, inform_time: &Vec<i32>) -> i32 {
        let mut max_time = 0;
        for &report in &tree[emp] {
            max_time = max_time.max(dfs(report, tree, inform_time));
        }
        inform_time[emp] + max_time
    }

    dfs(head_id, &tree, &inform_time)
}

#include <stdlib.h>
#include <string.h>

typedef struct Node {
    int* children;
    int count;
} Node;

int dfs(int emp, Node* tree, int* informTime) {
    int maxTime = 0;
    for (int i = 0; i < tree[emp].count; i++) {
        int time = dfs(tree[emp].children[i], tree, informTime);
        if (time > maxTime) maxTime = time;
    }
    return informTime[emp] + maxTime;
}

int numOfMinutes(int n, int headID, int* manager, int managerSize, int* informTime, int informTimeSize) {
    // Build tree: each manager -> list of direct reports
    Node* tree = calloc(n, sizeof(Node));
    int* childCount = calloc(n, sizeof(int));

    // First pass: count children per manager
    for (int i = 0; i < n; i++) {
        if (manager[i] != -1) childCount[manager[i]]++;
    }
    // Allocate children arrays
    for (int i = 0; i < n; i++) {
        tree[i].children = malloc(childCount[i] * sizeof(int));
        tree[i].count = 0;
    }
    // Second pass: populate children
    for (int i = 0; i < n; i++) {
        if (manager[i] != -1) {
            int mgr = manager[i];
            tree[mgr].children[tree[mgr].count++] = i;
        }
    }

    int result = dfs(headID, tree, informTime);
    for (int i = 0; i < n; i++) free(tree[i].children);
    free(tree); free(childCount);
    return result;
}

## Key Takeaways

- **Tree + DFS** is the natural solution for this hierarchy problem
- **Build adjacency list** from the manager array, with each manager pointing to their direct reports
- The answer is the **maximum path sum** from root to any leaf, weighted by informTime
- This is essentially: find the deepest path in a tree where each node has a weight
