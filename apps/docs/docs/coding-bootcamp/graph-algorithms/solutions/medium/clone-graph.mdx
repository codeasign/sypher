# Clone Graph — Solution

## Problem Overview

Deep copy a connected undirected graph. The cloned graph must be structurally identical but entirely new — no shared references with the original.

## Approach: BFS with HashMap

**Idea:** Use a hash map from original node index to cloned node index (or a mapping of neighbors). Traverse the graph with BFS, cloning each node before processing its neighbors.

**Algorithm:**
1. Create an array `clone` of size `n+1` where `clone[i]` will hold the cloned neighbor list for node `i`
2. Use BFS starting from node 1
3. For each node popped from the queue, create its neighbor list in `clone[node]` by iterating over the original neighbors
4. If a neighbor hasn't been visited yet, mark it visited and push it to the queue
5. Return the clone array (positions 1..n)

**Complexity Analysis:**
- **Time:** O(V + E) — we visit each node and edge once
- **Space:** O(V) — queue and clone storage

import java.util.*;

class Solution {
    public int[][] cloneGraph(int[][] adj) {
        int n = adj.length;
        if (n == 0) return new int[0][];
        int[][] result = new int[n][];
        boolean[] visited = new boolean[n];
        Queue<Integer> queue = new LinkedList<>();
        queue.offer(0);                            // Start BFS from node 0
        visited[0] = true;
        while (!queue.isEmpty()) {
            int node = queue.poll();
            result[node] = adj[node].clone();      // Copy the neighbor list
            for (int neighbor : adj[node]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.offer(neighbor);         // Enqueue unvisited neighbors
                }
            }
        }
        return result;
    }
}

from collections import deque

def clone_graph(adj: list[list[int]]) -> list[list[int]]:
    n = len(adj)
    if n == 0:
        return []
    result = [None] * n
    visited = [False] * n
    q = deque([0])                               # Start BFS from node 0
    visited[0] = True
    while q:
        node = q.popleft()
        result[node] = adj[node][:]              # Create a copy of neighbor list
        for neighbor in adj[node]:
            if not visited[neighbor]:
                visited[neighbor] = True
                q.append(neighbor)               # Enqueue unvisited neighbors
    return result

fn clone_graph(adj: Vec<Vec<i32>>) -> Vec<Vec<i32>> {
    let n = adj.len();
    if n == 0 { return vec![]; }
    let mut result = vec![vec![]; n];
    let mut visited = vec![false; n];
    let mut q = VecDeque::new();
    q.push_back(0);                                // Start BFS from node 0
    visited[0] = true;
    while let Some(node) = q.pop_front() {
        result[node] = adj[node].clone();          // Clone the neighbor list
        for &neighbor in &adj[node] {
            let nb = neighbor as usize;
            if !visited[nb] {
                visited[nb] = true;
                q.push_back(nb);                   // Enqueue unvisited neighbors
            }
        }
    }
    result
}

int** cloneGraph(int** adj, int adjSize, int* adjColSize, int* returnSize, int** returnColSize) {
    *returnSize = adjSize;
    *returnColSize = malloc(adjSize * sizeof(int));
    if (adjSize == 0) return NULL;
    int** result = malloc(adjSize * sizeof(int*));
    int* visited = calloc(adjSize, sizeof(int));
    int* queue = malloc(adjSize * sizeof(int));
    int front = 0, back = 0;
    queue[back++] = 0;                             // Start BFS from node 0
    visited[0] = 1;
    while (front < back) {
        int node = queue[front++];
        (*returnColSize)[node] = adjColSize[node];
        result[node] = malloc(adjColSize[node] * sizeof(int));
        for (int i = 0; i < adjColSize[node]; i++) {
            result[node][i] = adj[node][i];        // Copy neighbor value
            if (!visited[adj[node][i]]) {
                visited[adj[node][i]] = 1;
                queue[back++] = adj[node][i];      // Enqueue unvisited
            }
        }
    }
    free(visited); free(queue);
    return result;
}

            }
        }

func cloneGraph(adj [][]int) [][]int {
    n := len(adj)
    if n == 0 { return [][]int{} }
    result := make([][]int, n)
    visited := make([]bool, n)
    queue := []int{0}                              // Start BFS from node 0
    visited[0] = true
    for len(queue) > 0 {
        node := queue[0]
        queue = queue[1:]
        result[node] = make([]int, len(adj[node]))
        copy(result[node], adj[node])              // Copy neighbor list
        for _, neighbor := range adj[node] {
            if !visited[neighbor] {
                visited[neighbor] = true
                queue = append(queue, neighbor)    // Enqueue unvisited
            }
        }
    }
    return result
}

## Key Takeaways

- **BFS or DFS** both work for graph cloning — the key is the visited/cloned mapping
- A **hash map (or array)** from original to cloned node prevents infinite loops in cyclic graphs
- The graph is connected and undirected, so BFS from node 0 reaches all nodes
- Deep copy means creating new data structures, not sharing references
