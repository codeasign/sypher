## What This Is

A crash course in algorithmic coding patterns. Each module covers one
pattern вҖ” you'll learn the theory, solve curated problems of increasing
difficulty, and study idiomatic solutions in 6 programming languages.

## Why Coding Patterns Matter

Interview problems and real-world code challenges follow recurring
patterns. Master the pattern, and you can solve any problem in its
family вҖ” instead of memorizing hundreds of individual solutions.

## Skills Gained

- Recognize which pattern fits a given problem
- Implement two-pointer, sliding window, and other fundamental algorithms
- Write clean, correct code in Python, Java, C++, TypeScript,
  JavaScript, and Rust
- Test solutions against sample and hidden test cases via an embedded
  Judge0 code runner

## Prerequisites

Basic programming knowledge in any language. Familiarity with arrays,
strings, and hash maps is helpful but not required.

## How the Bootcamp Works

Each pattern module has three parts:

1. **Theory** вҖ” Learn the pattern: what it is, when to use it,
   complexity analysis, and a worked example.
2. **Exercises** вҖ” Solve problems grouped by difficulty (Easy вҶ’ Medium
   вҶ’ Hard). Write code in the embedded editor and run it against
   test cases.
3. **Solutions** вҖ” Study idiomatic solutions in all 6 languages
   side-by-side.

## Patterns

- **Arrays** вҖ” Traversal, in-place modification, prefix/suffix arrays, and simulation over arrays
- **Hash Maps / Hash Sets** вҖ” Frequency counting, fast lookup, and deduplication with O(1) average access
- **Prefix Sum** вҖ” Running sum precomputation for O(1) range queries and subarray sum problems
- **Two Pointers** вҖ” Using left/right converging or same-direction pointers to solve array and string problems efficiently (O(n) time, O(1) space)
- **Sliding Window** вҖ” Maintaining a window of elements to solve subarray/substring problems in O(n) time
- **Binary Search** вҖ” Finding elements in sorted arrays and searching the answer space in O(log n) time
- **Fast & Slow Pointers** вҖ” Floyd's cycle detection, finding middle nodes, and cycle-related problems
- **Linked List** вҖ” Pointer manipulation, reversal, cycle detection, and merging linked lists
- **Stack & Monotonic Stack** вҖ” LIFO structures for nested patterns, next greater/smaller element, and expression evaluation
- **Queue & Deque** вҖ” FIFO and double-ended structures for BFS, sliding windows, and scheduling
- **Tree DFS & BFS** вҖ” Recursive and level-order tree traversals for depth, path, and tree property problems
- **Binary Search Tree (BST)** вҖ” Ordered traversal, validation, search, and manipulation of BSTs
- **Heap / Priority Queue** вҖ” Using priority queues for Top K, median finding, and merging sorted data
- **Intervals** вҖ” Merging, inserting, and finding overlaps among ranges
- **Greedy** вҖ” Local optimum choices leading to global optimum for scheduling, jumps, and optimization
- **Dynamic Programming** вҖ” Breaking problems into overlapping subproblems with memoization for polynomial-time solutions
- **Backtracking** вҖ” Systematically searching for solutions by building candidates and pruning invalid branches
- **Trie** вҖ” Prefix tree for efficient word search, autocomplete, and pattern matching
- **Topological Sort** вҖ” DAG ordering for dependency resolution and course scheduling
- **Union Find (DSU)** вҖ” Disjoint set union for dynamic connectivity and component tracking
- **Graph Algorithms** вҖ” Adjacency lists, DFS/BFS traversal, cloning, connected components, and shortest paths
- **Matrix Traversal** вҖ” Direction vectors, spiral traversal, grid rotations, and in-place matrix operations
- **Bit Manipulation** вҖ” XOR tricks, bit masking, and shift operations for space/time optimization
- **Math & Number Theory** вҖ” GCD, prime sieving, modular arithmetic, and exponentiation
- **String Algorithms** вҖ” Parsing, pattern matching (KMP), encoding/decoding, and string manipulation
- **Simulation** вҖ” Following exact rules, managing state, and stepping through processes
- **Design** вҖ” Designing data structures (LRU/LFU caches, Min Stack, Browser History)
