## Production Hardening

### 1. Agent Fallback Chain

```python
# app/fallback.py
def routing_fallback(state: dict) -> str:
    if state.get("routing_decision") == "multi":
        return "technical"  # Default to technical if supervisor is unsure
    return state["routing_decision"]
```

### 2. Response Caching

```python
# app/cache.py
cache = {}  # Simple in-memory; replace with Redis in production

def cached_specialist(specialist_type: str, subject: str, description: str):
    key = f"{specialist_type}:{hash(subject + description)}"
    if key in cache:
        return cache[key]
    result = call_specialist(specialist_type, subject, description)
    cache[key] = result
    return result
```

## Deploy It

### API Endpoint

```python
# app/api.py
from fastapi import FastAPI
from app.runner import route_ticket
from pydantic import BaseModel

app = FastAPI(title="RouteBot API")

class TicketRequest(BaseModel):
    subject: str
    description: str
    email: str

@app.post("/api/v1/route")
def route(ticket: TicketRequest):
    result = route_ticket(ticket.subject, ticket.description, ticket.email)
    return {
        "ticket_id": result["ticket_id"],
        "routed_to": result["routing_decision"],
        "resolution": result["final_resolution"],
        "status": result["status"]
    }
```
