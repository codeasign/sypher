## Implementation

Below is a complete JavaScript implementation of the Observer pattern using a stock market domain. A `StockExchange` (Subject) tracks prices for AAPL, GOOGL, and MSFT. Three observers react to price changes: `PriceDisplay` shows updated prices, `AlertSystem` flags movements over 5%, and `PortfolioTracker` recalculates total portfolio value. The implementation uses ES2022+ class syntax with private fields and demonstrates OOP concepts (abstraction, encapsulation, inheritance, polymorphism) along with SOLID principles (SRP, OCP, LSP, ISP, DIP).

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────┐
//   │   Observer (duck-typed protocol)  │
//   ├──────────────────────────────────┤
//   │ + update(ticker, price)          │ ← OOP: Abstraction via protocol
//   └────────────────┬─────────────────┘    ISP: single-method interface
//                    ▲ implements
//   ┌────────────────┴─────────────────┐
//   │                │                 │
//   ┌────────────┐ ┌──────────┐ ┌──────────────┐
//   │PriceDisplay│ │AlertSys  │ │Portfolio     │ ← OOP: Polymorphism
//   ├────────────┤ ├──────────┤ ├──────────────┤    LSP: all implement
//   │ update()   │ │ update() │ │ update()     │     update()
//   └────────────┘ └──────────┘ └──────────────┘
//
//   ┌──────────────────────────────────┐
//   │       StockExchange              │ ← OOP: Encapsulation
//   ├──────────────────────────────────┤    SRP: manages observers
//   │ - #observers                     │     and prices only
//   │ - #prices                        │    OCP: new observers added
//   │ + attach(obs)                    │     without modifying subject
//   │ + detach(obs)                    │    DIP: depends on Observer
//   │ + setPrice(t, p)                 │     protocol, not concretions
//   │ + getPrice(t)                    │
//   └──────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — The Observer role is defined as a protocol (duck type): any object with an `update(ticker, price)` method can be an observer. The StockExchange depends only on this protocol, not on concrete types.
2. **Encapsulation** — StockExchange hides its internal observer list (`#observers`) and price map (`#prices`) behind public methods. Each observer encapsulates its own state (e.g., AlertSystem stores previous prices internally).
3. **Inheritance** — The StockExchange class does not extend a base Subject class (JavaScript uses composition here), but observers share a common protocol through structural typing.
4. **Polymorphism** — StockExchange iterates over its observer list and calls `update()` on each. The correct implementation runs per observer type without instanceof checks.

#### SOLID Principles Applied

5. **SRP** — StockExchange is responsible only for managing prices and notifying observers. Each observer has a single responsibility: PriceDisplay displays, AlertSystem alerts, PortfolioTracker tracks.
6. **OCP** — Adding a new observer (e.g., `ChartRenderer`) requires creating a new class with `update()` and registering it with `attach()`. The StockExchange class itself is never modified.
7. **LSP** — Any observer implementing the `update(ticker, price)` protocol can substitute for any other. The StockExchange treats all observers uniformly.
8. **ISP** — The Observer protocol is minimal: a single `update` method. No observer is forced to implement methods it does not need.
9. **DIP** — StockExchange depends on the abstract Observer protocol, not on concrete observer classes. Observers are injected via `attach()`.

### StockExchange (Subject)

```javascript
// Observer Pattern — JavaScript — Stock Market example
// =====================================================
// Participants: StockExchange (Subject), PriceDisplay,
// AlertSystem, PortfolioTracker (Concrete Observers)
// OOP: abstraction, encapsulation, inheritance, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// =====================================================

// ── Subject ──
class StockExchange {
  #observers = [];
  #prices = {};

  /** Register an observer (any object with update(ticker, price)) */
  attach(observer) {
    this.#observers.push(observer);
  }

  /** Unregister an observer */
  detach(observer) {
    const idx = this.#observers.indexOf(observer);
    if (idx !== -1) this.#observers.splice(idx, 1);
  }

  /** Notify all observers of a price change */
  #notifyObservers(ticker, price) {
    for (const obs of this.#observers) {
      obs.update(ticker, price);
    }
  }

  /** Set a new price and notify observers */
  setPrice(ticker, price) {
    this.#prices[ticker] = price;
    this.#notifyObservers(ticker, price);
  }

  /** Get the current price of a ticker */
  getPrice(ticker) {
    return this.#prices[ticker] ?? 0;
  }
}
```

### PriceDisplay (Concrete Observer)

```javascript
// ── Concrete Observer 1: PriceDisplay ──
class PriceDisplay {
  update(ticker, price) {
    console.log(`DISPLAY: ${ticker} is now $${price.toFixed(2)}`);
  }
}
```

### AlertSystem (Concrete Observer)

```javascript
// ── Concrete Observer 2: AlertSystem ──
class AlertSystem {
  #previousPrices = {};
  #threshold;

  constructor(thresholdPercent) {
    this.#threshold = thresholdPercent;
  }

  update(ticker, price) {
    if (this.#previousPrices[ticker] !== undefined) {
      const oldPrice = this.#previousPrices[ticker];
      const change = Math.abs(price - oldPrice) / oldPrice * 100;
      if (change >= this.#threshold) {
        const direction = price >= oldPrice ? 'up' : 'down';
        console.log(`ALERT: ${ticker} moved ${direction} ${change.toFixed(1)}% ` +
          `($${oldPrice.toFixed(2)} → $${price.toFixed(2)})`);
      }
    }
    this.#previousPrices[ticker] = price;
  }
}
```

### PortfolioTracker (Concrete Observer)

```javascript
// ── Concrete Observer 3: PortfolioTracker ──
class PortfolioTracker {
  #holdings = new Map();
  #exchange;
  #totalValue = 0;

  /**
   * @param {StockExchange} exchange - reference to query current prices
   */
  constructor(exchange) {
    this.#exchange = exchange;
  }

  /** Add a stock holding to the portfolio */
  addHolding(ticker, shares) {
    this.#holdings.set(ticker, shares);
  }

  /** Recalculate total portfolio value from all holdings */
  #recalculate() {
    this.#totalValue = 0;
    for (const [ticker, shares] of this.#holdings) {
      const price = this.#exchange.getPrice(ticker);
      this.#totalValue += price * shares;
    }
  }

  update(ticker, price) {
    if (this.#holdings.has(ticker)) {
      this.#recalculate();
      console.log(`PORTFOLIO: Updated value is $${this.#totalValue.toFixed(2)}`);
    }
  }

  getTotalValue() {
    return this.#totalValue;
  }
}
```

### Main / Driver

```javascript
// ── Main / Driver ──
const exchange = new StockExchange();

const display = new PriceDisplay();
const alerts = new AlertSystem(5.0);
const portfolio = new PortfolioTracker(exchange);

// Register observers
exchange.attach(display);
exchange.attach(alerts);
exchange.attach(portfolio);

// Set up portfolio holdings
portfolio.addHolding('AAPL', 10);
portfolio.addHolding('GOOGL', 5);
portfolio.addHolding('MSFT', 8);

// Price changes — observers react automatically
console.log('=== Stock Market Observer Demo ===');
console.log('');

exchange.setPrice('AAPL', 150.00);
exchange.setPrice('GOOGL', 2800.00);
exchange.setPrice('MSFT', 300.00);

console.log('');

// Detach display temporarily
exchange.detach(display);
exchange.setPrice('AAPL', 162.00);  // 8% change — triggers alert, no display
exchange.attach(display);

console.log('');

exchange.setPrice('GOOGL', 2750.00);  // < 5% change — no alert
exchange.setPrice('MSFT', 315.00);     // 5% change — triggers alert
```

### Language-Specific Notes

- **Private fields (`#`)**: JavaScript ES2022+ private fields (`#observers`, `#prices`, `#holdings`) enforce true encapsulation at the language level — they cannot be accessed from outside the class, even through reflection.
- **Duck-typed Observer protocol**: JavaScript does not have interfaces. The Observer role is defined by convention: any object with an `update(ticker, price)` method can be an observer. This is structural typing in practice.
- **Dynamic observer management**: The `attach()` and `detach()` methods allow observers to be added or removed at runtime, as demonstrated by detaching and re-attaching the display mid-execution.
- **Map vs Object**: The portfolio uses `Map` for holdings (keyed by ticker symbol) because it provides cleaner iteration with `for...of` and better performance for frequent additions and removals.
- **No external dependencies**: The implementation uses only ES2022+ built-in features — no libraries, no frameworks, no Node.js-specific APIs.
