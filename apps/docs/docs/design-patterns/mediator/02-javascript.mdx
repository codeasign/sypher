## Implementation

Below is a complete JavaScript (ES2022+) implementation of the Mediator pattern in the stock market domain. The `NasdaqExchange` mediator coordinates communication between `BuyAgent` and `SellAgent` instances, matching buy and sell orders without agents knowing about each other directly. The pattern decouples trading agents from the exchange, allowing orders to be matched and filled without direct peer-to-peer communication.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────────┐
//   │   StockExchange (base class)          │ ← OOP: Abstraction via base class
//   ├──────────────────────────────────────┤    ISP: focused mediator methods
//   │   registerAgent(agent)               │
//   │   placeBid(agent, sym, qty, price)   │
//   │   placeAsk(agent, sym, qty, price)   │
//   │   notify(agent, message)             │
//   │   broadcast(message)                 │
//   └────────────────┬─────────────────────┘
//                    ▲ extends
//   ┌────────────────┴─────────────────────┐
//   │   NasdaqExchange                      │ ← SRP: only handles order
//   │   #agents: Map                        │    matching and notification
//   │   #orderBook: Array                   │    OCP: new agent types work
//   │   #matchOrders()                      │    without changing exchange
//   │   notify(agent, message)              │    DIP: depends on TradingAgent
//   │   broadcast(message)                  │    protocol, not concretions
//   └────────────────┬─────────────────────┘
//                    │ coordinates
//   ┌────────────────┴─────────────────────┐
//   │   ┌────────────────┐  ┌────────────┐ │
//   │   │  BuyAgent      │  │ SellAgent  │ │ ← LSP: all agents follow
//   │   │  send(exch)    │  │ send(exch) │ │    TradingAgent protocol
//   │   │  onMessage(msg)│  │onMessage   │ │    Polymorphism: exchange
//   │   └────────────────┘  └────────────┘ │    calls onMessage uniformly
//   └───────────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — The `StockExchange` base class defines the mediator contract (registerAgent, placeBid, placeAsk, notify, broadcast). The `TradingAgent` base class defines the colleague contract (send, onMessage). Concrete classes implement the details while hiding internal state.
2. **Encapsulation** — Private fields (`#agents`, `#orderBook`, `#name`, `#symbol`, `#quantity`, `#maxPrice`, `#minPrice`) hide internal state from outside access. Agents have no knowledge of each other — all communication flows through the mediator.
3. **Inheritance** — `NasdaqExchange extends StockExchange`, `BuyAgent extends TradingAgent`, `SellAgent extends TradingAgent`. Subclasses inherit the base contract and override behavior.
4. **Polymorphism** — The exchange calls `agent.onMessage()` on any registered agent without knowing whether it is a `BuyAgent` or `SellAgent`. Each agent's `send()` method is called polymorphically.

#### SOLID Principles Applied

5. **SRP** — `NasdaqExchange` is responsible solely for order matching and agent notification. `BuyAgent` only creates buy orders. `SellAgent` only creates sell orders. Each class has a single, well-defined reason to change.
6. **OCP** — New agent types (e.g., `MarketMakerAgent`) can be added by extending `TradingAgent` and implementing `send()`. The `NasdaqExchange` mediator never needs modification to support new agent types.
7. **LSP** — Any subclass of `TradingAgent` can substitute for the base type. The exchange treats all agents uniformly through the `TradingAgent` protocol.
8. **ISP** — The `TradingAgent` base class defines only the methods every agent needs (`getName`, `getSymbol`, `send`, `onMessage`). No agent is forced to implement methods it does not use.
9. **DIP** — `NasdaqExchange` depends on the abstract `TradingAgent` protocol, not on concrete agent classes. Agents depend on the abstract `StockExchange` mediator, not on a concrete exchange.

### NasdaqExchange (Concrete Mediator)
```javascript
// ── Concrete Mediator ──
class NasdaqExchange extends StockExchange {
  #agents = new Map();
  #orderBook = [];

  /** Register a trading agent with the exchange */
  registerAgent(agent) {
    this.#agents.set(agent.getName(), agent);
    this.notify(agent, `Welcome to NasdaqExchange, ${agent.getName()}!`);
  }

  /** Place a buy order (bid) */
  placeBid(agent, symbol, quantity, price) {
    this.#orderBook.push({ agent, symbol, quantity, price, type: 'bid' });
    this.#matchOrders();
  }

  /** Place a sell order (ask) */
  placeAsk(agent, symbol, quantity, price) {
    this.#orderBook.push({ agent, symbol, quantity, price, type: 'ask' });
    this.#matchOrders();
  }

  /** Match buy and sell orders by symbol and price */
  #matchOrders() {
    for (const bid of this.#orderBook) {
      if (bid.type !== 'bid' || bid.quantity === 0) continue;
      for (const ask of this.#orderBook) {
        if (ask.type !== 'ask' || ask.quantity === 0) continue;
        if (bid.symbol === ask.symbol && bid.price >= ask.price) {
          const fillQty = Math.min(bid.quantity, ask.quantity);
          bid.quantity -= fillQty;
          ask.quantity -= fillQty;

          this.notify(
            bid.agent,
            `FILLED: Bought ${fillQty} shares of ${bid.symbol} at $${ask.price.toFixed(2)}`
          );
          this.notify(
            ask.agent,
            `FILLED: Sold ${fillQty} shares of ${ask.symbol} at $${ask.price.toFixed(2)}`
          );
          this.broadcast(
            `TRADE: ${bid.agent.getName()} x ${ask.agent.getName()} — ${fillQty} ${bid.symbol} @ $${ask.price.toFixed(2)}`
          );
        }
      }
    }
    // Remove fully filled orders from the book
    this.#orderBook = this.#orderBook.filter((o) => o.quantity > 0);
  }

  /** Send a private message to a specific agent */
  notify(agent, message) {
    agent.onMessage(message);
  }

  /** Send a public announcement to all registered agents */
  broadcast(message) {
    for (const agent of this.#agents.values()) {
      agent.onMessage(message);
    }
  }
}
```

### TradingAgent (Colleague / Base Class)
```javascript
// ── Colleague (base class) ──
class TradingAgent {
  #name;
  #symbol;

  constructor(name, symbol) {
    this.#name = name;
    this.#symbol = symbol;
  }

  getName() {
    return this.#name;
  }

  getSymbol() {
    return this.#symbol;
  }

  /** Place an order — subclass responsibility */
  send(exchange) {
    throw new Error('Method not implemented — subclass must override');
  }

  /** Receive a message from the mediator */
  onMessage(message) {
    console.log(`[${this.#name}] ${message}`);
  }
}
```

### BuyAgent (Concrete Colleague 1)
```javascript
// ── Concrete Colleague 1: BuyAgent ──
class BuyAgent extends TradingAgent {
  #quantity;
  #maxPrice;

  constructor(name, symbol, quantity, maxPrice) {
    super(name, symbol);
    this.#quantity = quantity;
    this.#maxPrice = maxPrice;
  }

  /** Submit a buy order to the exchange */
  send(exchange) {
    console.log(
      `${this.getName()}: Placing bid for ${this.#quantity} ${this.getSymbol()} @ $${this.#maxPrice.toFixed(2)}`
    );
    exchange.placeBid(this, this.getSymbol(), this.#quantity, this.#maxPrice);
  }
}
```

### Main / Driver
```javascript
// ── Main / Driver ──
const exchange = new NasdaqExchange();

// Create trading agents
const buyer1 = new BuyAgent('BuyAgent_AAPL', 'AAPL', 100, 150.50);
const seller1 = new SellAgent('SellAgent_AAPL', 'AAPL', 100, 150.50);
const buyer2 = new BuyAgent('BuyAgent_GOOGL', 'GOOGL', 50, 180.30);
const seller2 = new SellAgent('SellAgent_GOOGL', 'GOOGL', 50, 175.25);

// Register all agents with the exchange
console.log('=== Stock Market Mediator Demo ===\n');

exchange.registerAgent(buyer1);
exchange.registerAgent(seller1);
exchange.registerAgent(buyer2);
exchange.registerAgent(seller2);

console.log('');

// Submit orders — exchanges handle matching internally
buyer1.send(exchange);   // Bid AAPL 100 @ $150.50 — no matching ask yet
seller1.send(exchange);  // Ask AAPL 100 @ $150.50 — matches with buyer1

console.log('');

buyer2.send(exchange);   // Bid GOOGL 50 @ $180.30 — no matching ask yet
seller2.send(exchange);  // Ask GOOGL 50 @ $175.25 — matches immediately (bid >= ask)
```

### Language-Specific Notes

- **Private class fields (`#`)**: JavaScript ES2022+ private fields (e.g., `#agents`, `#orderBook`, `#name`, `#quantity`) provide true hard encapsulation — they cannot be accessed from outside the class, even through reflection or `Object.keys()`. This enforces the mediator's internal state isolation.
- **Base class as interface contract**: JavaScript lacks formal interfaces. The `StockExchange` and `TradingAgent` base classes act as interface contracts by defining methods that throw `Error` if not overridden — a runtime form of the interface guarantee.
- **Map for dynamic registries**: `Map` provides O(1) key lookups, preserves insertion order, and offers a clean `.values()` iterator — ideal for the agent registry where agents are added and looked up by name.
- **Array methods for order book management**: `.filter()` removes filled orders from the book in a single expression, while `for...of` loops and `.push()` provide clean order book manipulation without index tracking.
- **Template literals**: `${variable.toFixed(2)}` is used throughout for constructing readable, formatted log messages — fill notifications, trade announcements, and order placement all use template strings for clarity.
- **No external dependencies**: The implementation uses only ES2022+ built-in features — no libraries, no frameworks, no Node.js-specific APIs beyond `console.log`.
