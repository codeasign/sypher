## Implementation in C# 12 / .NET 8

The Memento pattern is a natural fit for C#, where nested classes, records, and collection expressions make the implementation both type-safe and concise. In this stock-market example, a `Portfolio` (originator) manages a collection of stock holdings and a cash balance. Before mutating operations, the caretaker (`PortfolioHistory`) can request a snapshot via `Save()`, which returns an opaque `PortfolioSnapshot` (memento) — a nested private class whose internals are visible only to the enclosing `Portfolio`. The caretaker never inspects the snapshot contents; it simply stores and retrieves them.

## How the Implementation Works

**Class diagram (pseudocode):**

```
Portfolio (Originator)
 ┣━ Dictionary<string, Holding> _holdings
 ┣━ decimal _cash
 ┣━ BuyTicker(symbol, shares, price)
 ┣━ SellTicker(symbol, shares)
 ┣━ Save() → PortfolioSnapshot
 ┣━ Restore(PortfolioSnapshot)
 ┗━ class PortfolioSnapshot (Memento, nested private)
      ┣━ IReadOnlyDictionary<string, Holding> _holdings
      ┗━ decimal _cash

PortfolioHistory (Caretaker)
 ┣━ Stack<PortfolioSnapshot> _snapshots
 ┣━ Push(PortfolioSnapshot)
 ┣━ Pop() → PortfolioSnapshot
 ┗━ CanUndo() → bool

Holding (immutable record struct)
 ┣━ string Symbol
 ┣━ int Shares
 ┗━ decimal TotalCost
```

**OOP and SOLID breakdown:**

- **Single Responsibility** — `Portfolio` owns trading logic and snapshot creation; `PortfolioHistory` manages the undo stack; `PortfolioSnapshot` is a passive data carrier with no behavior.
- **Open/Closed** — New snapshot strategies (delta snapshots, compressed snapshots) can be added by exposing an `ISnapshotStrategy` interface without modifying `Portfolio`.
- **Encapsulation** — `PortfolioSnapshot` is a nested private class inside `Portfolio`, so its constructor and data accessors are `internal`, invisible to any external code including `PortfolioHistory`.
- **Liskov Substitution** — The `Stack<T>` abstraction can be replaced with any other collection (queue, list, ring buffer) without affecting the originator or memento.

## Complete Code

The following program defines a portfolio that tracks stock holdings and cash, supports buy/sell operations, and records snapshots for undo via the Memento pattern.

### Main / Driver

```csharp
// Memento pattern — Stock Market Portfolio
// C# 12 / .NET 8

var portfolio = new Portfolio("Growth Portfolio", 1_000_000m);
var history = new PortfolioHistory();

portfolio.Print();

// Buy 100 shares of AAPL at $150
portfolio.BuyTicker("AAPL", 100, 150m);
history.Push(portfolio.Save());
Console.WriteLine("\nAfter buying 100 AAPL at $150:");
portfolio.Print();

// Buy 50 shares of GOOGL at $2800
portfolio.BuyTicker("GOOGL", 50, 2800m);
history.Push(portfolio.Save());
Console.WriteLine("\nAfter buying 50 GOOGL at $2800:");
portfolio.Print();

// Buy 200 shares of MSFT at $330
portfolio.BuyTicker("MSFT", 200, 330m);
history.Push(portfolio.Save());
Console.WriteLine("\nAfter buying 200 MSFT at $330:");
portfolio.Print();

// Sell 25 shares of GOOGL
portfolio.SellTicker("GOOGL", 25);
history.Push(portfolio.Save());
Console.WriteLine("\nAfter selling 25 GOOGL:");
portfolio.Print();

// Undo the sell operation
if (history.CanUndo())
{
    portfolio.Restore(history.Pop());
}
Console.WriteLine("\nAfter undo (restore before selling GOOGL):");
portfolio.Print();

// Undo the MSFT buy operation
if (history.CanUndo())
{
    portfolio.Restore(history.Pop());
}
Console.WriteLine("\nAfter second undo (restore before buying MSFT):");
portfolio.Print();
```

### Portfolio (Originator)

```csharp
public sealed class Portfolio
{
    private readonly Dictionary<string, Holding> _holdings = [];
    private decimal _cash;

    public string Name { get; }

    public Portfolio(string name, decimal initialCash)
    {
        Name = name;
        _cash = initialCash;
    }

    public void BuyTicker(string symbol, int shares, decimal price)
    {
        decimal cost = shares * price;
        if (cost > _cash)
            throw new InvalidOperationException($"Insufficient funds: need {cost:C}, have {_cash:C}");
        _cash -= cost;

        if (_holdings.TryGetValue(symbol, out var existing))
        {
            int totalShares = existing.Shares + shares;
            decimal totalCost = existing.TotalCost + cost;
            _holdings[symbol] = existing with { Shares = totalShares, TotalCost = totalCost };
        }
        else
        {
            _holdings[symbol] = new Holding(symbol, shares, cost);
        }
    }

    public void SellTicker(string symbol, int shares)
    {
        if (!_holdings.TryGetValue(symbol, out var holding))
            throw new InvalidOperationException($"No holdings found for {symbol}");
        if (shares > holding.Shares)
            throw new InvalidOperationException($"Cannot sell {shares} shares of {symbol}; only {holding.Shares} held");

        decimal avgCost = holding.TotalCost / holding.Shares;
        decimal proceeds = shares * avgCost;
        _cash += proceeds;

        int remaining = holding.Shares - shares;
        if (remaining == 0)
        {
            _holdings.Remove(symbol);
        }
        else
        {
            _holdings[symbol] = holding with { Shares = remaining, TotalCost = holding.TotalCost - proceeds };
        }
    }

    public PortfolioSnapshot Save() => new(new Dictionary<string, Holding>(_holdings), _cash);

    public void Restore(PortfolioSnapshot snapshot)
    {
        _holdings.Clear();
        foreach (var kvp in snapshot.GetHoldings())
            _holdings[kvp.Key] = kvp.Value;
        _cash = snapshot.GetCash();
    }

    public void Print()
    {
        decimal totalValue = _cash;
        Console.WriteLine($"Name: {Name}");
        Console.WriteLine($"Cash: {_cash:C}");
        if (_holdings.Count == 0)
        {
            Console.WriteLine("No holdings.");
            return;
        }
        foreach (var (symbol, holding) in _holdings)
        {
            decimal avgPrice = holding.TotalCost / holding.Shares;
            decimal marketValue = holding.TotalCost;
            totalValue += marketValue;
            Console.WriteLine($"  {symbol}: {holding.Shares} shares @ {avgPrice:C} avg (cost basis: {holding.TotalCost:C})");
        }
        Console.WriteLine($"Total account value: {totalValue:C}");
    }

    // Nested private Memento class — full encapsulation
    public sealed class PortfolioSnapshot
    {
        private readonly IReadOnlyDictionary<string, Holding> _holdings;
        private readonly decimal _cash;

        internal PortfolioSnapshot(Dictionary<string, Holding> holdings, decimal cash)
        {
            _holdings = holdings;
            _cash = cash;
        }

        internal IReadOnlyDictionary<string, Holding> GetHoldings() => _holdings;

        internal decimal GetCash() => _cash;
    }
}
```

### PortfolioHistory (Caretaker)

```csharp
public sealed class PortfolioHistory
{
    private readonly Stack<Portfolio.PortfolioSnapshot> _snapshots = [];

    public void Push(Portfolio.PortfolioSnapshot snapshot)
    {
        _snapshots.Push(snapshot);
    }

    public Portfolio.PortfolioSnapshot Pop()
    {
        return _snapshots.Pop();
    }

    public bool CanUndo() => _snapshots.Count > 0;
}
```

## Language-Specific Notes

- **Nested private class** — `PortfolioSnapshot` is declared as a `public sealed class` nested inside `Portfolio`. Its constructor and data accessors are marked `internal`, so only the enclosing `Portfolio` can create or inspect snapshots. The caretaker (`PortfolioHistory`) only references the type; it never reads the data.
- **Record struct for value objects** — `Holding` is declared as `readonly record struct Holding(...)` using C# 12 primary constructors. This gives value equality, immutability, and deconstruction for free, perfect for representing an immutable snapshot of a holding at a point in time.
- **Collection expressions** — The `_holdings` and `_snapshots` fields are initialized with `[]` (C# 12 collection expression syntax) instead of `new Dictionary<...>()` or `new Stack<...>()`, reducing boilerplate.
- **`with` expressions for non-destructive mutation** — When updating an existing holding, the code uses `existing with { Shares = totalShares, ... }` to produce a new immutable copy of the record struct rather than mutating the original. This is idiomatic C# for record types.
- **Stack-based undo** — `PortfolioHistory` wraps a `Stack<PortfolioSnapshot>`. The `Pop()` method retrieves the most recent snapshot, and `CanUndo()` checks whether any snapshots remain. This gives a natural depth-first undo stack without exposing the stack implementation.
- **Deep copy in Save()** — The `Save()` method creates a **new** `Dictionary<string, Holding>` by copying the existing entries. This ensures that subsequent mutations to the originator do not affect previously captured snapshots (defensive copy).
