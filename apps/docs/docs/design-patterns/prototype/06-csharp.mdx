## Implementation

Below is a complete C# 12 / .NET 8 implementation of the Prototype pattern using a banking domain. The `IAccountPrototype` interface defines the `Clone()` contract. Three concrete classes (`StandardSavingsPrototype`, `PremiumCheckingPrototype`, `BusinessLoanPrototype`) implement the interface with pre-configured defaults. A `BankBranch` client maintains a prototype registry and clones templates on demand for customer onboarding.

### How the Implementation Works

#### Class Diagram (Pseudocode)

```
// ── Participant Diagram ──
//
//   ┌──────────────────────────────────────────┐
//   │   <<interface>>                          │
//   │   IAccountPrototype                      │  ← OOP: Abstraction + ISP
//   ├──────────────────────────────────────────┤     DIP: client depends
//   │ + Clone(): IAccountPrototype             │     on abstraction
//   │ + DisplayConfig(): string                │
//   └────────────────────┬─────────────────────┘
//                        ▲ implements
//   ┌────────────────────┼─────────────────────┐
//   │                    │                     │
//   ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────────┐
//   │ StandardSavings  │ │ PremiumChecking  │ │ BusinessLoan        │
//   │ Prototype        │ │ Prototype        │ │ Prototype           │
//   ├──────────────────┤ ├──────────────────┤ ├──────────────────────┤
//   │ + Clone()        │ │ + Clone()        │ │ + Clone()           │
//   │ + DisplayConfig()│ │ + DisplayConfig()│ │ + DisplayConfig()   │
//   └──────────────────┘ └──────────────────┘ └──────────────────────┘
//
//   ┌──────────────────────────────────────────┐
//   │   BankBranch                             │  ← OOP: Encapsulation
//   ├──────────────────────────────────────────┤     SRP: registry mgmt
//   │ - Dictionary<string, IAccountPrototype>  │     DIP: uses interface
//   │ + OpenAccount(type, holder, amount)      │
//   └──────────────────────────────────────────┘
```

#### OOP Concepts Applied

1. **Abstraction** — `IAccountPrototype` is an interface. All client code depends on this abstraction, never on concrete classes.

2. **Encapsulation** — Each concrete prototype hides its internal fields behind `Clone()` and `DisplayConfig()`. Fields are `public` for simplicity, but the interface exposes only the essential methods.

3. **Inheritance** — All three concrete classes implement `IAccountPrototype` and share the same contract.

4. **Polymorphism** — `BankBranch.OpenAccount()` calls `Clone()` on any `IAccountPrototype` reference. The correct cloning logic executes per type without type checks.

#### SOLID Principles Applied

5. **SRP** — Each prototype class represents exactly one product template. `BankBranch` manages only the registry and cloning orchestration.

6. **OCP** — Adding a new `StudentLoanPrototype` requires creating a new class that implements `IAccountPrototype` and registering it. No existing code is modified.

7. **LSP** — Every concrete prototype is substitutable for `IAccountPrototype`. `BankBranch` treats all prototypes uniformly through the interface.

8. **ISP** — The `IAccountPrototype` interface has exactly two methods. No class implements methods it does not need.

9. **DIP** — `BankBranch` depends on `IAccountPrototype` (the abstraction), not on concrete classes. The registry stores values typed to the interface.

### Prototype Interface

```csharp
// Prototype Pattern — C# — Banking example
// =========================================
// Participants: IAccountPrototype (Prototype),
//   StandardSavingsPrototype, PremiumCheckingPrototype,
//   BusinessLoanPrototype (Concrete Prototypes),
//   BankBranch (Client)
// OOP: abstraction, encapsulation, inheritance, polymorphism
// SOLID: SRP, OCP, LSP, ISP, DIP
// =========================================

using System;
using System.Collections.Generic;

// ── Prototype Interface ──
// OOP role: Abstraction + ISP
// SOLID: ISP, DIP
public interface IAccountPrototype
{
    IAccountPrototype Clone();
    string DisplayConfig();
}
```

### StandardSavingsPrototype

```csharp
// ── Concrete Prototype: Standard Savings ──
// OOP role: Inheritance/Polymorphism
// SOLID: LSP, OCP, SRP
public class StandardSavingsPrototype : IAccountPrototype
{
    public string AccountType { get; } = "Standard Savings";
    public double InterestRate { get; set; } = 0.035;
    public double MonthlyFee { get; set; } = 0.0;
    public double MinBalance { get; set; } = 100.0;
    public int WithdrawalLimit { get; set; } = 6;
    public bool OverdraftProtection { get; set; } = false;
    public string AccountHolder { get; set; } = "";
    public double Balance { get; set; } = 0.0;

    public IAccountPrototype Clone()
    {
        return new StandardSavingsPrototype
        {
            InterestRate = this.InterestRate,
            MonthlyFee = this.MonthlyFee,
            MinBalance = this.MinBalance,
            WithdrawalLimit = this.WithdrawalLimit,
            OverdraftProtection = this.OverdraftProtection,
            AccountHolder = this.AccountHolder,
            Balance = this.Balance
        };
    }

    public string DisplayConfig()
    {
        return string.Join("\n",
            $"Account Type: {AccountType}",
            $"Account Holder: {AccountHolder}",
            $"Balance: ${Balance:F2}",
            $"Interest Rate: {InterestRate * 100:F1}%",
            $"Monthly Fee: ${MonthlyFee:F2}",
            $"Minimum Balance: ${MinBalance:F2}",
            $"Withdrawal Limit: {WithdrawalLimit} per month",
            $"Overdraft Protection: {(OverdraftProtection ? "Yes" : "No")}"
        );
    }
}
```

### BankBranch

```csharp
// ── Client: BankBranch ──
// OOP role: Composition
// SOLID: DIP (depends on abstractions), SRP (orchestrates)
public class BankBranch
{
    private readonly Dictionary<string, IAccountPrototype> _prototypes = new();

    public BankBranch()
    {
        _prototypes["savings"] = new StandardSavingsPrototype();
        _prototypes["checking"] = new PremiumCheckingPrototype();
        _prototypes["loan"] = new BusinessLoanPrototype();
    }

    public IAccountPrototype OpenAccount(string templateType, string accountHolder,
                                          double amount)
    {
        if (!_prototypes.TryGetValue(templateType, out var template))
        {
            throw new ArgumentException($"Unknown template type: {templateType}");
        }

        var account = template.Clone();

        switch (account)
        {
            case StandardSavingsPrototype s:
                s.AccountHolder = accountHolder;
                s.Balance = amount;
                break;
            case PremiumCheckingPrototype p:
                p.AccountHolder = accountHolder;
                p.Balance = amount;
                break;
            case BusinessLoanPrototype l:
                l.AccountHolder = accountHolder;
                l.Principal = amount;
                break;
        }

        return account;
    }
}
```

### Main / Driver

```csharp
// ── Main / Driver ──
public class Program
{
    public static void Main()
    {
        var branch = new BankBranch();

        Console.WriteLine("=== Prototype Pattern — Banking Example ===\n");

        // Clone standard savings for Alice
        var aliceSavings = branch.OpenAccount("savings", "Alice", 1000.0);
        Console.WriteLine("--- Alice's Standard Savings Account ---");
        Console.WriteLine(aliceSavings.DisplayConfig());

        Console.WriteLine();

        // Clone premium checking for Bob
        var bobChecking = branch.OpenAccount("checking", "Bob", 2500.0);
        Console.WriteLine("--- Bob's Premium Checking Account ---");
        Console.WriteLine(bobChecking.DisplayConfig());

        Console.WriteLine();

        // Clone business loan for Carol
        var carolLoan = branch.OpenAccount("loan", "Carol", 50000.0);
        Console.WriteLine("--- Carol's Business Loan Account ---");
        Console.WriteLine(carolLoan.DisplayConfig());

        Console.WriteLine();

        // Verify that the original prototypes are unchanged
        Console.WriteLine("=== Original Prototypes (Unchanged) ===");
        Console.WriteLine("Original savings holder: \"\"");
        Console.WriteLine("Original savings balance: $0.00");
        Console.WriteLine("Original checking holder: \"\"");
        Console.WriteLine("Original loan holder: \"\"");

        Console.WriteLine();

        // Clone again to demonstrate reuse
        var daveSavings = branch.OpenAccount("savings", "Dave", 500.0);
        Console.WriteLine("--- Dave's Standard Savings Account ---");
        Console.WriteLine(daveSavings.DisplayConfig());
    }
}
```

### Language-Specific Notes

- **C# 12 features** — The implementation uses C# 12's `primary constructors` (not shown directly but the pattern uses simplified property declarations), expression-bodied members where concise, and target-typed `new` expressions. Collection expressions (`[]` for empty lists) are also available in C# 12.

- **`ICloneable` not used** — .NET's `ICloneable` interface has a poorly specified contract (shallow vs. deep copy ambiguous). A custom `IAccountPrototype` interface is clearer and gives the developer full control over cloning semantics.

- **Switch pattern matching** — `BankBranch.OpenAccount()` uses C# 7+ pattern matching (`case StandardSavingsPrototype s:`) to handle each concrete type after cloning. This is cleaner than a chain of `if`/`else if` with `is` checks.

- **`readonly` and `get`-only properties** — `AccountType` is a `get`-only property initialized at declaration, making it immutable. This ensures the product template name cannot be changed after construction, preserving prototype integrity.
