# Setting Up Python — Hands On

⏱ 12 minutes

## Breaking It Down

### Step 1: Check What You Already Have

Open a terminal and run:

```bash
python3 --version
```

Three outcomes:

- You see `Python 3.12.x` or `Python 3.13.x` — you already have a modern Python. Move to Step 2.
- You see `Python 3.x` where `x < 12` — you have Python but it is too old for most AI libraries in 2026. Upgrade below.
- You see `command not found` — no Python installed. Proceed to Step 2.

Also check `pip`:

```bash
pip3 --version
```

This should show a version number and the Python version it belongs to, for example `pip 24.0 from .../python3.12/site-packages/pip`.

### Step 2: Install Python 3.12 or 3.13

**macOS** — use Homebrew:

```bash
# Install Homebrew if needed: https://brew.sh
brew install python@3.12
brew link --overwrite python@3.12
```

**Windows** — use the official installer:

1. Go to [python.org/downloads](https://www.python.org/downloads/) and download the latest Python 3.12 or 3.13 installer for Windows.
2. Run the installer. **Crucially**, check the box **Add Python to PATH** at the bottom of the first screen — this is unchecked by default and is the number one cause of setup frustration on Windows.
3. Click **Install Now**.

**Linux (Ubuntu/Debian)** — use deadsnakes PPA:

```bash
sudo apt update
sudo apt install software-properties-common
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt install python3.12 python3.12-venv python3.12-distutils
```

### Step 3: Verify the Installation

Run these commands and confirm they produce version output without errors:

```bash
python3 --version
# Python 3.12.x

pip3 --version
# pip 24.x from ... (python 3.12)
```

On Windows, the command may be `python` instead of `python3`:

```powershell
python --version
python -m pip --version
```

If `python3` is not found on Windows but `python` is, that is normal — the Windows installer creates `python.exe` not `python3.exe`. Use `python` throughout on Windows, or create an alias if you prefer consistency.

### Step 4: Run the REPL

The REPL (Read-Eval-Print Loop) is an interactive Python session. Launch it:

```bash
python3
```

You should see a prompt like `>>>`. Try some expressions:

```python
>>> 2 + 2
4

>>> print("AI engineering starts here!")
AI engineering starts here!

>>> exit()
```

Press `Ctrl+D` (macOS/Linux) or `Ctrl+Z` then `Enter` (Windows) to exit.

### Step 5: Write and Run Your First Script

Create a file called `hello_ai.py` with a text editor:

```python
# hello_ai.py — Your first Python script
print("Hello, AI engineering!")
```

Run it:

```bash
python3 hello_ai.py
```

You should see `Hello, AI engineering!` printed to the terminal.

### Step 6: Choose a Code Editor

For AI engineering, you want an editor with Python support, integrated terminal, and a good debugger. Two strong choices:

- **VS Code** — install it from [code.visualstudio.com](https://code.visualstudio.com/), then add the Python extension by Microsoft. This is the most common editor in AI engineering teams.
- **PyCharm Community Edition** — available at [jetbrains.com/pycharm](https://www.jetbrains.com/pycharm/). Heavier but has excellent Python-specific tooling built in.

Regardless of choice, configure your editor to use the Python interpreter you just installed. In VS Code, open the Command Palette (`Ctrl+Shift+P`), run `Python: Select Interpreter`, and pick the path that matches your `python3 --version` output.

## How It Works Internally

When you run `python3 hello_ai.py`, the operating system:

1. Looks up `python3` in the directories listed in the `PATH` environment variable.
2. Finds the `python3.12` executable (a symlink created during installation).
3. Launches a new process running the CPython interpreter — the reference implementation of Python written in C.
4. CPython reads your file, compiles it to bytecode, and executes it on its virtual machine.

If `python3` is not on your PATH, the OS cannot find it — this is the most common reason a working installation appears broken. The `brew link` or "Add Python to PATH" checkbox ensures this symlink is created.
