# Data Visualization

⏱ 10 minutes · Difficulty: Intermediate

## What You Will Learn

By the end of this lesson you will be able to:

- Create line plots, bar charts, histograms, and scatter plots with matplotlib
- Customize figure size, colors, labels, legends, and titles
- Plot pandas DataFrames directly with the pandas `.plot()` method
- Create multi-panel figures with subplots
- Save figures to PNG and SVG files

## Prerequisites

You should already understand:

- [NumPy](../numpy/overview) — matplotlib renders NumPy arrays; you will generate data and transformations
- [Pandas](../pandas/overview) — you will plot DataFrames directly and analyze trends
- [Lists](../lists/overview) — scatter plot coordinates come from lists or arrays

## Why This Matters

Data visualization is how you understand your data. Summary statistics hide distributions, outliers, and trends. A histogram of model confidence scores shows you the spread that a mean value collapses. A scatter plot of accuracy vs latency reveals tradeoffs you would miss in a table. A line chart of training loss over epochs tells you whether your model is learning, overfitting, or stuck.

In AI engineering, you will use plots constantly: to inspect evaluation results, compare model performance, debug training curves, and communicate findings to stakeholders. matplotlib is the foundational library — pandas, seaborn, and many other tools build on it.

## The Problem

You trained three model variants and have their accuracy over 100 evaluation batches. The raw numbers are columns in a CSV. You need to see whether model B converges faster, whether model C is unstable, and where the accuracy plateau begins. You cannot see these patterns in a table — you need a line plot.

## The Concept

matplotlib has two interfaces. The **pyplot** interface (`plt.plot()`, `plt.xlabel()`) is quick and interactive — good for exploration. The **object-oriented** interface (`fig, ax = plt.subplots()`; `ax.plot()`) gives you explicit control over every element in the figure and is the recommended approach for production code. This lesson uses the object-oriented interface.
