# Strings — Hands On

⏱ 12 minutes

## Breaking It Down

### Step 1: Creating Strings

```python
# string_creation.py
s1 = "Double quotes"
s2 = 'Single quotes — equivalent in Python'
s3 = """Triple quotes
span multiple lines
and preserve line breaks."""
s4 = '''Triple single quotes
work the same way.'''

# Empty string
empty = ""

# String with special characters — use backslash to escape
escaped = "She said, \"Python strings are easy!\""
path = "C:\\Users\\ai_engineer\\project"
```

In Python, single and double quotes are equivalent — choose one and be consistent. Triple quotes are useful for multi-line strings and docstrings.

### Step 2: Indexing and Slicing

```python
# indexing_slicing.py
text = "AIDeveloper"

# Indexing — get one character
print(text[0])   # A
print(text[2])   # D
print(text[-1])  # r — last character
print(text[-2])  # e — second-to-last

# Slicing — get a substring: text[start:stop:step]
# stop is exclusive (does not include stop index)
print(text[0:2])   # AI
print(text[2:])    # Developer — from index 2 to end
print(text[:2])    # AI — from start to index 2 (exclusive)
print(text[::2])   # ADeoe — every second character
print(text[::-1])  # repoleveDIA — reverse the string
```

Key rule: `s[start:stop]` includes index `start` but excludes index `stop`. Think of slice indices as positions **between** characters.

### Step 3: Essential String Methods

```python
# string_methods.py
raw = "  Hello, AI Engineer!  "

# Remove whitespace
print(raw.strip())        # "Hello, AI Engineer!"
print(raw.lstrip())       # "Hello, AI Engineer!  "
print(raw.rstrip())       # "  Hello, AI Engineer!"

# Split into list
csv_line = "name,role,experience"
parts = csv_line.split(",")
print(parts)              # ['name', 'role', 'experience']

# Join list into string
joined = " | ".join(parts)
print(joined)             # "name | role | experience"

# Find and replace
log = "ERROR: Model timeout"
print(log.find("Model"))  # 7 — starting index of "Model"
print(log.find("GPU"))    # -1 — not found
print(log.replace("ERROR", "WARNING"))  # "WARNING: Model timeout"

# Case operations
print("hello".upper())    # "HELLO"
print("WORLD".lower())    # "world"
print("hello world".title())  # "Hello World"
```

### Step 4: f-strings (Python 3.6+)

f-strings are the modern, readable way to build strings with embedded expressions:

```python
# fstrings_demo.py
model = "gpt-4"
tokens = 150
latency_ms = 2342

# Old style — concatenation
log1 = "Model: " + model + " | Tokens: " + str(tokens)

# f-string — variables in {}
log2 = f"Model: {model} | Tokens: {tokens} | Latency: {latency_ms}ms"
print(log2)
# Model: gpt-4 | Tokens: 150 | Latency: 2342ms

# Expressions inside f-strings
print(f"Cost: ${tokens * 0.00015:.4f}")  # Cost: $0.0225

# Formatting — :.2f means 2 decimal places
accuracy = 0.923456
print(f"Accuracy: {accuracy:.2%}")        # Accuracy: 92.35%
print(f"Accuracy: {accuracy:.3f}")        # Accuracy: 0.923
```

### Step 5: Build Something Real — Log Parser and Prompt Builder

```python
# log_parser.py — Extract fields from structured log lines

def parse_log_line(line):
    """Parse a structured log line into its components."""
    line = line.strip()
    if not line:
        return None

    # Split by whitespace: LEVEL DATE TIME key=value ...
    parts = line.split()

    level = parts[0]
    date = parts[1]
    time = parts[2]

    # Remaining parts are key=value pairs
    fields = {}
    for pair in parts[3:]:
        key, value = pair.split("=")
        fields[key] = value

    return {
        "level": level,
        "date": date,
        "time": time,
        "fields": fields,
    }

log_line = "INFO 2026-06-30 14:30:22 model=gpt-4 tokens=150 latency=2342ms"
result = parse_log_line(log_line)
print(result["level"])          # INFO
print(result["fields"]["model"])      # gpt-4
print(result["fields"]["tokens"])     # 150
print(result["fields"]["latency"])    # 2342ms

# ---

def build_prompt(system_message, user_message, examples=None):
    """Build a chat prompt from components."""
    parts = [f"System: {system_message}"]

    if examples:
        for ex in examples:
            parts.append(f"Example: {ex}")

    parts.append(f"User: {user_message}")
    return "\n\n".join(parts)

prompt = build_prompt(
    "You are a helpful assistant.",
    "Explain Python strings.",
    ["Example 1: Slicing", "Example 2: f-strings"],
)
print(prompt)
# System: You are a helpful assistant.
#
# Example: Example 1: Slicing
#
# Example: Example 2: f-strings
#
# User: Explain Python strings.
```

### Step 6: Performance Consideration

String concatenation with `+` in a loop is slow because each concatenation creates a new string. For building large strings, use a list and `join()`:

```python
# Slow — O(n²) due to repeated copying
result = ""
for i in range(1000):
    result += str(i) + ","

# Fast — O(n)
parts = []
for i in range(1000):
    parts.append(str(i))
result = ",".join(parts)
```
