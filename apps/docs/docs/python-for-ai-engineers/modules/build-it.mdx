## Breaking It Down

### Step 1: Importing from the Standard Library

```python
# stdlib_demo.py

# Single import
import os
print(os.getcwd())       # Current working directory

# Import with alias
import json as js
# js.load, js.dump, etc.

# Import specific names
from datetime import datetime, timedelta
now = datetime.now()
print(now.strftime("%Y-%m-%d %H:%M:%S"))

# Import everything (avoid in production — pollutes namespace)
# from math import *
```

Standard library modules you will use frequently in AI work:

| Module | Purpose |
|--------|---------|
| `os` | File paths, environment variables |
| `sys` | Command-line arguments, Python path |
| `json` | Read/write JSON |
| `csv` | Read/write CSV files |
| `re` | Regular expressions for text processing |
| `collections` | defaultdict, Counter, deque |
| `pathlib` | Modern file path handling |
| `math` | Mathematical functions |
| `statistics` | Mean, median, standard deviation |

### Step 2: Creating Your Own Module

Create a file called `ai_utils.py`:

```python
# ai_utils.py — Your own reusable module

def chunk_text(text, max_chars=1000):
    """Split text into chunks of max_chars length."""
    chunks = []
    for i in range(0, len(text), max_chars):
        chunks.append(text[i:i + max_chars])
    return chunks

def count_tokens(text):
    """Approximate token count (4 characters ≈ 1 token)."""
    return len(text) // 4

def truncate_to_max_tokens(text, max_tokens):
    """Truncate text to a maximum number of approximate tokens."""
    max_chars = max_tokens * 4
    if len(text) <= max_chars:
        return text
    return text[:max_chars] + "..."
```

Now import it from another file in the same directory:

```python
# use_utils.py — Imports from ai_utils.py
import ai_utils

text = "Python modules are useful for organising code. " * 100

chunks = ai_utils.chunk_text(text, max_chars=200)
print(f"Split into {len(chunks)} chunks")

token_estimate = ai_utils.count_tokens(text)
print(f"Approximate tokens: {token_estimate}")

shortened = ai_utils.truncate_to_max_tokens(text, max_tokens=50)
print(f"Truncated: {shortened[:50]}...")
```

### Step 3: The `if __name__ == "__main__"` Guard

Without this guard, code at the module level runs when you import the module:

```python
# training_utils.py

def load_dataset(path):
    print(f"Loading dataset from {path}...")
    return ["sample1", "sample2", "sample3"]

# Without guard — this runs on import!
# print("Module loaded!")  # ❌ Runs when imported

# With guard — only runs when executed directly
if __name__ == "__main__":
    print("Testing load_dataset...")
    data = load_dataset("test.csv")
    print(f"Loaded {len(data)} samples.")
```

Now:

- `python3 training_utils.py` runs the test block.
- `from training_utils import load_dataset` in another file does NOT run the test block.

This pattern is essential for any module that you both reuse and test independently.

### Step 4: Relative Imports (Inside Packages)

Inside a package, use relative imports to refer to sibling modules:

```python
# utils/__init__.py — makes utils a package
# utils/helpers.py
# utils/parsers.py

# Inside utils/parsers.py — relative import from sibling
# from .helpers import clean_text

# Inside main.py (at project root) — absolute import
# from utils.parsers import parse_log_line
```

### Step 5: Creating a Package

Create this structure:

```
my_pipeline/
├── __init__.py
├── loader.py
├── processor.py
├── exporter.py
```

`__init__.py`:

```python
# my_pipeline/__init__.py
from .loader import load_data
from .processor import process_data
from .exporter import export_results

__all__ = ["load_data", "process_data", "export_results"]
```

Now a script can import everything with one line:

```python
from my_pipeline import load_data, process_data, export_results
```

### Step 7: Build Something Real — Multi-Module Pipeline

```python
# data_ingestion.py
"""Module for loading data from multiple sources."""

import json
import csv

def load_json(path):
    with open(path, "r", encoding="utf-8") as f:
        return json.load(f)

def load_csv(path):
    with open(path, "r", encoding="utf-8") as f:
        return list(csv.DictReader(f))

if __name__ == "__main__":
    # Test loading
    import tempfile, os
    with tempfile.NamedTemporaryFile(mode="w", suffix=".csv", delete=False) as f:
        f.write("name,score\nAlice,0.95\nBob,0.87")
        tmp_path = f.name
    data = load_csv(tmp_path)
    print(f"Loaded {len(data)} records from CSV")
    os.unlink(tmp_path)
```

```python
# evaluation.py
"""Module for computing evaluation metrics."""

def accuracy(predictions, labels):
    """Calculate accuracy as fraction of correct predictions."""
    if len(predictions) != len(labels):
        raise ValueError("Predictions and labels must have the same length")
    correct = sum(1 for p, l in zip(predictions, labels) if p == l)
    return correct / len(predictions) if len(predictions) > 0 else 0.0

if __name__ == "__main__":
    preds = ["positive", "negative", "positive"]
    labels = ["positive", "positive", "positive"]
    acc = accuracy(preds, labels)
    print(f"Accuracy: {acc}")  # 2/3 = 0.667
```
