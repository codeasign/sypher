# AI Development & Production Python — Beginner Practice

⏱ 20 minutes · Difficulty: Beginner

## Exercise 1: LLM API Call

**Problem:** Write a function `ask_llm(question: str) -> str` that sends a question to an LLM API (Anthropic or OpenAI) and returns the answer text.

**Expected input:** `ask_llm("What is the capital of France?")`

**Expected output:** A string containing the answer (e.g., "The capital of France is Paris.")

**Hints:**
- Use the `anthropic` or `openai` SDK.
- Read the API key from an environment variable with `os.environ`.
- Create a client, call `messages.create()`, and extract the text from the response.

**Constraints:** Only use the standard library plus the chosen provider's SDK. Maximum 20 lines of code.

## Exercise 2: Simple Pytest Test

**Problem:** Write a pytest test for the following function:

```python
def classify_number(n: int) -> str:
    """Classify a number as positive, negative, or zero."""
    if n > 0:
        return "positive"
    elif n < 0:
        return "negative"
    return "zero"
```

**Expected input:** `test_classify_number.py` with test functions.

**Expected output:** All tests pass when run with `pytest test_classify_number.py -v`.

**Hints:**
- Name your test functions `test_*`.
- Use plain `assert` statements.
- Test at least: positive number, negative number, zero, edge case (large number).

**Constraints:** Use only pytest (no additional libraries). At least 4 test cases.

## Exercise 3: Dockerfile for a Python Script

**Problem:** Write a Dockerfile that packages the following Python script:

```python
# greet.py
import os

name = os.environ.get("NAME", "World")
print(f"Hello, {name}!")
```

**Expected output:** `docker build -t greet . && docker run -e NAME=Alice greet` outputs "Hello, Alice!"

**Hints:**
- Use `python:3.12-slim` as the base image.
- Copy just the script (not a full project).
- Use `CMD` for the default command.

**Constraints:** The Dockerfile must be 6 lines or fewer (not counting blank lines). Must use a slim base image.
