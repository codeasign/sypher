## Summary

Graph Search finds the shortest chain of connections between two points in a network, like finding the fewest number of friend-introductions needed to connect you to someone you don't know. It explores outward from the start, one ring of connections at a time — everyone directly connected first, then everyone connected to those people, and so on — and keeps a breadcrumb trail so it can trace back the exact route once it reaches the target. It stays fast because it never revisits a place it has already explored, so it slows down only as the network itself grows bigger.

## Pseudocode

```text
1. If the start and target are the same place, the path is just that one place. Done.
2. Keep a "already visited" list, starting with just the start.
3. Keep a "waiting line" of places to explore next, starting with just the start.
4. Keep a note for each place saying "which place led me here."
5. While the waiting line isn't empty:
   a. Take the next place out of the front of the line.
   b. Look at each of its direct connections.
   c. For any connection not yet visited: mark it visited, write down that this place led to it, and add it to the back of the waiting line.
6. If the target was never visited, report "no path."
7. Otherwise, starting at the target, keep following the "led me here" notes backward until you reach the start.
8. Reverse that trail so it reads from start to target, and report it.
```

## Worked Example

We'll trace a search from `A` to `E` through this five-place network: `A` connects to `B` and `C`; `B` connects to `A` and `D`; `C` connects to `A` and `D`; `D` connects to `B`, `C`, and `E`; `E` connects only to `D`. This only works because every connection counts as the same distance — one hop is one hop, no connection is worth more than another. If some connections were more costly than others, exploring ring by ring wouldn't guarantee the cheapest total route.

## Complexity

- **Best case:** If the start and target are literally the same place, the search is done before it even begins exploring — instant (`O(1)`).
- **Average case:** Typically you have to fan out through some real chunk of the network — visiting places and following their connections — before you reach the target, which costs time proportional to how many places and connections you touch along the way (`O(V + E)`, where `V` is places and `E` is connections).
- **Worst case:** If the target sits on the far edge of the network, or isn't connected to the start at all, the search has to visit every place and follow every connection before it can be sure — proportional to the total number of places plus the total number of connections (`O(V + E)`).
- **Space:** Keeping track of which places are visited, which place led to which, and the waiting line of places still to explore all grow with the number of places in the network, so it uses `O(V)` extra memory.

## Applications

- **Network routing and hop-count protocols** — routers finding the fewest number of hops between two points in a network topology rely on this exact ring-by-ring expansion.
- **Social-network "degrees of separation" features** — LinkedIn's "how you're connected" and Facebook's mutual-friend distance calculations are built on the same breadth-first spread outward from a person.
- **Web crawlers exploring sites breadth-first** — crawling outward from a seed page one link away, then two links away, and so on, to build an index level by level.
- **Puzzle and maze solvers** — finding the shortest sequence of moves in a maze or a sliding-tile puzzle, where every move is worth exactly the same "one step."
- **Peer-to-peer network flood and broadcast protocols** — determining the fewest hops a message needs to reach every node in a mesh network.

<CodeBlock language="python">{`from collections import deque


def graph_search(graph: dict[str, list[str]], start: str, target: str) -> list[str]:
    if start == target:
        return [start]

    visited: set[str] = {start}
    parent: dict[str, str] = {}
    queue: deque[str] = deque([start])

    while queue:
        node = queue.popleft()
        for neighbor in graph.get(node, []):
            if neighbor not in visited:
                visited.add(neighbor)
                parent[neighbor] = node
                queue.append(neighbor)

    if target not in visited:
        return []

    path: list[str] = []
    node = target
    while node != start:
        path.append(node)
        node = parent[node]
    path.append(start)
    path.reverse()
    return path`}</CodeBlock>

<CodeBlock language="c">{`#include <stdio.h>
#include <string.h>

#define MAX_NODES 50
#define MAX_NEIGHBORS 20
#define MAX_NAME_LEN 32

/*
 * C has no built-in hash map, so the graph is represented with parallel
 * fixed-size arrays instead: names[i] is the i-th node's name, neighbors[i]
 * lists its neighbor names in adjacency-list order, neighborCount[i] is how
 * many neighbors it has. visited/parent are tracked by node index rather
 * than by name; parent[i] is the index of the node that first discovered
 * node i during the BFS, or -1 if node i has no parent yet (the start node,
 * or a node never reached).
 */
int graphSearch(char names[][MAX_NAME_LEN], int neighborCount[],
                 char neighbors[][MAX_NEIGHBORS][MAX_NAME_LEN], int nodeCount,
                 const char *start, const char *target, char result[][MAX_NAME_LEN]) {
    if (strcmp(start, target) == 0) {
        strcpy(result[0], start);
        return 1;
    }

    int startIndex = -1;
    for (int i = 0; i < nodeCount; i++) {
        if (strcmp(names[i], start) == 0) { startIndex = i; break; }
    }
    if (startIndex == -1) {
        return 0;
    }

    int visited[MAX_NODES];
    int parent[MAX_NODES];
    for (int i = 0; i < nodeCount; i++) {
        visited[i] = 0;
        parent[i] = -1;
    }

    int queue[MAX_NODES];
    int queueHead = 0;
    int queueTail = 0;

    visited[startIndex] = 1;
    queue[queueTail++] = startIndex;

    while (queueHead < queueTail) {
        int nodeIndex = queue[queueHead++];

        for (int i = 0; i < neighborCount[nodeIndex]; i++) {
            const char *neighborName = neighbors[nodeIndex][i];
            int neighborIndex = -1;
            for (int j = 0; j < nodeCount; j++) {
                if (strcmp(names[j], neighborName) == 0) { neighborIndex = j; break; }
            }
            if (neighborIndex != -1 && !visited[neighborIndex]) {
                visited[neighborIndex] = 1;
                parent[neighborIndex] = nodeIndex;
                queue[queueTail++] = neighborIndex;
            }
        }
    }

    int targetIndex = -1;
    for (int i = 0; i < nodeCount; i++) {
        if (strcmp(names[i], target) == 0) { targetIndex = i; break; }
    }
    if (targetIndex == -1 || !visited[targetIndex]) {
        return 0;
    }

    int pathIndices[MAX_NODES];
    int pathLen = 0;
    int cur = targetIndex;
    while (cur != startIndex) {
        pathIndices[pathLen++] = cur;
        cur = parent[cur];
    }
    pathIndices[pathLen++] = startIndex;

    for (int i = 0; i < pathLen; i++) {
        strcpy(result[i], names[pathIndices[pathLen - 1 - i]]);
    }

    return pathLen;
}`}</CodeBlock>

<CodeBlock language="cpp">{`#include <algorithm>
#include <map>
#include <queue>
#include <string>
#include <vector>

std::vector<std::string> graphSearch(const std::map<std::string, std::vector<std::string>>& graph,
                                      const std::string& start, const std::string& target) {
    if (start == target) {
        return {start};
    }

    std::map<std::string, bool> visited;
    std::map<std::string, std::string> parent;
    std::queue<std::string> q;

    visited[start] = true;
    q.push(start);

    while (!q.empty()) {
        std::string node = q.front();
        q.pop();

        auto it = graph.find(node);
        if (it != graph.end()) {
            for (const std::string& neighbor : it->second) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    parent[neighbor] = node;
                    q.push(neighbor);
                }
            }
        }
    }

    if (!visited[target]) {
        return {};
    }

    std::vector<std::string> path;
    std::string node = target;
    while (node != start) {
        path.push_back(node);
        node = parent[node];
    }
    path.push_back(start);
    std::reverse(path.begin(), path.end());
    return path;
}`}</CodeBlock>

import java.util.ArrayList;
import java.util.Collections;
import java.util.Deque;
import java.util.HashMap;
import java.util.HashSet;
import java.util.List;
import java.util.Map;
import java.util.Set;

public class GraphSearch {
    public static List<String> graphSearch(Map<String, List<String>> graph, String start, String target) {
        if (start.equals(target)) {
            List<String> single = new ArrayList<>();
            single.add(start);
            return single;
        }

        Set<String> visited = new HashSet<>();
        Map<String, String> parent = new HashMap<>();
        Deque<String> queue = new ArrayDeque<>();

        visited.add(start);
        queue.add(start);

        while (!queue.isEmpty()) {
            String node = queue.poll();

            for (String neighbor : graph.getOrDefault(node, new ArrayList<>())) {
                if (!visited.contains(neighbor)) {
                    visited.add(neighbor);
                    parent.put(neighbor, node);
                    queue.add(neighbor);
                }
            }
        }

        if (!visited.contains(target)) {
            return new ArrayList<>();
        }

        List<String> path = new ArrayList<>();
        String node = target;
        while (!node.equals(start)) {
            path.add(node);
            node = parent.get(node);
        }
        path.add(start);
        Collections.reverse(path);
        return path;
    }
}`}</CodeBlock>

fn graph_search(graph: &HashMap<String, Vec<String>>, start: &str, target: &str) -> Vec<String> {
    if start == target {
        return vec![start.to_string()];
    }

    let mut visited: HashSet<String> = HashSet::new();
    let mut parent: HashMap<String, String> = HashMap::new();
    let mut queue: VecDeque<String> = VecDeque::new();

    visited.insert(start.to_string());
    queue.push_back(start.to_string());

    while let Some(node) = queue.pop_front() {
        if let Some(neighbors) = graph.get(&node) {
            for neighbor in neighbors.iter() {
                if !visited.contains(neighbor) {
                    visited.insert(neighbor.clone());
                    parent.insert(neighbor.clone(), node.clone());
                    queue.push_back(neighbor.clone());

            }
        }
    }

    if !visited.contains(target) {
        return Vec::new();

    let mut path = Vec::new();
    let mut node = target.to_string();
    while node != start {
        path.push(node.clone());
        node = parent.get(&node).unwrap().clone();
    }
    path.push(start.to_string());
    path.reverse();
    path
}`}</CodeBlock>

	if start == target {
		return []string{start}
	}

	visited := make(map[string]bool)
	parent := make(map[string]string)
	queue := []string{start}
	visited[start] = true

	for len(queue) > 0 {
		node := queue[0]
		queue = queue[1:]

		for _, neighbor := range graph[node] {
			if !visited[neighbor] {
				visited[neighbor] = true
				parent[neighbor] = node
				queue = append(queue, neighbor)
			}
		}
	}

	if !visited[target] {
		return []string{}
	}

	path := []string{}
	node := target
	for node != start {
		path = append(path, node)
		node = parent[node]
	}
	path = append(path, start)

	for i, j := 0, len(path)-1; i < j; i, j = i+1, j-1 {
		path[i], path[j] = path[j], path[i]
	}

	return path
}`}</CodeBlock>
