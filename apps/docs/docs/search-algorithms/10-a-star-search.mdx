## Summary

A* Search finds the cheapest route between two points in a map of connected places, using an educated guess to avoid wasting time exploring in the wrong direction. Think of a GPS choosing which street to try next: it considers both "how far have I already driven" and "roughly how much further does it look to the destination," and always tries the option that looks most promising overall. That educated guess is called a heuristic. With a good heuristic, A* can find the cheapest path while barely glancing at most of the map; with a poor one, it ends up checking almost everything.

## Pseudocode

```text
1. If the start or the goal isn't in the graph, report an empty path.
2. Keep a running tally of the cheapest cost found so far to reach each place, starting with 0 for the start.
3. Keep a "to explore" pile, ordered by: cost so far to reach this place, plus a guess at how much further to the goal. Put the start in it.
4. While the pile isn't empty, take out the place with the smallest "cost so far plus guess" total.
5. If this entry is out of date (a cheaper way to reach it was found after it was added), skip it and go back to step 4.
6. For each neighbor of the current place, work out the cost to reach that neighbor through the current place.
7. If that's cheaper than any way found before to reach the neighbor, remember this new cheaper cost, remember the current place as how we got there, and add the neighbor to the pile with its updated total.
8. Once the pile is empty, if the goal was ever reached, follow the "how we got there" trail backwards from the goal to the start, then reverse it to get the path in order. Otherwise report an empty path.
```

## Worked Example

We'll trace A* over one fixed weighted graph of 6 places, searching from `S` to `G`. Each place lists `h`, a straight-line-distance guess of how much further it is to `G`. This only works correctly if that guess never *overestimates* the true remaining distance — guessing low or exactly right is fine, guessing too high can trick A* into settling for a worse path. It also assumes no connection has a negative cost.

`G` now has the smallest total guess in the whole pile, so it's popped. Since we already know the cheapest way to reach every place along the way, we can just follow the "how we got here" trail backwards from `G` to `S` and reverse it: the winning route is `S -> A -> C -> D -> G`, at a total cost of `8` — cheaper than the direct-looking `S -> B -> C -> G` alternative, which would have cost `9`.

## Complexity

- **Best case** — if the heuristic guesses the exact remaining cost every time, A* walks almost straight down the winning path with barely any detours (`O(d)`, where `d` is the number of steps in that path).
- **Average case** — with a reasonably good heuristic, most of the graph gets ignored and only a modest slice of it is checked, using a priority queue to always grab the cheapest-looking option next (`O(E log V)`, the same shape as Dijkstra's algorithm, where `V` is places and `E` is connections).
- **Worst case** — with a poor or missing heuristic (guessing `0` everywhere), A* can't tell good options from bad ones and ends up checking almost everywhere, no better than plain Dijkstra's algorithm (`O(E log V)` with a priority queue).
- **Space** — it has to remember the cost and the "how I got here" trail for every place it has looked at, not just the ones on the final path (`O(V)`).

## Applications

- **GPS and map navigation** — turn-by-turn route planners find the fastest route by running a shortest-path search, often A* or a close variant, over a graph of roads weighted by travel time or distance.
- **Video game pathfinding** — engines such as Unity and Unreal Engine ship A*-based pathfinding on top of navigation meshes so NPCs and player-controlled units can walk around obstacles toward a target.
- **Robotics motion planning** — the ROS Navigation Stack's global planner searches an occupancy grid for a path from a robot's current position to a goal using A*/Dijkstra-style search.
- **Puzzle solving** — classic sliding-tile puzzles like the 8-puzzle and 15-puzzle are commonly solved with A*, using something like Manhattan distance as the heuristic.

<CodeBlock language="python">{`import heapq


def a_star_search(graph: dict[str, list[tuple[str, int]]], start: str, goal: str, heuristic: dict[str, int]) -> list[str]:
    if start not in graph or goal not in graph:
        return []
    if start == goal:
        return [start]

    g_score: dict[str, int] = {start: 0}
    parent: dict[str, str] = {}
    open_set: list[tuple[int, str]] = [(heuristic.get(start, 0), start)]

    while open_set:
        current_f, current = heapq.heappop(open_set)
        current_g = g_score.get(current, float("inf"))
        if current_f != current_g + heuristic.get(current, 0):
            continue  # stale entry, a better one was already processed

        for neighbor, weight in graph.get(current, []):
            tentative_g = current_g + weight
            if tentative_g < g_score.get(neighbor, float("inf")):
                g_score[neighbor] = tentative_g
                parent[neighbor] = current
                heapq.heappush(open_set, (tentative_g + heuristic.get(neighbor, 0), neighbor))

    if goal not in g_score:
        return []

    path: list[str] = []
    node: str | None = goal
    while node is not None:
        path.append(node)
        node = parent.get(node)
    path.reverse()
    return path`}</CodeBlock>

<CodeBlock language="c">{`#include <stdio.h>
#include <string.h>
#include <limits.h>

#define MAX_NODES 50
#define MAX_EDGES 20
#define MAX_NAME 32
#define MAX_OPEN 400

typedef struct { char label[MAX_NAME]; int f; } OpenEntry;

typedef struct {
    char names[MAX_NODES][MAX_NAME];
    char edgeTo[MAX_NODES][MAX_EDGES][MAX_NAME];
    int edgeWeight[MAX_NODES][MAX_EDGES];
    int edgeCount[MAX_NODES];
    int nodeCount;
} Graph;

int findNode(Graph *g, const char *name) {
    for (int i = 0; i < g->nodeCount; i++) {
        if (strcmp(g->names[i], name) == 0) return i;
    }
    return -1;
}

int getHeuristic(char heurNames[][MAX_NAME], int heurValues[], int heurCount, const char *node) {
    for (int i = 0; i < heurCount; i++) {
        if (strcmp(heurNames[i], node) == 0) return heurValues[i];
    }
    return 0;
}

int aStarSearch(Graph *g, const char *start, const char *goal,
                char heurNames[][MAX_NAME], int heurValues[], int heurCount,
                char pathOut[][MAX_NAME]) {
    int startIdx = findNode(g, start);
    int goalIdx = findNode(g, goal);
    if (startIdx == -1 || goalIdx == -1) return 0;
    if (strcmp(start, goal) == 0) {
        strcpy(pathOut[0], start);
        return 1;
    }

    int gScore[MAX_NODES];
    int hasG[MAX_NODES];
    int parent[MAX_NODES];
    for (int i = 0; i < g->nodeCount; i++) { hasG[i] = 0; parent[i] = -1; }
    gScore[startIdx] = 0;
    hasG[startIdx] = 1;

    OpenEntry open[MAX_OPEN];
    int openSize = 0;

    strcpy(open[openSize].label, start);
    open[openSize].f = gScore[startIdx] + getHeuristic(heurNames, heurValues, heurCount, start);
    openSize++;

    while (openSize > 0) {
        int minIdx = 0;
        for (int i = 1; i < openSize; i++) {
            if (open[i].f < open[minIdx].f) minIdx = i;
        }
        OpenEntry current = open[minIdx];
        open[minIdx] = open[openSize - 1];
        openSize--;

        int curIdx = findNode(g, current.label);
        int curG = hasG[curIdx] ? gScore[curIdx] : INT_MAX;
        int curH = getHeuristic(heurNames, heurValues, heurCount, current.label);
        if (current.f != curG + curH) continue; // stale entry, a better one was already processed

        for (int e = 0; e < g->edgeCount[curIdx]; e++) {
            const char *neighbor = g->edgeTo[curIdx][e];
            int weight = g->edgeWeight[curIdx][e];
            int neighborIdx = findNode(g, neighbor);
            int tentativeG = curG + weight;
            int existingG = hasG[neighborIdx] ? gScore[neighborIdx] : INT_MAX;
            if (tentativeG < existingG) {
                gScore[neighborIdx] = tentativeG;
                hasG[neighborIdx] = 1;
                parent[neighborIdx] = curIdx;
                strcpy(open[openSize].label, neighbor);
                open[openSize].f = tentativeG + getHeuristic(heurNames, heurValues, heurCount, neighbor);
                openSize++;
            }
        }
    }

    if (!hasG[goalIdx]) return 0;

    char tempPath[MAX_NODES][MAX_NAME];
    int pathLen = 0;
    int node = goalIdx;
    while (node != -1) {
        strcpy(tempPath[pathLen], g->names[node]);
        pathLen++;
        node = parent[node];
    }
    for (int i = 0; i < pathLen; i++) {
        strcpy(pathOut[i], tempPath[pathLen - 1 - i]);
    }
    return pathLen;
}`}</CodeBlock>

<CodeBlock language="cpp">{`#include <string>
#include <vector>
#include <map>
#include <queue>
#include <limits>
#include <algorithm>

std::vector<std::string> aStarSearch(
    std::map<std::string, std::vector<std::pair<std::string, int>>> &graph,
    const std::string &start, const std::string &goal,
    std::map<std::string, int> &heuristic) {

    std::vector<std::string> result;
    if (graph.find(start) == graph.end() || graph.find(goal) == graph.end()) {
        return result;
    }
    if (start == goal) {
        result.push_back(start);
        return result;
    }

    auto getH = [&](const std::string &node) {
        auto it = heuristic.find(node);
        return it != heuristic.end() ? it->second : 0;
    };

    std::map<std::string, int> gScore;
    std::map<std::string, std::string> parent;
    gScore[start] = 0;

    typedef std::pair<int, std::string> Entry; // (f, label)
    std::priority_queue<Entry, std::vector<Entry>, std::greater<Entry>> openSet;
    openSet.push({gScore[start] + getH(start), start});

    while (!openSet.empty()) {
        Entry top = openSet.top();
        openSet.pop();
        int currentF = top.first;
        std::string current = top.second;
        int currentG = gScore.count(current) ? gScore[current] : std::numeric_limits<int>::max();
        if (currentF != currentG + getH(current)) {
            continue; // stale entry, a better one was already processed
        }
        for (auto &edge : graph[current]) {
            const std::string &neighbor = edge.first;
            int weight = edge.second;
            int tentativeG = currentG + weight;
            int existingG = gScore.count(neighbor) ? gScore[neighbor] : std::numeric_limits<int>::max();
            if (tentativeG < existingG) {
                gScore[neighbor] = tentativeG;
                parent[neighbor] = current;
                openSet.push({tentativeG + getH(neighbor), neighbor});
            }
        }
    }

    if (gScore.find(goal) == gScore.end()) {
        return result;
    }

    std::vector<std::string> path;
    std::string node = goal;
    while (true) {
        path.push_back(node);
        auto it = parent.find(node);
        if (it == parent.end()) break;
        node = it->second;
    }
    std::reverse(path.begin(), path.end());
    return path;
}`}</CodeBlock>

<CodeBlock language="java">{`import java.util.ArrayList;
import java.util.Collections;
import java.util.HashMap;
import java.util.List;
import java.util.Map;
import java.util.PriorityQueue;

public class AStarSearch {

    private static class Node {
        final String label;
        final int f;

        Node(String label, int f) {
            this.label = label;
            this.f = f;
        }
    }

    public static List<String> aStarSearch(Map<String, List<Map.Entry<String, Integer>>> graph,
                                             String start, String goal,
                                             Map<String, Integer> heuristic) {
        List<String> result = new ArrayList<>();
        if (!graph.containsKey(start) || !graph.containsKey(goal)) {
            return result;
        }
        if (start.equals(goal)) {
            result.add(start);
            return result;
        }

        Map<String, Integer> gScore = new HashMap<>();
        Map<String, String> parent = new HashMap<>();
        gScore.put(start, 0);

        PriorityQueue<Node> openSet = new PriorityQueue<>((a, b) -> a.f - b.f);
        openSet.add(new Node(start, gScore.get(start) + heuristic.getOrDefault(start, 0)));

        while (!openSet.isEmpty()) {
            Node current = openSet.poll();
            int currentG = gScore.getOrDefault(current.label, Integer.MAX_VALUE);
            int currentF = currentG + heuristic.getOrDefault(current.label, 0);
            if (current.f != currentF) {
                continue; // stale entry, a better one was already processed
            }

            for (Map.Entry<String, Integer> edge : graph.getOrDefault(current.label, List.of())) {
                String neighbor = edge.getKey();
                int tentativeG = currentG + edge.getValue();
                if (tentativeG < gScore.getOrDefault(neighbor, Integer.MAX_VALUE)) {
                    gScore.put(neighbor, tentativeG);
                    parent.put(neighbor, current.label);
                    openSet.add(new Node(neighbor, tentativeG + heuristic.getOrDefault(neighbor, 0)));
                }
            }
        }

        if (!gScore.containsKey(goal)) {
            return result;
        }

        List<String> path = new ArrayList<>();
        String node = goal;
        while (node != null) {
            path.add(node);
            node = parent.get(node);
        }
        Collections.reverse(path);
        return path;
    }
}`}</CodeBlock>

<CodeBlock language="csharp">{`using System.Collections.Generic;

public class AStarSearch
{
    public static List<string> AStarSearch(
        Dictionary<string, List<(string Node, int Weight)>> graph,
        string start,
        string goal,
        Dictionary<string, int> heuristic)
    {
        var result = new List<string>();
        if (!graph.ContainsKey(start) || !graph.ContainsKey(goal))
        {
            return result;
        }
        if (start == goal)
        {
            result.Add(start);
            return result;
        }

        var gScore = new Dictionary<string, int> { [start] = 0 };
        var parent = new Dictionary<string, string>();
        var openSet = new PriorityQueue<string, int>();
        openSet.Enqueue(start, gScore[start] + GetHeuristic(heuristic, start));

        while (openSet.Count > 0)
        {
            openSet.TryDequeue(out string current, out int currentF);
            int currentG = gScore.TryGetValue(current, out int g) ? g : int.MaxValue;
            if (currentF != currentG + GetHeuristic(heuristic, current))
            {
                continue; // stale entry, a better one was already processed
            }

            if (graph.TryGetValue(current, out var edges))
            {
                foreach (var (neighbor, weight) in edges)
                {
                    int tentativeG = currentG + weight;
                    if (!gScore.TryGetValue(neighbor, out int existingG) || tentativeG < existingG)
                    {
                        gScore[neighbor] = tentativeG;
                        parent[neighbor] = current;
                        openSet.Enqueue(neighbor, tentativeG + GetHeuristic(heuristic, neighbor));
                    }
                }
            }
        }

        if (!gScore.ContainsKey(goal))
        {
            return result;
        }

        var path = new List<string>();
        string node = goal;
        while (node != null)
        {
            path.Add(node);
            node = parent.TryGetValue(node, out string p) ? p : null;
        }
        path.Reverse();
        return path;
    }

    private static int GetHeuristic(Dictionary<string, int> heuristic, string node)
    {
        return heuristic.TryGetValue(node, out int h) ? h : 0;
    }
}`}</CodeBlock>

<CodeBlock language="javascript">{`class MinHeap {
  constructor() {
    this.items = [];
  }

  get size() {
    return this.items.length;
  }

  push(priority, value) {
    this.items.push({ priority, value });
    let i = this.items.length - 1;
    while (i > 0) {
      const parent = (i - 1) >> 1;
      if (this.items[parent].priority <= this.items[i].priority) break;
      [this.items[parent], this.items[i]] = [this.items[i], this.items[parent]];
      i = parent;
    }
