Rust's trait system is a language-level embodiment of the Open/Closed Principle. Functions that accept `impl Trait` or `Box<dyn Trait>` are closed for modification -- they call trait methods without knowing the concrete type. New implementations extend the system without touching existing code.

## The Domain: File Compressor System

A file archiving utility needs to support multiple compression algorithms. Users choose between speed (gzip), broad compatibility (zip), or maximum compression (bzip2). New algorithms are added as the tool evolves.

### The Violation

The following example shows a `compress` function that uses a `match` on a compression type enum. Adding a new algorithm requires editing the match.

```rust
// VIOLATION: compress uses a match on CompressionType. Adding a new
// algorithm means editing this match and risking regressions in
// all existing compression paths.

use std::error::Error;
use std::fmt;

// VIOLATION: Adding a new variant to this enum means finding every match
// that handles it -- in this case, the compress function.
#[derive(Debug, Clone, Copy)]
enum CompressionType {
    Gzip,
    Zip,
    Bzip2,
}

#[derive(Debug)]
struct CompressionError(String);

impl fmt::Display for CompressionError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "Compression error: {}", self.0)
    }
}

impl Error for CompressionError {}

// VIOLATION: Monolithic compress function. Adding "lz4" or "xz" means
// adding a new match arm and retesting every existing algorithm.
fn compress(data: &[u8], algo: CompressionType) -> Result<Vec<u8>, Box<dyn Error>> {
    // VIOLATION: match on enum -- each new algorithm requires adding a new arm.
    // Forgetting to handle a new variant (if the enum is non-exhaustive) means
    // the match silently fails to compile or panics.
    match algo {
        CompressionType::Gzip => {
            // Simulated gzip compression
            println!("[GZIP] Compressing {} bytes with DEFLATE algorithm", data.len());
            let mut result = Vec::with_capacity(data.len() / 2);
            // Simplified compression simulation
            for chunk in data.chunks(2) {
                result.push(chunk.len() as u8);
                result.extend_from_slice(chunk);
            }
            Ok(result)
        }
        CompressionType::Zip => {
            // Simulated zip compression (store + deflate)
            println!("[ZIP] Compressing {} bytes with DEFLATE + store", data.len());
            let mut result = vec![0x50, 0x4B, 0x03, 0x04]; // ZIP local header
            result.extend_from_slice(data);
            Ok(result)
        }
        CompressionType::Bzip2 => {
            // Simulated bzip2 compression (Burrows-Wheeler + Huffman)
            println!("[BZIP2] Compressing {} bytes with BWT + Huffman", data.len());
            let mut result: Vec<u8> = data.to_vec();
            // Simplified: just reverse and prepend header
            result.reverse();
            let mut output = vec![0x42, 0x5A, 0x68]; // BZIP2 magic "BZh"
            output.extend_from_slice(&result);
            Ok(output)
        }
    }
}

fn main() -> Result<(), Box<dyn Error>> {
    let data = b"Hello, world! This is some sample data to compress.";

    println!("Original ({} bytes): {:?}", data.len(), &data[..20]);

    let gzipped = compress(data, CompressionType::Gzip)?;
    println!("Gzip compressed ({} bytes)", gzipped.len());

    let zipped = compress(data, CompressionType::Zip)?;
    println!("Zip compressed ({} bytes)", zipped.len());

    let bzipped = compress(data, CompressionType::Bzip2)?;
    println!("Bzip2 compressed ({} bytes)", bzipped.len());

    Ok(())
}
```

Expected output:

```bash
Original (53 bytes): [72, 101, 108, 108, 111, 44, 32, 119, 111, 114, 108, 100, 33, 32, 84, 104, 105, 115, 32, 105]
[GZIP] Compressing 53 bytes with DEFLATE algorithm
Gzip compressed (53 bytes)
[ZIP] Compressing 53 bytes with DEFLATE + store
Zip compressed (57 bytes)
[BZIP2] Compressing 53 bytes with BWT + Huffman
Bzip2 compressed (56 bytes)
```

### The Correct Implementation: Trait Dispatch

The solution defines a `Compressor` trait. Each compression algorithm implements the trait. A factory function or registry selects the compressor. Adding a new algorithm means implementing the trait in a new struct.

```rust
// CORRECT: Compressor trait defines the contract. Each algorithm is a
// separate struct. The compression pipeline is closed for modification.

use std::error::Error;
use std::fmt;

// --- Error type ---
#[derive(Debug)]
struct CompressionError(String);

impl fmt::Display for CompressionError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "Compression error: {}", self.0)
    }
}

impl Error for CompressionError {}

/// Result alias for compression operations.
type CompressResult = Result<Vec<u8>, Box<dyn Error>>;

// --- Trait contract ---
trait Compressor {
    /// Human-readable name of the compression algorithm.
    fn name(&self) -> &'static str;

    /// File extension used for this format (e.g., "gz", "zip").
    fn extension(&self) -> &'static str;

    /// Compress the input data.
    fn compress(&self, data: &[u8]) -> CompressResult;

    /// Decompress the input data (optional, default returns error).
    fn decompress(&self, data: &[u8]) -> CompressResult {
        Err(Box::new(CompressionError(
            format!("Decompression not implemented for {}", self.name())
        )))
    }
}

// --- Concrete implementations ---

struct GzipCompressor;

impl Compressor for GzipCompressor {
    fn name(&self) -> &'static str { "gzip" }
    fn extension(&self) -> &'static str { "gz" }

    fn compress(&self, data: &[u8]) -> CompressResult {
        println!("[GZIP] Compressing {} bytes with DEFLATE algorithm", data.len());
        let mut result = Vec::with_capacity(data.len() / 2);
        for chunk in data.chunks(2) {
            result.push(chunk.len() as u8);
            result.extend_from_slice(chunk);
        }
        Ok(result)
    }
}

struct ZipCompressor;

impl Compressor for ZipCompressor {
    fn name(&self) -> &'static str { "zip" }
    fn extension(&self) -> &'static str { "zip" }

    fn compress(&self, data: &[u8]) -> CompressResult {
        println!("[ZIP] Compressing {} bytes with DEFLATE + store", data.len());
        let mut result = vec![0x50, 0x4B, 0x03, 0x04]; // ZIP local header
        result.extend_from_slice(data);
        Ok(result)
    }
}

struct Bzip2Compressor;

impl Compressor for Bzip2Compressor {
    fn name(&self) -> &'static str { "bzip2" }
    fn extension(&self) -> &'static str { "bz2" }

    fn compress(&self, data: &[u8]) -> CompressResult {
        println!("[BZIP2] Compressing {} bytes with BWT + Huffman", data.len());
        let mut result: Vec<u8> = data.to_vec();
        result.reverse();
        let mut output = vec![0x42, 0x5A, 0x68]; // BZIP2 magic "BZh"
        output.extend_from_slice(&result);
        Ok(output)
    }

    fn decompress(&self, data: &[u8]) -> CompressResult {
        // Minimal decompression simulation
        if data.len() < 4 || data[0] != 0x42 {
            return Err(Box::new(CompressionError("Invalid bzip2 data".into())));
        }
        let mut result = data[3..].to_vec();
        result.reverse();
        println!("[BZIP2] Decompressed to {} bytes", result.len());
        Ok(result)
    }
}

// --- Registry for dynamic dispatch ---
struct CompressorRegistry {
    compressors: Vec<Box<dyn Compressor>>,
}

impl CompressorRegistry {
    fn new() -> Self {
        CompressorRegistry {
            compressors: Vec::new(),
        }
    }

    fn register(&mut self, compressor: Box<dyn Compressor>) {
        println!("Registered compressor: {}", compressor.name());
        self.compressors.push(compressor);
    }

    fn find_by_name(&self, name: &str) -> Option<&dyn Compressor> {
        self.compressors.iter()
            .find(|c| c.name() == name)
            .map(|c| c.as_ref())
    }

    fn list_compressors(&self) -> Vec<&str> {
        self.compressors.iter().map(|c| c.name()).collect()
    }
}

// --- Core pipeline -- closed for modification ---
fn compress_file(
    data: &[u8],
    compressor: &dyn Compressor
) -> CompressResult {
    println!("Compressing file with {}", compressor.name());
    let compressed = compressor.compress(data)?;
    println!(
        "  Original: {} bytes -> Compressed: {} bytes (ratio: {:.1}%)",
        data.len(),
        compressed.len(),
        (compressed.len() as f64 / data.len() as f64) * 100.0
    );
    Ok(compressed)
}

/// Process data through multiple compressors and return the best result.
fn find_best_compression(
    data: &[u8],
    registry: &CompressorRegistry
) -> Option<(String, Vec<u8>)> {
    let mut best: Option<(String, Vec<u8>)> = None;

    for compressor in &registry.compressors {
        if let Ok(compressed) = compressor.compress(data) {
            let is_better = match &best {
                Some((_, best_data)) => compressed.len() < best_data.len(),
                None => true,
            };
            if is_better {
                best = Some((compressor.name().to_string(), compressed));
            }
        }
    }

    best
}

fn main() -> Result<(), Box<dyn Error>> {
    // Setup registry
    let mut registry = CompressorRegistry::new();
    registry.register(Box::new(GzipCompressor));
    registry.register(Box::new(ZipCompressor));
    registry.register(Box::new(Bzip2Compressor));

    let data = b"Hello, world! This is some sample data to compress.";
    println!("Original ({} bytes)\n", data.len());

    // Use specific compressor via trait object
    if let Some(compressor) = registry.find_by_name("gzip") {
        let compressed = compress_file(data, compressor)?;
        println!("  Extension: .{}\n", compressor.extension());
    }

    // Compress with all registered compressors
    println!("--- All compressors ---");
    for name in registry.list_compressors() {
        if let Some(c) = registry.find_by_name(name) {
            let compressed = c.compress(data)?;
            println!(
                "  .{}: {} bytes",
                c.extension(),
                compressed.len()
            );
        }
    }

    // Find best compression
    println!("\n--- Finding best compression ---");
    if let Some((name, compressed)) = find_best_compression(data, &registry) {
        println!(
            "Best: {} ({} bytes vs original {} bytes)",
            name,
            compressed.len(),
            data.len()
        );
    }

    // --- Adding a new algorithm without modifying existing code ---
    //
    // New requirement: LZ4 compression for high-speed scenarios.
    // We create a new struct and register it. No existing code changes.

    println!("\n=== Adding LZ4 (no edits to existing code) ===");

    struct Lz4Compressor;

    impl Compressor for Lz4Compressor {
        fn name(&self) -> &'static str { "lz4" }
        fn extension(&self) -> &'static str { "lz4" }

        fn compress(&self, data: &[u8]) -> CompressResult {
            println!("[LZ4] Compressing {} bytes with LZ4 algorithm", data.len());
            // Simulated LZ4: simple byte-pair compression
            let mut result = Vec::new();
            let mut i = 0;
            while i < data.len() {
                let run_end = (i + 16).min(data.len());
                let literal_len = (run_end - i) as u8;
                result.push(literal_len);
                result.extend_from_slice(&data[i..run_end]);
                i = run_end;
            }
            Ok(result)
        }
    }

    registry.register(Box::new(Lz4Compressor));

    if let Some(lz4) = registry.find_by_name("lz4") {
        let compressed = compress_file(data, lz4)?;
        println!("  Extension: .{}", lz4.extension());
        println!("  Decompress not implemented (default trait method)");

        // Verify the default decompress returns an error
        match lz4.decompress(&compressed) {
            Ok(_) => println!("  Decompression succeeded (unexpected)"),
            Err(e) => println!("  Decompression: {}", e),
        }
    }

    // Now find best among all 4 compressors
    println!("\n--- New best (including LZ4) ---");
    if let Some((name, compressed)) = find_best_compression(data, &registry) {
        println!(
            "Best: {} ({} bytes vs original {} bytes)",
            name,
            compressed.len(),
            data.len()
        );
    }

    Ok(())
}
```

## Key Takeaways

- **Trait dispatch** is Rust's primary mechanism for OCP. The `Compressor` trait defines the contract. Any struct that implements it becomes available to the pipeline.
- **`Box<dyn Compressor>`** enables dynamic dispatch when the concrete type is not known at compile time. The `CompressorRegistry` holds trait objects.
- **Static dispatch with generics** (`fn compress<T: Compressor>(data: &[u8], c: &T)`) is an alternative when the concrete type is known at compile time. It monomorphizes to zero-cost abstraction but requires the caller to know the type.
- **Default trait methods** (like `decompress` in this example) allow trait evolution without breaking existing implementations. New compressors only need to implement `compress`; `decompress` is optional.
- **Enum dispatch vs trait objects**: If the set of compression algorithms is finite and known at compile time, an enum with a match is more efficient. If the set is extensible (plugins, user-provided), trait objects are the right choice. Both are OCP-compliant if the enum uses `#[non_exhaustive]` and the match has a wildcard arm.
- **Adding a compressor** is purely additive: define a new struct, implement `Compressor` for it, and add it to the registry. No existing code changes.
- **`find_best_compression`** demonstrates that operations composed from the closed core remain closed -- it works with any set of compressors without modification.
