# SOLID Principles

*Estimated completion time: 15–20 hours*

**Difficulty:** Intermediate

## What This Course Covers

This course covers the five SOLID principles — the object-oriented design principles that form the bedrock of clean, maintainable architecture. Each principle is taught violation-first: you see broken code, understand exactly what breaks when it needs to change, then see the corrected implementation.

## Why It Matters

Knowing a programming language is only the beginning. The difference between code that works and code that lasts comes down to design. SOLID principles help you write code that adapts to change without breaking, and give you a shared vocabulary for reviewing and discussing design decisions with other engineers.

## Skills You Will Gain

- Apply the five SOLID principles to write maintainable, loosely-coupled code
- Recognize SOLID violations during code review — by method names, class sizes, import counts, and parameter lists
- Understand when strict adherence to a principle creates unnecessary abstraction, and when it's worth it
- Write clean, testable, and extensible object-oriented code across multiple languages

## Course Sections

Master the five principles of object-oriented design:
- **Single Responsibility** — one class, one reason to change
- **Open/Closed** — open for extension, closed for modification
- **Liskov Substitution** — subtypes must be substitutable for their base types
- **Interface Segregation** — many specific interfaces are better than one general interface
- **Dependency Inversion** — depend on abstractions, not concretions

## Prerequisites

- Working knowledge of at least one object-oriented programming language (JavaScript, TypeScript, Python, Java, C#, or Rust)
- Understanding of classes, inheritance, interfaces, and polymorphism

## How to Use This Course

Each principle has multiple language-specific implementations so you can follow along in your preferred language or compare across languages. Start with the overview for each principle, then dive into the language-specific pages that interest you.
