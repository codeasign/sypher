# System Design Fundamentals

Welcome to the System Design Fundamentals course. This is a comprehensive, FAANG-level guide to system design — from the foundations to advanced interview preparation.

## Course Roadmap

<CourseCurriculum />

Each topic is structured as a guided learning path:

- **Mental Model** — Build intuition with stories and analogies
- **Deep Dive** — Understand internals, data structures, and algorithms
- **Architecture** — See how it fits in a production-scale system
- **Trade-offs** — Learn the hard choices and failure modes
- **Real World** — Study how companies like Netflix, Uber, and Google implement it
- **Interview** — Ace the system design interview with frameworks and phrase banks
- **Challenges** — Prove your knowledge with tiered exercises
